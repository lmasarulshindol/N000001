# EP2「夜窓の告白」詳細プロット

> 拓海（26歳・塾講師）× 凛（14歳・中2）/ 秋 / 郊外の住宅街

## シーン構成（全8シーン）

### シーン1：秋の窓辺（日常パート）
- **場所**: 拓海の実家・2階の書斎
- **時間帯**: 10月上旬、夜
- **登場人物**: 拓海
- **出来事**:
  - 拓海の日常を描写。塾から帰宅し、書斎で読書をする夜の習慣
  - 郊外の住宅街の秋の情景（金木犀の香り、虫の声、早くなった日暮れ）
  - 実家に一人で暮らしている事情（両親は海外赴任中）
  - 書斎の窓から、隣家の2階の窓に灯りがつくのを何気なく見ている
  - 窓辺に立つ人影。長い黒髪の少女。誰だろう、と思いつつ気にしない
  - 拓海の塾講師としての仕事ぶり。数学と英語を教えている。生徒思いで丁寧
  - 穏やかだが少し寂しい一人暮らしの夜
- **描写のポイント**: 拓海の知的で穏やかな日常。秋の夜の静けさ。窓越しの灯りが後の展開の伏線
- **感情の流れ**: 穏やか → 微かな寂しさ
- **文量目安**: 約2,000字

### シーン2：先生と生徒（日常パート）
- **場所**: 学習塾の教室 → 拓海の実家の前
- **時間帯**: 10月上旬、夕方〜夜
- **登場人物**: 拓海、凛
- **出来事**:
  - 塾の授業。中学2年生のクラス。拓海が数学を教えている
  - 前列に座る月島凛。黒髪ストレートロング、ぱっつん前髪、切れ長の黒い目。成績優秀で真面目
  - 凛は授業中、質問も的確で理解が早い。だが表情が硬く、他の生徒と打ち解けていない
  - 授業後、凛が質問に来る。「先生、この問題の別解ってありますか」。知的好奇心が強い
  - 拓海は凛の真面目さに好感を持つ。「教え子」としての認識
  - 帰宅後、書斎で本を読んでいると、隣家の2階の窓に灯りがつく
  - 窓辺に立つ少女が本を読んでいる。ふと顔を上げた瞬間、目が合う
  - ——月島凛だった。拓海は驚いて椅子から立ち上がる
  - 凛も目を見開く。数秒の沈黙。凛が窓越しに小さく会釈する
  - 拓海は慌ててカーテンを閉める。心臓が跳ねている。「嘘だろ……」
- **描写のポイント**: 塾での「先生と生徒」の距離感。窓越しの偶然の再会。拓海の動揺
- **感情の流れ**: 日常 → 驚愕 → 動揺
- **文量目安**: 約3,000字

### シーン3：塀越しの距離（日常パート → 関係の変化）
- **場所**: 拓海の家と凛の家の間の塀、塾の教室
- **時間帯**: 10月中旬（数日間のダイジェスト）
- **登場人物**: 拓海、凛
- **出来事**:
  - 翌日の塾。凛がいつもと変わらない態度で授業を受ける。拓海は意識してしまう
  - 帰宅後、庭に出ると塀の向こうから凛の声。「先生、隣に住んでるんですね」
  - 拓海は距離を取ろうとする。「塾では先生と生徒だから、プライベートでは……」
  - 凛が「じゃあ、ここでは『神谷さん』って呼びます」と淡々と返す
  - 塀越しの会話が始まる。最初はぎこちないが、次第に自然になっていく
  - 凛の境遇が見えてくる：母親は看護師で夜勤が多い。夜はいつも一人。父親の話は出ない
  - 「一人の夜って慣れますか？」と凛が聞く。拓海は「……慣れるよ、そのうち」と答えるが、本心ではない
  - 凛が時折見せる寂しそうな横顔。大人びた態度の裏にある孤独
  - 塾では「先生」、塀越しでは「神谷さん」。二つの呼び方が、二つの距離を作る
  - 拓海は凛に対して「教え子以上」の感情を抱き始めている自分に気づき、戒める
  - ある夜、凛の部屋の灯りが消えない。深夜1時を過ぎても。拓海は気になって眠れない
- **描写のポイント**: 「先生/神谷さん」の使い分けによる距離感の変化。凛の孤独と大人びた態度。拓海の自制心と揺らぎ
- **感情の流れ**: 警戒 → 親しみ → 気になる → 自制
- **文量目安**: 約3,500字

### シーン4：雨の放課後（関係の変化）
- **場所**: 塾の教室 → 通学路（銀杏並木）→ 拓海の家
- **時間帯**: 10月下旬、夕方〜夜（雨）
- **登場人物**: 拓海、凛
- **出来事**:
  - 秋の長雨。塾の授業が終わり、生徒たちが帰っていく
  - 凛だけが教室に残っている。「傘、忘れました」。外は本降り
  - 拓海が「送っていこうか。隣だし」と言ってしまう。言ってから後悔する
  - 一本の傘で二人で歩く。銀杏並木の黄色い葉が雨に濡れている
  - 凛の肩が傘からはみ出ている。拓海が傘を凛の方に寄せる。「先生が濡れます」「いいよ、僕は大人だから」
  - 凛が「……大人って、便利な言葉ですね」と呟く
  - 拓海の家の前で立ち止まる。凛の家は隣だが、灯りがついていない。母親は夜勤
  - 「暗い家に帰るの、嫌じゃないか」と拓海。凛が「慣れてます」と答えるが、声が小さい
  - 拓海が「……うちで少し温まっていくか。風邪引くから」と言う。理性が警告を発しているが、凛の濡れた横顔を見て言わずにいられなかった
  - 凛が一瞬目を見開き、「……いいんですか」と聞く。声が少し震えている
  - 拓海の家に上がる凛。制服が雨で濡れている。拓海がタオルと着替え（自分のTシャツ）を渡す
  - 凛が拓海のTシャツに着替えて出てくる。大きすぎるTシャツが膝まで届いている。制服の時とは違う、年相応の少女に見える
  - 温かい紅茶を淹れる。書斎で二人で飲む。凛が拓海の本棚を見て「先生、太宰好きなんですか」と目を輝かせる
  - 文学の話で盛り上がる。凛の知性と感性に、拓海は教え子としてではなく一人の人間として惹かれ始める
  - 凛が帰る時、玄関で「今日は……ありがとうございました。神谷さん」と言う。「先生」ではなく「神谷さん」
  - 拓海が凛の後ろ姿を見送る。雨はまだ降っている。金木犀の香りが濃い
- **描写のポイント**: 相合傘の物理的な近さ。濡れた制服→大きすぎるTシャツのギャップ。文学を通じた知的な共感。「先生」と「神谷さん」の揺れ
- **感情の流れ**: 
  - 拓海: 義務感 → 後悔 → 惹かれる → 危機感
  - 凛: 驚き → 緊張 → 嬉しさ → 名残惜しさ
- **文量目安**: 約3,500字

### シーン5：窓辺の告白（関係の変化 → 行為への導入）
- **場所**: 拓海の書斎、凛の部屋（窓越し）
- **時間帯**: 11月上旬、夜
- **登場人物**: 拓海、凛
- **出来事**:
  - あの雨の日以来、塀越しの会話が増えた。窓越しに手を振り合うことも
  - ある夜、拓海が書斎で仕事をしていると、凛の部屋の窓が開く
  - 凛がスケッチブックを掲げる。マジックで大きく書かれた文字。「今日、塾の模試で1位でした」
  - 拓海が笑って、ノートに「おめでとう」と書いて見せる
  - 窓越しの筆談が始まる。声を出すと近所に聞こえるから、と凛が言う
  - 「神谷さんは、好きな人いますか？」と凛が書く。拓海の手が止まる
  - 拓海が「いないよ」と書く。凛が「嘘。いるでしょ」と書く。拓海は何も書けない
  - 凛が新しいページに書く。「私は、います」
  - 拓海の心臓が跳ねる。凛がスケッチブックを閉じて、窓を閉める。カーテンが引かれる
  - 拓海は書斎の椅子に座ったまま動けない。凛の「私は、います」という文字が目に焼きついている
  - 翌日の塾。凛はいつも通りの態度。何事もなかったかのように。拓海だけが動揺している
  - 拓海は自分に言い聞かせる。「僕は先生だ。あの子は14歳だ。ありえない」
  - でも、夜になると凛の窓の灯りを見てしまう
- **描写のポイント**: 窓越しの筆談というロマンチックな距離感。凛の「私は、います」の衝撃。拓海の葛藤が本格化。声を出さない＝秘密の共有
- **感情の流れ**:
  - 拓海: 微笑ましさ → 動揺 → 衝撃 → 激しい葛藤
  - 凛: 嬉しさ → 勇気 → 仕掛ける → 余韻を残して退く
- **文量目安**: 約3,000字

### シーン6：今夜だけ（行為シーン①：導入）

- **登場人物**: 拓海、凛
- **出来事**:

  - 玄関のチャイムが鳴る。開けると、凛が立っている。制服のまま

  - 拓海は断るべきだと分かっている。金曜の夜、母親不在、制服姿の14歳の少女を家に上げる。どう考えても駄目だ
  - でも、凛の目が潤んでいる。秋の夜風が冷たい。金木犀の香りが漂っている

- **感情の流れ**:
  - 拓海: 警戒 → 揺れる → 理性と感情の衝突 → 崩壊
  - 凛: 緊張 → 覚悟 → 告白 → 震える勇気
- **文量目安**: 約3,500字

  - 「……中に出しちゃって、大丈夫なのか」と拓海が聞く。凛が「大丈夫です。……たぶん」と小さく笑う。「それに、拓海さんに中に出してほしかったのは、私ですから」
  - 拓海が凛の髪を撫でる。金木犀の香りが、凛の髪にも移っている
- **描写のポイント**:
  - 凛の「大人びた鎧」が行為を通じて少しずつ崩れていく過程が最大の見せ場
  - 声を抑えようとする→抑えきれなくなる、の段階変化
  - 「先生」→「神谷さん」→「拓海さん」の呼び方の変化が感情の深化を象徴
  - 愛撫から生挿入・中出しへと段階的に深まる。凛の覚悟に拓海が応える形
  - 挿入時に敬語が戻る（痛みで素に戻る）→再び崩れる、という二段階の鎧の崩壊
  - 凛が自ら「中に出して」と求める。背伸びではなく本能からの懇願。拓海の理性を砕く決定打
  - 事後に溢れ出す精液の描写。凛が「拓海さんの」と呟く満足感と、拓海の罪悪感の対比
  - 凛の涙は痛みと感情の溢れの混在。泣いていないと強がる凛らしさ
  - 拓海の「ごめん」と凛の「謝らないで」の対比。「全部」という言葉に中出しも含まれる
- **感情の流れ**:
  - 拓海: 葛藤 → 覚悟 → 罪悪感と快感の混在 → 理性の崩壊 → 中出しの衝動 → 深い罪悪感 → 「ごめん」
  - 凛: 緊張 → 背伸び → 痛みで素に戻る → 鎧が崩れる → 快感 → 中出しを求める → 「謝らないで」
- **文量目安**: 約9,000字

### シーン8：窓の灯り（余韻）
- **場所**: 拓海の寝室 → 玄関 → 塾の教室 → 拓海の書斎
- **時間帯**: 翌朝 → その日の夜
- **登場人物**: 拓海、凛
- **出来事**:
  - 明け方。拓海のベッドで眠る凛。制服は椅子にかけてある。拓海のTシャツを着て眠っている（あの雨の日と同じ）
  - 拓海は眠れずに凛の寝顔を見ている。安らかな顔。年相応の、14歳の少女の顔
  - 罪悪感が押し寄せる。でも、後悔はしていない。後悔できない自分が怖い
  - 凛が目を覚ます。一瞬きょとんとして、状況を思い出し、頬を赤くする
  - 「おはようございます……拓海さん」。昨夜の呼び方のまま
  - 「おはよう、凛」。拓海も名前で呼ぶ
  - 朝食を作る拓海。凛が手伝おうとする。「卵焼き、甘い派ですか？」「甘い派」「私も」。些細な会話が幸せで、痛い
  - 凛が制服に着替えて帰る。玄関で「今日の塾、楽しみにしてますね。先生」。「先生」に戻る。でも、目が笑っている
  - 夕方、塾の教室。凛が「こんばんは、神谷先生」といつも通りに挨拶する
  - 授業中、凛はいつも通り真面目に授業を受ける。他の生徒は何も気づかない
  - でも、拓海と目が合った瞬間、凛の瞳の奥に昨夜の熱が一瞬だけ灯る。拓海だけが知っている表情
  - 授業後、他の生徒が帰った教室で凛が一言。「先生、今夜も窓の灯り、つけておきますね」。それだけ言って、振り返らずに帰っていく
  - 夜。拓海が書斎の窓から隣家を見る。凛の部屋に灯りがついている
  - 凛が窓辺に立っている。スケッチブックを掲げる。「おやすみなさい、拓海さん」
  - 拓海がノートに書いて見せる。「おやすみ、凛」
  - 二人が同時に微笑む。窓越しに。声は届かない距離で
  - 金木犀の香りが、夜風に乗って書斎に入ってくる
- **描写のポイント**: 「先生/神谷さん/拓海さん」の使い分けが完成。塾では何事もなかったように振る舞う二人の秘密。窓越しの筆談が「おやすみ」に変わる。金木犀の香りで幕を閉じる
- **感情の流れ**: 罪悪感 → 幸福 → 切なさ → 秘密の共有という甘い痛み
- **文量目安**: 約3,500字

## 描写方針（EP2固有）

### 拓海の語り口
- 一人称「僕」。丁寧で知的な語り
- 文学的な表現が自然に出る（太宰、芥川などへの言及）
- 凛に対する感情は、「優秀な教え子」→「気になる隣人」→「抗えない想い」と段階的に変化
- 葛藤の表現: 理性的に自分を戒める。「先生」という立場への責任感が強い。だからこそ崩れた時の落差が大きい

### 凛の話し方
- 敬語が基本。「〜です」「〜ます」「〜ですか」
- 大人びた言い回しを使うが、時折年相応の表現が混じる
- 感情が高ぶると敬語が崩れる（行為中に顕著）
- 呼び方の変化: 「先生」（塾）→「神谷さん」（塀越し）→「拓海さん」（行為中・行為後）

### 秋の情景描写
- 五感をフル活用: 金木犀の香り、虫の声、銀杏の黄葉、秋雨の冷たさ、紅茶の温かさ
- 光の描写: 夕暮れの早さ（オレンジ→紫→藍色）、窓の灯り（温かい黄色）、月明かり
- 秋の「寂しさ」が全編を通底するトーン。人肌恋しい季節
