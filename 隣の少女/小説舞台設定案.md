# 隣の少女

> 現代日本を舞台にした短編エロ小説集。主人公・ヒロイン共に複数。それぞれ別々の場所・シチュエーションでストーリーを展開するオムニバス形式。
> ヒロインは **小学生・中学生・高校生・大学生・社会人** の各年代から1名ずつ。

---

## 1. 作品コンセプト

### 基本設計
- **ジャンル**: 現代日本・短編エロ小説集（オムニバス）
- **舞台**: 現代日本の様々な街・住宅地
- **形式**: 1話完結型の短編集。各話に異なる主人公とヒロインが登場する
- **視点**: 男性主人公の一人称視点（各話ごとに主人公が異なる）
- **共通テーマ**: 「隣に住んでいる少女」との関係が発展していく物語
- **ヒロインの年齢構成**: 小学生・中学生・高校生・大学生・社会人（各1名）

### 作品のトーン
- 日常の延長線上にあるリアリティを重視
- 「隣人」という距離感――近いのに一線がある関係から、その一線を越える瞬間を描く
- 各話ごとに異なる雰囲気（甘い・背徳的・切ない・コミカル・退廃的など）を持たせる
- ヒロインの年齢によって **主人公の葛藤の質** が大きく変わることが作品の核
- 行為シーンは丁寧に描写し、省略・暗転しない

## 2. 世界観・共通設定

### 舞台の時代
- 現代日本（202X年）
- スマートフォン・SNS・コンビニなど現代の生活インフラが存在する世界

### 「隣」の定義
各話で「隣」の意味合いが異なるバリエーションを持たせる:

| パターン | 説明 | 距離感 |
|----------|------|--------|
| 隣の家 | 一戸建ての隣家 | 窓越し・庭越しに見える距離 |
| 隣の部屋 | アパート・マンションの隣室 | 壁一枚の近さ。生活音が聞こえる |
| 隣の部屋（同居型） | シェアハウス・寮の隣室 | 共有スペースで毎日顔を合わせる |

### 季節と情景
各話に季節感を持たせ、情景描写で雰囲気を演出する:

| 季節 | 情景の特徴 | エロティックな要素 |
|------|-----------|-------------------|
| 春 | 桜、引っ越しシーズン、新生活 | 新しい出会い、薄着になり始める |
| 夏 | 花火、蝉の声、入道雲、熱帯夜 | 薄着・肌の露出、汗、開け放った窓 |
| 秋 | 金木犀の香り、夕暮れの早さ | 寂しさ、人肌恋しい季節 |
| 冬 | 雪、クリスマス、年末年始 | 寒さを理由にした密着、暖房の効いた部屋 |

各話のヒロインは「隣に住む少女（女性）」。**小学生・中学生・高校生・大学生・社会人** の5つの年代から1名ずつ。年齢による身体的・精神的な違いを丁寧に描き分ける。

### ヒロイン①：海野 澪（うみの みお）【小学生】
**← 主人公①・瀬戸一真の隣人**

| 項目 | 詳細 |
|------|------|
| 年齢 | 11歳（小学5年生） |

| 髪色 | 太陽に透ける明るい栗色 |
| 目の色 | 海を映したような深い青緑。大きくてくりっとした瞳 |
| 身長 | 140cm |
| 体重 | 32kg |
| スリーサイズ | B62 / W53 / H65 |
| 身体的特徴 | まだ幼さの残る華奢な体つき。健康的に日焼けした肌。膝に絆創膏を貼っていることが多い。胸はほぼ平ら |

- **性格**: 天真爛漫で好奇心旺盛。人見知りしない。感情がストレートに顔に出る。怖いもの知らずだが、夜は少し寂しがり
- **境遇**: 夏休みの間だけ、東京に住む母親の元を離れ、海沿いの町の祖母の家に預けられる。両親は離婚しており、母は仕事が忙しい。祖母の家の隣が一真の古民家
- **主人公との関係**: 海で溺れかけたところを一真に助けられ、「かずまにぃ」と懐く。カメラに興味を持ち、毎日のように一真の家に遊びに来る。無邪気な信頼と好意が、一真を追い詰めていく
- **物語のキーワード**: ひと夏の秘密、無邪気と背徳、海と太陽、期限のある関係

## 7. 性行為シーンの描写方針

> **各エピソードのクライマックス。主人公とヒロインの関係性・個性が最も濃密に表現される場面。**

### 基本設計思想
- 行為シーンは **省略・暗転しない**。五感を使って丁寧に描写する
- 各カップルの **年齢差・関係性・性格** による反応の違いを最大限に描き分ける
- 現代日本が舞台のため、行為に至る心理的ハードルや背徳感をリアルに描く
- ファンタジー要素がないぶん、**生々しい肉体描写と心理描写** で魅せる
- ヒロインの年齢が低いほど、**主人公の葛藤と行為に至る過程** に紙幅を割く

### 描写の三層構造

**① 導入（行為に至るまで）**
- 二人の関係がどう変化してきたかの振り返り
- 「一線を越える」瞬間のトリガー（偶然の接触、告白、雰囲気に流される等）
- 主人公側の葛藤（年齢差、立場、道徳観）
- ヒロイン側の心理（緊張、期待、覚悟、好奇心、誘惑）

**② 行為本体（最も紙幅を割くパート）**
- 身体描写: 肌の質感、体温、吐息、汗、表情の変化を五感で描く
- 反応描写: 声（抑えようとする、漏れる、叫ぶ）、仕草、無意識の動き
- 段階的変化: 接触 → 愛撫 → 行為の深化 → クライマックスの流れを丁寧に
- 主人公の視点: ヒロインの反応を見つめる描写（読者は主人公の目を通してヒロインを見る）
- 会話: 行為中の言葉（懇願、名前を呼ぶ、恥ずかしい言葉）が関係性を深める

**③ 余韻（行為後）**
- 事後の静けさ、汗が引いていく感覚
- ヒロインの表情・言葉（ここに本音が出る）
- 二人の関係が行為前と行為後でどう変化したか
- その後の日常への回帰（または変化）

## 9. 各エピソードのプロット概要

### EP1「夏色のシャッター」（一真×澪 / 夏 / 小学生）

**起**: 夏。海沿いの町に暮らす一真の隣家に、夏休みの間だけ祖母の家に預けられた澪がやってくる。海で溺れかけた澪を一真が助けたことがきっかけで、澪は「かずまにぃ」と懐く。

**承**: 一真のカメラに興味を持った澪が毎日のように遊びに来る。海辺での撮影、虫取り、花火、縁側での夕涼み。澪の屈託のない笑顔にカメラを向けるうち、一真はファインダー越しに澪の無防備な姿を意識し始める。「夏が終わったら帰る」という期限が、日々を特別なものにしていく。

**転**: 夏休み最後の夜。「かずまにぃ、澪のこと忘れないで」と暗室に入ってきた澪。薄暗い赤い光の中で、澪が「写真、撮って。澪だけの、特別なやつ」と言う。一真の中で、カメラマンとしての欲望と、男としての欲望が混ざり合う。

**結**: 夏の終わり、澪が町を去る。一真の手元には、あの夜の記憶だけが残る。翌年の夏、玄関のチャイムが鳴る。少しだけ背が伸びた澪が立っている。

### EP2「夜窓の告白」（拓海×凛 / 秋 / 中学生）

**起**: 秋。拓海は書斎の窓から、隣家の2階の窓に灯りがつくのを毎晩見ている。ある日、その窓辺に立つ少女が自分の塾の生徒・月島凛だと気づく。

**承**: 塾では優等生の凛が、家では一人で夕食を食べ、窓辺で本を読んでいる姿。拓海は講師として距離を取ろうとするが、凛の方から「先生、隣に住んでるんですね」と声をかけてくる。塀越しの会話が日課になる。凛が時折見せる寂しそうな横顔に、拓海は教え子以上の感情を抱き始める。

**転**: 秋の夜、母親が夜勤で不在の凛が「先生、今夜だけ隣にいてください」と拓海の家を訪れる。制服のまま、金木犀の香りをまとって。「私、子供じゃないです」と背伸びする凛の瞳が潤んでいる。拓海の理性が崩れる。

**結**: 翌日、塾で「こんばんは、神谷先生」といつも通りに挨拶する凛。だが目が合った瞬間、二人だけが知る秘密が空気を変える。帰り道、凛からのメッセージ。「今夜も、窓の灯り、つけておきますね」

### EP3「隣の部屋の甘い罠」（蓮×ひなた / 春 / 高校生）

**起**: 春。蓮のマンションの隣室に、父親の転勤で引っ越してきた一家の娘・ひなたが挨拶に来る。明るく人懐っこいひなたに戸惑う蓮。

**承**: Wi-Fiの設定、勉強を教えてほしい、料理のおすそ分け……何かと理由をつけて蓮の部屋を訪れるひなた。両親の帰りが遅い日は特に頻繁で、蓮は迷惑に思いつつも、次第にひなたの存在が日常に溶け込んでいく。制服姿で来ることもあれば、部屋着の短パンにTシャツという無防備な格好のことも。ある雨の夜、ひなたが「一人だと怖い」と蓮の部屋に転がり込んでくる。

**転**: 薄着のまま蓮の部屋で眠ってしまうひなた。目が覚めた蓮の腕の中で、ひなたが「……知ってたよ、蓮さんがいつも気にしてくれてたの」と囁く。「蓮さんのこと、ただのお隣さんだと思ってないよ」

**結**: 行為後、桜が散る窓の外を二人で眺める。「隣の部屋でよかった」とひなたが笑う。翌朝、制服に着替えて「行ってきます」と手を振るひなた。蓮の日常が、決定的に変わった春。

### EP4「壁越しのラブレター」（悠斗×こはる / 冬 / 大学生）

**起**: 冬。悠斗の隣室に越してきたこはるは演劇学科の大学生で、毎晩発声練習をする。壁が薄いアパートで悠斗は眠れない日々。苦情を言いに行くが、泣きそうな顔で謝るこはるに強く言えない。

**承**: お詫びの手料理をきっかけに交流が始まる。壁越しにこはるの練習を聞くのが悠斗の日課になる。ある夜、壁越しに聞こえてきたのは台本の朗読ではなく、こはるの切ない喘ぎ声だった。

**転**: 翌日、顔を合わせた二人の間に気まずい空気。こはるが「……聞こえてた？」と真っ赤になる。悠斗が「壁、薄いから」と答えると、こはるは「じゃあ……本物の声、聞かせてあげようか」と。

**結**: 行為後、壁の薄さを嘆いていた二人が「この壁のおかげだね」と笑い合う。冬の朝、二人分の布団で目覚める。こはるが「ねえ、隣の部屋じゃなくて、同じ部屋に住まない？」と。

### EP5「インクと肌の匂い」（透×紗夜 / 冬 / 社会人）

**起**: 冬。タワーマンションの隣室に越してきた紗夜が、透の玄関前でファンレターを渡そうとしているところに鉢合わせる。紗夜が出版社の文芸編集者であること、そして透の作品の熱狂的なファンであることを知り、透は興味を持つ。

**承**: 透は紗夜を「次の作品の参考に」と称して観察し始める。紗夜の読書の好み、仕草、表情の変化を記録する。紗夜は透に見られることに戸惑いながらも、憧れの作家に「見てもらえる」快感を覚え始める。仕事帰りにワインを持って透の部屋を訪れるのが習慣になる。

**転**: 執筆が行き詰まった夜、透が紗夜に「君の反応を、もっと近くで観察させてほしい」と告げる。紗夜は「……先生の小説のためなら」と応じるが、二人とも「取材」が建前だと気づいている。

**結**: 行為後、透は原稿に向かう。紗夜がモデルの新作は、透の最高傑作になる。紗夜は完成した原稿を読み、「これ、私ですね」と微笑む。「君以外の誰でもない」と透が答える。紗夜は編集者として、この原稿を世に出す決意をする。
