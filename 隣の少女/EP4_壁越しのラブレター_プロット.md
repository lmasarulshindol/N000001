  - こはるの体が大きく痙攣する。中に出された熱で絶頂が押し寄せる。「悠斗くんっ……！」。悠斗の名前を叫ぶ声が、狭い部屋に響き渡る。壁越しに聞いたどの声よりも、美しい声
  - しばらく繋がったまま動けない。二人の荒い呼吸だけが聞こえる。石油ストーブの炎が揺れている
  - 悠斗がゆっくりと体を離す。繋がりが解けた瞬間、こはるの中から精液がとろりと溢れる。シーツに染みが広がる
  - こはるがそれを見て頬を赤くする。「……いっぱい出たね」「……言うな」「えへへ。悠斗くんの、あったかかった」
  - 「……中に出しちゃったけど、大丈夫か」と悠斗が聞く。こはるが「大丈夫。……たぶん。日にち的に」と小さく笑う。「それに、悠斗くんの声が聞きたかったの。中に出す時の声。……すごく、良い声だった」
  - 悠斗がこはるを抱きしめる。「お前の声のほうがいいよ」「……ありがと」
  - 二人で抱き合ったまま、石油ストーブの温もりに包まれている。窓の結露が、外のイルミネーションの光をぼんやり映している
- **描写のポイント**:
  - 「声」がテーマ。壁越しの声→本物の声→行為中の声→中出し時の声、と段階的に変化
  - こはるの声の豊かさが行為中に最大限発揮される。演劇で鍛えた声帯が快感をダイレクトに伝える
  - コミカルなやりとりが行為中も健在。「隣に聞こえる」「隣は俺の部屋」のくだり
  - 二人とも初めて同士。不器用さと初々しさ
  - 愛撫から生挿入・中出しへと段階的に深まる。こはるが「悠斗くんの声、聞かせて」と求める。声をテーマにしたEP4ならではの中出しの経緯
  - 事後の精液描写。こはるの「悠斗くんの、あったかかった」と、中出し時の声への言及
  - 石油ストーブの温もりと冬の冷たさの対比
- **感情の流れ**:
  - 悠斗: 緊張 → コミカル → 夢中 → 理性の崩壊 → 中出し → 幸福
  - こはる: 恥ずかしさ → 大胆 → 快感 → 中出しを求める → 幸福
- **文量目安**: 約9,000字

### シーン8：同じ部屋で（余韻）
- **場所**: 悠斗の部屋 → アパートの共用廊下 → 悠斗の部屋
- **時間帯**: 翌朝 → 大晦日
- **登場人物**: 悠斗、こはる
- **出来事**:
  - 朝。悠斗の狭いシングルベッドで二人で目覚める。こはるが悠斗の腕の中にいる
  - こはるが「おはよう、悠斗くん」と笑う。寝癖がひどい。悠斗が「髪やばいぞ」と笑う。こはるが「見ないで！」と枕で顔を隠す
  - 悠斗が朝食を作る。卵かけご飯とインスタント味噌汁。こはるが「悠斗くんの朝ごはん、初めて食べる」と嬉しそう
  - 「ねえ、悠斗くん」「ん」「昨日の私の声、どうだった？」「……どうって」「演技じゃない本物の声。……点数つけるなら」「100点」「……えへへ」
  - こはるが帰る。共用廊下を数歩。「じゃあ、また壁越しにね」と手を振る。悠斗が「おう」と返す
  - 壁越しの日常に戻る。でも、全てが違って聞こえる。こはるの鼻歌、足音、独り言。全部が愛おしい
  - 大晦日の夜。悠斗がギターを弾いている。壁越しにこはるが聴いている
  - 悠斗が弾き語りで歌う。こはるへの気持ちを込めた曲。壁越しのラブレター
  - 曲が終わると、壁越しにこはるの声が聞こえる。泣いている。「……ずるい。泣かせないでよ」
  - 悠斗が壁に手を当てる。壁の向こうから、こはるも壁に手を当てている気がする
  - しばらくして、玄関のチャイムが鳴る。開けると、こはるが立っている。目が赤い。パジャマ姿
  - 「隣の部屋じゃなくて、同じ部屋に住まない？」
  - 悠斗が笑う。「……うちの部屋、6畳しかないぞ」
  - こはるが悠斗に抱きつく。「6畳で十分。壁越しじゃなくて、隣にいたい」
  - 年越しの鐘が遠くから聞こえる。結露した窓の向こうに、冬の星空が広がっている
  - 新しい年が始まる。壁一枚の距離が、ゼロになった冬
- **描写のポイント**: 壁越しの弾き語りがラブレター（タイトル回収）。「同じ部屋に住まない？」で壁越しの距離がゼロに。年越しの鐘で幕を閉じる
- **感情の流れ**: 幸福 → 照れくささ → 日常の変化 → 壁越しのラブレター → 「同じ部屋に」
- **文量目安**: 約3,500字

## 描写方針（EP4固有）

### 悠斗の語り口
- 一人称「俺」。テンポが良く軽快
- 音楽的なリズム感のある文体。比喩に音楽用語が混じる（「テンポが狂う」「アンサンブルが合う」等）
- お調子者だが、こはるの前では素直になる。ツッコミ役になることが多い
- こはるに対する感情は、「うるさい隣人」→「気になる声の持ち主」→「好き」と段階的に変化
- 声への執着。こはるの声の変化を細かく描写する

### こはるの話し方
- 「悠斗くん」と呼ぶ。タメ口
- 感情表現が豊かで声が大きい。「えへへ」が口癖（EP3のひなたとは違い、もう少し大人っぽい「えへへ」）
- 天然ボケ。ずれたことを言って悠斗にツッコまれる
- 演劇をやっているので、感情を込めた台詞回しが自然に出る。行為中は特に顕著
- 恥ずかしい時は声が小さくなる（普段が大きいだけに落差が目立つ）

### 冬の情景描写
- 五感をフル活用: 木枯らし、結露した窓、石油ストーブの匂い、鍋の湯気、イルミネーション
- 音の描写を重視: 壁越しの声、ギターの音色、年越しの鐘、冬の静寂
- 冬の「寒さ」と部屋の「温かさ」の対比。人肌の温もりが際立つ季節
- 古いアパートの生活感。安くて狭いけど、二人でいれば温かい
