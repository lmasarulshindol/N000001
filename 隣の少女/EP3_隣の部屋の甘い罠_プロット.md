## シーン構成（全8シーン）

### シーン1：引きこもりの春（日常パート）
- **場所**: 蓮の1LDKマンション
- **時間帯**: 4月上旬、昼
- **登場人物**: 蓮
- **出来事**:
  - 蓮の日常。リモートワークで自宅に籠もるWebエンジニアの生活
  - 昼過ぎに起きて、パソコンの前に座り、コーヒーを飲みながらコードを書く
  - 部屋は整頓されているが生活感がない。カーテンは閉めがち。外出は最低限
  - 窓の外は桜が満開。だが蓮は花見に行く相手もいない
  - 隣室から引っ越しの物音。業者の声、段ボールを運ぶ音。「また隣人が変わるのか」と思う程度
  - 蓮の内向的な性格。人付き合いが苦手。女性経験はほぼない。アニメと深夜ラジオが友達
  - 仕事は真面目にこなすが、プライベートは空っぽ。25歳にして枯れた生活
  - 夜、隣室から微かに音楽が聞こえる。壁が薄いわけではないが、静かな夜は少し聞こえる
- **描写のポイント**: 蓮の引きこもりがちな日常。春なのに閉じた生活。後のひなたの登場との対比
- **感情の流れ**: 無気力 → 微かな寂しさ（自覚なし）
- **文量目安**: 約2,000字

- **時間帯**: 4月中旬〜下旬（数日間のダイジェスト）
- **登場人物**: 蓮、ひなた
- **出来事**:

- **描写のポイント**: ひなたが蓮の閉じた日常を「侵食」していく過程。無防備な距離感。蓮の中に芽生える意識
- **感情の流れ**: 迷惑 → 慣れ → 楽しさ → 意識し始める → 自制
- **文量目安**: 約3,500字

### シーン8：隣の部屋でよかった（余韻）
- **場所**: 蓮の部屋 → マンションの共用廊下
- **時間帯**: 翌朝
- **登場人物**: 蓮、ひなた
- **出来事**:
  - 朝。雨は上がっている。窓から朝日が差し込む。桜の花びらが濡れた地面に散っている
  - 蓮のベッドで二人で目覚める。ひなたが蓮の腕の中にいる
  - ひなたが「おはよう、蓮さん」と笑う。いつもの太陽みたいな笑顔。でも、昨夜を知っている二人の間には新しい空気がある
  - 蓮が朝食を作る。目玉焼きとトースト。ひなたが「蓮さんの朝ごはん、初めて」と嬉しそう
  - 二人で食べながら、昨夜のことを少し照れくさく振り返る
  - 「蓮さん、今日から私のこと何て呼ぶの？」「……ひなた」「えへへ、名前で呼んでくれた」
  - ひなたが窓の外を見る。桜が散っている。「桜、もう終わりだね」
  - 蓮が「また来年咲くよ」と言う。ひなたが「来年も一緒に見れるかな」「……見れるだろ、隣だし」
  - ひなたが蓮に抱きつく。「隣の部屋でよかった」
  - ひなたが自分の部屋に帰る準備をする。パジャマのまま。共用廊下は数歩の距離
  - 玄関で「じゃあ、行ってきます」と手を振るひなた。制服ではなくパジャマだけど、その言葉が自然に出る
  - 蓮が「行ってらっしゃい」と返す。ひなたが隣の部屋のドアを開けて入る
  - 蓮が自分の部屋に戻る。壁の向こうから、ひなたが鼻歌を歌っているのが聞こえる
  - 蓮はモニターの前に座る。いつもの席。いつもの画面。でも、世界が違って見える
  - カーテンを開ける。春の光が部屋に差し込む。桜の花びらが風に舞っている
  - 蓮の日常が、決定的に変わった春
- **描写のポイント**: 朝の光と桜。「隣の部屋でよかった」がタイトル回収。蓮がカーテンを開ける＝閉じていた世界が開く象徴。壁越しの鼻歌で幕を閉じる
- **感情の流れ**: 幸福 → 照れくささ → 日常の変化への喜び
- **文量目安**: 約3,000字

## 描写方針（EP3固有）

### 蓮の語り口
- 一人称「俺」。内省的でツッコミが多い
- オタク的な比喩が混じる（「ラノベの展開」「フラグが立った」等。ただし多用しすぎない）
- 自分を卑下しがち。「俺みたいな陰キャに」「現実はそんな甘くない」
- ひなたに対する感情は、「眩しい隣人」→「気になる存在」→「好き」と段階的に変化
- 初体験の緊張と不器用さを正直に語る

### ひなたの話し方
- 「蓮さん」と呼ぶ。敬語ではないがさん付け
- 明るくてテンポが良い。「えへへ」が口癖
- 感情が素直に出る。嬉しい時は声が弾み、寂しい時は声が小さくなる
- 甘え上手。「ねえ」「お願い」が自然に出る

### 春の情景描写
- 桜：満開→散り始め→葉桜。物語の進行と桜の変化がリンク
- 雨：春の嵐。行為シーンのBGM
- 光：カーテン越しの春の光、モニターの青白い光、朝日
- 蓮がカーテンを閉めている→最後に開ける、という変化が物語全体のアーク
