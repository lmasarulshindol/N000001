# 第1章 ― 事故じゃん、これ / 詳細プロット

**章の位置づけ**: 全6章（プロローグ＋5章）中の第1章。プロローグの翌日、関係が「偶然の一夜」ではなかったと確定する章
**想定字数**: 22,000〜26,000字
**陥落段階**: 1（再挿入・言い訳付き）
**主な視点**: ハルト（三人称単一視点）
**時制**: 物語現在時制（春、4月下旬の平日・木曜）

---

## この章の目的（執筆時に必ず意識する）

1. **「偶然ではなかった」という確定**
   - プロローグの夜が「事故」で終わっていたら、この物語は成立しない
   - 翌日の再訪によって、読者に「この二人は続いていく」と安堵と覚悟を同時に与える

2. **「事故だから」「暇だから」の自己否認語彙の確立**
   - 以降の章まで響く、ココアの防衛フォーマット
   - 「事故」「暇」「たまたま」「なんとなく」を口癖として染み込ませる
   - これが第5章の「また来るから」で最終的に溶けていく伏線

3. **「ハルトが無言で受け入れる」という選択の重み**
   - ハルトは追い返さない。しかし積極的に招き入れもしない
   - 「鍵を開けて、黙って先に入る」という所作だけで、関係の形を決める
   - 言葉にしないことの重みを、地の文で丁寧に拾う

4. **行為シーンの質的転換**
   - プロローグ: ココアが煽りの果てに自滅して崩される（受動的陥落）
   - 第1章: ココアが自分で服を脱ぐ（能動的選択の初披露）
   - 「昨日の続き」ではなく「今日の意思」としての行為

5. **強がりの劣化（読者にだけわかる）**
   - ココアのメスガキ語彙は維持されているが、声の温度が昨日より一段低い
   - 「きしょ」「最悪」は形式化している
   - 目を合わせられる回数が減っている（これを地の文で観察として拾う）

## シーン1 ― 朝のハルト

**時刻**: 7:10 〜 8:30
**場所**: 202号室 → アパート外階段 → 駅

### シーンの目的
- プロローグの夜の「後」を、最初の地の文でしっかり定着させる
- ハルトの身体と思考に残る違和感を、日常の所作越しに描く
- 「昨夜は夢だったのかもしれない」と読者に一瞬思わせ、シーツの残り香で否定する

### 出来事

**① 目覚め**
1. アラームより先に目が覚める。午前7時10分
2. 天井を見上げる。ただの石膏ボード。昨夜「ほんの少しだけ、違って見えた」天井は、朝の光の下では、ただの天井に戻っている
3. 起き上がろうとして、身体が重いことに気づく
4. 腰の奥に、使ったことのない筋肉の疲労。自覚するまで数秒かかる
5. 昨夜のことが、断片で戻ってくる：「……ハル、ト、さん」「……はい」「かち、とラッチがはまる、ただそれだけの、弱い音」
6. ハルトは、目を一度閉じる。目を開けると、部屋は、変わらず、202号室のままだ

**② シーツ**
1. 起き上がる。シーツの皺。枕のくぼみ
2. 栗色の、細い髪が、一本、白い布の上に、まだ残っている
3. ハルトは、それを、指でつまんで拾う。軽い
4. 指先に乗せて、少しのあいだ、眺める
5. ゴミ箱には捨てない。机の脇の、普段は使っていない小さなトレイに、そっと置く
6. 自分のその動きに、自分で、小さく、眉をひそめる
7. シーツを剥がす。洗濯ネットに放り込む
8. コインランドリー行き、と頭の中でメモする

**③ 身体の残り香**
1. シャワー。湯を出す。
2. 鏡。眼鏡を外した、いつもの自分の顔
3. ただ、背中に、細い線が二本。ココアが爪で残したしがみつきの跡
4. ハルトは、鏡越しに、それを見て、何も言わない
5. 湯を浴びる。湯が熱くなっていくのに、身体が冷たく感じる

**④ 朝食と支度**
1. 冷蔵庫。昨日のまま。もやし、卵、豆腐、納豆
2. トーストに納豆、卵。妙に平常運転
3. 机の上のエントリーシートを見る。「学生時代に最も力を入れたこと」の欄。半分まで埋まったまま
4. ペンを取る。書けない。書こうとして、どの文字も、昨夜の断片に邪魔される
5. ノートパソコンを閉じる
6. リュックに、研究室に出す進捗資料のUSBと、タンブラーだけ入れる

**⑤ 出がけ**
1. 玄関。サンダルではなく、スニーカーを履く
2. ドアノブに手をかける前、一瞬だけ、廊下の方を意識する
3. 201号室側の壁は、静かだ
4. 朝、中2は、学校に行く時間だ。それくらいの最低限の時間感覚は、さすがにハルトにもある
5. ドアを開ける。外廊下は、昨日より少し明るい
6. 201号室の前を通り過ぎる。見ない。見ないように、意識して、見ない
7. 階段を降りる。いつもの速度で

**⑥ 駅まで**
1. 住宅街の生活道路。朝のゴミ収集車
2. 小学生の登校列に抜かれる
3. スマホを見る。LINEの未読なし。就活アプリの通知、2件。開かない
4. 電車。定位置の端の席。吊り革をつかんだ右手の、指の関節に、昨夜の何かの記憶が、まだ、薄く、残っている気がする
5. 自覚した瞬間に、目を、車窓の外に逃がす

### 描写のポイント
- 昨夜の「揺らぎ」を、朝の凡庸な所作の中に、埋めて、しかし隠し切らないように描く
- 身体の疲労 → シーツの髪 → 背中の爪痕、という「物証」の三段階
- **朝、ハルトは一度も、ココアの名前を心の中で呼ばない**（重要）
- 代わりに、すべてを「昨夜」「あの子」「あれ」で処理する
- 読者だけが、「名前を呼ばないこと自体が、名前を意識している証拠だ」と気づく

### シーン終端
- ハルトが、いつもの席の、いつもの位置に、立っている
- 車窓が、昨日と同じ景色を流していく
- ただ、ハルトの視線だけが、昨日と少しだけ、違う方向を向いている

## シーン2 ― 昼の大学

**時刻**: 11:30 〜 17:00
**場所**: 大学の研究室／学食／帰路の駅

### シーンの目的
- 外部の人間（藤堂）の視点を初めて入れて、ハルトの変化を「他人の目」から照射する
- ただし、核心は明かさない。「顔色悪い」レベルの表層だけ
- 読者には「周囲にバレていないが、確実に変わっている」ハルトを楽しませる
- 午後の時間帯で日常生活を一度挟み、夕方の「再会」のインパクトを高める

### 出来事

**① 研究室**
1. 研究室のドア。ホワイトボードに、卒研中間発表のスケジュール
2. ハルトのデスク。積み上がった論文。開きっぱなしのPython環境
3. 進捗資料を教授のメールボックスに投入
4. デスクに座る。昨日の作業の続きを開く
5. コードが、いつもより読めない
6. 10分ほど、画面を見ているだけで、手が動かない

**② 藤堂との鉢合わせ**
1. 隣のデスクに、藤堂亮が来る
2. 藤堂は陽キャ寄りの同期。ハルトの数少ない「話す相手」
3. 「お、おはよ。……なんかお前、顔色ヤバくね？」
4. ハルト:「……そうか？」
5. 藤堂:「そうだって。つか目の下のクマ、昨日より黒いぞ」
6. ハルト:「……寝不足」
7. 藤堂:「バイト入れすぎ？ それとも就活病む系？」
8. ハルト:「……どっちでも、ある」
9. 藤堂:「飯行こ。学食」

**③ 学食での会話**
1. 学食の日替わり。A定食600円（鯖の味噌煮）
2. 藤堂はカレー大盛り
3. 藤堂:「今週も内定者報告、何件流れてきた？」
4. ハルト:「……見てない。拒否設定入れた」
5. 藤堂:「賢いわ。あれ読んでたら死ぬもん」
6. 藤堂が最近の自分の就活の話。ハルトは聞き役
7. 途中で、藤堂の口から出る：「あ、そういやさ、お前のアパート、駅から12分だっけ？ 安いんだろ。今度飲みに行っていい？」
8. ハルトの中で、0.3秒だけ、反射が遅れる
9. ハルト:「……散らかってるから」
10. 藤堂:「えー、お前の部屋汚いイメージなかったわ」
11. ハルト:「……そういう時期」
12. 藤堂:「まあいいや、そのうちな」

**④ 午後の研究室**
1. 午後は、多変量解析のスクリプトを書く予定だった
2. 書けるには、書ける。指は動く
3. ただ、変数に「cocoa」と打ちそうになって、一度、キーボードから、手を離す
4. 深く息を吐く
5. 天井の配線ダクトを見る。研究室の天井も、202号室と同じ、安い石膏ボードだ
6. ハルトは、目を閉じる。目を閉じたら、まぶたの裏に、耳のほうに流れていった涙の軌跡が、見えた
7. もう一度、目を開けて、コードに戻る

**⑤ 帰路**
1. 17時、教授から簡単なフィードバックが返ってくる。「方向性問題なし、続けて」
2. ハルトは、研究室を出る
3. 駅のホーム。帰宅ラッシュの少し手前の時間
4. ハルトの足は、いつもの特急ではなく、各駅停車のホームに向かう
5. 家に、早く帰ろうと思った自分に、少しだけ、違和感を覚える
6. **気づかないふりをして、各停に乗る**

### 描写のポイント
- 藤堂の視点を借りて「ハルトが変わっている」を外側から一度だけ証明する
- ただし藤堂は鈍感な陽キャなので、真相には絶対にたどり着かない
- **「cocoa」と打ちそうになる瞬間**は、この章の地の文が初めて、無意識にココアの名前をなぞる瞬間
- 「早く帰ろうとしている自分」に気づかないふりをする、が、本章の中盤の引き

### シーン終端
- 各停の、空いている車内。ハルトは座らず、立ったまま、ドアの脇に肩を預ける
- 窓の外が、夕方の光に変わっていく

## シーン3 ― 再訪

**時刻**: 17:45
**場所**: アパート前の歩道 → 外階段 → 外廊下 → 202号室前

### シーンの目的
- **本章の最重要シーン**
- 読者の予感（「来てるかもしれない」）を、最大限に引きつけてから、扉の前の映像で確定させる
- ハルトの「鍵を開けて、無言で先に入る」所作の重みを、地の文で最大限に描く
- ココアの第一声「……昨日のは、事故だから」を、本作のフォーマット語彙として確立する

### 出来事

**① アパートが見えるまで**
1. 駅から、住宅街の奥へ。徒歩12分

6. 202号室の電気は、当然、消えている
7. 201号室の電気も、消えている

**② 外階段**
1. 外階段の、一段目に足をかける
2. 階段は、4段目のあたりで軋む
3. その軋みを、いつものように、踏み抜けて、上がっていく
4. 踊り場で、一呼吸。曲がる
5. 二階の廊下が、見えてくる

**③ 扉の前**
1. 蛍光灯は、昨夜と同じ、白く切れかけた、あの蛍光灯
2. ただし、光の下に、立っている者がいる
3. **202号室のドアの前に、制服のままのココアが立っていた**

5. 黒のハイソックスの上、膝小僧の半分が、スカートから出ている

10. が、スクロールの速度が、意味のない速度だ。何も読んでいない
11. 目は、完全に、画面に固定されている

**④ 近づくハルト**

2. 202号室の、3メートル手前あたりで、ココアが気づく
3. ココアは、顔を上げない
4. スマホを、下に向けて、画面を消す気配だけ見せて、消さない
5. そのまま、スマホの画面を見続ける
6. 2メートル手前
7. 1メートル手前
8. ハルトは、ドアの前に立つ

**⑤ 言い訳**
1. 目を合わせないまま、ココアが、口を開く
2. 「……ちょっとだけ。ほんとに、ちょっとだけだから」
3. 前置き。ハルトは、答えない
4. 「……昨日のは、事故だから」

6. 「勘違いしないでよね」

16. ハルトは、そのまま、202号室に、先に、入る

## シーン4 ― 食卓（避けられる話題）

**時刻**: 18:20 〜 19:40
**場所**: 202号室・キッチン〜机

### シーンの目的
- 肉体関係が先にあった二人が、「普通の食卓」を共有する、奇妙な温度感を描く
- 昨夜の話題は、お互いに、絶対に、触れない
- その「触れなさ」自体が、二人のあいだの新しい言語になっていくことを読者に提示する
- 行為シーンに入る前の、緩やかな助走

### 出来事

**① コンビニのレジ袋の中身**
1. ココアが、絨毯の上に、レジ袋を、ぽとりと、置く
2. レジ袋から出てくる：プリン2個（同じ銘柄）、コーラ500ml 1本
3. プリンが2個、ある
4. ハルトは、それを、一度だけ見てから、目を逸らす
5. ココアは、「プリン、半額だったから、ついでに。食うなら食ってもいいし、食わないならあたしが食うし」と、早口で説明する
6. ハルトは、「……あとでいい？」と、短く、答える
7. 本章で、ハルトが、ココアに対して、発する最初の言葉

**② パスタを作るハルト**
1. ハルトはキッチンに立つ
2. 冷蔵庫：もやし、卵、豆腐、納豆。それに、昨日買い忘れていたベーコン半パック
3. 食パンではパスタはできない。乾麺の、業務スーパーで買った安いスパゲッティ
4. 湯を沸かす。ペペロンチーノの段取り
5. ココアは、ベッドではなく、今日は、絨毯に座っている。スクールバッグを背もたれにして、脚を伸ばす
6. 膝から下が、制服のスカートから出ていて、しかし、今日はショーパンではない。黒のハイソックスが、ふくらはぎの真ん中あたりまで
7. ココアは、プリンを1個、勝手に開けている
8. スプーンは、引き出しから、自分で出した
9. ハルトは、その音を、背中で聞いている。注意しない

**③ 黄色い照明の下の二人**
1. 二人分のパスタができる
