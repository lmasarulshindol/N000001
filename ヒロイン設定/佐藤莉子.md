## ステータス（初登場時 = 入店4ヶ月・初接客後）

| 項目 | 値 | 補足 |
|------|-----|------|
| 性欲 | **3** | ほぼない。快楽は「仕事の副作用」。体が反応しても心が追いつかない |
| 性知識 | **6** | 研修で一通り叩き込まれた。技術として知っている。感情的な理解は薄い |
| 感度 | **7** | 小さい体に高い反応。本人の意図に反して声が出る |
| 羞恥心 | **4** | 仮面で押さえている。営業中は低い。素に戻った瞬間だけ跳ね上がる |
| 積極性 | **8** | 接客のプロ意識。自分から仕掛ける・誘う・主導する。仕事だから |
| 信頼度 | **2** | 客への信頼はほぼゼロ。「おじさん」は全員駒。花音にだけ5程度 |
| 経験値 | **経験あり** | 「はじめて。」撮影で処女喪失済み。以降、日常的に接客 |

### ステータスから読み取れる振る舞い

- **積極性8 × 信頼度2**: プロとして完璧に動くが、心は預けない。一線の壁
- **感度7 × 性欲3**: 体は反応するが、快楽を「欲しい」とは思わない。仮面が剥がれる瞬間
- **性知識6 × 10歳**: 技術は知っている。でも行為の「意味」は半分しか分かっていない
- **羞恥心4（通常）→ 8（亀裂時）**: 仮面をかぶっている間は平気。素に戻ると急に恥ずかしい

### 特殊項目

| 項目 | 補足 |
|------|------|
| 仮面強度 | 高。剥がれるのは体が限界を超えた瞬間のみ |
| 母への想い | 全行動の根底。内面独白に1文だけ出る |
| 体格ペナルティ | 133cm/28kg。挿入時に「入りにくさ」「圧」が物理的に描写される |
