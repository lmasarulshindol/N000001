# 美月編 第2話 プロット「おっとりメイドの初めてのご主人様♡」

---

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | おっとりメイドの初めてのご主人様 |
| 視点 | 橘美月 一人称「私」 |
| 時系列 | 10月4日（土）鳳翔祭初日 9:00〜12:30 |
| お相手 | **中村悠真**（3年・文化部系・穏やかで礼儀正しい）。3人組の一人として来店するが、カフェでは中村だけが美月のテーブルに残り、午前中ずっと1対1で接客される |
| テーマ | 「全部計画通り♡」のはずだった —— たくらみの裏の本音が、初めてのご奉仕で溢れ出す |

## 第1話からの引き継ぎ

- 美月はメイド喫茶の発案者。夏休みにカフェで雑誌を読んで思いつき、莉子と二人で形にした
- リハーサルで神代にフェラの「お手本」を披露。練習の成果で上手い
- リハーサル後の空き教室で一人、「知らない人に中出しされるのって……」と妄想して濡れていた
- 不安はゼロ。あるのは期待と好奇心。「私のたくらみ、始まるよ」で第1話を終えた
- 神代への好意あり。でもメイドリーダーとしての自分を優先している
- おっとりした笑顔の裏に「たくらみ」の顔を隠している。計算高いが、それは「みんなが幸せになる場所を作りたい」という善意から

## 前編テーマ

**「メイドリーダーの完璧な午前 —— でも余裕の笑顔の下で、身体がじわじわ火照っていく」**

美月は完璧だった。開場から午前中盤まで、カフェで余裕の接客をこなし、他のメイドたちを見守り、アドバイスし、この場所の「母」のように微笑んでいた。でも——あかりの声が個室から漏れてきた瞬間、美月の太ももがぴくりと震えた。「そろそろ私も♡」——リーダーの笑顔のまま、美月はフェラブースへ向かう。

## 前編 シーン構成

### ■ Scene 1：開場——私の舞台が幕を開ける（9:00〜9:30）

**場所：カフェスペース**

- 開場直前。美月が最終チェックを仕切る。

- 「はい、美月チェック完了 みんな、準備いい？」
- 9人のメイドが整列。それぞれの表情を見る。
  - あかり：緊張しながらも目がきらきら。ポニーテールがぴょこぴょこ。
  - ひなた：袖を握りしめて震えている。でも前を向いている。
  - 莉子：腕を組んでクール。でも耳がほんの少し赤い。
  - るる：テンション最高潮。「まじやばーい」
- *みんなかわいい 最高のメイド軍団*
- 神代が入口に立つ。フロアマネージャーの腕章。全員を見渡して——
- 「似合ってる。全員」
- 美月が一歩前に出る。巻き髪をさらりとかきあげて。
- 「ありがとう 神代くんにそう言ってもらえると、嬉しいな♡」
- にこり。穏やかな笑顔。目だけがほんの少し妖しく光る。
- *神代くんの目が一瞬こっちに来たの、気づいてるよ*

- 開場。「お帰りなさいませ、ご主人様♡」9人の声が重なる。
- *さぁ 私のたくらみ、始まるよ*

**描くこと：**
- 美月＝リーダー兼発案者。「この場所を作った」という特権的な視点
- メイド全員を見守る「母」の視線
- 神代への小さな好意（でもリーダーモードで抑える）
- 開場の高揚感を「計画が動き始めた」喜びとして描く

- *知らない人の♡ 初めて♡*

- *あ♡*
- *……神代くんと、味が違う♡*
