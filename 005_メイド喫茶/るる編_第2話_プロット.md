# るる編 第2話 プロット「ギャルメイドは止まれない」

---

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | ギャルメイドは止まれない |
| 視点 | 桃園るる 一人称「あたし」 |
| 時系列 | 10月4日（土）鳳翔祭初日 9:00〜16:30 |
| テーマ | テンション全開で走り続けるギャルメイドが、5人の個室を駆け抜けた先で初めて「ぽかん」とする——深く考えない強さと、考えないから見えないものの話 |
| 行為範囲 | 全オプション対応（フェラ複数＋個室5回＝全メイド最多） |
| お相手 | 特定の一人ではなく複数。5人の別々の男子と、それぞれ1回ずつ |

---

## 第1話からの引き継ぎ

- るるは即決で全オプション対応を志願。「面白そうなことに真っ先に飛び込む」がモットー
- メイド服をハサミで短くカットし、ギャルメイド爆誕。自信しかない
- リハーサルでは一番に手を上げて神代にフェラ。余裕でこなした
- ただし第1話ラストで「本番は知らない男の……」が一瞬よぎり→即座に打ち消した
- 「不安を認めない。それが桃園るるの強さ」
- 深く考えない。立ち止まらない。楽しい方に走る。怖いことは考えない

---

## 前編テーマ

**「楽しいしか出てこない——フェラも個室も全部やばい。あたし何でもできるじゃん♡」**

開場直後からフルスロットルのるるが、フェラブースを飛び越えて個室に突入し、午前だけで3人と本番をこなす。「楽しい」「やばい」「もっと」——感情のアクセル全開。でも3人目の後、一瞬だけ……静かになる瞬間がある。それを振り払って、昼休憩に突入する。
