# 律編 第2話 プロット「男前メイドの長い一日」

---

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | 男前メイドの長い一日 |
| 視点 | 相沢律 内心「俺」／セリフ「私」（矯正） |
| 時系列 | 10月4日（土）鳳翔祭初日 9:00〜16:30 |
| テーマ | 男前にこなすけど、身体は正直——行為を重ねるうちに「です♡」が棒読みじゃなくなっていく、認めたくない変化の物語 |
| 行為範囲 | フェラブース＋個室4回（午前2回＋午後2回） |
| お相手 | 複数の客。特定の相手への恋愛感情はなし。全員を「仕事」としてこなす——はずだった |

## 第1話からの引き継ぎ

- 律はるるに負けたくない一心で全オプション対応を志願
- ボーイッシュショートにカチューシャ。腕まくりメイド。意外と似合う
- メイド口調が壊滅的。「お帰りなさいませ、ご主人様♡……です……」→棒読み
- リハーサルで神代にフェラ。「いくぞ」で始める男前スタイル。サバサバこなしたが、頬が赤かった
- 「ドキドキなんてしてない」→してる。でも認めない
- 千歳に「かっこいいです」と言われて素直に嬉しかった
- 「やるからには全力。負けねぇ」で第1話を終えた

## 前編テーマ

**「いくぞ——じゃなくて、失礼しますね♡。男前メイドの午前中は、効率と照れの戦い」**

律は迷わない。カフェは苦手だからフェラブースへ直行。知らない男の前に膝をつき、「いくぞ」——じゃない、「失礼しますね……ご主人様♡」。棒読み。でもフェラ自体はサクッと手際よく、口内射精まで男前に完遂する。個室も「奥、行くか？」——じゃなくて「奥に……行きませんか♡」。頑張って敬語を使うたび、自分の声に照れる。そして——予想外に声が出る自分に、一番驚いているのは律自身。

## 前編 シーン構成

### ■ Scene 1：開場——カフェなんて得意じゃねーし（9:00〜10:00）

- 開場。9人のメイドが「お帰りなさいませ、ご主人様♡」と声を揃える。

- カフェを少しだけ担当。最初の客にメニューを出す。
  - 「ご注文は……何にしますか。……ご主人様」
  - *……なんだこのむず痒さ。鳥肌立つ。*
  - 客「お、おすすめは？」
  - 「コーヒー。ブラックで」
  - *——あ。メイドっぽくねえ。*
  - 「……あ、えっと。コーヒーが……おすすめ、です♡」

- 隣のブースからるるの「んっ♡ んっ♡」が聞こえる。もう1人目を始めている。
- *……桃園、はえーな。*
- *負けてらんねぇ。*

- *——よし。いくぞ。*
- *……じゃなくて。*
- 深呼吸。
- 「……いらっしゃいませ♡ ご主人様♡」

**描くこと：**
- カフェ接客の壊滅的なぎこちなさ＝律がフェラブースを選ぶ動機

### ■ Scene 2：初めての本番フェラ——男前スタイル開幕（10:10〜10:25）

**場所：フェラブース**

- パーテーションの中。客が椅子に座っている。緊張している。
- 律が膝をつく。スカートが床に広がる——が、律の膝のつき方は「正座」に近い。スポーツ選手が膝をつく感じ。
- 「失礼しますね……ご主人様♡」

- ベルトに手を伸ばす。指先は——
- *震えてねぇ。大丈夫。リハーサルでやった。神代相手にできたんだ。知らねぇ奴相手でも同じだ。仕事だ。*
- ファスナーを下ろす。取り出す。
- ——リハーサルの神代のとは、形が違う。
- *……ふーん。人によって違うんだな。当たり前か。*
- 感傷に浸る間もなく、律はさっさと始める。
- 先端を唇で包む。舌を絡める。深く咥える。
- リハーサルで覚えた手順。でも律のフェラには独特の特徴がある。
  - **速い**。テンポがいい。焦らさない。
  - **手を使う**。片手で根元を握り、口と手の連動で効率よく刺激する。
  - **無駄がない**。上目遣いで甘えるとか、唇で転がすとか、そういう「演出」がない。実直に、気持ちよくさせることだけに集中する。
- *神代のとき「男前フェラ」って言われた。まあ——そうだろうな。俺のフェラは「仕事」だ。*
- 客が声を漏らす。「っ……すごい……上手い……」
- *当たり前だ。やるからにはちゃんとやる。*
- 「んっ……ん……」
- ——口の中が、熱い。知らない男の味。しょっぱくて、少し苦い。
- *……別に。なんとも思ってない。味がするだけだ。*
- 客が限界を告げる。律は加速する。手と口の連動。
- 口内射精。律が全部受け止める。
- ごくん——と飲み込んで。
- 「ん。ごちそうさま」
- ——メイド口調、忘れてる。
- *あ。*
- 慌てて言い直す。「……ごちそう、さまでした♡ ……ご主人様♡」
- 客が呆気にとられている。「か、かっこいい……」
- *かっこいいって言われても。メイドなんだけど。*
- でも——頬が、ほんの少しだけ赤い。

**描くこと：**
- 律のフェラスタイルの具体的特徴——速さ、効率、無駄のなさ＝「男前フェラ」の正体
- 他メイドとの差別化：陽菜の丁寧さ、るるの甘えた声に対して、律は「実直」
- メイド口調を忘れる＝素が出る瞬間
- 「かっこいい」に照れる＝第1話の千歳の言葉との連続
