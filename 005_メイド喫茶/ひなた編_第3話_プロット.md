## 第2話からの引き継ぎ

- 初日、初客・悠人さんに初手コキ・初フェラ・初挿入（対面座位）・初中出しをすべて捧げた
- フェラ中、隣のブースと個室から**あかり先輩の声**が聞こえた。甘くて、元気で、恥ずかしげもなく大きい声
- 初日の個室では人生最大の声を出したが、それは「我慢できなかった」偶然。意図的な大胆さではなかった
- 午後は「もっと上手くなりたい」「フェラ好きかも♡」と口にした。足は震えなくなった
- 初日成績：フェラ複数回、個室2回。148cmの小さなメイドが、一歩踏み出した日
- 閉店後「あたし……変われました。みんなのおかげです」——でも声はまだ小さい
- **身体の変化**：初日の経験で感度は上がっている。本人は「大胆になった」と意識しているが、身体はまだ初日の余韻を覚えている

## 前編テーマ

**「あかり先輩の声を、あたしの鎧にする——初日より、自分から」**

2日目の朝。ひなたは鏡の前で小さく「お帰りなさいませ、ご主人様♡」と練習する。あかり先輩の、あの元気な声。初日は客に選ばれるのを待っていた。今日は——**自分からオプションを言う**。声は借り物でも、一歩は本物。

## 前編 シーン構成

### ■ Scene 1：2日目の朝——あかり先輩の声（8:30〜9:00）

**場所：更衣室・鏡の前**

- 2日目の朝。ひなたは初日より15分早く来た。メイド服に袖を通す手に迷いがない。
- 鏡の前。おさげを直す。カチューシャを触る。
- 小さく口を開く。「……お、お帰りなさいませ……ご主人様♡」
- *……小さい。やっぱり小さい。*
- 目を閉じて、思い出す。初日。フェラブース。隣から聞こえた声。
- *「んっ あっ ご主人様♡」——あかり先輩の声。あんなに大きくて、あんなに元気で。*
- *あの声なら、怖くないみたいだった。*
- もう一度。今度は少しだけ——腹に力を入れる。お姉ちゃんがライブの前にやってたやつ。
- 「お帰りなさいませ、ご主人様♡」
- *……少しだけ、大きくなった？*
- あかりが更衣室の扉を開ける。「ひなたちゃん、おはよ♡ 今日もかわいい♡」
- ひなたがびくっとする。「あ、あかり先輩、おはようございます……」
- あかり「昨日すごかったもんね♡ 初めてのご主人様に全部あげちゃって♡」
- ひなた、顔が赤い。「……が、がんばりました……」
- あかり「今日はもっとできるよ♡ ひなたちゃん、声出していいんだよ♡ 小さくても、出していいの♡」
- ひなた「……え？」
- あかり「あたしね、最初も恥ずかしかった。でも——出したら、楽になった♡ ご主人様も喜んでくれるし♡」
- ポニーテールをぶんっと振る。あかりの笑顔。
- *……あかり先輩の声、借りていいのかな。*
- *……借りてみよう。あたしには、まだあたしの声が足りない。*
- 開店直前。9人整列。ひなたは隅じゃなく、**列の真ん中寄り**に立つ。初日は端だった。

**描くこと：**
- 「声＝鎧」のコンセプト導入
- あかりからの一言が、2日目の起爆剤（あかり編第4話 Scene 1 と同時刻の別視点）
- 列の位置の変化＝小さな自信の外化

### ■ Scene 2：開場——初日より大きい「お帰りなさいませ」（9:00〜10:00）

- *……出た。あかり先輩の真似、した。* 
- *胸がどきどき。恥ずかしい。でも——止まらなかった。*
- 2日目は初日より客が多い。口コミ。「小さなメイド」「初日で個室まで」
- 1人目の客。2年・少しチャラい。「ひなたさん指名で来た」
- ひなた、一歩踏み出す。足は震えていない。
- 「お帰りなさいませ、ご主人様♡ ひなたが、ご案内します♡」——**あかり先輩のノリで、元気に**。
- テーブルへ。メニュー表を出す。
- 初日は「紅茶が美味しいです」としか言えなかった。
- 今日は——深呼吸。あかり先輩の声を思い出す。
- 「ご主人様♡ 今日は……紅茶もいいですけど……♡ ひなた、昨日より上手くなったので……♡ 手コキとか、フェラとか……♡ えっと……奥のお部屋も……♡ ご、ご案内、できます……♡」

- 「……は、はい♡ 昨日、初めて全部やったので……♡ 今日は——今日は、あたしから、ご提案、します……♡」
- *「あたしから」——初日は言えなかった。* 
- 客が笑う。「じゃあ全部頼むわ。順番に」
- ひなた、小さく頷く。おさげが揺れる。
- *……全部。初日と同じ。でも今日は——あたしが選んだ。*
- **10:00** カフェを短く済ませ、フェラブース①へ（Scene 3へ）

**合間に他メイド（背景）：**

- 口に含む。舌を動かす。初日の悠人さんのときより——深い。

- 「んっ……♡ ご主人様♡ ……ここ、気持ちいい、ですか……♡」

**描くこと：**

**場所：個室2（初日と同じ部屋。思い出の場所）**

- 「……んっ……♡ ご主人様……♡ 入って、きました……♡」

### ■ Scene 5：フェラブース②——あかり先輩の嬌声で、スイッチが入る（11:10〜11:45）★前編クライマックス・前編終わり

## 後編テーマ

**「借り物の声を返す日——あたしの声で、リードする」**

午後。あかり先輩の声は個室からカフェまで響き続けている。ひなたはもう真似だけでは足りないと知った。午後の客——**悠人さんが再来**するなら、初日の恩返しに、あたしの声で全部奉仕する。初日より大胆。初日より大きい。でも初日より、**あたしらしい**。

- *……お姉ちゃんの声は、借りなくていい。自然と大きい。*
- *あたしは——あたしの中に、大きい声を探す。*
- 美月「ひなたちゃん、午前すごかったよ♡ 自分から個室誘ったんでしょ♡」
- 「……はい。後半、あかり先輩の——本番の声、聞いちゃって……」

### ■ Scene 2：悠人さん、再来——初日のご主人様（13:00〜14:00）

- 「……お帰りなさいませ、ご主人様♡」
- 声が——**初日の悠人さんへの出迎えより、大きい。** 借り物混じり。
- 悠人「ひなたさん。昨日お世話になった。今日も来ちゃった」

- 初日は「ただいま」と言ってもらって泣きそうになった。今日は——笑っている。

- 紅茶。初日と同じ。

- 「……あかり先輩の声、借りてます♡ でも——悠人さんには、あたしの本音、出したい、です♡」

- 口内射精。飲み込む。
- 個室2へ。初日と同じ部屋。
- 「……今日は……対面座位、じゃなくて……♡ 後ろから、も……できます……♡ 昨日より、大胆、です……♡」

**同時刻の外部（軽く）：**

**描くこと：**

### ■ Scene 3：3人目——あたしの声で、リードする（14:30〜15:30）

- 声——**あかり先輩の真似はもうしない。**

- あかりへの感謝＝「声を借りた」物語の完結

- お姉ちゃんへの報告＝第1話からの支え線の着地
- 恋は描かない。**自己変革の完結**

## ひなた編の独自性（第1〜3話通し）

ひなた編は**「恋」ではなく「自己変革」**の物語。

- 第1話：お姉ちゃんに背中を押されて一歩踏み出す
- 第2話：初めてのご主人様（悠人）に全部捧げて「できる」と知る
- 第3話：あかりの**嬌声**でスイッチが入り、**自分の声で感じる**ようになる

148cm・声が小さい・おとなしい——「変われない」と思っていた子が、**メイド喫茶という舞台**で、**声という武器**を手に入れる。あかりは「恋のヒロイン」、ひなたは「成長のヒロイン」。

## 描写方針

| 項目 | 方針 |
|------|------|
| 視点 | 小鳥遊ひなた一人称「あたし」。内面は短い文。自己否定→小さな肯定 |
| 声の描写 | **四段階**：①消え入りそう ②あかり真似（鎧・演技） ③**嬌声トリガーでスイッチON** ④本声（感じている声） |
| スイッチON | **11:30**が核心。嬌声の台詞→太ももの熱→奉仕の質が変わる→客が「急にすごくなった」→イきかけ。初日の焦りとの対比必須 |
| 嬌声 | あかり編第4話後編から引用可。「翔太くんっ♡」「イく♡」等。**メイド口調ではない本音**であることが重要 |
| 身体 | 148cm・おさげ・そばかす・萌え袖。小ささは変わらない。**態度**が変わる |
| あかり真似 | 完璧コピーにしない。不器用で、時々裏返る。客に「デカくなった」と言わせる |
| 悠人さん | 恋愛ではない。**恩と成長の鏡**。初日の優しさを、ひなたが午後の後輩に返す |
| ♡の使い方 | 初日並み（控えめ多め）。大胆になるにつれ少し増える。絶頂時は♡♡まで |
| 行為描写 | 初日より**能動的**な動詞（誘う・引く・提案する）。体位は対面＋後背位 |
| 他メイド | 合間に短く。あかり＝声の師匠。美月＝見守り。るる＝声デカい仲間 |
| 神代 | 短い声かけ程度。ひなた編は神代視点に切り替えない |
| 閉店 | 恋の余韻ではなく**自己肯定**で締める |
