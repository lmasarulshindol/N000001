# 莉子編 第2話 プロット「完璧なメイドの不完璧な午後」

---

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | 完璧なメイドの不完璧な午後 |
| 視点 | 椎名莉子 一人称「私」（内心も「私」） |
| 時系列 | 10月4日（土）鳳翔祭初日 9:00〜16:30 |
| テーマ | 制度設計者が自分の制度に飲み込まれる——「カフェのみで通す」はずだった完璧主義の鎧に、午後、初めてヒビが入る |
| 行為範囲 | 午前カフェ専念→午後フェラブース2回（個室なし） |
| お相手 | 特定の一人ではなく、フェラブースに来た2人の別々の客 |

---

## 第1話からの引き継ぎ

- 莉子は美月と共にメイド喫茶を企画した制度設計の頭脳。父親が学校の理事。完璧主義
- 「全オプション対応」に自分の名前を入れたが、「カフェのみで通す」と自分に言い聞かせていた
- リハーサルには「品質チェック」名目で見学のみ。フェラの練習はしていない
- メニュー表の「スペシャルオプション」が視界の端でちらちらしていた
- 手の震え、お腹の奥の熱、太ももの締め——全て「気にしてなんかいない」で封じている
- 神代に「似合ってる」と言われて耳が赤くなった
- 完璧主義の「鎧」の下に好奇心と欲望がある

---

## 前編テーマ

**「完璧なカフェ接客の向こう側で、声が聞こえる——制度を設計した私だけが、制度の外に立っている」**

午前中、莉子は完璧だった。紅茶を淹れ、クールな接客で「指名」を集め、サブリーダーとして全体を見渡していた。でも——フェラブースから漏れる声、個室へ消えていくメイドたち、あかりの甘い喘ぎ。莉子が自ら設計した「制度」が、莉子を置いて動き始めている。カフェのカウンターで紅茶のカップを持つ手が、震えている。

## 前編 シーン構成

### ■ Scene 1：開場——私が設計した場所（9:00〜9:30）

**場所：カフェスペース**

- 開場直前。美月が最終チェックを仕切る横で、莉子はメニュー表の最終校正をしている。
- 誤字がないか。価格の整合性。オプションの導線。全部、莉子が設計した。
- 「……問題ない。完璧よ」
- 腕を組む。カウンターの前に立つ。167cmの長身にメイド服。ショートカットボブにカチューシャ。凛々しい。
- 9人のメイドが整列。「お帰りなさいませ、ご主人様♡」
- 莉子の声は低い。感情を載せない。「お帰りなさいませ、ご主人様♡」——♡だけが不自然に浮いている。
- *……この「♡」を付けろと言ったのは私。メニュー表のガイドラインに書いた。「親しみを出すために語尾に♡を付けること」と。*
- *自分で書いたルールに従っているだけ。別に、恥ずかしいわけじゃない。*
- 神代がカウンターに入る。フロアマネージャーの腕章。ホワイトボードを準備している。
- 莉子がカウンターの内側からメニュー表の最終版を神代に渡す。
- 「これ、最終版。変更点は赤字でマークしてある」
- 神代「ありがとう、椎名。……いつも助かる」
- 「……当然でしょ。私がやらなきゃ誰がやるの」
- 耳が、少し赤い。でも髪が短いから隠せない。
- *……「助かる」って。別に、褒められたくてやってるわけじゃない。*
- 美月が「莉子ちゃん、午前中はカフェ担当でいいよね？♡」と確認する。
- 「ええ。午前中はカフェの品質管理に専念する。オプションスペースは……午後の状況を見て判断するわ」
- 冷静。論理的。完璧な返答。
- *「午後の状況を見て判断」——逃げ道を残した。*
- *別に逃げてなんかいない。合理的な判断をしているだけ。*

**描くこと：**
- 莉子＝制度設計者としての誇り。メニュー表もルールも全部自分が作った
- 神代への小さな反応（耳が赤い）＝第1話からの引き継ぎ
- 「午後の状況を見て判断」＝本人は合理的だと思っているが、実際は迷っている
- 腕を組む防御姿勢＝感情を鎧で覆う莉子の象徴

### ■ Scene 2：カフェ接客——クールメイドの需要（9:30〜10:50）

**場所：カフェスペース**

- 最初の客。2年生の文学少年。
- 莉子がテーブルに案内する。動きに無駄がない。
- 「いらっしゃいませ、ご主人様。本日の紅茶はアールグレイとダージリンの二種をご用意しています」
- 「♡」を落とした。意図的に。
- *全部の語尾に♡を付ける必要はない。品のある接客には、適度な距離感が必要。これも私が設計した接客マニュアルに書いた。*
- 紅茶を淹れる。手つきが美しい。ティーポットを高い位置から注ぐ。莉子の家は茶道を嗜む家系。
- 文学少年が見惚れる。「……綺麗な所作ですね」
- 「……当然でしょ。紅茶を雑に扱うメイドなんていないわ」
- ——でもほんの少し、口角が上がった。
- *褒められた。紅茶の淹れ方を。……別に嬉しくない。当然のことだから。*
- 2人目の客。3年生の物静かな男子。
- 莉子のテーブルが「当たり」だと廊下で噂になっている。
- 「クールなメイドさんがいる」「毒舌だけど紅茶が美味い」「話してると頭良いのが分かる」
- *……需要があるの？ この接客スタイルで？*
- *他のメイドはみんな甘い声で「ご主人様♡」を連発してるのに。私は——*
- 莉子のカフェ接客スタイル：
  - 紅茶を淹れる所作が美しい
  - 会話は知的。客の趣味（読書、将棋、科学）に合わせて深い話ができる
  - 毒舌混じりだが不快感がない。「……ふん。まあまあの選択ね」
  - 「あーん♡」はしない。代わりにティーカップを両手で差し出す。指先が長く白い
  - 時折、口元だけで微笑む。笑わない莉子がふっと笑うと——客が固まる
- 3人目の客。1年生。莉子の前で緊張してカップを倒す。
- 紅茶がこぼれる。莉子が瞬時にナプキンで受け止める。完璧な反応。
- 「……落ち着いて。紅茶はまだあるわ。もう一杯、淹れてあげる」
- 1年生が「す、すみません……」
- 「謝らなくていい。……ご主人様が紅茶を楽しんでくれれば、それでいいの」
- ——「ご主人様♡」ではなく「ご主人様」。♡なし。でも声が少しだけ柔らかくなった。
- 1年生の顔が赤くなる。「……メイド喫茶って、こういうのもあるんですね」
- *「こういうの」……クールな接客のこと？*
- *私が設計した「スペシャルオプション」じゃなくて、こういう紅茶の時間を楽しんでくれる人がいる。*
- *……それはそれで、悪くない。*
- でも——10:00を過ぎた頃から、カフェの向こうから声が聞こえ始めた。
- フェラブースの方角。るるの甲高い「んっ んっ♡」。律の短い息。つむぎの「えへへ……大きいですね♡」。
- 莉子の紅茶を持つ手が、ぴくりと止まった。
- *……聞こえてる。当然よ。あの程度の防音で完全に遮音できるわけがない。設計段階で分かっていたこと。*
- *パーテーションの素材を段ボールから合板に変更する案もあったけれど、予算の都合で——*
- 思考を「制度設計」に逃がしている。声が聞こえるたびに、莉子は「設計上の問題」として処理しようとする。
- 10:50。あかりの声が個室から聞こえてきた。
- 「あっ あっ ご主人様♡」
- 甘くて、切実で、幸せそうな声。カフェスペースまで届いている。
- 莉子のカップが——揺れた。
- 紅茶の水面がかすかに波打つ。手が、震えている。
- *……あかりさんの声。個室で。本番を。*
- *あの声は——「演技」じゃない。本物の。*
- *……別に。気にしてなんかいない。*
- 太ももをきゅっと閉じた。メイド服のスカートの下が、うっすらと熱い。
- *これは——紅茶の温もりが移っただけ。そう。紅茶が熱いから。*
- 合理化。莉子は自分の身体の反応を、全て「別の理由」に置き換えようとする。

**合間に描く他メイドの状況（莉子の視界の端で）：**
- るるが個室から出てきた。「やっば♡ まじ楽しい♡」と飛び跳ねている。
  - *……品位の欠片もないわね。でも——楽しそう。*
- 陽菜がフェラブースに膝をついている。静かに。丁寧に。
  - *篠宮さんは朝からずっとフェラブースにいる。私はカフェにいる。*
  - *同じ「全オプション対応」のはずなのに。*
- ひなたが壁の隅から一歩踏み出して、フェラブースへ向かっていった。小さな背中。
  - *……あの子でさえ、行ったのね。*

**描くこと：**
- 莉子のカフェ接客の「質の高さ」と独自の需要＝知的でクールなメイド
- 他メイドの声・行動が莉子の身体を揺さぶっている
- 感情を「制度設計の語彙」で合理化する莉子の癖（防音の問題、予算の問題にすり替える）
- あかりの声で手が震える＝鎧の下の好奇心・欲望が反応している
- 他メイドが次々とオプションに向かう中、カフェに留まっている莉子の焦り

### ■ Scene 3：孤独なカウンター——みんなが行ってしまう（10:50〜12:00）

**場所：カフェスペース**

- 11時を過ぎると、カフェに残っているメイドが減っていく。
- るるは個室。陽菜はフェラブース。ひなたもフェラブースへ。あかりは個室。律は個室。つむぎも個室へ消えた。
- 美月はカフェに残っていたが——11:15、ついにフェラブースへ移動し、中村の手を引いて個室に消えた。
- カフェスペースに残っているのは——莉子だけ。
- *……私だけ。*
- カウンターで紅茶を淹れる。客はまだいる。莉子のカフェ接客を求めて来る客が途切れない。
- でも莉子の意識は、紅茶のカップの向こう側——フェラブースと個室の方向に引っ張られている。
- 声が聞こえるたびに、手が止まる。
- るるの声。あかりの声。美月の……おっとりした、でもいつもより甘い声。
- *……美月まで。*
- *美月は「リーダー」じゃなかったの。私と同じで、管理する側じゃなかったの。*
- *なのに——あの声。あの声は、「管理者」の声じゃない。*
- 美月が個室に入ったことで、莉子の中の「設計者の立場」が揺らぐ。
- *美月と私は共同で企画した。二人でメニューを作り、制度を設計し、リハーサルを仕切った。*
- *美月は「体験する側」に行った。私は「設計する側」に留まっている。*
- *それは——正しい選択のはず。誰かが全体を見ていなければ。誰かがカフェの品質を担保しなければ。*
- *……でも。*
- フェラブースの方から、陽菜の「……ん♡ ……全部、いただきました♡ ご主人様♡」という静かな声が聞こえた。
- 莉子の指先が、紅茶のカップの縁をなぞる。無意識に。唇に触れるように。
- *……篠宮さんの声が、変わってきてる。朝より——自信がある。*
- *私はずっと同じ場所にいて、同じ紅茶を淹れてる。*
- *変わったのは——カフェの空気だけ。みんながいなくなって、声だけが聞こえる。*
- 11:30。神代が集計のためにカウンターに来た。
- 「椎名、午前の中間集計だが——フェラブースの稼働率が予想より高い。個室も3部屋フル回転だ」
- 「……そう。想定の120%ね。オプション需要が見積もりより高かった」
- 冷静な分析。データで語る。莉子の得意分野。
- 「カフェの方は？」
- 「リピート率が高い。私のテーブルだけで6人目。……紅茶の品質が評価されているようね」
- 「椎名のおかげだな」
- 「……ふん。当然でしょ」
- 耳が赤い。
- 神代がホワイトボードに「カフェ：椎名6人」と書き込む。隣に「フェラ：篠宮5人」「個室：桃園3回」と並んでいる。
- ——莉子の目が、ホワイトボードの数字に吸い寄せられた。
- *「カフェ：椎名6人」。悪くない数字。カフェ接客としてはトップ。*
- *でも隣の数字は——「フェラ：篠宮5人」。「個室：桃園3回」。*
- *同じ1日の中で。同じメイドとして。*
- *私はカップを6つ並べた。篠宮さんは5人の男のモノを咥えた。桃園さんは3人の男に中出しされた。*
- *……数字で比較するものじゃない。それは分かってる。*
- *でも——数字で比較してしまう自分がいる。*
- 莉子の完璧主義が、データとして突きつけられる。
- *私が設計した制度。私が書いたメニュー表。「スペシャルオプション」の価格設定も、時間配分も、全部私が考えた。*
- *——なのに、その制度を体験しているのは、みんなで。私だけが外にいる。*
- *料理を作ったシェフが、一口も食べていない。*
- 腕を組む。きつく。爪が二の腕に食い込む。

**描くこと：**
- カフェに「一人残される」莉子の孤独感。設計者が制度の外に取り残されている
- ホワイトボードの数字＝莉子の完璧主義を刺激する具体的なデータ
- 美月が個室に行ったことの衝撃＝「共同設計者」すら体験する側に行った
- 神代との事務的な会話の裏にある感情の揺れ
- 「料理を作ったシェフが一口も食べていない」＝莉子の苛立ちの核心

### ■ Scene 4：昼休憩——みんなの声を聞く（12:00〜12:30）

### ■ Scene 5：午後への扉——でも唇が乾いている（12:30〜13:00）

- *午後もカフェ。計画通り。*
- 午後の客が入ってくる。莉子のテーブルに案内する。
- 「いらっしゃいませ、ご主人様。午後の紅茶はアッサムも加わりましたわ」
- 完璧な接客。クールな微笑。
- でも——手が、震えている。ティーポットを持つ指先が、かすかに。
- 客が「あれ、手、震えてない？」
- 「……気のせいよ。ポットが熱いだけ」
- *嘘。ポットは適温。私の手が震えてるのは——*
- フェラブースの方角から、声が聞こえる。
- あかりの「ご主人様♡ いらっしゃいませ♡」。陽菜の静かな「……ん♡」。

- *カフェの品質管理が私の仕事。オプションスペースは——*

- 朝、自分で言った言葉が反響する。
- *「午後の状況を見て判断」。*
- *午後になった。状況は見えている。*
- *みんなが行って、みんなが変わった。カフェに残ったのは私だけ。*
- *……「判断」するのは——いつ？*

## 莉子編の独自性

莉子編 第2話が他ヒロインのエピソードと決定的に異なるのは、**「午前中ずっと何もしなかった」こと**。

あかりは開場30分でフェラブースへ。陽菜は開場直後からフェラブースに志願。美月はカフェから自然にフェラへ。るるは10分でフェラ→個室直行。ひなたですらフェラブースに向かった。

莉子だけが——午前中ずっと、カフェのカウンターで紅茶を淹れていた。

**制度を設計した人間が、制度の外にいた。**

メニュー表を書いた指先が、紅茶のカップを握ったまま動けなかった。「全オプション対応」と自分で書いたのに、カフェから一歩も出なかった。

午後14:00。全メイド中最も遅いフェラブースデビュー。そしてたった2回。

でもその「たった2回」が、莉子の鎧にヒビを入れた。抑圧していた分だけ、反応は誰よりも激しかった。2回のフェラで太ももが止まらないほど震えた。「別に、普通だったわ」と平静を装ったが、帰り道で唇に触れている。

**完璧主義者が、初めて「不完璧」を受け入れた日。**

それが、椎名莉子の初日。

## 第3話への布石

第2話の莉子は「午前カフェ→午後フェラ2回」で初日を終えた。完璧主義の鎧にヒビが入り、「不完璧が嫌じゃなかった」という新しい感覚を得た。

**2日目（10月5日）の莉子に何が起きるか：**
- 午前からフェラブースに入るか？　「午前はカフェ」のルールを自分で破れるか？
- フェラの技術は上がるか？　1回目→2回目でPDCA改善した莉子は、3回目以降さらに分析的になる
- **個室デビューはあるか？**——フェラで「声が漏れた」莉子が、挿入されたらどうなるのか
- 神代との関係は？　「フェラ：椎名1回」の文字が消えないように、莉子の中の神代への意識も消えない
- 「完璧なフェラ」を目指す莉子と、「やさしいフェラ」の陽菜、「たくらみフェラ」の美月——三者三様のスタイルが2日目に交差する

**莉子の2日目のテーマ：**
- 「不完璧」の先にあるもの。完璧主義者が完璧を捨てたとき、何が残るのか
- 制度設計者が制度の中で「気持ちいい」と言えるようになるのか
- 鎧を脱いだ莉子は——どんな顔で笑うのか

## 描写方針

| 項目 | 方針 |
|------|------|
| 内心描写 | 「制度」「数字」「合理性」「データ」「設計」の語彙で構成。感情を認めようとしない。身体の反応を別の理由に置き換える |
| 防御姿勢 | 腕を組む場面を頻出させる。思考が揺らぐたびに腕を組み直す |
| 口癖 | 「……別に」「当然でしょ」「気にしてなんかいない」の繰り返しが、逆に気にしていることを示す |
