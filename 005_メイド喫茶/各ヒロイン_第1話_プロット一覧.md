## 1. 陽菜編 第1話「あたしがメイドになる理由」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 篠宮陽菜 一人称「あたし」（心の中）/「私」（人前） |
| テーマ | 恋人がいるのに、他の男に奉仕するメイドになる理由 |
| 核心 | 「優くんに見守られているから、怖くない」——でも「優くん以外の人にされる」ことへの未知の感情 |

### シーン構成

#### Scene 1：企画会議（9/8 放課後）
- 美月の「メイド喫茶やらない？」を聞いた瞬間、陽菜の心臓が跳ねる。
- メニュー表を見る。*生フェラ。生挿入。中出し。*
- 隣の席に優くんがいる。優くんの顔をちらりと見る。穏やかな顔。何を考えてるか分からない。
- *優くんは……あたしがこれをやるの、どう思うんだろう。*
- メイド志願者の手が上がっていく。あかりちゃん、るるちゃん、律ちゃん……。
- 陽菜は手を上げない。上げられない。*あたしがメイドになったら、優くん以外の男の人に……*
- でも黒板を見つめている。メニュー表を。頬が赤い。心臓がドキドキしている。
- *気になってる。認めたくないけど、気になってる。*

#### Scene 2：放課後——優くんへの相談
- HRが終わった後。教室に残る。優くんの机の横に立つ。
- 袖をちょんと引く。「……優くん」
- 「あのね……メイドのこと。あたし……やりたいかもしれない」
- 「でも怖い。知らない男の人に、あたしの身体を……」
- 「優くんのじゃない人に、触られたり……されたりするの、優くんは嫌じゃないの？」
- 優くんが静かに聞いている。穏やかな顔。
- 「……嫌だよ」——陽菜の心臓が止まりかける。
- 「嫌だけど、陽菜がやりたいなら、俺がちゃんと見てるから」
- *ちゃんと見てる。*
- その言葉で覚悟が決まる。桜が咲くような笑顔。
- あかりの元へ走って「メイド、まだ募集してる？」

#### Scene 3：LINEグループの夜
- スマホを抱えてベッドに座る。グループのメッセージを追う。
- るるの「オプションの練習とかするの？笑」を見て、お腹がきゅっとなる。
- 美月の「神代くんでしょ♡」を見て、指が止まる。
- *優くんが……リハーサルの相手……？*
- *優くんのを、あたしが咥えるの……？ それとも、他のみんなも……？*
- 嫉妬。不安。でも、ちょっとだけ——期待。
- *……あたしが一番上手くなりたい。優くんのこと、あたしが一番知ってるんだから。*

#### Scene 4：準備期間——メイド服とリハーサル
- メイド服が届く。着てみる。鏡を見る。
- *……メイドさんだ。あたしが。*
- 清楚感が際立つ。ロングの黒髪がメイド服によく似合う。
- 「……かわいい」自分で言って赤面する。
- リハーサルの日。優くんが相手。
- 他のメイドが順番にフェラの練習をする。陽菜は見ている。
- あかりが優くんの前に膝をつく。つむぎが。るるが。
- *他の子が、優くんのを咥えてる。*
- 嫉妬が胸を焼く。でも——同時に、不思議な熱を感じる。
- 陽菜の番。優くんの前に膝をつく。目が合う。
- 「……優くん。あたし、頑張るから」
- 初めて優くん以外の人にも奉仕することを前提にした練習。でも今は優くんだけ。

#### Scene 5：開場直前（10/4 朝）
- メイド服に着替える。最後のボタンを留める。
- 鏡を見る。覚悟を決めた顔。
- 神代が「似合ってる。全員」と言う。陽菜の耳が真っ赤になる。
- *今日から、あたしは他の人にもご奉仕する。優くんの前で。*
- *怖い。でも——優くんが見ていてくれるなら。*
- 「……行ってきます。優くん」
- 心の中で、そう呟いた。

### 感情曲線
```
動揺 → 相談 → 決意 → 嫉妬 → 覚悟
```

## 2. るる編 第1話「メイド喫茶？ まじやばーい♡」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 桃園るる 一人称「あたし」 |
| テーマ | 考えるより先に行動するギャル。でも実は「楽しい」の裏に一瞬だけ不安がある |
| 核心 | 「やばーい！」の一言で全部乗り越える。ギャルの強さと可愛さ |

### シーン構成

#### Scene 1：企画会議（9/8）
- 美月の提案を聞いた瞬間「まじ？ やばくない？ やばーい！」
- メニュー表を見て1秒で理解。「生フェラとか中出しとか！ まじ！？ テンション上がるんだけど！」
- 志願の手を真っ先に上げる。迷い？ ない。
- 「全オプション対応で あたし何でもやるー」
- 男子が「マジかよ桃園……」と引いてるけど気にしない。
- *だって楽しそうじゃん？ メイド服かわいいし、いろんな男子と遊べるし。*
- ……でもこの時点で、「いろんな男子と遊ぶ」の具体的な内容は想像が追いついていない。

#### Scene 2：LINEグループの夜
- 「オプションの練習とかするの？笑」を送ったのはるる。
- ノリで送った。でも送った後、ちょっとだけ考える。
- *フェラの練習……あたし、したことあるけど、人前では初めてだな。*
- *つーか神代がリハの相手？ あいつ、そういう顔してないのに。ウケる。*
- *でも……いっぱい練習しちゃお まじテンション上がるー*
- るるは基本的にポジティブ。ネガティブな思考が1秒以上続かない。

#### Scene 3：準備期間——メイド服改造
- メイド服が届く。着る。「かわいー」
- スカートが長い。「これじゃ全然見えないじゃん」
- ハサミとミシンで裾を詰める。太ももが全開。
- 「こっちの方が絶対いい あたしの脚、見ないと損だし」
- 谷間も見せ気味にフリルを調整。
- 美月に見せる。「るるちゃん、すごいアレンジだね♡」「でしょ」

#### Scene 4：リハーサル
- リハーサルで一番に手を上げる。「あたしが先やるー」
- 神代の前に膝をつく。「じゃ、いくね」
- 慣れた手つきでベルトを外す。*お、こいつ意外とデカいじゃん。*
- フェラを始める。上手い。経験がある。甲高い声。「んっ んっ♡」
- 他のメイドが唖然としている。あかりが目を丸くしている。
- 「え、るるちゃんすごい……」
- 「えへへ まあ経験値はあるからね」
- ——ただ、リハーサルが終わった後。
- 一人で更衣室にいるとき。ふっと静かになる瞬間がある。
- *本番……知らない男のを咥えるのか。*
- *まあ、あたしは大丈夫。いつも通りやればいいし。*
- 不安を感じたことを認めない。感じてない、ということにする。それがるるの強さ。

#### Scene 5：開場直前
- メイド服（改造済み）で整列。テンション最高潮。
- 「まじやばーい 開店だよ開店」
- 隣の千歳の肩を叩く。「千歳も気合い入れてこー」千歳が「ひっ」と肩を縮める。
- 神代が「似合ってる。全員」。るる「きゃー 神代くん普通にイケメンー♡」
- *さあ 今日は最高に楽しい日にするよ*

### 感情曲線
```
即決 → ノリノリ → (一瞬の静寂) → テンションMAX → 開場
```

## 3. 莉子編 第1話「完璧な企画書」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 椎名莉子 一人称「私」 |
| テーマ | 企画の裏側。制度設計の頭脳。完璧主義の鎧の下にある好奇心 |
| 核心 | 「理性で制度を作り、感情を排除する」——はずが、メイド服を着た瞬間に揺れる |

### シーン構成

#### Scene 1：美月との密談（9/8 HR前）
- 企画会議の前日〜当日朝。美月と二人で計画を詰めていた。
- 美月「メイド喫茶、やっぱりやりたい」莉子「……本気なのね」
- 莉子が制度設計を一人で作り上げる。同意書、衛生管理、予算、申請書類。
- *感情ではなく、システムで安全を担保する。それが私の仕事。*
- 美月「莉子ちゃんって、やっぱりすごいね♡」
- 莉子「……当然でしょ」（耳が赤い）
- 莉子が美月を発案者にした理由：美月の「おっとり」が場の空気を柔らかくする。莉子が最初に言ったら「強制」に見える。
- *あの子の笑顔があれば、みんな「やってみようかな」と思える。私の顔では無理。*

#### Scene 2：企画会議での発言
- 美月が提案。莉子が制度説明。完璧な連携。
- 藤堂の「風紀的に——」に即座に切り返す。気持ちいい。
- 神代を指名する。「あなた、フロアマネージャーやって」
- *真面目で、柔軟で、メイドたちが安心できる人。……条件を満たすのは神代くんだけ。*
- 神代「……わかった」——この短い返答に、莉子は信頼を感じる。
- *……ふん。殊勝な答えね。*（少しだけ口角が上がった）

#### Scene 3：LINEグループ——数字と感情
- 衣装費の計算をLINEに送る。「一着3,500円×9着＝31,500円 予算内」
- 数字を扱っているときは落ち着く。感情を排除できるから。
- るるの「オプションの練習するの？笑」を見て、指が止まる。
- *……練習。私も、するの？*
- 美月の「神代くんでしょ♡」に「同意」とだけ返す。
- でもスマホを置いた後、莉子の手が震えていた。
- *……別に。ただの制度確認よ。リハーサルは品質管理のために必要。それだけ。*

#### Scene 4：準備期間——メイド服と鎧
- メイド服を着る。鏡の前。ショートカットにカチューシャ。背筋を伸ばす。
- *……悪くない。*
- 脚が長いから、ミニスカートが映える。自覚はある。
- でも——このメイド服を着て、知らない男の前で膝をつくことを想像すると——
- 腕を組む。いつもの防御姿勢。
- *感情を排除しなさい。これはシステムの一部。私がメイドとして最高品質のサービスを提供すれば、企画全体の評価が上がる。それだけ。*
- リハーサルには参加しない（カフェ担当なのでオプションの練習は不要、と自分に言い聞かせる）。
- でも他のメイドがリハーサルしているのを、企画運営の名目で「監督」として見ている。
- あかりが神代の前で膝をつくのを見て、莉子の目が一瞬揺れる。

#### Scene 5：開場直前
- 完璧な準備。チェックリストを全項目クリア。
- 腕を組んで教室を見渡す。*完璧。*
- 神代が「似合ってる。全員」。莉子「……ふん。別に嬉しくないけど」（耳が赤い）
- *今日は「カフェのみ」で通す。オプションには出ない。私はサブリーダーだから、全体を見る立場。*
- ——そう決めているはずなのに。
- メニュー表の「スペシャルオプション」の文字が、視界の端でちらちらしている。
- *……別に。気にしてなんかいないわ。*

### 感情曲線
```
計画 → 完璧 → (動揺) → 鎧 → (ちらちら)
```

## 4. ひなた編 第1話「小さなメイドさんの大きな決心」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 小鳥遊ひなた 一人称「あたし」 |
| テーマ | クラスで一番おとなしい子が、なぜメイドに志願したのか |
| 核心 | 「変わりたい」——お姉ちゃんに背中を押されて、一歩踏み出す勇気 |

### シーン構成

#### Scene 1：企画会議
- メニュー表を見て固まる。心臓がバクバクする。
- *生フェラ……中出し……こんなの、あたしにできるわけ……*
- 周りの反応を見ている。男子が歓声。女子が動揺。あかりちゃんが教壇でポニーテールの先をいじっている。
- *あかりちゃんも悩んでるんだ。……あかりちゃんでも悩むんだ。*
- 「……あの、メイドさんは、誰がやるんですか……？」
- 自分でも驚く。声が出た。核心の質問が、あたしの口から。
- *なんで聞いたの、あたし。……気になったから？ あたしも……やりたいの？*

#### Scene 2：お姉ちゃんとの電話（9/8 夜）
- 家に帰って、大学生のお姉ちゃんに電話する。
- 「お姉ちゃん……文化祭でね、メイド喫茶やることになって……」
- お姉ちゃんは明るくてオープン。「えー！ いいじゃん！ ひなた、やりなよ！」
- 「でも……あたし、おとなしいし、声小さいし……メイドなんて似合わないよ……」
- 「似合う似合わないじゃなくて、やりたいかやりたくないかでしょ？」
- 沈黙。
- 「……やりたい、かもしれない」
- 「じゃあやりな ひなたは一歩踏み出したら絶対できる子だから」
- 「……うん。……やる。あたし、メイドやる」
- 電話を切った後、枕に顔を埋めて「うわあああ」と小さく叫んだ。

#### Scene 3：志願
- 翌日。あかりに「メイド、まだ募集してる……？」と聞く。声が震えている。
- あかり「してるしてる！ ひなたちゃん、やってくれるの！？」
- 「……はい。あたし……変わりたくて」
- あかりが嬉しそうに名前を書き足す。*あかりちゃん、あたしの名前書いてくれた。*

#### Scene 4：メイド服と不安
- メイド服が届く。着る。袖がぶかぶか。裾を握りしめる。
- 鏡を見る。*小さい……メイドさんっぽくない……*
- でもお姉ちゃんに写真を送ったら「かわいい！！！！」の連打。
- リハーサル。神代の前に立つ。膝が震えている。
- 「あの……あたし……初めてで……」
- 神代「大丈夫。ゆっくりでいい」
- *神代くん、優しい。……この人がフロアマネージャーでよかった。*
- 手コキの練習から始める。手が小さくて届かない。でも一生懸命。

#### Scene 5：開場直前
- メイド服の袖を折り返す。カチューシャをつける。
- 鏡の中の「メイドひなた」は、昨日までの「小鳥遊ひなた」とは少し違う顔をしている。
- *あたしにもできる。……できるよね、お姉ちゃん。*
- 神代が「似合ってる。全員」。ひなたの目がきらっと光る。
- 「……が、がんばりますっ♡」小さいけど、確かな声。

### 感情曲線
```
固まる → 質問 → お姉ちゃん → 志願 → 震える → 「がんばります」
```

## 5. 美月編 第1話「おっとりメイドのたくらみ♡」

- *さぁ 私のたくらみ、始まるよ*

## 6. 千歳編 第1話「眼鏡の奥のわたし」

|------|------|
| 視点 | 如月千歳 一人称「私」 |
| テーマ | 照れ屋の文学少女が、物語の中の世界に自分を重ねてメイドになる |
| 核心 | 「本の中の女の子は、いつも勇敢だった」——千歳は自分も物語の登場人物になりたい |

### シーン構成

#### Scene 1：企画会議
- メニュー表を見て、頭が真っ白になる。
- でも——文学少女の脳が、別の回路で動く。
- *これは……物語だ。メイドと客の、一夜限りの物語。*
- 本の中で読んだシーンが蘇る。官能小説。図書室の奥で読んでいた、誰にも言えない秘密。
- *私……あの本の女の子みたいに、なれるのかな。*
- 志願しようかどうか、ずっと迷っている。手が上がらない。
- でも——莉子の「手コキまで」というオプション設定を聞いて。
- *手コキまで、なら……私にも……*
- おずおずと手を上げる。声は出ない。でも手は上がった。

#### Scene 2：夜——本棚の前で
- 自室の本棚。官能小説が何冊か並んでいる。表紙を裏返して隠してある。
- 一冊取り出して、パラパラとめくる。
- *この子は……メイドの格好をして、ご主人様にご奉仕する。*
- *私も明日から、こうなるの……？*
- 眼鏡を外す。世界がぼやける。本の文字が読めなくなる。
- 眼鏡をかけ直す。世界が鮮明になる。
- *眼鏡を外すと、世界がぼやけて……怖くなくなる気がする。*
- この感覚を、千歳は覚えておく。

#### Scene 3：メイド服——ギャップ
- メイド服を着る。眼鏡をかけたまま。
- 鏡を見て息を呑む。*……誰、この人。*
- おとなしそうな文学少女の顔と、ミニスカートのメイド服。ギャップがすごい。
- *……でも、嫌じゃない。*
- 眼鏡を外してみる。鏡の中の自分がぼやける。ぼやけた自分は——ちょっとだけ、本の中の女の子に似ている気がした。

#### Scene 4：リハーサル——手コキ
- リハーサル。千歳の担当は「手コキまで」。
- 神代の前に座る。手が震えている。
- 「あの……私、初めてで……ごめんなさい」
- 指が長い。ピアノを弾いていた指。
- 触れる。形を確かめるように。恐る恐る動かす。
- *……本で読んだ通り……こう……*
- 不器用だけど、一生懸命。神代が「大丈夫」と言ってくれる。
- *……神代くんがいるなら、怖くないかもしれない。*
- でも——手を動かしているうちに、千歳の太ももがぴくぴくと震え始めた。
- *……なにこれ。私の身体が……勝手に……*

#### Scene 5：開場直前
- メイド服。眼鏡。本を一冊、メイド服のポケットに忍ばせている。お守り代わり。
- 「似合ってる。全員」——千歳の眼鏡の奥が潤む。
- *物語が、始まる。*
- 「……が、がんばり……ます……」蚊の鳴くような声。でも、確かに。

### 感情曲線
```
真っ白 → 本の記憶 → ギャップ → 震える手 → 「物語が始まる」
```

## 7. つむぎ編 第1話「えへへ、メイドさんです♡」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 宮前つむぎ 一人称「つむぎ」/「わたし」 |
| テーマ | 天然。深く考えない。でもそれが逆にすごい |
| 核心 | 「みんなが楽しそうだから、つむぎも楽しい♡」——周りが悩んでいることを、つむぎは自然体で超えていく |

### シーン構成

#### Scene 1：企画会議
- メニュー表を見る。
- *生フェラ……えっと……お口でするやつ？ 中出し……中に出すってこと？*
- 理解はしている。でも感情的な動揺が薄い。
- *みんなすごいドキドキしてるみたい。あかりちゃん、顔赤い。莉子ちゃんは腕組んでる。千歳ちゃんは固まってる。*
- *つむぎは……どうだろう？ ドキドキ……するかな？ うーん……ちょっとだけ？*
- 志願。「はーい つむぎもやりたいです」ぽわんとした声。
- 教室が「えっ」となる。つむぎのイメージと「全オプション対応」のギャップ。
- 「えへへ、楽しそうだなって思って」——本気。深い理由はない。

#### Scene 2：LINEグループ
- 「練習……するんですか？ ドキドキしちゃいます……」
- 素直。飾らない。でもドキドキはちゃんとしている。
- *神代くんが相手なんだ。神代くん、優しそうだから大丈夫かな*

#### Scene 3：メイド服——ぱつぱつ
- メイド服を着る。
- 胸元が明らかにきつい。Gカップがフリルの向こうでぱつぱつに張っている。
- 「……あれ？ つむぎのメイド服、ちょっと小さいかも？」
- 美月「それ、全員同じサイズだよ……つむぎちゃんが大きいの♡」
- 「えへへ……そうなんだ♡」本人は気にしていない。
- ボタンが一つ閉まらない。谷間が丸見え。
- 莉子「……開き直ってるわけじゃないのよね、あれ」律「天然だろ」

#### Scene 4：リハーサル——えへへ
- 神代の前に膝をつく。
- 「えへへ つむぎ、初めてだけど頑張ります」
- ゆっくり、丁寧に、口に含む。不慣れだけど素直。
- 「……んっ ……大きい……つむぎのお口、入るかな……♡」
- 天然のコメントが周りを脱力させる。
- リハーサル後。「えへへ……おちんちんって、あったかいんですね♡」
- 全員「「「……」」」

#### Scene 5：開場直前
- メイド服（ぱつぱつ）で整列。
- 「似合ってる。全員」——つむぎがにっこり。「えへへ ありがとうございます」
- *今日はいっぱいご奉仕するんだ 楽しみ*
- 深い葛藤はない。つむぎは自然体で、全部を受け入れる。それがつむぎの強さ。

### 感情曲線
```
ぽわん → 志願 → ぱつぱつ → えへへ → 「楽しみ」
```

## 8. 律編 第1話「メイドなんてガラじゃないけど」

### 基本情報

| 項目 | 内容 |
|------|------|
| 視点 | 相沢律 一人称「俺」→「私」（矯正中） |
| テーマ | ボーイッシュな子がメイド服を着る。似合わないはずなのに、意外と似合う。その戸惑い |
| 核心 | 「メイド口調なんて無理だろ」——でもやると決めたら負けず嫌いが発動する |

### シーン構成

#### Scene 1：企画会議
- メニュー表を見て、腕を組む。
- *ふーん。エロメイド喫茶か。思い切ったな、橘と椎名。*
- 志願する。「おれ——私もやる。全オプションで」
- 理由は単純。「面白そうだから」。あと、負けず嫌い。るるが即決で手を上げたのを見て、*負けたくない*と思った。
- *メイドなんてガラじゃねーけど。やるからには全力だろ。*

#### Scene 2：LINEグループ——予行演習
- 「練習つーか予行演習は必要だろ ぶっつけ本番は事故る」
- これは律の本音。*エロいことするならちゃんと練習しろ。仕事なんだから。*
- 合理的。感情より効率。でも——
- 「神代くんでしょ♡」の流れを見て。
- *神代か。……まあ、あいつなら安心だな。変なことしないし。*
- *……って、「安心」って何だよ。別にドキドキしてねーし。*
- ドキドキしている。認めない。

#### Scene 3：メイド服——戸惑い
- メイド服を着る。鏡の前。
- 「…………」
- ボーイッシュショートにカチューシャ。腕まくりしたメイド服。
- *似合わないだろ、こんなの。*
- ——でも、鏡の中の自分は、思ったより悪くない。
- 凛々しさと可愛さが同居している。ボーイッシュメイドの破壊力。
- *……ちょっとだけ、かわいいかもしれない。*
- その思考を即座に打ち消す。*いや、かわいいとか言ってんじゃねーよ。*

#### Scene 4：リハーサル——男前フェラ
- 神代の前に膝をつく。
- 「いくぞ」——メイド口調じゃない。
- ベルトを外す。迷わない。さっとやる。
- 口に含む。男前。効率的。でも丁寧。
- 「……ん。こんなもんか」あっさり。でも頬が赤い。
- リハーサル後。るるに「律ってサバサバしすぎじゃない？」と言われる。
- 「うるせーな。仕事だろ」
- *……仕事。そう。仕事だ。ドキドキなんてしてない。*
- してる。

#### Scene 5：開場直前——メイド口調の練習
- 開場前のメイド口調練習。
- 「お帰りなさいませ、ご主人様♡……です……」
- 棒読み。
- あかりに「律ちゃんもっと可愛く♡」と言われる。
- 「お帰りなさいませっ、ご主人様♡っ」——力み過ぎ。
- るるが爆笑。千歳が「か、かっこいいです……」と呟く。
- 「似合ってる。全員」——律の耳が赤くなる。
- *……ふん。似合ってるって言われたぐらいで嬉しくなってんじゃねーよ。*
- 嬉しい。

### 感情曲線
```
「面白そう」 → 即決 → 戸惑い → 男前 → 「です」 → (嬉しい)
```
