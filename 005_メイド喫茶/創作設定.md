## ♡（ハート）の使用ルール

> **♡は味付け。地の文・内心・プロット箇条書きには原則つけない。**

### つけていい場面

| 場面 | 例 | 頻度 |
|------|-----|------|
| 定番のご挨拶 | 「お帰りなさいませ、ご主人様♡」 | 1回の挨拶に1個まで |
| サービスの決め台詞 | 「あーん♡」「失礼しますね、ご主人様♡」 | その行為の開始時 |
| クライマックスの喘ぎ | 「中に出して……ご主人様♡」 | **1セリフ1個**。連打しない |
| キャラ固有のたくらみ（美月・るる） | 誘うとき、計算高く笑うとき | 控えめに |

### つけない場面

- 地の文・ナレーション・プロットの箇条書き（セリフ引用を除く）
- 内心モノローグ（`*……*`）——感情のピーク時のみ例外可
- 日常の雑談・説明・質問（「何飲む？」「5組！」など）
- **「ご主人様」の後に助詞が続く形**（× `ご主人様の` → ○ `ご主人様の`）
- 連続♡（♡♡♡）——禁止

### キャラ別の目安（1シーンあたり）

| キャラ | ♡の多さ |
|--------|---------|
| あかり・るる | やや多め（でもセリフ1個まで） |
| 美月 | 少なめ・計算されたタイミング |
| つむぎ | 稀 |
| 莉子・千歳・ひなた・陽菜 | ご主人様呼びと初声のみ |
| 律 | 棒読み練習・照れのときだけ |
