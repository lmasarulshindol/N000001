# 陽菜編 第2話 プロット「恋する唇のご奉仕」

---

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | 恋する唇のご奉仕 |
| 視点 | 篠宮陽菜 一人称「あたし」（内心）/「私」（人前・メイド時） |
| 時系列 | 10月4日（土）鳳翔祭初日 9:00〜16:30 |
| テーマ | フェラブースだけを選んだ陽菜が、恋人への愛情と同じ熱量で、すべてのお客様に「世界一やさしい口づけ」を捧げる |
| 行為範囲 | 終日フェラブースのみ（個室には入らない） |
| お相手 | 特定の一人ではなく複数のお客様。誰に対しても同じ愛情を注ぐ |

---

## 第1話からの引き継ぎ

- 陽菜は優くんの幼馴染かつ恋人。優くん以外の男性に奉仕することへの葛藤を抱えてメイドになることを決意した
- 優くんの「嫌だけど、陽菜がやりたいなら、俺がちゃんと見てるから」という言葉が陽菜の支え
- リハーサルで他の女の子が優くんにフェラする姿に嫉妬しながらも、「あたしが一番上手くなりたい」という強い意志を持つ
- 開場直前、優くんと目が合い、心の中で「行ってきます」と呟いて初日を迎えた
- 恋人への愛と、メイドとしての覚悟が「矛盾したまま重なっている」状態

---

## 前編テーマ

**「知らない人のモノを咥える恐怖を、恋人への愛で塗りつぶす——陽菜が見つけた、たったひとつの奉仕のかたち」**

開場直後、陽菜はフェラブースに立つ。個室には行かない。挿入はしない。「あたしにできるのはここまで」——その線を引いたのは陽菜自身。でも「ここまで」の中で、陽菜は誰にも真似できない奉仕を始める。優くんにするように、優しく、丁寧に、愛を込めて。知らない男の子のモノを、恋人のもののように大切に扱う。

---

## 前編 シーン構成

### ■ Scene 1：フェラブースの前——あたしの持ち場（9:00〜9:40）

**場所：カフェスペース→フェラブース前**

- 開場。9人のメイドが「お帰りなさいませ、ご主人様♡」と声を揃える。
- 陽菜の声は震えなかった。でも——握りしめた拳は白くなっていた。
- 配置確認。美月がフロアを仕切る。
  - 「陽菜ちゃんは、フェラブースの担当でいい？♡」
  - 「……はい。あたし、そこにします」
- 陽菜が自分でフェラブースを選んだ理由：
  - *個室は……まだ無理。知らない人と、優くん以外の人と、最後まで……するのは、今日はまだ——*
  - *でも、フェラなら。口だけなら。あたしの唇で、ご主人様を気持ちよくすることなら——*
  - *……できる。できるはず。リハーサルで、優くん相手にちゃんとできたんだから。*
- フェラブースのパーテーションの前に立つ。5つ並んだ半個室。隣のブースではるるがすでに「じゃ、いくね♡」と客を引き入れている。
- 陽菜のブースはまだ空。椅子とクッション。膝をつくための場所。
- 優くんがカウンターにいる。視線を感じる。ちらりと見た。目が合った。
- 優くんが小さく頷いた。——それだけで、指先の震えが止まった。
- *……見ていてね、優くん。*
- *あたし、ここで頑張るから。*

**描くこと：**
- 陽菜が自分の意志で「フェラブースのみ」を選んだ理由——個室への恐怖と、フェラなら自分にもできるという覚悟の線引き
- 優くんの存在が精神的支柱であること
- 他メイドが動き始める中、陽菜だけがまだ「待っている」静かな緊張

### ■ Scene 2：最初のお客様——恐怖と発見（9:40〜10:10）

**場所：フェラブース**

- 最初のお客様。2年生の男子。眼鏡をかけた、おとなしそうな子。
- 美月がカフェから案内してくれた。「陽菜ちゃんが、とっても丁寧にご奉仕してくれるよ♡」
- 男子が入ってくる。陽菜と目が合う。二人とも緊張している。
- 陽菜がパーテーションの中に招き入れる。椅子に座らせる。
- 膝をつく。メイド服のスカートが床に広がる。黒髪が肩から流れ落ちる。
- 「失礼しますね……ご主人様♡」
- 手を伸ばす。ベルトに触れる。——指先が震えている。
- *……優くんじゃない人の。知らない人の。あたしが今から——*
- ファスナーを下ろす。取り出す。
- ——止まった。
- *違う。優くんのと、違う。形も、大きさも、匂いも。*
- 当たり前のことなのに、その当たり前が陽菜の心を打った。
- これは優くんのものじゃない。知らない男の子のモノが、あたしの目の前にある。
- *怖い。やっぱり怖い。*
- *でも——*
- 見上げた。お客さんの顔を。
- 男の子が、陽菜よりも震えていた。目を泳がせて、手が膝の上で拳を作っている。耳が真っ赤。
- *……この人も、怖いんだ。*
- *あたしと同じくらい、ドキドキしてるんだ。*
- その瞬間、陽菜の中で何かが切り替わった。
- *——大丈夫。*
- *あたしが、安心させてあげなきゃ。*
- 両手でそっと包む。温かい。生きている温度。
- *優くんのときと同じように。優くんにするときと同じ気持ちで。丁寧に。やさしく。*
- 先端に唇を当てる。キス。そっと。
- 「……ん♡」
- ゆっくり咥えていく。舌を使う。裏側をなぞる。リハーサルで覚えた通り。
- でも——リハーサルのときより、もっと丁寧に。もっとゆっくりに。
- *この人を気持ちよくしてあげたい。怖い思いをさせたくない。*
- *優くんにキスするみたいに。優くんを大切にするみたいに。*
- *あたしの唇で、この人を——幸せにしたい。*
- 男の子が小さく声を漏らす。「っ……すごい……優しい……」
- その一言で、陽菜の目が潤んだ。
- *優しい、って。あたしのこと。*
- *——これで、いいんだ。*
- *あたしは、優くんを愛してるから。優くんに注ぐ愛情と同じもので、この人に奉仕してる。*
- *それは裏切りじゃない。あたしが愛を知ってるから、この人にも愛を注げるんだ。*
- 頭をゆっくり動かす。髪が揺れる。唇を窄めて、舌で包んで、吸って。
- 男の子が限界を告げる。陽菜は離さない。
- 口内射精。全部受け止める。ごくん、と静かに飲み込む。
- 「……ん♡ ……全部、いただきました♡ ご主人様♡」
- 口元を指先でそっと拭う。見上げる。黒髪の隙間から、大きな瞳で。
- 涙の跡が、一筋だけ頬に光っていた。
- 「……ありがとうございました♡ ……また来てくださいね♡」
- 男の子が真っ赤な顔で「ありがとうございました」と何度も頭を下げて出ていった。

**描くこと：**
- 「知らない人のモノ」と「優くんのモノ」が違うことへの衝撃——でもそこから「この人も人間で、怖がっている」という共感へ
- 陽菜の奉仕スタイルの誕生：恋人への愛情と同じ熱量で、すべてのお客様に注ぐ
- 「優しい」と言われて泣く＝自分の奉仕が正しかったと確信する瞬間
- 裏切りではなく、愛を知っているからこそできる奉仕

### ■ Scene 3：2人目、3人目——恋する唇の確立（10:10〜11:00）

**場所：フェラブース**

- 2人目のお客様。3年生。少し慣れた感じの男子。
- 陽菜は同じように膝をつく。同じように丁寧に。
- 取り出したとき、最初の客よりも大きい。一瞬怯む。
- *……大きい。でも——大丈夫。ゆっくり、ゆっくり。*
- 先端にキス。舌で舐める。唾液でたっぷり濡らしてから、ゆっくり咥えていく。
- 陽菜のフェラには特徴がある。
  - 急がない。お客様のペースではなく、陽菜のペースで。
  - 両手を添える。片手は根元を、もう片手はお客様の太ももに。触れているだけ。「ここにいるよ」という合図。
  - 時折見上げる。大きな瞳で。長い睫毛が濡れている。
  - 口を離すたびに、先端に小さなキスを落とす。
- *優くんにするときと同じ。ゆっくり、丁寧に、あたしの全部で。*
- 3年生が「お前……すごいな……こんなフェラ初めてだ」と呻く。
- 口内射精。飲み込む。口元を拭う。微笑む。
- 「……ご主人様♡ たくさん出してくれて、嬉しいです♡」
- 3人目。1年生。陽菜より年下。ほとんど声も出せないくらい緊張している。
- 陽菜が手を取る。「大丈夫ですよ♡ ……私がゆっくりしますから♡」
- この子には特にゆっくり。先端だけを唇で包んで、舌先でくるくると。
- 1年生が30秒で限界を迎える。
- 「ごっ ごめんなさい 早くて——」
- 陽菜が口元を拭いながら、ふんわり笑う。
- 「謝らないでください♡ ……気持ちよくなってくれたなら、私はそれだけで嬉しいです♡」
- 1年生の目が潤む。「……天使みたいな人だ」
- *てっ 天使……！？*
- 陽菜の耳が真っ赤になる。

**合間に聞こえる他ブースの音：**
- 隣のブースからるるの甲高い「んっ んっ♡」。もう3人目。速い。
- 律の「次」という短い声。サクサクこなしている。
- つむぎの「えへへ……ご主人様、大きいですね♡」というぽわんとした声。

**描くこと：**
- 陽菜のフェラスタイルの具体的な特徴——急がない、手を添える、見上げる、キスを落とす
- 他のメイドとの明確な差別化：るるの速さ、律の効率、つむぎの天然に対して、陽菜は「丁寧さ」
- 「天使」と呼ばれて照れる陽菜＝清楚な外見と献身的な奉仕のギャップがお客様に刺さっている
- 人数をこなすことで自信がついていく過程

### ■ Scene 4：優くんの視線——あたしを見ていてくれる人（11:00〜11:30）

**場所：フェラブース→カフェスペース（小休止）**

- 3人目を送り出した後、陽菜がブースを出て水を飲みに行く。
- カウンターに優くんがいる。ホワイトボードに各メイドの対応状況を書き込んでいる。
- 陽菜が近づく。「……優くん、お水ください」
- 優くんがペットボトルを渡してくれる。指先が触れる。
- *……あったかい。優くんの手。*
- 「……どうだ？ 大丈夫か？」
- 「うん……大丈夫。あたし、ちゃんとできてるよ」
- 水を飲む。ほっと息をつく。唇が少し赤くなっている。3人分のご奉仕の跡。
- 優くんがそれに気づいている。気づいているけど、何も言わない。
- *優くんは見てる。あたしの唇が赤くなってるのも、あたしが他の男の人のを咥えてたのも、全部。*
- *それでも「大丈夫か」って聞いてくれる。*
- *——ずるい。好き。大好き。*
- 「……ねえ、優くん」
- 「ん？」
- 小さな声。二人にしか聞こえない距離。
- 「あたしね……フェラしてるとき、ずっと優くんのこと考えてる」
- 優くんの目が一瞬だけ揺れる。
- 「優くんにするみたいに、丁寧に、大事にしてるの。……そうしたら、怖くなかった」
- 「……そうか」
- 「怒ってる？」
- 「怒ってない」
- 「嫉妬してる？」
- 長い沈黙。優くんが目を逸らす。
- 「……してる」
- 陽菜の心臓がとくん、と跳ねた。
- *嫉妬してるんだ。優くん。あたしが他の人にフェラしてることに。*
- *——嬉しい。ごめんね、嬉しいなんて。でも嬉しい。*
- 「……ごめんね。でも、あたしの唇は優くんのだよ。……今日だけ、ちょっとだけ、みんなに貸してるだけ」
- 優くんが小さく笑った。困ったような、でも優しい笑顔。
- 「……戻れ。次の客が来てる」
- 「うん♡」
- ——この小さなやりとりが、陽菜の午前中を支えた。

**合間に描く他メイドの状況（陽菜の視界の端で）：**
- あかりが高橋の手を引いてフェラブースから出てきた。頬が赤い。目がきらきら。カウンターの前で「……神代くん。あかり、我慢できなくなっちゃった♡」と言って、個室へ消えていった。
- *あかりちゃん、個室に行くんだ。……あたしは、まだ行けない。行かない。でも——あかりちゃんが眩しい。*

**描くこと：**
- 優くんとの距離感：人前では「水をもらう」だけの事務的な接触。でも言葉の裏に二人だけの感情が流れている
- 「唇は優くんのもの、今日だけ貸してる」＝陽菜なりの矛盾の折り合いのつけ方
- 優くんの嫉妬＝第1話の「嫌だよ」の延長線。でも陽菜を止めない
- あかりの個室行きを見て「あたしはまだ」と自覚する＝個室に行かない選択の再確認

### ■ Scene 5：午前の終わり——唇が覚えたもの（11:30〜12:30）

**場所：フェラブース→カフェスペース**

- 午前の後半。4人目、5人目のお客様。
- 陽菜のフェラに口コミが広がり始めている。
  - 廊下で「フェラブースの黒髪ロングの子がやばい」「すげー優しい」「天使」という声。
  - 「篠宮さん指名で」という客が現れ始める。
- 5人目。おとなしい2年生。
- 陽菜が膝をつく。もう最初のような震えはない。
- *あたしの唇が、覚えてきてる。*
- *人によって形が違う。大きさが違う。反応する場所が違う。*
- *でも、みんな同じように温かくて、みんな同じように「気持ちいい」って言ってくれる。*
- 丁寧に舌を使う。裏筋をゆっくりなぞりながら、先端を唇で転がす。
- 時折深く咥えて、喉の奥で包み込む。涙がにじむ。でも離さない。
- *……5人目。あたし、今日5人の男の人のを咥えた。*
- *優くん以外の人に。知らない人に。*
- *でも——全部、愛情を込めた。全部、優くんにするみたいに大切にした。*
- *それだけは、嘘じゃない。*
- 口内射精。飲み込む。口元を拭う。微笑む。
- 「……ご主人様♡ ありがとうございました♡」
- 5人目を見送って、ブースを出る。
- 個室3の方向から、あかりの声が聞こえている。「あっ あっ ご主人様♡」。甘くて、切実で、幸せそうな声。
- 陽菜の太ももが、ぴくり、と震えた。
- *あかりちゃんの声。あかりちゃん、気持ちよくなってるんだ。個室で。*
- *……あたしは、まだあっちには行かない。今日は、唇だけ。*
- *でも——*
- 太ももをきゅっと閉じる。メイド服のスカートの下が、じんわりと熱い。
- *……フェラしてるだけなのに。あたし、こんなに……*
- *だめ。考えちゃだめ。*
- 昼休憩。おにぎりを食べながら、陽菜はふと自分の唇に触れた。
- *赤くなってる。ちょっとだけ腫れてる。5人分のご奉仕の跡。*
- *……優くんに見せたら、どんな顔するだろう。*
- 優くんがカウンターで集計をしている横顔を、遠くから見つめた。
- *……午後も、頑張ろう。あたしの唇で。あたしの愛で。*

**描くこと：**
- 陽菜のフェラが「技術」として成長していることを描写。ただし技巧ではなく「丁寧さ」の深化
- 口コミで評判が立つ＝「やさしいフェラ」「天使」の噂
- あかりの個室の声で太ももが疼く＝身体は正直。でも理性で止める
- 5人のフェラを終えた唇に触れる＝自分の身体の変化を実感する
- 午前の着地：恐怖を乗り越え、自分だけの奉仕スタイルを確立した

## 後編テーマ

**「あたしの唇は、全部愛でできてる——フェラブースの天使が見つけた、自分だけの答え」**

午後の部。陽菜のフェラは口コミで評判になり、指名客が途切れない。「黒髪ロングの天使」と呼ばれ始めた陽菜は、一人ひとりに同じ愛情を注ぎ続ける。疲れても、唇が痺れても、膝が痛くても——恋人への愛を源泉に、献身は止まらない。そして閉店後、優くんの前でだけ、「メイド」ではない「篠宮陽菜」に戻る。

## 後編 シーン構成

### ■ Scene 1：午後の部——指名の列（12:30〜13:30）

**場所：フェラブース**

- 午後の部開場。口コミ効果で客足が午前より多い。
- 陽菜のブースの前に、小さな列ができている。
- 「あの……黒髪の長い子にお願いしたいんですけど」
- 「篠宮さん、いますか？」
- 陽菜が驚く。*あたしを指名してくれてる。あたしのフェラを、求めてくれてる。*
- 美月がにこりと微笑む。「陽菜ちゃん、人気者ね♡」
- *嬉しい。……嬉しいけど、変な感じ。あたしのフェラが「商品」として求められてる。*
- でも陽菜はその違和感を飲み込んで、膝をつく。
- 6人目。7人目。
- 同じように丁寧に。同じように愛を込めて。
- 一人ひとり、形が違う。匂いが違う。声が違う。
- でも陽菜は全員に同じことをする。
  - 先端にキスを落としてから始める。
  - 両手を添える。片手は根元を、もう片手はお客様の手に重ねる。
  - ゆっくり、ゆっくり、唇と舌で包み込む。
  - 時折見上げる。潤んだ瞳で。「気持ちいいですか？♡ ご主人様♡」
  - 口内射精を全部受け止めて、飲み込んで、最後に先端にもう一度キス。
  - 「……ありがとうございました♡」
- 7人目のお客様が帰り際に振り返って言った。
- 「……なんか、彼女にしてもらってるみたいだった」
- 陽菜の心臓が跳ねた。
- *彼女にしてもらってるみたい。*
- *——うん。そうだよ。あたし、恋人にするみたいにしてるから。*
- *あたしにとって、これはご奉仕じゃない。……ううん、ご奉仕なんだけど。*
- *恋人に捧げるキスと同じものを、全部の人にあげてるの。*
- *だからあたしのフェラは優しいの。だから「天使」なんて呼ばれるの。*
- *あたしの愛の源泉は——全部、優くんだから。*

## 陽菜編の独自性

あかりには高橋がいた。美月には中村がいた。でも陽菜には「この人」がいない。13人全員が「この人」。全員に同じだけの愛を注ぐ。その愛の源泉は——恋人の神代優。

**一人の恋が、13人への愛になる。**

それが、篠宮陽菜のメイドとしての在り方。

## 第3話への布石

第2話の陽菜は「フェラブースのみ」で1日目を戦い抜いた。しかしその体験を通じて、「唇だけでなく全身で相手を気持ちよくしてあげたい」という思いが芽生えた。

**2日目（10月5日）の陽菜は通常接客のフルコースに挑む。**
- カフェでの接客（笑顔・会話）
- フェラブース（1日目で確立した「天使のフェラ」）
- 個室ブース（★デビュー）——フェラで培った「やさしさ」を本番でも

動機は恐怖の克服ではなく、**「もっと相手を気持ちよくしてあげたい」**という愛情の拡張。陽菜らしい、恋人への愛を源泉にした献身が、唇から全身へと広がっていく。
