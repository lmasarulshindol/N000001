# ヒロインステータス定義

N000001 共通のキャラクター数値体系。**好感度は採用しない**（性欲・信頼のどちらかが高ければ身体は許しやすいため）。  
数値は **1〜10**。高いほどその性質が強い。

---

## 1. 恒常ステータス（キャラシート用）

| ステータス | 意味 | 低いとき | 高いとき |
|-----------|------|---------|---------|
| **性欲** | 性的欲求の強さ・頻度 | 余裕がある・誘惑に強い | 焦り・求める・我慢が効かない |
| **性知識** | 用語・体位・避妊等の理解（頭の中） | 無知・誤解・照れ隠し | 的確な言葉・余裕のある反応 |
| **感度** | 身体反応の出やすさ・強さ | 時間がかかる・抑えられる | すぐ濡れる・声が漏れる・絶頂が早い |
| **羞恥心** | 恥じらい・拒否・赤面の度合い | 大胆・開放的 | 視線を逸らす・口ごもる・泣きそう |
| **積極性** | 自分から踏み出すか | 受け身・待つ・言われるまで動かない | 誘う・主導する・先に手を出す |
| **信頼度** | 相手を身体に預けられるか | 警戒・抵抗・条件付き | 無防備・委ねる・重い行為も許す |

### 経験値（段階）

数値ではなく **段階ラベル** で管理する。

| 段階 | 説明 |
|------|------|
| `未経験` | 挿入・本番なし |
| `口のみ` | 口での経験あり |
| `経験あり` | 本番経験あり（回数は少なめ可） |
| `慣れ` | 繰り返し経験・身体が覚えている |

---

## 2. シーン中のみ（キャラシートには書かない）

執筆・ゲーム進行中に変動する一時値。泡児島ノベルゲームの `intimateSession` に相当。

| 名称 | 意味 |
|------|------|
| 興奮度 | その場の高ぶり（0〜100） |
| 絶頂回数 | シーン内カウンタ |
| 同行動回数 | 同じプレイの繰り返し（羞恥の低下に使う） |

---

## 3. ステータス同士の関係

```
性知識 ── 頭の理解
経験値 ── 身体の慣れ     ← 別軸。「知ってるけど未経験」が作れる
感度   ── 生まれつき／体質

羞恥心 ── その場の恥
積極性 ── 自分から動くか
性欲   ── 欲の強さ
信頼度 ── 相手への預け度  ← 好感度の代わり。「何をどこまで許すか」
```

### よく使う分岐の例

| 見たい描写 | 見るステータス |
|-----------|---------------|
| 初々しい拒否 | 羞恥心↑ 経験値=未経験 |
| 知識はあるのに震える | 性知識↑ 羞恥心↑ 経験値=未経験 |
| 口では強がるが体が先 | 積極性↑（言葉） 感度↑ 羞恥心（事後）↑ |
| 使命として淡々と | 性知識↑ 積極性↑ 羞恥心中〜高 |
| 信頼がないと中出し不可 | 信頼度＋経験値 |

---

## 4. 数値の目安

| 帯 | 範囲 | ラベル |
|----|------|--------|
| 低 | 1〜3 | ほぼない／極端に弱い |
| 中 | 4〜6 | 平均的・状況次第 |
| 高 | 7〜10 | 顕著・キャラの核 |

**10** は「この作品内でトップクラス」を意味する。全員に10を付けない。

---

## 6. 採用しないもの

| 候補 | 理由 |
|------|------|
| 好感度 | 性欲・信頼と結果が重複しやすい。恋愛ラインが主題の作品のみ別途「恋心」を検討 |
| 堕落度 | 羞恥心の経時低下＋経験値で代用可能 |
| 部位別感度 | 必要なキャラのみ補足欄に記載（例: 胸1.2倍） |
