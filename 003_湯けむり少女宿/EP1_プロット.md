# EP1「湯けむりの向こうに」詳細プロット

> 藤堂悠一（32歳・SE）× 白鷺千紗（14歳・少女仲居）/ 秋 / 東北の温泉旅館「鶴乃湯 花菱」

---

## シーン構成（全8シーン）

### シーン1：辿り着いた秘湯（チェックイン）
- **場所**: バス停 → 温泉街の坂道 → 旅館「鶴乃湯 花菱」玄関
- **時間帯**: 10月中旬、午後3時過ぎ。紅葉が盛り
- **登場人物**: 藤堂悠一、白鷺千紗
- **出来事**:
  - バスを降りた悠一。乗客は自分一人。バス停には古びた木の看板「鳴子鶴乃湯温泉郷」。山の冷気と硫黄の匂いが頬を撫でる
  - 東京を朝に出て、新幹線と在来線とバスを乗り継いで約4時間。スマホの電波は1本。都会の喧騒が嘘のように静か
  - 温泉街は寂れている。廃業した旅館の看板が色褪せて残る。しかし紅葉に染まった山並みは息を呑むほど美しい。渓流のせせらぎが耳に心地よい
  - 石畳の坂道を登ると「鶴乃湯 花菱」の門構え。打ち水された玄関。提灯に灯が入り始めている
  - 玄関の引き戸を開けると、畳と藺草の匂い。奥から下駄の音。そして——
  - 正座で出迎える少女。深いお辞儀。「ようこそお越しくださいました。鶴乃湯花菱でございます」。黒髪を夜会巻きにまとめ、臙脂色の着物をきりりと着こなした少女——白鷺千紗（14歳）
  - 悠一の第一印象：若い。あまりにも若い。「仲居」の姿をした少女。戸惑い。だが、その所作の美しさに一瞬言葉を失う
  - 千紗が悠一の靴を揃え、スリッパを差し出し、荷物を預かる。一連の動作に無駄がない
  - 帳場で記帳。千紗が宿帳を差し出す。悠一は「藤堂悠一」と書く。千紗が名前を確認し、「藤堂さまでございますね。本日は千紗がお世話させていただきます」
  - 客室「山吹の間」へ案内。千紗が先を歩く。廊下の板張りに足袋の音。窓の外は紅葉と渓流。千紗が襖を開け、部屋の説明をする。12畳の和室。床の間に秋の掛け軸と一輪の野菊
  - 悠一は畳に腰を下ろす。窓を開けると渓流の音と、山の冷気と紅葉の匂い。肩の力がすとんと抜ける感覚。いつ以来だろう、こんなに静かな場所にいるのは
- **描写のポイント**: 都会→秘湯の場面転換で読者を「非日常」に引き込む。五感描写をフル動員（硫黄の匂い、渓流の音、紅葉の色、冷たい山の空気、畳の感触）。千紗の第一印象——年齢への戸惑いと、仲居としての所作の見事さのギャップ。悠一の疲労が「静けさ」に触れて初めて自覚される瞬間
- **感情の流れ**: 疲労・無感動 → 自然への小さな感動 → 千紗への戸惑い → 静けさへの安堵
- **文量目安**: 約3,000字

### シーン2：抹茶と和菓子（お茶出し）
- **場所**: 客室「山吹の間」
- **時間帯**: 午後3時30分
- **登場人物**: 悠一、千紗
- **出来事**:
  - 千紗が抹茶と栗きんとんの和菓子を運んでくる。漆塗りの盆。茶碗の置き方、菓子皿の向き——すべてが丁寧
  - 千紗が客室の隅で正座し、茶筅で抹茶を点て始める。シャカシャカという小気味よい音が部屋に響く。悠一は思わず見入る
  - 「お茶をどうぞ」。千紗が茶碗を両手で差し出す。指先が白く、細い。琥珀色の瞳がまっすぐに悠一を見ている
  - 悠一が一口飲む。抹茶の苦味と栗きんとんの甘さ。「……美味いな」。素直な感想が口をつく。千紗の表情がほんの一瞬だけ緩む
  - お茶を飲みながら、ぽつりぽつりと会話。千紗が旅館や温泉街の説明をする。「この時期の紅葉が一番でございます」「渓流沿いの露天風呂からの眺めは格別でございます」
  - 悠一が「若いのに、よくできるね」と言ってしまう。千紗の眉がわずかに動く。「……恐れ入ります。まだまだ修行中でございます」。丁寧だが、少しだけ硬くなった声
  - 悠一は自分の言葉を後悔する。余計なことを言った。プロとして働いている相手に「若いのに」は失礼だ
  - 千紗が温泉の案内をする。「大浴場と露天風呂がございます。よろしければ、お背中をお流しいたしましょうか」
  - 悠一が戸惑う。「背中を……？ いや、一人で大丈夫だよ」。千紗が「かしこまりました」と引き下がるが、どこか残念そうな気配が一瞬よぎる
  - 千紗が退室する。襖を閉める音が静か。悠一は一人になり、窓の外の紅葉を見る。「背中を流す」という申し出が頭に残っている。あの少女が？ 温泉で？
- **描写のポイント**: 抹茶を点てる千紗の「プロの所作」と14歳の「華奢な指先」の対比。悠一の失言（「若いのに」）で二人の間に微妙な緊張が走る——千紗のプロ意識を傷つけた罪悪感。「背中を流す」提案が次のシーンへの伏線。和菓子と抹茶の味覚描写で「五感の癒し」を強調
- **感情の流れ**: リラックス → 千紗の所作への感心 → 失言の後悔 → 「背中を流す」への戸惑い・好奇心
- **文量目安**: 約2,500字

### シーン3：背中を流す（入浴シーン）
- **場所**: 大浴場 → 露天風呂
- **時間帯**: 午後4時
- **登場人物**: 悠一、千紗
- **出来事**:
  - 悠一が大浴場へ向かう。脱衣所には誰もいない。平日で客が少ないのだ
  - 浴場に入る。湯けむりが立ち込める。石造りの大きな浴槽。ナトリウム硫酸塩泉の透明な湯が掛け流しになっている。窓の外は紅葉
  - 体を洗い、湯に浸かる。じわりと体に沁みる温かさ。硫黄の匂い。悠一は目を閉じ、長い溜息をつく。肩こり、腰痛、眼精疲労——全身の凝りを湯が溶かしていく
  - 露天風呂へ移動。渓流沿いの岩風呂。頭上は紅葉のトンネル。赤、橙、黄色の葉が風に揺れる。山の冷気と湯の温かさのコントラスト
  - しばらく一人で浸かっていると、背後から静かな声。「藤堂さま、失礼いたします」。千紗だ
  - 振り返ると、千紗が浴場の入り口に立っている。仕事用の薄い着物に着替え、袖を肩の上でたすき掛けにしている。白い腕が露わになっている
  - 「やはり、お背中をお流しいたします。お部屋でのご遠慮は、お疲れが溜まっていらっしゃる証でございます」。千紗が小さな桶と手拭いを持って近づいてくる
  - 悠一が「いや、本当にいいから——」と言いかけるが、千紗はすでに洗い場に膝をつき、準備を始めている。「遠慮なさらず。これも私の仕事でございますので」
  - 悠一は観念して洗い場に腰を下ろす。千紗が背後に回る。温かい湯を掬い、悠一の背中に流す
  - 千紗の手が石鹸を泡立て、悠一の背中に触れる。小さな手。だが力加減が絶妙。肩甲骨の周り、首の付け根、腰——凝りの溜まっている場所を的確にほぐしていく
  - 「藤堂さま、ずいぶんお疲れのご様子でございます。こちらの筋が張っていらっしゃいます」。千紗の指が肩の凝りを見つける。プロの触診のような手つき
  - 湯けむりの中、少女の手が大人の男の背中を洗う。石鹸の泡が背中を滑る。千紗の指先が脊椎に沿って下りていく。腰まで。丁寧に、一つ一つの凝りをほぐすように
  - 悠一は目を閉じている。千紗の手の温かさと、湯けむりと、渓流の音。体の力が抜けていく。いつの間にか、こんなにも体が強張っていたのだと気づく
  - 「おばあちゃんに教わりました」と千紗がぽつりと言う。「お客さまの体を流すのは、心を流すのと同じだって」
  - 悠一の胸に小さな感動が走る。14歳の少女の手で、自分は今、洗われている。それは単なる入浴介助ではなく、不思議な「癒し」だった
  - 背中を流し終え、千紗が湯を掛けて泡を洗い流す。「ごゆっくりお浸かりくださいませ。夕食は六時にお部屋へお持ちいたします」。千紗が一礼して去る
  - 悠一は露天風呂に戻り、紅葉を見上げる。手のひらを見る。自分の体が少し軽くなった気がする
- **描写のポイント**: 温泉描写の最大の見せ場。湯けむり・紅葉・渓流の三点セットで「和の空間美」を構築。千紗の入浴介助が「プロの仕事」であることを強調しつつ、少女の手が大人の男の背中に触れる官能性を漂わせる。「おばあちゃんの教え」で千紗の人間性が垣間見える。悠一の体の「強張り」が物理的にほぐされる＝心の壁がほぐされる暗喩
- **感情の流れ**: 一人の安堵 → 千紗の登場への狼狽 → 観念 → 手の温かさへの驚き → 癒しの実感 → 小さな感動
- **文量目安**: 約3,500字

### シーン4：夕食と布団敷き（距離が縮まる）
- **場所**: 客室「山吹の間」
- **時間帯**: 午後6時〜午後8時30分
- **登場人物**: 悠一、千紗
- **出来事**:
  - 夕食の時間。千紗が懐石料理を一品ずつ運んでくる。先付から始まり、椀物、お造り、焼物（岩魚の塩焼き）、煮物、ご飯と止め椀
  - 千紗が料理の説明をする。「こちらは地元の渓流で獲れた岩魚でございます」「山菜は今朝、裏山で採ったものでございます」。料理長の仕事を誇るように、丁寧な口ぶり
  - 悠一が「美味い」と声を上げる。岩魚を一口食べた瞬間。コンビニ弁当とファストフードで麻痺していた舌が、久しぶりに本物の味を感じる
  - 千紗が「ありがとうございます。料理長に伝えましたら喜びます」と微かに笑う。初めて見る笑顔。控えめだが、温かい
  - 地酒を勧められ、悠一は一合だけ頼む。千紗が徳利と猪口を運び、酌をする。華奢な手が徳利を傾ける仕草。悠一は受け取りながら、千紗と目が合う
  - 酒が入り、悠一の口が軽くなる。仕事の愚痴がぽろぽろと出る。プロジェクトの炎上、眠れない夜、週末も返上で働いた日々。千紗は正座のまま、静かに聞いている
  - 「すまない、仲居さんにこんな話をしても……」。悠一が我に返る。千紗が首を小さく横に振る。「お客さまのお話を聞くのも、仲居の務めでございます」
  - 千紗がぽつりと自分のことを話す。「私も……ここに来る前は、寂しい場所にいました。村に同い年の子が一人もいなくて」。祖母のことを少しだけ。故郷の廃校のことを少しだけ
  - 二人の間に、不思議な共感が生まれる。「居場所がなかった者同士」の静かな共鳴
  - 夕食が終わる。千紗が膳を下げ、少し経ってから戻ってくる。「お布団を敷かせていただきます」
  - 千紗が手際よく布団を敷く。シーツを整え、掛け布団の角を揃える。枕元に水差しと湯呑みを置く
  - 布団を敷き終えた千紗が正座する。一瞬の沈黙。そして——
  - 「藤堂さま。当館では……夜のお世話も承っております」。千紗の声が少し低くなる。「添い寝、湯もみ、あるいは……夜伽。ご希望がございましたら」
  - 悠一が言葉を失う。「夜伽」。その言葉の意味を理解するのに数秒かかる。この少女が——夜伽を？
  - 千紗の表情は淡々としている。仕事の説明をしている、というプロの顔。だが琥珀色の瞳の奥に、かすかな緊張がある
  - 悠一の頭が混乱する。倫理、道徳、常識——いくつもの「ダメだ」が頭を巡る。だが、背中を流してくれた千紗の手の温かさを体が覚えている
  - 「……添い寝、だけ。それなら」。悠一の口が勝手に動く。自分でも驚いている
  - 千紗が小さく頷く。「かしこまりました。少しお待ちくださいませ。湯浴みを済ませてまいります」。千紗が静かに退室する
  - 悠一は一人、布団の上に座り込む。何をしているんだ自分は。だが、断りきれなかった。あの琥珀の瞳に
- **描写のポイント**: 夕食の味覚描写で「五感の再起動」を進める。酒の力で悠一が本音を漏らし、千紗も自分の境遇を少しだけ開示——心理的距離が縮まる。「夜伽の案内」が物語の分岐点。悠一の葛藤を丁寧に描きつつ、「添い寝だけ」という中間選択で段階的な心の変化を表現。千紗が「プロの顔」で夜伽を案内する姿と、瞳の奥の緊張のギャップ
- **感情の流れ**: 食の感動 → 酒と会話で心が開く → 千紗への共感 → 「夜伽」の言葉への衝撃 → 葛藤 → 「添い寝だけ」の決断 → 自分への困惑
- **文量目安**: 約3,500字

### シーン5：夜伽——導入（添い寝から始まる夜）
- **場所**: 客室「山吹の間」
- **時間帯**: 午後9時
- **登場人物**: 悠一、千紗
- **出来事**:
  - 襖が静かに開く。千紗が戻ってくる。湯浴みを終え、髪を下ろしている。腰まで届く黒髪が、湯上がりの湿り気でしっとりと光っている
  - 着物は薄い白地の浴衣一枚。帯は簡素な貝の口結び。浴衣の下に肌着を着ていないのが、襟元のラインから透けて分かる
  - 部屋は行灯の灯りだけ。障子越しに月明かり。渓流のせせらぎが遠くに聞こえる
  - 千紗が悠一の布団の傍に正座する。「添い寝でございますね」。確認するような声。悠一が頷く
  - 千紗が掛け布団をめくり、悠一の隣に滑り込む。布団の中に石鹸と湯上がりの匂いが満ちる。千紗の体温が近い
  - 仰向けに並ぶ二人。天井を見ている。沈黙。渓流の音だけが流れる
  - 「……千紗さん」「千紗で構いません」「千紗。こういうのは、慣れてるのか？」。悠一が聞いてしまう
  - 千紗の沈黙。数秒。「……慣れるものではございません。ですが、お客さまに安らいでいただくのが、私の務めです」。声は静かだが、揺らぎがない
  - 千紗が寝返りをうち、悠一のほうを向く。琥珀の瞳が行灯の灯りを映している。「藤堂さまは、お優しい方ですね」
  - 「どうして」「……『添い寝だけ』とおっしゃいましたから」。千紗の声に感情が混ざる。いつもの淡々とした仲居の声ではなく、年相応の少女の声
  - 悠一の胸が締まる。この子は「添い寝だけ」を頼む客が珍しいのだろうか
  - 千紗が悠一の浴衣の袖に触れる。「おやすみなさいませ。朝までおそばにおります」
  - 悠一が目を閉じる。千紗の体温が布団越しに伝わってくる。規則正しい寝息……ではない。千紗も起きている
  - どれくらい経っただろう。悠一は眠れない。千紗の匂いが近すぎる。湯上がりの石鹸の香りに混じる、少女の体温そのもののような甘い匂い
  - 悠一が寝返りをうつ。千紗のほうを向く。月明かりに照らされた千紗の横顔。目を閉じている。だが睫毛が微かに震えている。起きている
  - 「千紗」。小さな声で呼ぶ。千紗の目が開く。至近距離。琥珀の瞳が月明かりで金色に見える
  - 「……眠れませんか」と千紗。「ああ」と悠一。「きみが隣にいると、眠れない」
  - 千紗の頬がほんのり色づく。行灯の灯りでも分かる。「……では」。千紗が悠一の胸元に手を置く。薄い浴衣越しに、小さな手の温もり
  - 「もう少し、近くに参りましょうか」。千紗がそっと体を寄せる。悠一の腕の中に収まるように。千紗の頭が悠一の胸に触れる。黒髪の匂い。細い体の輪郭。心臓の鼓動が伝わる距離
  - 悠一の手が千紗の背中に触れる。浴衣越しに感じる華奢な肩甲骨。この子は、こんなに小さい
  - 「添い寝だけ」。自分で決めた境界線。だが千紗の体温に包まれていると、その線がどんどん曖昧になっていく
  - 千紗が悠一の胸に顔を埋めたまま、小さく呟く。「藤堂さま。……もし、お望みでしたら」。声が震えている。仲居の「仕事」としての案内ではない。千紗個人の声
  - 悠一の理性と欲望の攻防。この子に触れたい。もっと。だが——
  - 悠一の手が千紗の顎に触れ、顔を上げさせる。月明かりに照らされた千紗の顔。潤んだ琥珀の瞳。唇がわずかに開いている。呼吸が浅い
  - 悠一が千紗の額にキスをする。額に。唇ではなく。「……ありがとう。もう少しだけ、こうしていてくれ」
  - 千紗の目が大きくなる。そして——微笑む。今日一番の、柔らかい笑み。「はい」
  - だが、その夜の温もりが、悠一の中の何かを確実に変えていく。千紗の寝息が規則正しくなる頃、悠一は自分の心臓が千紗の温もりに向かって傾いているのを感じている

## EP1固有の描写方針

### 和のエロティシズム五原則

1. **衣擦れの音**: 浴衣の帯を解く音、肩から滑り落ちる布の音が最大の官能装置
2. **月明かり**: 行灯の灯りと障子越しの月光だけで肌を照らす。蛍光灯やLEDは存在しない
3. **湯の記憶**: 温泉の湯の滑らかさが肌に残り、行為中の肌の質感に反映される
4. **渓流の音**: 行為中のBGMは渓流のせせらぎと虫の声。二人の呼吸と肌の音がそれに重なる
5. **畳と布団**: 和室の布団の上という「場」がベッドとは異なる親密さを生む

### 千紗の描き方

- **仲居としての千紗**: 凛とした所作、丁寧な言葉遣い、プロ意識。14歳とは思えない落ち着き
- **素の千紗**: 恥じらい、緊張、祖母への思い。行為中に仲居の言葉遣いが崩れ始める瞬間が最大のポイント
- **二面性の交差**: 「お客さまのすべてを受け止めなさい」という祖母の教えが、仲居の仕事と行為の両方を橋渡しする
- 行為中のセリフ: 敬語が基本だが、快感が高まると「……っ」「あ……」と言葉が崩れる。完全にタメ口にはならない
- 涙: 痛みではなく、自分でも名前がつけられない感情からの涙。初めての経験の衝撃

### 悠一の描き方

- 三人称・悠一視点。内面描写多め
- 「仕事疲れの男が癒されていく」過程を五感で追体験させる
- 千紗への感情: 戸惑い → 感心 → 共感 → 慈しみ → 欲望 → 行為後の穏やかな愛着
- 理性の崩壊を段階的に描く: ①入浴介助を断る（理性）→ ②添い寝だけと決める（妥協）→ ③千紗に触れてしまう（衝動）→ ④挿入・中出し（没入）
- 行為中は「大人の余裕」と「千紗への配慮」を見せる。乱暴にならない。千紗のペースに合わせる

- **シーン1〜2（チェックイン・お茶）**: 千紗100%（仲居が客をリード）
