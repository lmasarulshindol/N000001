# EP0「山百合の仲居」詳細プロット

> 白鷺千紗（14歳・少女仲居）視点 / 秋田の山村 → 東北の温泉旅館「鶴乃湯 花菱」/ 春〜秋

---

## EP0概要

EP0はヒロイン・白鷺千紗の前日譚。過疎の山村で祖母に育てられた少女が、温泉旅館「鶴乃湯 花菱」に入館し、仲居として成長していく姿を描く。行為シーンはなく、千紗の日常と内面——祖母との絆、仲居としてのプロ意識の芽生え、夜伽への覚悟——に集中する。

ラストシーンでは、千紗が初めて夜伽を含む担当客を受け持つことになり、藤堂悠一の到着を玄関で待つ場面で幕を閉じる。EP1「湯けむりの向こうに」の冒頭へ直結する構成。

**文量目安**: 15,000〜20,000字（行為シーンなし）
**視点**: 千紗の一人称的三人称（千紗視点。丁寧で少し古風な語り口）
**行為シーン**: なし
**重要事項**: 千紗は処女。EP1で悠一が初めての夜伽の相手となる。

## シーン構成（全6シーン）

### シーン1：山百合の咲く村（秋田の山村・祖母との暮らし）
- **場所**: 秋田県北部の過疎集落。千紗と祖母の古い木造家屋。裏山。廃校になった小学校
- **時間帯**: 春〜初夏。早朝から昼過ぎ。山百合が咲き始める季節
- **登場人物**: 白鷺千紗（14歳）、祖母・白鷺ハツ（76歳）
- **出来事**:
  - 朝。千紗が祖母より先に起き、囲炉裏に火を入れる。水を汲み、味噌汁を作る。祖母が教えてくれた「お客さまをもてなすように、自分の家族をもてなしなさい」という教えを守っている
  - 千紗の日常の描写。村には同い年の子どもがいない。小学校は3年前に廃校になった。中学校はバスで40分の隣町。片道の通学で毎朝5時半に起きる
  - 朝食後、祖母の着付けの稽古。千紗に浴衣の着付けを教える祖母。「帯は心臓の高さ。それより上だと苦しい、下だと崩れる」。祖母の手は皺だらけだが、帯を結ぶ手つきは淀みがない
  - 祖母が千紗に抹茶の点て方を教える場面。茶筅の振り方、手首の使い方。「お湯は八十度。沸騰させたらあかん」。千紗が一服点てると、祖母が目を細めて飲む。「上手になったねえ」
  - 祖母の昔話。「おばあちゃんはね、若い頃に鶴乃湯の花菱さんで仲居をしていたんだよ」。温泉旅館の仲居だった頃の思い出。お客さまの背中を流したこと、布団を敷いたこと、朝食のお膳を運んだこと。「一期一会でね。お客さまにとっては一泊の旅でも、お迎えするほうにとっては一生に一度のご縁なの」
  - 祖母の教え「お客さまの体を流すのは、心を流すのと同じ」。千紗はまだその意味を完全には理解していないが、祖母の声の響きが胸に残る
  - 裏山への散歩。山百合が咲いている。祖母が「千紗は山百合に似ているねえ。目立たないところで静かに咲いて、でも見つけた人はみんな息を呑む」と言う
  - 夕方。祖母が咳き込む。最近、咳がひどくなっている。千紗が背中をさする。祖母は「なんでもないよ」と笑うが、千紗は気づいている——祖母の手が、以前より冷たくなっていること
  - 夜。布団の中で千紗が考える。父からの仕送りはもう2ヶ月届いていない。祖母の薬代。中学校の教材費。米の残りはあと一袋。不安が、山百合の匂いと一緒に暗闇の中に溶けていく
- **描写のポイント**: 千紗の原風景。過疎の山村の美しさと寂しさの同居。祖母との二人暮らしの温かさと脆さ。着付け・抹茶・礼法を教える祖母の姿が、千紗の「仲居としての才能」の原点であることを印象づける。山百合がEP0全体のモチーフ。「一期一会」と「お客さまの体を流すのは心を流すのと同じ」——二つの祖母の教えがこの物語の通奏低音になる。千紗の語りは14歳の少女の目線だが、祖母の影響で古風な言い回しが自然に混じる
- **感情の流れ**: 穏やかな日常の幸福感 → 祖母の教えへの敬意と憧れ → 山百合への愛着 → 祖母の体調への不安 → 将来への漠然とした恐怖
- **文量目安**: 約3,500字

### シーン2：旅立ち（花菱への入館）
- **場所**: 秋田の集落の家 → バス停 → 在来線の車窓 → 鳴子鶴乃湯温泉郷の温泉街 → 旅館「鶴乃湯 花菱」玄関 → 女将の居間 → 寮室
- **時間帯**: 初秋（9月）。早朝の出発から午後の到着まで
- **登場人物**: 千紗、祖母・ハツ、女将・鶴見楓（45歳）、同室の先輩仲居・小鳥遊ひなた（15歳）
- **出来事**:
  - 冒頭——祖母の入院。村の診療所から隣町の病院へ。検査の結果、長期の治療が必要と告げられる。入院費は月に8万円。父からの仕送りは途絶えている
  - 花菱の女将・鶴見楓から祖母宛に手紙が届く。「ハツさんのお孫さんが仲居見習いとしてうちで働く気はないか」という誘い。祖母がかつて花菱で仲居をしていた縁。女将は祖母の教え子筋にあたる
  - 祖母が千紗に花菱の話をする。「昼はお客さまのお世話。夜は……夜のお仕事もあるんだよ」。祖母の口ぶりは穏やかだが、目が少し伏せられる。千紗はまだ「夜のお仕事」の具体的な内容をはっきりとは理解していない
  - 千紗の決断。祖母の入院費のため。「おばあちゃんが元気でいてくれるなら、千紗はなんでもする」。祖母が千紗の手を握る。その手がとても冷たい。「辛くなったら、いつでも帰ってきなさい」
  - 出発の朝。千紗が家の前に立つ。裏山に山百合が最後の花をつけている。祖母は病院から電話で見送る。声が震えている。千紗はバス停まで走る。泣かないように。振り返らないように
  - バスと在来線を乗り継ぐ道中。車窓の景色が秋田の山並みから宮城の山間へと変わっていく。千紗は祖母から持たされた風呂敷包み一つを膝の上に抱えている。中身は着替えと、祖母の簪と、抹茶の入った小さな缶
  - 温泉街の第一印象。バスを降りると、硫黄の匂いが鼻をつく。寂れた街並み。廃業した旅館の看板が色褪せて残っている。だが、山の紅葉が始まりかけていて、空気は澄んでいる。渓流のせせらぎが聞こえる。「おばあちゃんの温泉街も、こんな風だったのかな」
  - 花菱の門構え。打ち水された石畳。趣のある木造建築。提灯には「鶴乃湯 花菱」の文字。千紗は門の前で深呼吸する
  - 女将・鶴見楓との面談。女将は40代の美しい女性。着物姿に品がある。「千紗ちゃん、ようこそ。ハツさんにはお世話になったのよ」。穏やかだが、眼光は鋭い
  - 女将からの説明。「うちでは昼のおもてなしと、夜のお世話がございます」。昼の仲居としての仕事——出迎え、お茶出し、入浴案内、給仕、布団敷き。そして夜伽の説明。「添い寝から夜伽まで、お客さまのご希望に応じてお世話をするの。これは強制ではないけれど、うちの収入の大きな柱」
  - 千紗が「夜伽」の意味をここで初めて正確に理解する。動揺。だが顔には出さない。祖母の入院費のこと。月の仕送りのこと。「……やらせてください」。声が少し震えるが、目は伏せない
  - 女将が千紗をじっと見る。「まずは昼の仲居として一人前になること。夜のことはそれからよ」。千紗がほっとする——猶予がある
  - 寮室への案内。6畳の相部屋。もう一つの布団が敷かれている。同室は先輩仲居の小鳥遊ひなた（15歳）。明るくて人懐こい。「千紗ちゃん？ わあ、可愛い！ 今日からよろしくね！」。千紗は人見知りで上手く返事ができないが、ひなたの屈託のなさに少し肩の力が抜ける
  - 夜。寮室の布団の中。畳の匂い。隣のひなたの寝息。天井の木目を見つめながら、千紗は祖母のことを考える。「おばあちゃん、千紗はちゃんとやれるかな」。窓の外から渓流の音。硫黄の匂い。故郷とは違う山の匂い。千紗は風呂敷から祖母の簪を取り出し、胸に抱いて目を閉じる
- **描写のポイント**: 千紗の「旅立ち」を紀行文的に描く。秋田の山村→バス→在来線→温泉街→花菱と、場所が移るごとに千紗の世界が広がっていく感覚。女将との面談で「夜伽」の意味を知る場面が物語上の転換点——千紗が何を引き受けようとしているのかを読者と千紗が同時に理解する構成。ひなたの登場で重くなりすぎた空気が和らぐ。祖母の簪が千紗の「お守り」として機能し始める
- **感情の流れ**: 祖母への心配と使命感 → 出発の寂しさ（泣きそうだが泣かない） → 温泉街への好奇心と不安 → 女将の前での緊張 → 「夜伽」を知った衝撃と覚悟 → ひなたへの戸惑いと安堵 → 夜の孤独と祖母への想い
- **文量目安**: 約3,500字

### シーン3：白い蕾（研修期間・仲居の基礎）
- **場所**: 花菱の大広間（稽古場）、客室、大浴場、中庭、温泉街
- **時間帯**: 入館から1ヶ月間。9月〜10月初旬。朝から夕方の研修時間
- **登場人物**: 千紗、女将・鶴見楓、先輩仲居・小鳥遊ひなた、他の少女スタッフ2〜3名
- **出来事**:
  - 研修初日。女将が千紗を含む新人に研修内容を説明する。「一ヶ月で、お辞儀の仕方からお茶の点て方、お布団の敷き方まで。身体に染み込ませてもらいます」
  - 礼法の稽古。正座の仕方、お辞儀の角度（会釈15度、敬礼30度、最敬礼45度）、歩き方（畳の縁を踏まない、廊下は左側を歩く、足袋の音を立てない）。千紗の正座が最初から群を抜いている。他の少女たちが30分で足が痺れる中、千紗は1時間座っていても姿勢が崩れない。「祖母に毎日正座でお茶をいただいていましたから」
  - 着付けの稽古。仲居の着物——臙脂色の着物に白い半襟、銀鼠色の帯。帯の結び方は「お太鼓結び」。千紗は自分で着物を着られるだけでなく、帯の結び目が完璧に整っている。女将が目を見張る。「ハツさんの仕込みね」
  - 抹茶を点てる稽古。大広間に茶道具が並べられ、一人ずつ点てていく。他の少女たちが茶筅の扱いに苦戦する中、千紗は祖母に教わった通りに一服点てる。細かい泡が均一に立ち、色も香りも申し分ない。女将が一口飲んで、「……これは教えることがないわ」と苦笑する
  - 千紗が他の少女たちに「生意気」と見られ始める。「なんでもできるんだ、千紗ちゃんは」。嫌味ではないが、距離を感じる。千紗は人見知りで自分から話しかけるのが苦手。昼休みに一人で中庭の池の鯉を眺めている
  - ひなたが仲裁役になる。「千紗ちゃんはおばあちゃんに教わったから上手いだけで、威張ってるわけじゃないよー」。ひなたが千紗を他の少女たちの輪に引き入れる。少しずつ打ち解け始める
  - 配膳の練習。お盆の持ち方、襖の開け閉め、料理の置き方（左にご飯、右に汁物）。千紗は所作は完璧だが、声が小さい。「『お料理をお持ちいたしました』——千紗、声が小さい。お客さまに聞こえなかったら意味がないの」。女将に何度も指摘される。千紗が唇を噛む。負けず嫌いの性分が顔を出す。夜、寮室で一人、声出しの練習をする
  - 入浴介助の練習。大浴場で少女同士が背中を流し合う。千紗の手つきが丁寧すぎて時間がかかる。ひなたに背中を流してもらいながら、ひなたが「千紗ちゃん、力加減うまいね。でも速さも大事だよ。お客さまを待たせちゃうから」とアドバイス
  - 研修の合間に温泉街を歩く。地元のおばあちゃんたちが千紗に声をかける。「あら、花菱の新しい子？ 可愛いねえ」。漬物を分けてくれるおばあちゃん。山菜の見分け方を教えてくれるおばあちゃん。千紗は祖母を思い出して胸がきゅっとする。でも、この街のおばあちゃんたちの温かさに救われる
  - 研修の最終日。女将の前で通し稽古。出迎えからお茶出し、入浴案内、給仕、布団敷きまで。千紗が一通りやり終えると、女将が頷く。「合格。明日から客前に出なさい」。千紗の目が潤む。泣かない。でも、嬉しい
- **描写のポイント**: 研修のディテールで「温泉旅館の仲居」という仕事のリアリティを構築する。千紗の「祖母直伝の所作」が他の少女を凌駕する一方、声の小ささという弱点もある——完璧ではないからこそ成長の余地がある。ひなたが千紗の社交面を補完するバディ的存在。温泉街のおばあちゃんたちが千紗の「第二の祖母」のような存在になっていく。仲居の仕事の一つ一つ（正座・着付け・配膳・入浴介助）を五感で描写し、読者に「花菱のおもてなし」の空気を体感させる
- **感情の流れ**: 研修への緊張と気負い → 祖母の教えが活きる手応え → 孤立感（生意気と思われる不安） → ひなたの助けへの感謝 → 声の小ささへの悔しさと克服への努力 → 温泉街のおばあちゃんたちへの親しみ → 合格の喜び（静かな達成感）
- **文量目安**: 約3,500字

### シーン4：凛と立つ（仲居としての日常）
- **場所**: 花菱の玄関、客室、大浴場、露天風呂、厨房前、寮の朝湯
- **時間帯**: 研修修了後。10月。朝湯から夜の就寝まで（仲居の1日を通しで描く）
- **登場人物**: 千紗、ひなた、他の少女スタッフ、宿泊客（中年の夫婦など名前のない客）、料理長
- **出来事**:
  - 早朝。朝湯の時間。少女たちだけの露天風呂。湯けむりの向こうに紅葉の山が見える。朝日が差し込み、湯面が金色に光る
  - 朝湯での会話。素の千紗が見える時間。ひなたが「ねえねえ、昨日のお客さん面白かったよー」と話しかけ、千紗が小さく笑う。他の少女たちとの距離も縮まっている。一人の少女が「千紗ちゃん、髪きれい。濡れると色が変わるね」。千紗の髪が湯に濡れると藍がかった光沢を帯びる。千紗は照れて首まで湯に沈む
  - 朝食のまかない。料理長が作ってくれる焼き魚と味噌汁。千紗が「美味しい」と素直に言う。料理長が目を細める。「千紗ちゃんの味噌汁も美味いぞ。秋田の味だな」
  - ミーティング。女将が今日の宿泊客を発表する。千紗の担当は中年の夫婦。「記念日のご旅行だそうです。粗相のないように」
  - チェックイン。千紗が玄関で正座して出迎える。深いお辞儀。「ようこそお越しくださいました」。声はまだ少し小さいが、以前より通るようになっている。中年の夫婦が千紗を見て目を丸くする。「まあ、若い仲居さんね」
  - お茶出し。客室で千紗が抹茶を点てる。シャカシャカという茶筅の音。客の奥さまが「まあ、お上手ね。どこで習ったの？」。千紗が「祖母に教わりました」と答える。少し誇らしそうに
  - 入浴案内。大浴場への案内。千紗が「お背中をお流しいたしましょうか」と尋ねる。奥さまが「お願いするわ」と言う。千紗が入浴着に着替え、奥さまの背中を流す。「まあ……上手ねえ。気持ちいい」。凝りの場所を見つける千紗の指。「こちらの筋がお疲れのようです」。祖母の教え——「体を流すのは心を流すのと同じ」——を実践している千紗の姿
  - 夕食の給仕。懐石料理を一品ずつ運ぶ。岩魚の塩焼き、山菜の天ぷら、芋煮。千紗が料理の説明をする。「こちらは裏山で今朝採れた茸でございます」。ご主人が「若いのに本格的だなあ」と感心する。千紗の眉がわずかに動く——「若いのに」という言葉が少しだけ引っかかる。でも、「ありがとうございます」と微笑む。その微笑みにご主人と奥さまが顔を見合わせる。「いい笑顔ね」
  - 布団敷き。千紗がシーツを整え、掛け布団の角を揃える。枕元に水差しを置く。一連の動作に無駄がない
  - 夜。仕事を終えた千紗が寮に戻る。疲れているが、充実感がある。ひなたが「今日のお客さん、千紗ちゃんのこと褒めてたよー。帳場でご主人が『あの子は将来すごい仲居になるよ』って」。千紗の頬がほんの少し赤くなる
  - 就寝前。祖母への手紙を書く。便箋に万年筆。「おばあちゃんへ。お仕事がんばっています。おばあちゃんの教えが役に立っています。抹茶を褒めていただきました。お背中を流すのも、お客さまに喜んでいただいています。お体はいかがですか。お薬はちゃんと飲んでいますか。今月のお給料を同封します。寒くなってきたので、温かくしてください。千紗より」
  - 手紙を封筒に入れ、仕送りの現金を同封する。封をしてから、千紗は窓の外を見る。紅葉の山と、湯けむりと、星空。「おばあちゃん。千紗は大丈夫だよ」。声に出さずに呟く

### シーン5：夜の意味（夜伽への覚悟）
- **場所**: 花菱の廊下、寮室、ひなたとの布団の中、女将の居間、中庭
- **時間帯**: 入館から3ヶ月後。11月。夕方〜深夜
- **登場人物**: 千紗、女将・鶴見楓、ひなた
- **出来事**:
  - 夕方。千紗が廊下を歩いていると、ひなたが客室から出てくるのとすれ違う。ひなたは夜伽を終えたばかり。髪を下ろしている。浴衣が少し乱れている。ひなたが千紗に気づき、いつもの笑顔を見せる。「あ、千紗ちゃん。おはよー」。だが、その笑顔の奥に千紗は何かを見る——疲れなのか、それとも別の何かなのか、名前をつけられない感情

  - 「少し、考えさせてください」。千紗が小さく頭を下げる。女将が「ええ、もちろん」と微笑む
  - 夜。寮室で布団に入った千紗が、ひなたに聞く。「ひなた先輩。……夜伽って、どんな感じですか」。暗闘の中、ひなたが少し黙る。そして、「うーん……怖くはないよ。最初はちょっと痛いけど。でもね、優しいお客さまだと、ちゃんと気持ちいいの」。ひなたの声がいつもより大人びている
  - 「お客さまに、体を……見られるんですよね」。千紗の声が小さくなる。ひなたが「うん。でもね、暗いから。行灯の灯りだけだし。それに、お客さまも緊張してるんだよ。私たちだけじゃない」。少しの間。「千紗ちゃん、大丈夫だよ。千紗ちゃんの背中の流し方を見てたら分かる。お客さまのことを大事にできる子だもん」
  - 千紗がひなたの言葉を噛みしめる。「おばあちゃんに教わりました。お客さまのすべてを受け止めなさい、って」。その「すべて」の意味が、今ようやく輪郭を持ち始める。祖母もかつて花菱の仲居だった。祖母も——夜伽を。千紗の胸が締まる。「おばあちゃんも、昔はこうだったのだろうか」

  - 翌朝。千紗が女将の居間を訪ねる。「女将さん。……夜のお世話も、やらせてください」。声が震えている。でも、目は伏せない。琥珀の瞳がまっすぐに女将を見ている
  - 女将がしばらく千紗を見つめ、静かに頷く。「わかったわ。ピルの服用を始めましょう。次にふさわしいお客さまが見えたら、千紗に担当をお願いするから」

- **描写のポイント**: EP0最大の心理ドラマ。千紗が「夜伽」を具体的に受け入れるまでの内面の葛藤を丁寧に描く。ひなたの「夜伽帰り」の姿を千紗の目で捉える場面が、夜伽のリアリティを行為描写なしで暗示する技法。ひなたとの夜の会話——布団の中の親密さが少女たちの信頼関係を象徴。「おばあちゃんのすべてを受け止めなさい」が、仲居の仕事から夜伽へと意味を拡張する瞬間。祖母もかつて同じ道を歩んだのではないか——この推察が千紗にとって最後の覚悟の引き金になる。覚悟を決めた後の「足の震え」が、千紗がまだ14歳の少女であることを忘れさせない
- **感情の流れ**: ひなたの姿への複雑な感情（憧れ・不安・共感） → 女将の言葉への動揺 → 仕送りの現実への焦り → ひなたとの対話で少し安心 → 祖母の教えの「すべて」の意味に気づく衝撃 → 眠れない夜の葛藤 → 覚悟の表明（震えながらも）→ 覚悟の後の足の震えと自分を奮い立たせる凛とした姿
- **文量目安**: 約3,500字

### シーン6：明日の客（藤堂悠一を迎える朝）
- **場所**: 寮の露天風呂（朝湯）、花菱のミーティング室、千紗の寮室（着替え）、花菱の玄関
- **時間帯**: シーン5から数日後。10月中旬の朝。晴れ。紅葉が盛り
- **登場人物**: 千紗、女将・鶴見楓、ひなた、他の少女スタッフ
- **出来事**:

- **感情の流れ**: 朝湯の静謐と自己確認 → ミーティングでの緊張（初めての男性一人客・夜伽の可能性） → ひなたの手の温もりへの感謝 → 着付けの集中と覚悟 → 打ち水の一つ一つに込める「おもてなし」 → 正座で待つ間の静かな緊張 → バスの音と靴音で高まる鼓動 → 覚悟と不安が混じった一瞬

## EP0固有の描写方針

### 文体・視点

- **視点**: 千紗視点の三人称。千紗の内面に深く入り込むが、「私」は使わない。千紗の思考・感覚をそのまま地の文に溶かす
- **語り口**: 丁寧で少し古風。祖母の影響が文体にも滲む。「でございます」調が千紗の思考にも混じる
- **テンポ**: EP1より静かでゆったり。行為シーンがないぶん、日常の描写と内面の揺れに紙幅を割く

### 通奏低音としての祖母の教え

| 教え | 登場シーン | 意味の変化 |
|------|-----------|-----------|
| 「一期一会でございます」 | シーン1（原点）→ シーン5（覚悟）→ シーン6（出迎え） | 祖母の人生訓 → 夜伽を含む全人格的なおもてなしの覚悟 → 悠一との出会いへの祈り |
| 「お客さまの体を流すのは、心を流すのと同じ」 | シーン1（教え）→ シーン4（実践） | 入浴介助の心得 → 千紗の仲居としてのアイデンティティ |
| 「お客さまのすべてを受け止めなさい」 | シーン5（再解釈） | 仲居の心得 → 夜伽を含む「すべて」への意味拡張 |

### モチーフ：山百合

- シーン1: 裏山に咲く山百合。祖母が「千紗は山百合に似ている」と言う
