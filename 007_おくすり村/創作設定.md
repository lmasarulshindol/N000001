## 世界観

### 時代・舞台

- **時代**: 現代日本（夏休み前〜夏休み期間がメイン）
- **国**: 日本
- **舞台**: 最寄りの町まで車で2時間かかる山間部の集落。携帯の電波も一部不安定。

### 「おくすり村」の秘密

- **水源汚染**: 数十年前、かつての製薬会社が不法投棄した未知の実験薬品が地下水脈に溶け出している。
- **水の影響**: 
  - **女性**: 第二次性徴の著しい早熟化（8〜9歳で初潮・胸の発育など）。そして**慢性的な性的興奮（発情状態）**が引き起こされる。
  - **男性**: 精力増強と回復力の向上。
- **村の認識**: 村人はこれを「うちの村の水は特別」「こういう体質になる水」として半ば伝統のように受け入れている。
- **隠蔽体質**: 県の保健所などが水質調査を試みたこともあるが、「うちの水は綺麗だ」「余計な検査は不要」と村ぐるみで拒否している。

### 村の「常識」とルール

| 項目 | 内容 |
|------|------|
| 発情への対処 | 「体が熱くなったら誰かに抜いてもらう」のが当たり前。風邪をひいたら薬を飲むのと同じ感覚の「体調管理」。 |
| 羞恥心の欠如 | 誰と誰がした、どこでした、ということを隠さない。日常会話として「今日熱かったから〇〇くんに手伝ってもらった」と話す。 |
| 学校・保健室 | 学校の保健室は「処置室」の役割を兼ねている。体調不良（発情）を訴えれば、当番の男子生徒や保健医が処置を行う。 |
| 避妊と妊娠 | 村の診療所が低用量ピルを無償配布。基本は服用するが、「子供は村の宝」という価値観が強いため、妊娠を望む子（中学生など）には止めない。 |
| 場所の自由度 | 納屋の藁の上、川原の木陰、縁側の昼下がり、学校の空き教室など、どこでも行為が行われる。 |

## エロス描写の方向性

### 1. 田舎の風景 × エロス
- 蝉の声、風鈴の音、川のせせらぎ、強い日差しと濃い緑。
- のどかで美しい日本の原風景の中で、堂々と行われる性行為のギャップを描く。
- 汗ばんだ肌、麦わら帽子、浴衣、スクール水着、縁側での昼寝などの要素を多用する。

### 2. 「体調管理」としての受容
- 背徳感や罪悪感で悩むのは最初だけ。「熱い」「苦しい」という身体的症状から解放される「気持ちよさ」「安心感」を強調する。
- 行為中も「いやらしいこと」をしているというより、「お互いに助け合っている」「気持ちよくしてあげている」という感覚。
- 声の描写も「あえぎ声」だけでなく、「はぁ、楽になった……」「冷たくて気持ちいい」といった生理的な安堵感を含ませる。

### 3. バリエーション
- **少女同士（百合）**: ヒナとの夜の「手伝い」。指や舌を使ったスキンシップ。安心感と親密さ。
- **保健室・教室**: 学校という日常空間での処置。同級生や上級生との行為。
- **野外**: 川遊びの途中、神社の裏手など、自然の中での解放的な行為。
- **複数人**: 誰とでも行為をするため、複数人での「手伝い」も自然に発生する。

### 4. 結衣の変化（三段階）
- **第1段階（拒絶と戸惑い）**: 「おかしい」「こんなの嫌だ」。でも体は熱くなる。
- **第2段階（妥協と依存）**: 「苦しいから仕方なく手伝ってもらう」。ヒナや特定の相手への依存。
- **第3段階（完全な受容）**: 「気持ちいいからしてほしい」。自ら相手を求め、村の常識に染まりきる。

## 展開イメージ（全6話想定）

| 話数 | 概要 |
|------|------|
| 第1話 | 引っ越し初日〜数日。ヒナとの同居生活。村の異質さに気づく。ヒナの発情を手伝い、自らも水を飲む。 |
| 第2話 | 結衣の体に変化が訪れる。耐えきれず、保健室で初めての「処置」を受ける（男子生徒相手）。 |
| 第3話 | 村の日常に溶け込み始める。川遊びや縁側で、同級生たちと自然に行為に及ぶようになる。 |
| 第4話 | 母親の秘密を知る。母もまた村の男と交わっているのを目撃し、親子で村の常識を受け入れる。 |
| 第5話 | 夏祭りの夜。非日常の空間で、複数人での行為や、より濃厚な奉仕を経験する。 |
| 第6話 | 夏休み終盤。完全に村の少女となった結衣。都会へ帰るという選択肢は消え、ここで生きていく決意と快楽。 |
