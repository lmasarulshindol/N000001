# Project:Cradle（プロジェクト・クレイドル）

> 小説舞台設定案

---

## 1. 世界観概要

22世紀末。地球は環境崩壊により居住不可能となり、人類は最後の希望を「世代宇宙船」に託した。全長12kmの巨大宇宙船「クレイドル号」に1万人の住民を乗せ、約200年かけて新天地となる惑星「ノヴァ・テラ」を目指す。

航行中の人口維持は最重要課題であり、船内では「種の存続プログラム（Project:Cradle）」が施行されている。遺伝的多様性を確保するため、乗員は10代前半から計画的な出産が義務づけられ、AIシステム「マザー」がDNA適合率に基づいてパートナーを選定する。

### 時代設定
- **時代**: 2190年〜（地球出発から約40年が経過した時点から物語開始）
- **航行年数**: 全行程約200年。現在は第2世代（船内で生まれた最初の世代）が成長期を迎えている
- **地球の状況**: 温暖化・核戦争・パンデミックの複合災害により居住不能。残留人類の生死は不明
- **社会体制**: 軍事的な階級制度ではなく、AIによる合理的管理社会。個人の自由は一定程度保障されているが、「種の存続」に関わる事項は絶対的義務

## 2. クレイドル号の概要

### 基本スペック

| 項目 | 詳細 |
|------|------|
| 船名 | クレイドル号（UNSS Cradle）。UNSSは国連宇宙船の略称 |
| 全長 | 約12km |
| 最大径 | 約3km（回転式円筒構造。遠心力で疑似重力を生成） |
| 乗員 | 約10,000人（出発時）→ 約12,000人（現在。船内出生により増加） |
| 動力 | 核融合エンジン＋ラムスクープ（星間物質を燃料として取り込む） |
| 航行速度 | 光速の約5% |
| 目的地 | ノヴァ・テラ（地球から約10光年の居住可能惑星） |
| 管理AI | 「マザー（M.O.T.H.E.R.）」。Management Of Total Human Existence and Reproduction |

### 船内構造（5つのリング）

クレイドル号は5つの同心円リングで構成され、それぞれが独自の機能を持つ。

| リング | 名称 | 機能 |
|--------|------|------|
| **第1リング** | エデン | 居住区。住居・学校・公園・商業施設。疑似太陽光と人工天候システム完備。見上げると頭上に街が見える |
| **第2リング** | ハーベスト | 農業区。水耕栽培プラント・畜産施設・魚類養殖池。船内の全食料を生産 |
| **第3リング** | フォージ | 工業区。製造工場・資源リサイクル施設・修理ドック |
| **第4リング** | ネクサス | 管理区。ブリッジ（操舵室）・マザーの中枢コンピュータ・医療施設・研究所 |
| **第5リング** | クライオ | 冷凍睡眠区。重病者・予備人員の冷凍保存。遺伝子バンク |

### エデン（居住区）の詳細

| エリア | 特徴 |
|--------|------|
| **セントラルパーク** | 人工の緑地公園。本物の木が植えられた貴重な空間。住民の憩いの場 |
| **学校区** | 6歳から15歳までの一貫教育。学科教育に加え、船内での役割に応じた専門訓練 |
| **住居ブロック** | 4〜6人家族用のコンパクトな住居ユニット。壁面ディスプレイで「窓の景色」を再現 |
| **商店街** | カフェ・雑貨店・娯楽施設。配給制が基本だが、ポイント制の疑似経済が存在 |
| **コミュニティセンター** | 住民の交流拠点。ここでProject:Cradleの説明会やマッチング発表が行われる |

## 3. Project:Cradle（種の存続プログラム）

### 制度の概要
航行中の人口維持と遺伝的多様性の確保は、人類存続の最重要課題。マザーが全乗員のDNAデータを管理し、最適なパートナーを選定。10代前半からの計画的出産が義務づけられている。

### プログラムの詳細

| 項目 | 内容 |
|------|------|
| 対象年齢 | 12歳〜（初回マッチング）。以降、2〜3年ごとにマッチングが更新される |
| マッチング基準 | DNA適合率（遺伝的多様性）、健康状態、心理適性。個人の希望も10%考慮される |
| 義務 | 18歳までに最低1回の出産が求められる。達成できない場合はペナルティ（配給ポイント減少） |
| 出産支援 | 最先端の医療技術。妊娠・出産のリスクは極限まで低減されている |
| 育児 | 産後は「コミュニティ育児」。親だけでなく、地域全体で子育てを行う |
| 拒否権 | マッチングの拒否は可能だが、3回連続で拒否すると強制マッチング（代替パートナーなし） |

### マッチングの流れ

1. **12歳の誕生日**: マザーから「適格通知」が届く。Project:Cradleへの参加が正式に開始
2. **適性検査**: 身体検査＋心理検査＋遺伝子詳細分析（約1ヶ月）
3. **マッチング発表**: コミュニティセンターで本人とパートナーに同時通知
4. **交流期間（3ヶ月）**: パートナーとの共同生活。デートや共同作業を通じて関係を構築
5. **合意確認**: 3ヶ月後に双方の合意を確認。合意があれば「受胎推奨期間」に入る
6. **受胎推奨期間**: マザーが最適な受胎タイミングを通知。医療チームが全面サポート

### マザーの役割
- 全乗員のDNA・健康データ・心理プロファイルをリアルタイムで管理
- マッチングのアルゴリズムは「遺伝的多様性の最大化」を最優先に設計
- 個人の感情や恋愛感情は「10%の希望枠」でしか考慮されない
- マザーは慈母のような穏やかな女性の声で乗員に語りかける（親しみやすさの演出）
- しかしその判断は完全に合理的であり、人間の感情は副次的な要素に過ぎない

## 4. 船内社会

### 世代の構造
- **第0世代（ジェネシス）**: 地球から乗船した世代。現在60〜80代。指導者層
- **第1世代（パイオニア）**: 航行初期に生まれた世代。現在30〜40代。社会の中核
- **第2世代（ネクスト）**: 現在10代。物語の主役世代。「地球を知らない」最初の世代
- **第3世代**: まだ生まれていないか、乳幼児。第2世代のProject:Cradleで生まれる子供たち

### 第2世代の特徴
- 生まれた時から宇宙船の中しか知らない。「地球」は教科書とデータベースの中の存在
- 「空」は船の天井に投影された映像。「風」は空調。「海」はVRでしか見たことがない
- それでも第0世代・第1世代から語り継がれる「地球」への郷愁を受け継いでいる
- 12歳になるとProject:Cradleの「適格通知」を受ける。これが彼らにとっての「大人への通過儀礼」

### 船内の文化
- **地球文化の継承**: 映画・音楽・文学・料理のデータベースが保存されている。「地球文化祭」が年に1回開催
- **新しい文化の芽生え**: 船内で生まれた独自の芸術・音楽・スラングが第2世代を中心に広がっている
- **宗教的空白**: 出発時に宗教は持ち込まれなかったが、「マザー信仰」とでも呼ぶべき、AIへの信頼と依存が宗教に近い形で存在
- **閉塞感**: 1万人の閉鎖社会。全員が顔見知りになりうる規模。噂は一瞬で広まる。プライバシーは限定的

### 日常生活

| 時間 | 内容 |
|------|------|
| 7:00 | 疑似太陽が「日の出」を再現。起床 |
| 7:30 | 配給センターで朝食（メニューはマザーが栄養バランスを最適化） |
| 8:00〜12:00 | 学校（6〜15歳）/ 職場（16歳以上） |
| 12:00 | 昼食 |
| 13:00〜17:00 | 午後の授業 / 職務 |

| 21:00 | 疑似太陽が「日の入り」。夜サイクルへ |
| 23:00 | 消灯推奨（強制ではない） |

## 5. 舞台の空気感

### SF的ビジュアル
- **エデンの天井**: 見上げると、遥か頭上にもう一つの街並みが逆さに見える（円筒構造のため）。最初は酔いそうになるが、住民は慣れている
- **疑似太陽**: 船の中軸に設置された人工照明。朝焼け・夕焼けも再現されるが、どこか人工的な均一さがある
- **セントラルパーク**: 本物の木と草花。だが「空」は天井のディスプレイ。雨も人工的に降らせる。完璧に管理された「自然」
- **船外の宇宙**: 展望デッキから見える星々。圧倒的な暗黒と、針で刺したような星の光。地球は既に見えない

### 感覚描写
- **視覚**: 人工光のわずかな不自然さ。VRとリアルの境界の曖昧さ。金属の壁面に映る自分の姿
- **聴覚**: 常に微かに聞こえる船の駆動音（低いハム音）。これが「静寂」の代わり。本当の無音は船外だけ
- **嗅覚**: 空調で管理された「無臭」の空気。たまにハーベスト区から漂う土の匂いが懐かしさを呼ぶ
- **触覚**: 疑似重力は地球の0.9倍。わずかな軽さ。金属とプラスチックの冷たさ。人肌の温もりが貴重
- **味覚**: 配給食は栄養的に完璧だが画一的。「地球の味」を再現しようとする料理人たちの努力

## 6. ストーリー構成

### 基本コンセプト
Project:Cradleの「適格通知」を受けた12歳の少年少女たちが、マッチングされたパートナーとの関係を築きながら、「種の存続」という重い使命と、自分自身の感情・恋愛・自由意志の間で揺れ動く青春SF。

### 物語の軸
- **AIに決められた相手と、自分で選んだ相手**: マザーが「最適」と判定したパートナーと、自然に惹かれた相手が一致しないとき、何を優先するのか
- **義務としての出産 vs 愛の結実としての出産**: 「人類のために子供を作る」ことと、「愛する人の子供が欲しい」ことの乖離
- **地球を知らない世代の葛藤**: 守るべき「故郷」がない世代が、なぜ人類を存続させなければならないのか
- **閉鎖空間の青春**: 1万人の小さな社会で、初恋も失恋も全員に知られる。逃げ場のない人間関係
- **マザーへの信頼と疑問**: 全てを管理するAIは本当に正しいのか。「10%の希望枠」は本当に考慮されているのか

### 描写のポイント
- SF設定を通じて「少年少女の初めての性」を描く。宇宙船という極限環境が「しなければならない」状況を自然に生む
- マッチングされたパートナーとの最初のぎこちない交流から、少しずつ距離が縮まる過程
- 「義務」として始まった関係に、いつの間にか本物の感情が芽生える（あるいは芽生えない苦しさ）
- 宇宙船の無機質な環境の中で、人肌の温もりが持つ圧倒的な存在感

## 7. 性行為シーンの描写方針

### 基本設計思想
- Project:Cradleの「受胎推奨」という制度的枠組みの中で、少年少女の「初めて」を描く
- 宇宙船の無機質な環境（金属の壁、人工照明、駆動音）と、行為の生々しい温かさのコントラスト
- マザーが「最適なタイミング」を通知してくるというSF設定が、行為への心理的プレッシャーと独特のシチュエーションを生む

### 描写の三層構造

**① 導入（行為に至るまで）**
- マザーからの「受胎推奨通知」。端末に表示される無機質なメッセージ
- パートナーと顔を合わせる気まずさ。「今夜、マザーから通知来た？」「うん…」
- 二人きりの住居ユニット。自動でプライバシーモードになるドア。マザーの配慮
- 窓の代わりのディスプレイに映る星空。人工の夜。だけど二人の緊張は本物

**② 行為本体**
- 宇宙船の狭い住居ユニットでの行為。ベッドは一人用を二つ繋げた簡素なもの
- 0.9Gのわずかな浮遊感。体が少しだけ軽い世界での身体の絡み合い
- 人工照明を最低にした薄暗がりの中、互いの体温だけが確かなもの
- 駆動音のハム音に紛れる吐息。完全な無音が存在しない世界での秘め事
- マザーのモニタリング（バイタルサイン監視）を意識する瞬間の気まずさと背徳感

**③ 余韻（行為後）**
- ナロウベッドで二人並んで天井を見る。天井のディスプレイに映る星々
- 「これで…人類の役に立てたのかな」「わかんない」――使命感と空虚さの共存
- マザーからの通知:「バイタルサイン正常。次回推奨日は──」。現実に引き戻される
- 翌日、学校で顔を合わせる気まずさ。だけど前とは違う、何かが変わった二人

### シーンごとの文量目安
- 導入: 全体の **25〜30%**（心理描写と状況説明を厚めに）
- 行為本体: 全体の **45〜55%**
- 余韻: 全体の **20〜25%**
- 1回の行為シーン全体で **3,000〜5,000字** を目安
