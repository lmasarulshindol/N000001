# EP0「銀色の髪の少女」詳細プロット

> 12歳の誕生日を迎えようとしている桐谷凛花の視点から、船内の日常、亡き父への想い、適格通知の受信、適性検査、マッチング発表、交流期間、そして受胎推奨通知までを描く。蒼真視点のEP1と同じ出来事を、凛花の内面から追体験する前日譚。「大丈夫」という言葉が嘘の鎧から、共有された弱さを経て、新しい意味を獲得するまでの物語。温室の花が土に根を張るように、閉じた心がひとつだけ開く扉を見つける第0話。

## シーン構成

### シーン1：温室の少女

**場所**: ハーベスト区・植物温室
**時間帯**: 放課後（17:00〜18:30頃）。疑似太陽が西日に傾く時間帯
**登場人物**: 凛花、温室管理の老人（第0世代・名前なし／セリフ1〜2行程度）
**文量目安**: 約2,500〜3,000字

#### 出来事
- 放課後、凛花がハーベスト区の植物温室にやって来る。ここが彼女の日課であり、聖域
- 土に触れ、苗に水をやり、葉の状態を確認する。手つきは慣れている。誰に教わったのか——父だ
- 回想: 6歳の頃、父・桐谷裕也が凛花をこの温室に連れてきた記憶。「ここだけ、ちょっとだけ地球っぽい気がするだろ？」と笑っていた父の声。土の匂い、緑の匂い、湿った空気——船内のどこよりも「生きている」場所
- 父は外壁補修作業中の事故で死んだ。凛花が6歳の時。船内では珍しい「事故死」。マザーの管理下で人が死ぬことの異常さを、凛花は幼いながらに刻まれた
- 母・沙耶のこと。医療部門に勤める母は優しいが、父の死後どこか空虚だ。笑顔の奥に常に翳りがある。マザーからの再マッチング勧告を47回拒否し続けていることを、凛花は知っている——端末の通知を偶然見てしまったことがある
- 母は凛花に「お父さんの話」をほとんどしない。聞けば答えるが、自分からは語らない。凛花にとって、父の記憶はこの温室に凝縮されている
- 温室管理の老人（第0世代）がすれ違いざまに「今日も来たのかい」と声をかける。凛花は明るく「はい！」と返す。しっかり者の笑顔。だが老人が去ると、表情が少し緩む。ここでは仮面を外せる
- トマトの苗に話しかける凛花。「もうすぐ12歳だよ。適格通知、来ちゃうね」。苗は答えない。駆動音のハムだけが響く
- 温室の窓（ガラスパネル）越しに、ハーベスト区の水耕栽培プラントの無機質なパイプラインが見える。「自然」と「人工」の境界線。凛花はいつもこの景色を見て、自分がどちら側の人間なのか考える

#### 描写のポイント
- 温室の五感描写を徹底する。土の匂い、湿度の高い空気、植物の緑、水滴の音、トマトの苗の手触り——エデンの住居区とは明確に異なる「生きた空間」
- 凛花の「外面」と「内面」のギャップを最初のシーンから明確にする。老人への明るい返事と、一人になった時の静かな表情
- 父の記憶は温かいが、同時に痛い。懐かしさと喪失が同居する
- 母の再マッチング拒否47回という数字の重さを、凛花がどう受け止めているか。「お母さんはお父さんを愛していた」という確信と、「自分はマザーの言う通りに誰かと結ばれるのか」という不安の萌芽
- 「適格通知」への意識をさりげなく提示。凛花にとって12歳の誕生日は喜びではなく、覚悟の期限

#### 感情の流れ
- 温室に入った瞬間の安堵（「帰ってきた」感覚） → 植物の世話をしながらの穏やかさ → 父の記憶への回想（温かさと痛みの同居） → 母のことを考える時のやるせなさ → 老人との会話での「いつもの自分」 → 一人に戻った時の静けさ → 適格通知への意識がよぎる小さな不安 → 駆動音のハムを聴きながら、「大丈夫」と自分に言い聞かせる

### シーン2：適格通知

**場所**: エデン・住居ブロック／桐谷家の住居ユニット → ネクサス（管理区）・医療施設内の検査室
**時間帯**: 12歳の誕生日の朝（7:00）〜適性検査日（翌週の午前中）
**登場人物**: 凛花、母・桐谷沙耶、医療スタッフ（名前なし）、マザーの声
**文量目安**: 約3,000〜3,500字

#### 出来事
- 12歳の誕生日の朝。凛花の誕生日は蒼真より3ヶ月早い
- 母が朝食を用意してくれている。配給ポイントで手に入れた合成果物のジュースと、少しだけ豪華な朝食。二人暮らしの小さな祝い
- 母が「お誕生日おめでとう、凛花」と笑う。その笑顔はいつもより少しだけぎこちない。12歳の誕生日が何を意味するか、母は知っている
- 朝食中、凛花の端末にマザーからの通知。柔らかな女性の声。「おめでとうございます、桐谷凛花さん。本日より、あなたはProject:Cradleの適格者となりました」
- 母の手が一瞬止まる。ジュースのグラスを持ったまま。だがすぐに笑顔を作る。「凛花なら大丈夫よ」
- 凛花も笑い返す。「大丈夫だよ、お母さん」。二人の「大丈夫」が重なる。二人とも嘘をついている。凛花はそれに気づいている
- 凛花の内面: 母が47回の再マッチング勧告を拒否し続けていること。父を愛していたから。では自分はどうなるのか——マザーに選ばれた誰かを、愛せるのか？　母のように？
- 端末に「適性検査のご案内」が表示される。身体検査・遺伝子詳細分析・心理検査

**── 場面転換 ──**

- 翌週。ネクサスの医療施設。白い壁、金属の検査台、淡い青色のライト。エデンの温もりとは対照的な冷たさ
- 身体検査。全裸になる検査着に着替える。マザーのカメラが見えない場所から見ている気配。凛花は制服を脱ぎながら、自分の体を見る。まだ子供の体。これで「母親になれ」と言われるのか
- 遺伝子分析。採血。凛花のDNAの中に、この船のどこかの男の子と「最適」になる何かが刻まれている。それが怖いのか、安心なのか、わからない
- 心理検査。マザーが直接質問する。「桐谷凛花さん。あなたにとって"家族"とは何ですか？」
- 凛花は一瞬、父の顔を思い浮かべる。温室で笑う父の顔。次に母の横顔。夜、端末を見つめている母の背中。再マッチング勧告を閉じる、その指先
- 凛花の回答: 「……守りたいもの、です」。自分でも意外な答えだった
- マザーの次の質問。「あなたは将来、パートナーを愛せると思いますか？」。凛花は3秒間沈黙してから答える。「わかりません。でも、大丈夫です」——また、「大丈夫」
- 検査終了後の廊下で、同い年の少年少女とすれ違う。みんな同じ検査を受けに来ている。目が合うと互いに気まずそうに逸らす。凛花だけは「しっかり者」のスイッチを入れて小さく微笑む。知り合いの女子に「大丈夫だった？」と声をかけられ、「うん、大丈夫だよ！」と明るく返す

#### 描写のポイント
- 母と凛花の二人きりの朝食。父の不在が三人目の椅子として存在する
- 二重の「大丈夫」——母の「大丈夫」と凛花の「大丈夫」が鏡合わせ。互いの嘘を見抜いているが、指摘しない優しさ（あるいは臆病さ）
- ネクサスの検査室の無機質さ。エデンの「人工的な温かさ」すらない、純粋な「機能」の空間
- マザーの質問が凛花の核心を突く。「家族」と「愛」——父を失い、母の拒否を知る凛花にとって、これ以上に重い質問はない
- 「守りたいもの」という回答に込められた、凛花自身も自覚していない想い
- 検査後の廊下での「しっかり者スイッチ」。仮面の着脱をシーン内で見せる

#### 感情の流れ
- 誕生日の朝の複雑な感情（嬉しさと怖さの混在） → 母との「大丈夫」の交換（温かさと切なさ） → マザーの通知を聞いた時の胃の底が冷たくなる感覚 → 検査室での緊張と屈辱（品定めされている感覚） → マザーの質問に父を想い出す痛み → 「大丈夫です」と答える時の空虚さ → 廊下で仮面をつけ直す。「しっかり者の桐谷凛花」を演じる疲労感

### シーン3：マッチング発表——「藤堂蒼真」

**場所**: エデン・コミュニティセンター大ホール
**時間帯**: 適性検査から約1ヶ月後。放課後（16:00頃）
**登場人物**: 凛花、蒼真、同世代の少年少女たち（数十名）、進行役の大人、マザーの声
**文量目安**: 約3,000〜3,500字

#### 出来事
- コミュニティセンター大ホール。適格者たちが緊張した面持ちで席についている。凛花は友人の女子数人と固まって座っている。外面は平静。「ドキドキするね〜」と笑っているが、手は膝の上で握りしめている
- 周囲の会話。「適合率高い方がいいよね？」「誰が来るんだろう」「イケメンだといいな」。凛花は相槌を打ちながら、内心では「誰と組まされるんだろう」という不安を噛みしめている
- 「組まされる」——自分の中でその言葉が出たことに凛花は気づく。「選ばれる」ではなく「組まされる」。母が47回拒否したものに、自分はこれから従う
- マザーの声で一組ずつマッチング結果が読み上げられていく。名前を呼ばれたペアが前に出る。歓声を上げる子、泣き出す子、無表情の子。凛花は一組ごとに心臓が跳ねる
- 「桐谷凛花さん」——名前を呼ばれた瞬間、周囲の音が遠くなる
- 「あなたのパートナーは——藤堂蒼真くんです。DNA適合率、92.7パーセント」
- 凛花が蒼真を探す。ホールの端の方。暗い藍色の髪の少年。一人で座っている。表情が読めない。——「あ、あの子か」。いつも一人でいる静かな男子。クラスでほとんど話したことがない。接点は皆無に近い
- 92.7パーセント。高いのか低いのか、凛花にはわからない。ただ、数字で自分の運命が決められた感覚がある
- 指示に従い、蒼真の隣に座る。互いの顔をまともに見られない。蒼真は前を向いたまま微動だにしない。凛花は盗み見る——灰がかった青い目。曇り空みたいな色。何を考えているのか、まったくわからない
- 交流期間の説明を受ける。週3回の交流日。3ヶ月間。その間に互いを知り、合意確認に至る。凛花は説明を聞きながら、「3ヶ月でこの人を知れるのか」と考える
- 発表後、蒼真が一人で帰ろうとする。凛花は友人たちに「ちょっと待ってて」と言って蒼真を追いかける。声をかける。「あの、藤堂くん……よろしくお願いします」
- 声は明るくしたつもりだった。でもわずかに震えていたかもしれない。蒼真は振り返る。一瞬の沈黙。「……よろしく」。それだけ。蒼真は去っていく
- 凛花の内面: 蒼真への第一印象は「暗そう」「何を考えているかわからない」「愛想がない」。でも——「自分だって本当の自分を見せていないのだから、この人もそうかもしれない」と思う。仮面を被っている者は、他人の仮面に敏感だ

#### 描写のポイント
- 大ホールの描写。普段はイベント会場として賑わう空間が、独特の緊張感に包まれている。人工照明がいつもより明るく感じる
- 凛花の「グループの中にいる自分」を描写。友人たちとの会話に混ざりながら、内面では孤独
- マザーの声がホールに響く音響描写。天井から降ってくるような声。名前を呼ばれた瞬間の心臓の音
- 蒼真の第一印象を凛花の目を通して描く。暗い藍色の髪、灰がかった青い目、表情の薄さ。EP1で蒼真が凛花を見た描写と対になる
- 「よろしくお願いします」を言いに行く凛花の行動力。これは「しっかり者」の仮面か、それとも凛花自身の意志か——凛花自身にもわからない
- 蒼真の「……よろしく」の素っ気なさと、凛花の微妙な落胆

#### 感情の流れ
- 発表前の不安を友人との会話で紛らわせる（仮面） → 名前を呼ばれた瞬間の空白（聴覚が遠のく感覚） → 92.7%という数字を突きつけられる → 蒼真を見た時の「あの子？」という戸惑い → 隣に座った時の気まずさ（金属の椅子の冷たさ、蒼真との物理的距離の近さ） → 「よろしくお願いします」を言いに行く時の小さな覚悟 → 蒼真の素っ気ない返事に「やっぱり暗い」と思いつつも、「でも自分もそうだ」と気づくかすかな共感

### シーン4：交流期間——変わっていく距離（凛花視点）

**場所**: エデン・セントラルパーク → 商店街のカフェ → ハーベスト区の植物温室
**時間帯**: マッチング発表後の2〜3週間（複数日のダイジェスト）
**登場人物**: 凛花、蒼真
**文量目安**: 約3,500〜4,500字

#### 出来事

**第1回（セントラルパーク）:**
- マザーの指示で最初の交流日。セントラルパークのベンチ。凛花は少し早めに来て待っている（しっかり者）
- 蒼真がやって来る。「……」「……」。沈黙。凛花が話題を振る。「好きな食べ物とかある？」「……別に」
- 会話が壊滅的に続かない。凛花は内心焦る。いつもなら誰とでも話せるのに。蒼真の寡黙さが凛花の社交スキルを無力化する
- 蒼真がぽつりと星の話を始める。「あの天井のディスプレイ、デネブの位置が0.3度ずれてる」。凛花は「へえ」としか返せない
- だが、凛花は気づく。蒼真の目が星の話をする時だけ、灰色から澄んだ青に変わること。曇り空が晴れるように。「この人、星が本当に好きなんだ」
- 帰り道、凛花は母に「どうだった？」と聞かれ「大丈夫だよ、普通の人だった」と答える。嘘ではない。でも「普通」でもない気がした

**第3回（商店街のカフェ）:**
- カフェで向かい合って座る。前回・前々回より少しだけ自然に話せるようになった
- 凛花がカフェのメニューについて語り出す。「ここのハーブティー、温室のミントを使ってるんだよ。私が世話してる子」。蒼真の目が少し動く
- 蒼真が「世話してるって、植物の？」と聞く。「へえ」以外の反応。凛花は小さく嬉しくなる
- 凛花はハーブティーの話から温室の話へ。自分でも気づかないうちに饒舌になっている。蒼真は黙って聞いている——だが、退屈そうではない
- 凛花の内面: 「あれ、私、この人にしか話してないことを話してる」。温室のことを友人に話したことはあるが、こんなに熱く語ったのは初めてだ

**第5回（ハーベスト区の植物温室）:**
- 凛花が決心して蒼真を温室に連れていく。自分の聖域に人を入れるのは初めて。緊張する
- 温室の扉を開けた瞬間、蒼真が足を止める。「……何、この匂い」。凛花は「本物の土の匂い」と答える
- 蒼真が息を呑んでいる。灰がかった青い目が見開かれて、温室の緑を映している。凛花はその表情に驚く——星の話をしている時とは違う、もっと無防備な表情
- 凛花がいつものように植物の世話を始める。蒼真に説明しながら。「この子はトマト。もうすぐ実がなるの」「こっちはバジル。ちょっと元気ないけど」
- 水やりの途中、凛花の髪ゴムが切れる。低い位置で結んでいた銀灰色の長髪がばさりと肩から背中に落ちる
- 凛花は「あ、ゴム切れた」と手で髪を押さえようとする。振り返ると、蒼真が一瞬——ほんの一瞬、見とれている。目が少し見開かれて、灰色が青に変わって、凛花の髪を見ている
- 凛花はドキッとする。心臓が跳ねる。なぜ跳ねたのか自分でもわからない。「ど、どうしたの？」「……いや、別に。切れたなら、これ使えば」蒼真が自分のリストバンドを差し出す。凛花は「えっ、いいの？」と受け取り、不器用に髪をまとめる。蒼真のリストバンドが髪に触れている。それだけのことなのに、耳が熱い
- 帰り道。ハーベスト区からエデンに戻る通路。蒼真が言う。「桐谷さんは……あの温室が好きなんだな」
- 凛花は答える。いつもの明るい声ではなく、静かな声で。「……うん。お父さんが連れてきてくれた場所だから」
- 言ってしまった後に気づく。父のことを、この人に話した。初めての「本音」。蒼真は何も聞き返さなかった。ただ「そうか」と言って、少し歩幅を緩めた。並んで歩く。駆動音のハムが通路に反響している

#### 描写のポイント
- 交流回ごとの距離感の変化を、物理的な距離で表現。第1回: ベンチの端と端 → 第3回: テーブル越しだが身を乗り出して話す → 第5回: 温室で自然に隣を歩く
- 凛花の視点から見た蒼真の変化。無表情の奥にある感受性を、凛花が少しずつ発見していく過程
- 温室シーンの五感描写。土の匂い、植物の緑、水の音、湿度の高い空気。ここは凛花が「本当の自分」でいられる唯一の場所——そこに蒼真を招き入れる意味
- 髪ゴムが切れるシーンの映像的な美しさ。銀灰色の長髪が緑の温室の中で広がる。蒼真の視線。凛花の動揺
- 「お父さんが連れてきてくれた場所」——凛花にとって最も大切な記憶を、ほぼ無意識に蒼真と共有した瞬間
- 蒼真の「そうか」の一言が、凛花にとってどれほど心地よかったか。根掘り葉掘り聞かず、ただ受け止める蒼真の寡黙さが、初めて「優しさ」に見える

#### 感情の流れ
- 第1回: 義務感と気まずさ。社交スキルが通じない焦り → 蒼真の「星の目」への小さな発見 / 第3回: 温室の話をする時の自然体。「あれ、私、素に近い」という気づき / 第5回: 聖域に人を入れる緊張 → 蒼真の反応への驚きと嬉しさ → 髪ゴムのシーンでの動揺（理由がわからないドキドキ） → 父のことを話してしまった後の「やってしまった」感と、同時にどこか楽になった感覚 → 並んで歩く帰り道の静かな充足。「嫌じゃない。この人と歩くのは、嫌じゃない」

### シーン5：受胎推奨通知

**場所**: エデン・住居ブロック／桐谷家の住居ユニット（凛花の部屋）
**時間帯**: 交流期間3ヶ月目の夜（21:00頃）
**登場人物**: 凛花、蒼真（端末越しの通話）、マザーの声（通知）
**文量目安**: 約2,500〜3,000字

#### 出来事
- 交流期間の合意確認が完了し、「受胎推奨期間」に入った翌日の夜
- 凛花は自室のベッドの上で端末を見ていた。天井のディスプレイには星空が映っている——蒼真が教えてくれた星座の位置を、最近は確認するようになった自分に気づく
- マザーからの通知音。端末の画面が切り替わる。柔らかな女性の声。「桐谷凛花さん。明後日が最適な受胎推奨日です。住居ユニットB-7が確保されています。詳細は端末をご確認ください」
- 凛花は端末を見つめたまま固まる。わかっていた。適格通知を受けた日から、この瞬間が来ることは。でも——「明後日」。具体的な日付が目の前にある
- 端末の画面を見つめる。「住居ユニットB-7」「受胎推奨日」「最適」。一文字一文字が目に刺さる。自分の体が、マザーのスケジュールに管理されている
- 母のことを考える。母はこの通知を48回受け取ったのだろうか——いや、最初の1回は受け入れた。父と。そしてその後の47回を拒否し続けた。凛花を産んだ時、母はどんな気持ちだったのだろう
- 蒼真からメッセージが届く。テキスト。「……通知、来た？」。たった5文字。凛花は端末を握りしめる
- 返信しようとする。指が動かない。「大丈夫だよ」と打とうとして、消す。「うん」と打って、消す。3分間、画面を見つめる
- 3分後、「うん。来たよ」と送る。それだけで精一杯
- 蒼真から「怖い？」。凛花は端末を見つめる。テキストでは嘘をつけてしまう。だから——音声通話に切り替える。コール音。蒼真が出る
- 凛花の声は平静を装っている。でも自分でわかる、微かに掠れている。「怖くないよ。大丈夫」。いつもの「大丈夫」。自動的に口から出る言葉
- 通話の向こうで、蒼真の呼吸が聞こえる。駆動音のハムが通話のノイズに混じる。蒼真が言う。「俺も……大丈夫じゃないかも」
- 凛花は息を呑む。「大丈夫じゃない」——その言葉を、誰かの口から聞いたのは初めてだ。みんないつも「大丈夫」と言う。母も、友人も、自分も。誰も「大丈夫じゃない」とは言わない。この船の中で「大丈夫じゃない」は存在しないことになっている
- なのに蒼真が言った。正直に。不器用に。凛花は目の奥が熱くなるのを感じる
- 「……なんだ、藤堂くんも同じなんだ」。小さく笑う。声が少し震えたかもしれない。でも不思議と、この震えは恥ずかしくない
- 蒼真が「……ああ」とだけ返す。沈黙。でもこの沈黙は痛くない。温室の帰り道みたいに、ただ二人でいる静けさ
- 凛花が「明後日……ね」と言う。蒼真が「ああ」と答える。それだけ。約束は成立する

#### 描写のポイント
- マザーの通知の無機質さと、凛花の体温の対比。「受胎推奨」という言葉が持つ冷たさ
- 凛花の部屋の描写。小さな部屋。母の気配が壁の向こうにある。母はこの通知のことを知っているのか。凛花は聞けない
- テキストメッセージの「沈黙の重さ」。3分間返信できなかった凛花の内面を丁寧に
- 音声通話に切り替えた理由——テキストでは嘘をつけてしまうから。文字で「大丈夫」と打つのは簡単。声で言うのは難しい。だからこそ声を選んだ。だがそれでも「大丈夫」と言ってしまう凛花の性（さが）
- 蒼真の「大丈夫じゃないかも」が凛花にとっての転換点。「大丈夫」という鎧を着続けてきた凛花に、初めて「鎧を脱いでもいい」と示した言葉
- 通話越しの駆動音のハムが二人を繋いでいる描写。同じ船の中の、別々の部屋で、同じ音を聞いている

#### 感情の流れ
- 通知を受けた瞬間の凍結（思考が止まる） → 母のことを考える（47回の拒否の重み） → 蒼真のメッセージに胸が締まる → 3分間返信できない葛藤 → 音声通話に切り替える小さな勇気 → 「大丈夫」と言ってしまう自分への諦め → 蒼真の「大丈夫じゃない」に衝撃を受ける → 目の奥が熱くなる → 小さく笑う（泣きたいのに笑ってしまう） → 沈黙の中の安堵。「この人となら、大丈夫じゃなくても大丈夫かもしれない」

### シーン6：端末の光

**場所**: エデン・住居ブロック／桐谷家の住居ユニット（凛花の部屋）
**時間帯**: シーン5の通話直後。深夜（22:00頃）
**登場人物**: 凛花（独り）
**文量目安**: 約1,500〜2,000字

#### 出来事
- 蒼真との通話を切った後。凛花はベッドの上で仰向けになっている
- 天井のディスプレイの星空を見つめる。デネブ。蒼真が「0.3度ずれてる」と言っていた星。凛花は笑う。あの日は何のことかわからなかったのに、今は星の名前を知っている
- 明後日、自分は蒼真と肌を重ねる。「種の存続」のために。マザーが決めた日に、マザーが用意した部屋で、マザーが最適と判定した相手と。——全部、マザーだ。自分の意志は、どこにあるのだろう
- 母のことを考える。母は父と出会った時、何を思ったのだろう。マザーに選ばれた相手を、いつ好きになったのだろう。それとも——最後まで「義務」だったのだろうか。いや、それはない。母は47回も拒否した。父を失った後に。あれは「愛」以外の何でもない
- では自分は？　蒼真を——好き、なのか？
- まだ「好き」ではない。たぶん。「好き」がどういうものなのか、凛花にはまだわからない
- でも。温室で蒼真が息を呑んだ顔を覚えている。銀灰色の髪に見とれた蒼真の青い目を覚えている。「大丈夫じゃないかも」と正直に言ってくれた声を覚えている。帰り道で歩幅を合わせてくれたことを覚えている
- 「嫌じゃない」。それは「好き」とは違うかもしれない。でも、この船の中で、マザーに管理された世界で、自分が「嫌じゃない」と思えること——それは、小さな奇跡なのかもしれない
- 端末の画面が暗くなる。オートスリープ。星空のディスプレイだけがぼんやりと光る。駆動音のハムが、凛花の鼓動と重なる
- 凛花は端末を両手で持ち上げ、胸に抱く。蒼真との通話の温もりがまだ残っている気がする。お守りのように
- 目を閉じる。瞼の裏に、温室の緑と、蒼真の青い目が浮かぶ
- 唇が動く。声にならない声で——

> 「大丈夫。……大丈夫じゃなくても、大丈夫」

- 嘘の「大丈夫」でも、強がりの「大丈夫」でもない。蒼真と共有した「大丈夫じゃない」を受け入れた上での、新しい「大丈夫」。完璧でなくていい。震えていてもいい。それでも、大丈夫
- 銀灰色の髪が枕に広がる。星空のディスプレイの光が、その髪を淡く照らしている。駆動音のハムが子守唄のように低く響く
- 凛花は眠りに落ちる。明後日の夜を、夢に見るかもしれない

#### 描写のポイント
- シーン1の温室から始まった物語が、凛花の部屋で閉じる。「外の聖域（温室）」から「内面の安堵（自室）」への帰還
- 天井の星空は蒼真との接点。最初は意味のなかった星の名前が、今は蒼真の声と結びついている——交流期間で二人が積み上げたものの象徴
- 「大丈夫」の三段変化をこのシーンで完結させる。①嘘の大丈夫（シーン1〜4）→ ②蒼真との「大丈夫じゃない」の共有（シーン5）→ ③新しい意味の「大丈夫」（シーン6）
- 母の47回の拒否=「愛」。凛花はまだ蒼真を「好き」とは言えないが、母の「愛」を理解しかけている
- 端末を胸に抱くラストイメージ。蒼真との繋がりの象徴としての端末。マザーの管理ツールでもあるそれが、凛花にとっては「お守り」になる逆転
- EP1シーン6（行為シーン）に直結するラスト。だが凛花はまだ怖い。震えている。それでも「大丈夫」——これがEP0の着地点

#### 感情の流れ
- 通話後の余韻（温かさと不安の混在） → 蒼真との記憶を一つずつ辿る → 「好き」なのかという自問（答えは出ない） → 母の愛を想う → 「嫌じゃない」が「奇跡」かもしれないという気づき → 端末を胸に抱く → 新しい「大丈夫」を口にする → 穏やかな安堵。怖さは消えない。でも一人じゃない → 眠りに落ちる

## EP0固有の描写方針

### テーマ
「嘘の"大丈夫"をまとった少女が、"大丈夫じゃない"を共有できる相手と出会う」

### 「大丈夫」の変遷

| シーン | 「大丈夫」の種類 | 意味 |
|--------|------------------|------|
| シーン1 | 自分に言い聞かせる大丈夫 | 不安を押し殺す一人の呪文 |
| シーン2 | 母との交換する大丈夫 | 互いの嘘を見抜きながら共有する優しい嘘 |
| シーン3 | 友人の前の大丈夫 | しっかり者の仮面としての社交辞令 |
| シーン4 | 蒼真の前の大丈夫 | 少しずつ薄くなる仮面。本音が漏れ始める |
| シーン5 | 通話での大丈夫 → 蒼真の「大丈夫じゃない」 | 嘘の大丈夫が、正直な「大丈夫じゃない」で解かれる |
| シーン6 | 新しい「大丈夫」 | 不完全さを受け入れた上での、自分自身への許し |

### 文体
- 一人称（凛花視点）。EP1の蒼真視点と対になる
- 凛花の語り口は蒼真より言葉数が多く、感情の機微を言語化しようとする。だが本当に大事なことは言葉にならず、「……」や体の反応（心臓が跳ねる、耳が熱い、目の奥が痛い）で表現される
- 短文と長文のリズム: 「しっかり者」の凛花が語る部分はテンポよく。一人の凛花が内面を語る部分はゆっくりと

### SF環境描写の活用
- **温室**: 凛花にとっての「地球」。土の匂い、緑、湿度。他のどのシーンよりも五感描写を濃くする
- **人工太陽**: 均一すぎる光。凛花はその不自然さに気づいているが、温室の中では人工太陽すら「本物」に見える
- **頭上の街**: 見上げると逆さまの街並み。凛花は時々「あの逆さまの自分も、笑っているのかな」と考える
- **駆動音のハム**: 全シーンの通奏低音。凛花にとっては「父がいた頃から変わらない音」。唯一の不変
- **0.9Gの浮遊感**: 凛花にとっての「普通」。だが温室で土に触れている時だけ、なぜか重力が「ちゃんとある」ように感じる

### 色彩設計
- **凛花のカラー**: 銀灰（髪）、琥珀（目）。月光と蜂蜜。冷たさと温かさの中間
- **温室のカラー**: 緑（植物）、茶（土）、透明（水滴）。船内で唯一の「地球の色」
- **蒼真のカラー**: 凛花の目を通して。藍色（髪）、灰青→青（目）。感情が動くと色が変わる不思議な目
- **マザーのカラー**: 白（検査室）、青（インジケーター）、無色（声）。温度のない色
