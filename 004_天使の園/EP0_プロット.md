# EP0「白樺の門」詳細プロット

> 白河澪（13歳・中等部2年）の聖エステル学院入学から、学園生活への順応、「天使の園」への選抜告知、天使としての準備を経て、初めての楽園館訪問の直前までを描く。行為シーンはなし。澪の日常と内面——母への想い、閉鎖空間の美しさと不穏さ、自ら鳥籠に入っていく少女の心理——に集中するエピソード。

---

## EP0の位置づけ

- **EP0**: 入学〜天使への選抜〜楽園館へ向かう直前まで（本プロット）
- **EP1**: 初めての楽園館訪問。鷹宮隆一郎との添い寝＋軽い身体接触（第三天使の範囲内）
- **EP2以降**: ランクアップと深化

EP0はEP1（旧EP1のシーン4〜8に相当）の前日譚として、澪が「普通の少女」から「天使」に変わる境界線の手前までを丁寧に描く。読者にとっては、聖エステル学院の表の美しさと裏の不穏さを体感し、澪という少女に感情移入するための導入エピソードとなる。

---

## シーン構成（全6シーン）

### シーン1：入学の日
- **場所**: 正門前〜白樺の並木道〜校舎棟〜講堂〜寮棟（白百合館）
- **時間帯**: 4月上旬、午前10時〜午後。春の柔らかい陽射し
- **登場人物**: 澪、母・白河紗英（見送り）、学院の教師、学院長（壇上）、新入生たち、ルームメイト・藤堂彩音
- **出来事**:
  - 山道を登ってきたタクシーが聖エステル学院の正門前に到着する。標高800m。空気が冷たく澄んでいる。門柱に天使のレリーフ。白樺の並木道が、赤レンガの校舎へ向かって真っ直ぐに伸びている
  - 澪が車を降りる。制服姿（白いブラウス、紺のプリーツスカート、赤いリボン）。春風が腰まで届く黒髪を揺らす。初めて見る景色に息を呑む——こんなに美しい場所があるのかと
  - 母・紗英との別れ。紗英は涙ぐみながら「澪、あなたは才能があるの。この学校で立派になって」と言う。澪は微笑んで「大丈夫。お母さんこそ、体に気をつけて」と答える
  - 母の手の描写。かつてピアノを奏でた細い指は、今は関節が腫れて変形している。第二関節が外側に曲がり、薬指はまっすぐ伸びない。澪はその手を両手で包む。手の甲に浮いた青い血管。骨ばった指先の冷たさ。「必ず、恩返しするから」——声には出さない誓い
  - 車が去っていく。後部座席の窓越しに母が手を振っている。澪は振り返らず、白樺の並木道を歩き始める。背筋を伸ばして。革靴が砂利道を踏む音だけが静寂の中に響く
  - 赤レンガの校舎が見えてくる。ステンドグラスの嵌った窓、時計塔、手入れされた花壇に水仙と桜草。教師に案内され、校舎内へ。大理石の廊下、磨き上げられた木の手すり、壁に掛かった宗教画。別世界に足を踏み入れた感覚
  - 講堂での入学式。20名の新入生。洗練された所作、高級な靴、手入れされた爪。全員が裕福な家庭の令嬢に見える。澪だけが場違い——だが、誰よりも姿勢が良く、誰よりも静かに座っている
  - 学院長の壇上からの訓示:「この学院は、あなたたちを守り、育てる場所です。外の世界の喧噪から離れ、ここで本当の淑女になりなさい」。柔和な笑み。その目が一瞬だけ新入生を値踏みするように動いたことに、澪は気づかない
  - 入学式後、教師に案内されて寮棟・白百合館へ。2人部屋。白い壁、木のベッド、小さな机、窓の外に白樺の木。質素だが清潔で、陽当たりが良い
  - ルームメイト・藤堂彩音との出会い。彩音は外交官の娘で、明るく社交的な少女。栗色のショートボブ、くりくりした大きな目。澪より3cmほど高い。初日から「澪ちゃん！　よろしくね！」と距離を詰めてくる。澪の警戒心を自然に溶かしていく存在
  - 夜、ベッドの中で澪は天井を見つめる。清潔なシーツの匂い。母のことを想う。「お母さん、今頃一人で夕飯を食べてるのかな」。少しだけ目が潤む。隣のベッドから彩音の寝息が聞こえる。知らない場所で知らない人の寝息を聞きながら眠る——初めての夜
- **描写のポイント**: 聖エステル学院の圧倒的な美しさを五感で描写し、「美しい鳥籠」の第一印象を確立する。母の手の描写を丁寧に行い、澪の動機（母を助けたい）を読者の心に刻む。学院長の値踏みの一瞥を伏線として忍ばせる。彩音との出会いを温かく描き、後の二重生活の切なさへの布石とする
- **感情の流れ**: 緊張と決意 → 母との別れの寂しさ → 学園の美しさへの驚嘆 → 入学式での場違い感 → 彩音との出会いに少し安堵 → 初夜の孤独と母への想い
- **文量目安**: 約3,500〜4,000字

### シーン2：学園生活（日常ダイジェスト）
- **場所**: 校舎棟（教室・音楽室）・食堂・礼拝堂・庭園・大浴場・白百合館
- **時間帯**: 4月〜6月の2ヶ月間をダイジェスト。朝〜夜までの日常を断片的に描写
- **登場人物**: 澪、彩音、同級生たち、教師陣（ピアノ教師含む）
- **出来事**:
  - 学院の日常サイクルが始まる。6時起床、礼拝堂で朝の礼拝、朝食、午前授業、昼食、午後授業、自由時間、夕食、自習、入浴、22時消灯。このリズムが少しずつ澪の体に染みていく
  - 授業の風景。通常教科に加えてマナー・礼法の授業（お辞儀の角度は15度・30度・45度の三段階、食事作法、歩き方の指導）。芸術教育ではピアノ・ヴァイオリン・バレエ・絵画が必修
  - ピアノの授業。澪が弾き始めた瞬間、教室の空気が変わる。ショパンの練習曲。教師が手を止めて聴き入る。同級生たちが振り返る。演奏が終わると教師が「……白河さん、あなた、どなたに師事していたの？」と訊く。「母に教わりました」と澪が答える。教師が目を見開く。独学に近い環境でこの腕前——才能だけでは説明がつかない、膨大な練習量の賜物
  - 食堂での食事。上品な料理。澪は美味しいと思う。けれど、母の味噌汁が恋しい。家では二人きりの食卓だった。ここには120人がいるのに、あの食卓よりも寂しく感じる瞬間がある
  - 彩音との友情が深まっていく。昼休みに庭園のベンチで一緒にお弁当のデザートを食べる。放課後に二人で図書室で勉強する。彩音は明るくて話が尽きない。「澪ちゃんってさ、すっごく綺麗なのに自分でわかってないでしょ」と笑う。澪が戸惑う。そういうことを言われた経験がない
  - 携帯電話のない生活。外の情報が極端に少ない。母への手紙は週に1通。便箋に万年筆で丁寧に書く。「お母さん、学校は楽しいです。ピアノの授業で先生に褒められました」——本当のことだけを書く。嘘はまだ必要ない。返事が届くまで10日。母の体調が気になるが、確認する術がない
  - 大浴場での入浴シーン。同学年の少女たちと湯船に浸かる。湯気の中に白い肌がぼんやりと浮かぶ。少女たちの他愛ない会話。「今日の礼法の授業、先生厳しくなかった？」「バレエの筋肉痛がひどい〜」。彩音が澪の隣に来て「澪ちゃん、肌きれい〜。すべすべ」と腕に触れてくる。澪の華奢な体——発育途上の薄い胸、浮き出た鎖骨、細い腰、白い肌。澪は少し恥ずかしそうにしながらも、この無邪気な時間が嫌いではない
  - ある夜、消灯後。澪が窓から外を見ると、森の奥にぼんやりと灯りが見える。オレンジ色の温かい光。「ねえ彩音ちゃん、あの灯りって何？」と小声で聞く。彩音が起き上がって窓を覗く。「さあ……先輩が『立入禁止の別棟がある』って言ってたけど。研修施設かなんかじゃない？」。澪は気にしながらも、それ以上追及しない。灯りは毎晩ではない。ときどき灯り、ときどき消えている
  - 礼拝堂でのピアノ演奏。放課後、澪は一人で礼拝堂を訪れるようになる。隅に置かれたアップライトピアノ。古い楽器だが、澪にとっては母のピアノ教室のピアノと同じ手触り。誰もいない礼拝堂で、ステンドグラスの光を浴びながらピアノを弾く。ドビュッシーの「月の光」。ショパンのノクターン。弾いているとき、澪の表情が変わる。普段の礼儀正しい微笑みが消え、無防備で、剥き出しの感情が顔に浮かぶ。ここだけが澪の「素」でいられる場所
  - 季節が移ろう。白樺の若葉が深い緑に変わり、庭園の薔薇が咲き始め、蝉の声が遠くから聞こえるようになる。澪はこの閉じた世界に少しずつ馴染んでいく。美しく、規律正しく、安全な——ように見える場所
- **描写のポイント**: 学院の「表の顔」の魅力を存分に描く。授業・友情・食事・入浴と多角的に日常を提示し、読者をこの世界に没入させる。ピアノの授業での実力発覚が、後の学院長の目に留まる伏線。彩音との友情は後の二重生活との対比で効く。森の灯り＝楽園館の伏線をさりげなく。礼拝堂のピアノは澪のアイデンティティの象徴として繰り返す。入浴シーンで澪の身体を読者に見せておく（EP1以降の行為シーンとの対比）。携帯電話のない閉鎖性を強調
- **感情の流れ**: 新鮮さと戸惑い → ピアノで認められる喜び → 彩音との友情に安心 → 母への手紙に込める想い → 学院生活への順応と小さな幸せ → 森の灯りへの微かな疑問 → 礼拝堂のピアノで得る安らぎ → 季節の移ろいの中で閉塞感と安心が共存
- **文量目安**: 約4,000〜5,000字

### シーン3：特別奨学生プログラムの告知
- **場所**: 校舎棟の廊下〜学院長室〜校舎棟の廊下
- **時間帯**: 6月中旬、午後の授業後（16時頃）。梅雨の晴れ間
- **登場人物**: 澪、学院長、寮監、上級生（先輩天使・すれ違うのみ）
- **出来事**:
  - 放課後、教師から「学院長がお呼びです」と告げられる。澪は何か悪いことをしただろうかと緊張する。思い当たる節はない。成績は上位。規律も守っている
  - 学院長室。重厚な木の扉をノックして入る。深いカーペット、壁一面の書棚、大きなデスク、窓の外に庭園の緑。学院長が柔和な笑みで「かけなさい」と促す。革張りの椅子に浅く腰掛ける澪
  - 「白河さん、あなたの成績と品行、大変素晴らしいです。ピアノの腕前も、教師陣が絶賛しています」。褒められて澪は戸惑う。面と向かって褒められることに慣れていない
  - 「実は、あなたを『特別奨学生プログラム』に推薦したいのです」。学院長が説明する——学院を支援してくださる方々、文化人や実業家の方々に、ピアノ演奏や食事の席で接待をする。「接待」と言っても、お食事の席でピアノを弾いたり、会話のお相手をしたり。あなたの教養と品性があれば、きっと喜んでいただける
  - 寮監が横から補足する。「先輩方も何人か参加しています。最初は緊張するかもしれませんが、すぐに慣れますよ」。優しい口調。けれど、澪には感じ取れない微かな翳りがその目の奥にある
  - 「このプログラムに参加していただければ、通常の学費免除に加えて月々のお小遣いをお渡しできます。それから、卒業後の進路支援も」。金額を聞いた瞬間、澪の心臓が跳ねる。この金額があれば——母の治療費。ピアノ教室の家賃。月々の生活費。母が夜遅くまで生徒を教えなくてもよくなる
  - 澪の中で天秤が傾く。迷いはある。知らない大人の前でピアノを弾く。食事の席で会話をする。それだけなら——「やらせてください」。自分の声が思ったよりもはっきり聞こえて、澪自身が驚く
  - 学院長が微笑む。「良い決断です。あなたにぴったりの方を紹介しましょう」。その微笑みの意味を、澪はまだ知らない
  - 学院長室を出た廊下。澪は胸に手を当てる。鼓動が速い。嬉しいのか、不安なのか、よくわからない。ただ、「お母さんを助けられる」という思いだけが確かにある
  - 廊下の向こうから、上級生の少女が歩いてくる。高等部の生徒。長い髪を丁寧に編み込んでいる。その少女が澪を見て——一瞬だけ足を緩め、哀れむような、懐かしむような、複雑な目を向けた。そしてすぐに視線を外し、すれ違っていく。澪は立ち止まる。今の目は何だったのか。気のせいだと思おうとする。気のせいだと思えない
- **描写のポイント**: 「天使の園」への入り口を、徹底的に美化された言葉で描く。学院長の巧みな話術——褒め→提案→報酬の段階的提示。澪が「自分の意思で」選んだと思わせる誘導の構造。母への想いが澪の判断の軸であること。上級生の一瞥は「先輩天使」が新人を見る視線——哀れみと諦め。この違和感が読者に不穏さを伝える
- **感情の流れ**: 緊張と不安 → 褒められて戸惑い → 「接待」への困惑 → 報酬額を聞いて心が揺れる → 母を想って決断 → 不安と期待の混在 → 上級生の目に微かな違和感 → 言語化できない不穏さを抱えて寮へ戻る
- **文量目安**: 約3,000〜3,500字

### シーン4：天使の準備
- **場所**: 楽園館（初訪問）・楽園館内の研修室・準備室
- **時間帯**: 6月下旬、放課後（自由時間）。晴れた午後
- **登場人物**: 澪、楽園館管理人（中年女性）、先輩天使（高等部1年・名前は出さない）
- **出来事**:
  - 寮監に連れられ、昼間の楽園館を初めて訪れる。普段は「立入禁止の別棟」としか知らなかった建物。森の小道を通って到着。赤レンガの2階建て、アイアンのアーチに薔薇。昼間見ると、校舎と同じ様式だが小ぶりで、品の良い洋館のように見える。あの夜に窓から見えた灯りの正体がここだと知る
  - 楽園館の管理人が迎える。中年の女性。穏やかだが事務的な口調。「今日はお仕事の前の説明と研修です。何も怖いことはないわ」
  - 研修室で「天使の園」の詳細説明を受ける。天使のランク制度——第三天使（セラフ・トワ）は食事・会話の接待と添い寝まで。第二天使はさらに踏み込んだ奉仕。第一天使は全ての奉仕。大天使は専属。管理人は淡々と説明するが、「添い寝」「奉仕」という言葉を使うとき、ほんの少しだけ声のトーンを下げる
  - 澪の反応。「接待」だと思っていたものの全貌が、少しずつ形を持ち始める。食事とピアノだけではない。「添い寝」——知らない男の人の隣で眠る。体がわずかに強張る。しかし、第三天使の範囲はそこまで。それ以上のことはしなくていい。……今は
  - パトロンとの接し方の研修。食事中のマナー（すでに授業で学んでいる内容をより実践的に）。会話の心得——パトロンの話を聞くこと、自分から質問すること、笑顔を絶やさないこと。「あなたたちは特別なお客様をおもてなしする、この学院の天使なのよ」
  - 先輩天使が研修に同席している。高等部1年の少女。落ち着いた物腰。澪を見る目に、先日の廊下の上級生と似た翳りがある。けれど、口調は優しい。「最初は食事と会話だけよ。鷹宮様はとても紳士的な方だから、怖いことは何もないわ。ただ……」——少し言い淀む。「添い寝があるかもしれないけれど、隣で横になるだけ。大丈夫」。その「大丈夫」に、微かな力みを澪は感じ取る
  - 天使の衣装を初めて手にする。白いシルクのキャミソール。細い肩紐。膝丈のレースのガウン。素足に履くサテンのスリッパ。銀の十字架のイヤリング。薄く、軽く、肌に触れるとひんやりと冷たい。制服とは正反対の質感。澪は衣装を指先で撫でる。これを着て、知らない男の人と……
  - 管理人が「一度着てみましょう」と促す。準備室の鏡の前で着替える。制服を脱ぎ、衣装を纏った自分を鏡で見る。白い衣装が澪の黒髪と菫色の瞳を際立たせている。鎖骨が露出し、肩のラインが剥き出しになる。「……これが、天使」と呟く。鏡の中の自分が、つい数分前の「生徒」とは別の存在に見える
  - 管理人が「綺麗よ」と言う。先輩天使が「似合ってる」と小さく微笑む。澪は「ありがとうございます」と返すが、胸の奥がざわついている
  - 制服に着替え直して楽園館を出る。森の小道を寮へ戻る。昼間の道は明るく、鳥の声が聞こえる。けれど澪の足取りは行きよりも少し重い
- **描写のポイント**: 楽園館の昼の姿（夜とは異なる穏やかな印象→EP1で夜の姿を見たときの対比に効く）。天使のランク制度を読者に提示する場面だが、澪の心理反応を通じて「システムの不気味さ」を伝える。先輩天使の「大丈夫」に込められた欺瞞と善意の混在。天使の衣装を初めて着る場面＝「鏡」のモチーフ。制服→天使の衣装→制服という往復が、二重生活の予行演習になっている
- **感情の流れ**: 好奇心と緊張 → 楽園館の外観に少し安堵（思ったより普通の建物） → ランク制度の説明で不穏さを感じる → 「添い寝」に体が強張る → 先輩天使の優しさに少し救われる → 衣装に触れて現実感が増す → 鏡の中の「天使」に戸惑い → 制服に戻る安堵と、元には戻れない予感
- **文量目安**: 約3,000〜3,500字

### シーン5：二重生活の予感
- **場所**: 庭園（天使の庭）・白百合館（寮室）・礼拝堂
- **時間帯**: 6月下旬のある日。午後の自由時間〜夕方
- **登場人物**: 澪、彩音
- **出来事**:
  - 自由時間。彩音と庭園を散策している。6月の庭園は薔薇が満開。噴水の水しぶきが午後の光に虹を作る。白い東屋のベンチに並んで座る二人
  - 彩音が足をぶらぶらさせながら言う。「この学校、すごく綺麗だけどさ。なんか閉じ込められてる感じしない？」。澪が「そう？」と返す。彩音は「だってさ、携帯もないし、外出も月イチだし。世界から切り離されてるみたい。綺麗な檻っていうか」。澪は薔薇を見つめたまま少し笑う。「でも、守られてるとも言えるよね」。彩音が「うーん、そうかなあ」と首を傾げる
  - 澪の内面。彩音の言葉が胸に刺さっている。綺麗な檻。そうかもしれない。でも澪は自分からこの檻に入った。入るしかなかった。母の治療費。学費。「特別奨学生プログラム」に参加すれば、月々の仕送りができる。その安堵は本物だ。けれど、「添い寝」という言葉が頭の隅にこびりついている
  - 彩音が「澪ちゃんは？　この学校、好き？」と聞く。澪は少し考えて「うん。好きだよ。……彩音ちゃんがいるし」と答える。彩音が嬉しそうに笑う。「えへへ。私も澪ちゃんがいるからがんばれる」。二人の間に流れる穏やかな午後の時間。この時間を守りたいと澪は思う。この子には何も知られたくない
  - 寮室に戻る。彩音が「先にお風呂行ってくるね〜」と出ていく。一人になった澪は、枕の下に手を伸ばす。そこにはまだ何もない。けれどもうすぐ、ここに報酬の封筒が入ることになる。まだ一度も受け取っていない「お給料」。それを受け取ったら、もう引き返せない気がする
  - 夕方、澪は一人で礼拝堂に向かう。放課後の礼拝堂は無人。西日がステンドグラスを透かして、床に赤と青と金の光を落としている。壁のステンドグラスに描かれた天使が、夕陽を受けて燃えるように輝いている
  - アップライトピアノの前に座る。鍵盤に指を置く。ショパンのノクターン第2番。弾き始める。変ホ長調の柔らかな旋律が礼拝堂に満ちる。弾いているうちに、感情が溢れてくる。母の変形した手。彩音の無邪気な笑顔。「天使」の白い衣装。知らない男の人の隣で眠ること
  - 涙がこぼれる。鍵盤の上に落ちる。止められない。何に泣いているのかわからない。怖いのか。悲しいのか。それとも、まだ何も始まっていないのに、もう後戻りできないと感じているからか
  - 曲が終わる。鍵盤の上に指を置いたまま、しばらく動かない。ステンドグラスの天使が無言で見下ろしている。澪は涙を拭い、深呼吸する。立ち上がる。「大丈夫。私は大丈夫」。——母の口癖。母がいつも体の痛みを隠して言っていた言葉を、澪も使うようになっている
- **描写のポイント**: 彩音との日常会話が「檻」のメタファーを自然に導入する。澪の内面——母への想いと不安の葛藤を丁寧に。まだ何も起きていないのに「もう戻れない」と感じる心理がこのシーンの核心。礼拝堂のピアノが澪の唯一の感情の吐口。ステンドグラスの天使＝聖なる天使と、楽園館の「天使」の二重の意味。母の口癖「大丈夫」を澪が無意識に継承していることが、母子の絆と自己犠牲の連鎖を暗示する
- **感情の流れ**: 彩音との穏やかな時間 → 「綺麗な檻」という言葉に胸が痛む → 彩音には何も知られたくない決意 → 枕の下の「まだ何もない空間」への不安 → 礼拝堂のピアノで感情が溢れる → 涙 → 「大丈夫」と自分を立て直す → 静かな覚悟
- **文量目安**: 約3,000〜3,500字

### シーン6：今夜、楽園館へ
- **場所**: 校舎棟の廊下〜食堂〜白百合館（寮室）
- **時間帯**: ある日の夕方〜消灯後（22:25）。6月下旬
- **登場人物**: 澪、寮監、彩音
- **出来事**:
  - 放課後。校舎から寮へ向かう廊下で、寮監が澪に声をかける。「白河さん」。穏やかだが有無を言わせない声。「今夜が初めてのお仕事です。楽園館へ。22時30分に廊下でお待ちしています」。それだけ言って、寮監は去っていく
  - 澪の心臓が凍る。足が止まる。廊下の窓から差し込む夕日が澪の横顔を照らしている。ついにこの日が来た。頭ではわかっていた。いつか来ると。でも「いつか」は今夜だった
  - 数秒間、立ち尽くす。そして歩き出す。何事もなかったかのように。足が震えているのを、スカートの裾が隠してくれている
  - 夕食。食堂で彩音の向かいに座る。献立はハンバーグとコーンスープとサラダ。好きなメニューのはずだけれど、何を口に入れても味がしない。フォークを持つ手が微かに震えている。彩音が首を傾げる。「澪ちゃん、顔色悪くない？　今日ずっとぼんやりしてるし」。澪は「ちょっと頭痛がして……」と微笑む。彩音が「保健室行く？」と心配する。「大丈夫。寝たら治ると思う」
  - 自習時間。教科書を開くが、文字が頭に入らない。時計を見る。19時半。あと3時間。ノートに万年筆を走らせるが、何を書いているのか自分でもわからない
  - 入浴。湯船に沈む。湯気の向こうに他の少女たちの笑い声が聞こえる。澪は自分の体を見下ろす。白い肌。薄い胸。細い腕。今夜、この体が知らない男の人の隣に横たわる。湯の温かさが心まで届かない
  - 消灯。22時。部屋の灯りが消える。暗闇。天井を見つめたまま、目を閉じられない。布団の中で、両手を胸の前で組んでいる。祈るように。何に祈っているのかわからない。母のこと。彩音のこと。明日の自分のこと
  - 隣のベッドから彩音の寝息が聞こえてくる。穏やかな、規則正しい呼吸。この子は何も知らない。知らなくていい。知らないままでいてほしい
  - 枕元の時計を確認する。暗闇に目が慣れてきた。22時20分。あと5分
  - 22時25分。澪は音を立てずにベッドを抜け出す。制服はハンガーにかかっている。暗がりの中、手探りで袖を通す。ブラウスのボタンを留める指先が冷たい。リボンは——いらない。夜の廊下にリボンは必要ない
  - 彩音の寝顔を見る。月明かりが窓から差し込んで、彩音の栗色の髪を淡く照らしている。無防備な寝顔。口が少し開いている。いつもの彩音
  - 「行ってくるね」。心の中で呟く。声には出さない
  - ドアノブに手をかける。金属の冷たさが指に伝わる。一度だけ、深く息を吸う
  - ドアを開ける。暗い廊下の向こうに、寮監の影がぼんやりと見える

  - ——ここで、EP0は終わる。
- **描写のポイント**: 時間の経過を「告知→夕食→自習→入浴→消灯→ベッドを抜け出す」と段階的に描き、不可逆的に「その時」が近づいてくる緊張感を作る。澪の恐怖は直接的な独白ではなく、「味がしない食事」「震える手」「閉じられない目」「冷たい指先」など身体反応で表現する。彩音の無邪気さが残酷なまでのコントラスト。ラストの「行ってくるね」は声に出さない——澪の秘密がここから始まる。ドアノブの冷たさで終わることで、EP1への強いブリッジを作る
- **感情の流れ**: 告知の衝撃（心臓が凍る） → 表面上の平静を装う → 夕食で味がしない → 時間が過ぎるのが怖い → 入浴で自分の体を意識する → 消灯後の暗闇で恐怖と覚悟が交錯 → 彩音の寝息を聞いて孤独を感じる → 「行ってくるね」——静かな決意 → ドアノブの冷たさに触れて、EP1へ
- **文量目安**: 約3,500〜4,500字

## EP0 描写方針

### 基本コンセプト
- **視点**: 三人称・澪視点寄り。澪の五感と心理を通じて世界を描く。他キャラの内面には踏み込まない
- **文体**: やや抒情的な文語体。五感描写を多用。心理描写は直接的な独白よりも、身体反応や環境描写を通じて暗示する
- **時制**: 過去形基調
- **色調**: 学園の昼の場面は白・青・緑を基調とした明るい色彩。礼拝堂は七色のステンドグラスの光。シーン6に向かうにつれて暗い色調（夕暮れのオレンジ→消灯後の暗闇→月明かりの青白さ）へ移行
- **行為シーン**: なし。EP0は澪が天使として楽園館に向かう直前で終わる。身体的な接触はない
- **音楽**: ピアノの音色が通奏低音として全シーンに響く。澪が弾く曲（ショパン、ドビュッシー）が感情を代弁する

### 重要モチーフ
- **母の手**: 変形した関節、浮いた血管。澪の全ての行動の動機の象徴。シーン1で刻み、シーン5でフラッシュバック
- **ピアノ**: 澪のアイデンティティ。母との絆。唯一無防備になれる場所。シーン2（授業）とシーン5（礼拝堂）で繰り返す
- **鏡**: シーン4で天使の衣装を着た自分を見る場面。「生徒」から「天使」への変容の予感
- **白樺の並木道**: シーン1の入学で歩いた道。美しく静謐な道が、EP1では深夜に歩く森の道へと変貌する
- **ステンドグラスの天使**: 礼拝堂の聖なる天使と、楽園館の「天使」。シーン5で澪が涙を流す場面で、この二重性が暗示される
- **「大丈夫」**: 母の口癖であり、澪が自分に言い聞かせる言葉。自己犠牲の連鎖の象徴
- **森の灯り**: シーン2で窓から見える楽園館の灯り。正体を知る前と知った後で意味が変わる
- **ドアノブ**: シーン6のラスト。冷たい金属の感触が、「日常」と「非日常」の境界線を象徴する

## EP0→EP1 接続メモ

EP0のラスト（シーン6）の状況:
- 澪は制服姿で寮の部屋を出たところ。暗い廊下で寮監と合流
- 天使の衣装はまだ着ていない（楽園館で着替える）
- 彩音は寝ている。澪が出たことに気づいていない
- 時刻は22時25〜30分

EP1はここから直接続き、以下を描く:
- 夜の森の小道を歩いて楽園館へ
- 天使の衣装に着替え
- 鷹宮隆一郎との食事会・ピアノ演奏・添い寝（第三天使の範囲内）
- 翌朝の二重生活とエピローグ
