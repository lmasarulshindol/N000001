# EP1「鳥籠の白鍵」詳細プロット

> 白河澪（13歳・中等部2年）の聖エステル学院入学から、「天使の園」への選抜、そして初めての楽園館訪問（第三天使としての初任務＝パトロン・鷹宮隆一郎との添い寝）までを描く。清楚な学園生活と楽園館の夜のコントラストが最大の演出軸。

---

## シーン構成（全8シーン）

### シーン1：白樺の門（入学）
- **場所**: 聖エステル学院・正門〜並木道〜校舎棟
- **時間帯**: 4月上旬、午前10時。春の柔らかい陽射し
- **登場人物**: 澪、母・白河紗英（見送り）、学院の教師
- **出来事**:
  - 山道を登ってきた車が聖エステル学院の正門前に到着する。白樺の並木道が真っ直ぐに伸びている。門柱には天使のレリーフ
  - 澪が車を降りる。制服（白いブラウス、紺のプリーツスカート、赤いリボン）に身を包んだ姿。春風が長い髪を揺らす
  - 母・紗英との別れ。紗英は涙ぐみながら「澪、あなたは才能があるの。この学校で立派になって」と言う。澪は微笑んで「大丈夫。お母さんこそ、体に気をつけて」と答える
  - 母の手の指——かつてピアノを弾いていた細い指が、今は関節が腫れて変形している。澪はその手を両手で包む。「必ず、恩返しするから」という無言の誓い
  - 車が去っていく。澪は振り返らず、白樺の並木道を歩き始める。背筋を伸ばして。足音だけが響く静謐な空間
  - 赤レンガの校舎が見えてくる。ステンドグラス、時計塔、手入れされた花壇。絵画のように美しい学園風景。澪は息を呑む
  - 教師に案内され、校舎内へ。大理石の廊下、磨き上げられた木の手すり、壁に掛かった宗教画。別世界に足を踏み入れた感覚
  - 講堂での入学式。20名の新入生。全員が裕福な家庭の令嬢に見える。澪だけが場違いな存在——だが、誰よりも姿勢が良く、誰よりも静かに座っている
  - 学院長（壇上）の訓示:「この学院は、あなたたちを守り、育てる場所です。外の世界の喧噪から離れ、ここで本当の淑女になりなさい」。柔和な笑み。だが、その目が一瞬だけ新入生を「値踏み」するように動いたことに、澪は気づかない
- **描写のポイント**: 聖エステル学院の圧倒的な美しさを五感で描写。澪の母との関係（貧しい母子家庭・母の手の描写）。「鳥籠」に自ら入っていく少女の姿。学院長の不穏な一瞥（読者への伏線）
- **感情の流れ**: 緊張と決意 → 母との別れの寂しさ → 学園の美しさへの驚嘆 → 新生活への微かな希望
- **文量目安**: 約3,000〜3,500字

### シーン2：白百合館の日々（寮生活・友人）
- **場所**: 寮棟（白百合館）・教室・食堂・庭園
- **時間帯**: 入学から2ヶ月間の日常（4月〜5月）をダイジェストで
- **登場人物**: 澪、ルームメイト・藤堂彩音（あやね）、同級生たち、教師陣
- **出来事**:
  - 寮室の描写。2人部屋。白い壁、木のベッド、小さな机、窓の外に白樺の木。質素だが清潔で、陽当たりが良い
  - ルームメイト・藤堂彩音との出会い。彩音は外交官の娘で、明るく社交的。初日から「澪ちゃん」と呼んで距離を詰めてくる。澪の警戒心を自然に溶かしていく存在
  - 学院の日常：朝6時起床→礼拝→朝食→授業→昼食→午後授業→自由時間→夕食→自習→入浴→22時消灯。このサイクルが繰り返される規律正しい生活
  - 授業の風景。通常教科に加えて、マナー・礼法の授業（お辞儀の角度、食事作法、歩き方）。芸術教育（ピアノ・ヴァイオリン・バレエ・絵画）。澪はピアノの授業で教師を驚かせる
  - 食堂での食事。全員が揃って「いただきます」。上品な料理。澪は母の手料理を思い出す。ここの食事は美味しいが、母の味噌汁が恋しい
  - 庭園での散策。彩音と二人で「天使の庭」を歩く。四季の花、噴水、ベンチ。彩音が「この学校、すごく綺麗だけど、ちょっと閉じ込められてる感じしない？」と笑う。澪も笑うが、心のどこかで同意している
  - 携帯電話のない生活。外の情報が極端に少ない。母への手紙は週に1通。返事が届くまで10日かかる。母の体調が気になるが、確認する術がない
  - 大浴場での入浴シーン。同級生たちと湯船に浸かる。少女たちの裸体が湯気の中にぼんやりと浮かぶ。彩音が「澪ちゃん、肌きれい〜」と無邪気に触れてくる。澪の華奢な体の描写（発育途上の薄い胸、細い腰、白い肌）
  - ある夜、消灯後。澪が窓から外を見ると、森の奥にぼんやりと灯りが見える。「あれは何？」と彩音に聞くと、「さあ……立入禁止の別棟があるらしいよ。先輩が言ってた」。澪は気にしながらも、すぐに忘れる
  - 礼拝堂での朝の礼拝。ステンドグラスから差し込む光の中、讃美歌を歌う少女たち。澪はこの時間が好きだと気づく。閉じた世界だが、美しく、穏やかで、安全な——ように見える場所
- **描写のポイント**: 学院の「表の顔」の魅力を存分に描く。澪と彩音の友情（後の二重生活で効いてくる）。閉鎖空間の美しさと息苦しさの両面。森の奥の灯り＝楽園館の伏線。入浴シーンで澪の身体を読者に見せる（後の行為シーンとの対比）
- **感情の流れ**: 新鮮さと戸惑い → 彩音との友情に安心 → 学院生活への順応 → 微かな閉塞感 → 森の灯りへの小さな疑問
- **文量目安**: 約4,000〜4,500字

### シーン3：特別奨学生（選抜の告知）
- **場所**: 校舎棟・学院長室
- **時間帯**: 6月中旬、午後の授業後（16時頃）
- **登場人物**: 澪、学院長、寮監
- **出来事**:
  - 放課後、教師から「学院長がお呼びです」と告げられる。澪は何か悪いことをしただろうかと緊張する
  - 学院長室。重厚な木の扉。ノックして入ると、深いカーペット、壁一面の書棚、大きなデスク。学院長が柔和な笑みで迎える
  - 「白河さん、あなたの成績と品行、とても素晴らしいです。ピアノの腕前も、教師陣が絶賛しています」。褒められて澪は戸惑う
  - 「実は、あなたを『特別奨学生プログラム』に推薦したいのです」。学院長が説明する：学院を支援してくださる方々——文化人や実業家の方々に、ピアノ演奏や食事の席で接待をする。お礼として、通常の学費免除に加えて月々のお小遣いと、卒業後の進路支援が得られる
  - 澪が戸惑う。「接待、ですか……」「ええ。お食事の席でピアノを弾いたり、会話のお相手をしたり。あなたの教養と品性があれば、きっと喜んでいただけます」
  - 寮監が補足する。「先輩方も何人か参加しています。最初は緊張するかもしれませんが、すぐに慣れますよ」。優しい口調。だが目が笑っていない（澪は気づかない）
  - 「月額のお小遣い」の金額を聞いて、澪の心が揺れる。母の治療費。ピアノ教室の収入だけでは足りない。自分が少しでも助けになれるなら——
  - 「……やらせてください」。澪が答える。学院長が微笑む。「良い決断です。あなたにぴったりの方を紹介しましょう」
  - 学院長室を出た廊下で、澪は胸に手を当てる。鼓動が速い。嬉しいのか、不安なのか、よくわからない。ただ、「お母さんを助けられる」という思いだけが確かにある
  - 寮に戻る途中、上級生の少女とすれ違う。その少女は澪を見て、一瞬だけ哀れむような目をした。澪は気のせいだと思って通り過ぎる
- **描写のポイント**: 「天使の園」への入り口を、徹底的に美化された言葉で描く。学院長の巧みな話術（褒め→提案→報酬の提示）。澪が「自分の意思で」選んだと思わせる誘導構造。母への想いが澪の判断を曇らせる。上級生の一瞥＝「天使」の先輩が新人を見る目（伏線）
- **感情の流れ**: 緊張 → 困惑 → 迷い → 母を想って決断 → 不安と期待の混在 → 微かな違和感（上級生の目）
- **文量目安**: 約3,000〜3,500字

### シーン4：天使の衣装（楽園館への道行き）
- **場所**: 白百合館（寮室）→ 森の小道 → 楽園館
- **時間帯**: 6月下旬、初任務の夜。22時消灯後
- **登場人物**: 澪、寮監、楽園館管理人
- **出来事**:
  - 消灯後。ベッドの中で目を開けている澪。隣のベッドでは彩音が寝息を立てている。今夜が「初めての接待」の日
  - 22時30分。約束の時間。澪は音を立てずにベッドを抜け出す。制服のまま、上にカーディガンを羽織る。彩音の寝顔を見る。「行ってくるね」と心の中で呟く。彩音には何も言えない
  - 廊下に出ると、寮監が待っている。「こちらへ」。無言で導かれる。夜の寮棟の廊下。月明かりが窓から差し込んで、床に光の矩形を作っている。澪の上履きがリノリウムの床を踏む音だけが響く
  - 裏口から外へ。6月の夜気。少し肌寒い。白樺の木々が月光に青白く浮かんでいる。寮監の懐中電灯が小道を照らす
  - 森の中の小道を歩く。昼間は散策路として知っている道だが、夜は別の場所のよう。虫の声。草を踏む足音。澪の心臓がどんどん速くなる
  - 木々の間から、暖かい灯りが漏れている。楽園館。澪が初めて近くで見る建物。赤レンガの2階建て。校舎と同じ様式だが、より小ぶりで、窓からオレンジ色の光が溢れている。入口にアイアンのアーチ。薔薇が絡まっている
  - 寮監が「ここまでです。中で指示を受けてください」と言って去る。澪は一人で扉を押す
  - 楽園館の内部。玄関ホールは学院の校舎よりも遥かに豪奢。大理石の床、シャンデリア、壁に掛かった油彩画（天使を描いたもの）。花の香りとアロマが混ざった、甘く重い空気
  - 楽園館の管理人（中年女性）が澪を迎え、準備室へ案内する。「今夜が初めてね。大丈夫、何も難しいことはないわ」
  - 準備室で「天使の衣装」に着替える場面。制服を一枚ずつ脱いでいく澪。赤いリボンを外す。ブラウスのボタンを外す。スカートを下ろす。下着姿になった自分を鏡で見る——学院の「生徒」が消えていく感覚
  - 天使の衣装：白いシルクのキャミソール（肩紐が細く、鎖骨と肩のラインが露出する）、膝丈のレースのガウン、素足にサテンのスリッパ。髪を下ろし、耳元に小さな銀の十字架のイヤリング。管理人が「綺麗よ」と呟く
  - 鏡に映った自分。数分前まで制服を着ていた中学生が、まるで別人のように見える。白い衣装が澪の黒髪と菫色の瞳を際立たせている。「これが、天使……」と澪が呟く
  - 管理人が最終確認:「今夜のお相手は鷹宮様。とても紳士的な方よ。最初はお食事と会話。あなたはピアノをお弾きになるんでしょう？　きっと喜ばれるわ。それから……添い寝をお願いされるかもしれないけれど、それだけよ。何も怖いことはないから」
  - 「添い寝」という言葉に、澪の体がわずかに強張る。だが、頷く。「はい」
- **描写のポイント**: 夜の学園の美しさと不穏さ。「生徒→天使」への変身過程を衣装の変化で丁寧に描く（制服を脱ぐ→天使の衣装を纏う）。楽園館の豪奢さと校舎との対比。管理人の「何も怖いことはない」が逆に不安を煽る構造。鏡に映った自分を見る場面＝アイデンティティの揺らぎ
- **感情の流れ**: 緊張と恐怖 → 夜の森の美しさに一瞬心が鎮まる → 楽園館の豪華さへの圧倒 → 着替えの羞恥と非日常感 → 鏡の中の「天使」への戸惑い → 「添い寝」への不安 → 覚悟
- **文量目安**: 約4,000〜4,500字

### シーン5：食事会とピアノ（パトロンとの対面）
- **場所**: 楽園館・ダイニングルーム → サロン（グランドピアノのある部屋）
- **時間帯**: 同夜、23時頃
- **登場人物**: 澪、鷹宮隆一郎
- **出来事**:
  - ダイニングルームへ通される。円卓にキャンドル。テーブルには二人分のセッティング。フレンチのフルコース。ワイングラスには赤ワイン（澪のグラスにはノンアルコールのぶどうジュース）
  - 鷹宮隆一郎が立ち上がって迎える。ダークグレーのスーツ。銀の時計。白髪混じりの髪がきちんと整えられている。低い声で「初めまして。鷹宮です」と名乗る。笑顔は穏やかで、威圧感はない
  - 澪が緊張しながら挨拶する。「白河澪です。今夜は……よろしくお願いします」。声が微かに震えている
  - 鷹宮が椅子を引いてくれる。「緊張しなくていい。今夜はただの食事会だ。美味しいものを食べて、少しお話をするだけ」
  - 食事が始まる。鷹宮は食事中、澪の緊張を解くように穏やかに話しかける。「ピアノを弾かれるそうですね。どんな曲が好きですか？」「お母様もピアニストだったと聞きました」
  - 母の話題に、澪の目が少し潤む。「母は……手を壊してしまって。今はピアノ教室を」。鷹宮が静かに頷く。「そうですか。大変でしたね」。その声に嘘がないように聞こえて、澪は少しだけ心を開く
  - 鷹宮が自分の話もする。「私の妻もピアニストでした。10年前に亡くしましてね」。遠い目。澪は「そうだったんですか……」と返す。二人の間に、音楽を通じた共感が芽生える——ように見える
  - 食事を終え、隣のサロンへ移動。部屋の中央にグランドピアノ（スタインウェイ）。鷹宮が「弾いてみませんか」と促す
  - 澪がピアノの前に座る。鍵盤に指を置いた瞬間、緊張が溶ける。ショパンのノクターン第2番。澪の指が鍵盤の上を滑り始める
  - 演奏する澪の描写。天使の白い衣装、キャンドルの灯り、黒いピアノ、細い指が鍵盤を叩く。目を閉じて弾いている澪の横顔は無防備で、美しい
  - 鷹宮がソファに座って聴いている。その目に浮かぶ感情は複雑——音楽への感動、亡き妻への郷愁、そして少女への欲望が渾然と混ざっている。だが表情は穏やかなまま
  - 演奏が終わる。沈黙。鷹宮が静かに拍手する。「素晴らしい。……妻の演奏を思い出しました」。その声がわずかに掠れている。澪は「ありがとうございます」と頬を染める
  - 鷹宮が「もう1曲、連弾しませんか」と提案する。隣に座る。澪は鷹宮の大きな手が鍵盤に置かれるのを見る。二人でシューベルトの軍隊行進曲を弾く。鷹宮のパートは正確だが温かみがある
  - 連弾中、鷹宮の肩が澪の肩に触れる。大人の男の体温。香水の匂い（ウッディな香り）。澪は心臓が跳ねるが、演奏に集中することで平静を保つ
  - 連弾が終わる。鷹宮が「楽しかった。……そろそろ、奥の部屋に行きましょうか」と立ち上がる。穏やかだが、有無を言わせない口調
- **描写のポイント**: 鷹宮の「紳士的な導入」を丁寧に描く（食事→会話→音楽→身体的距離の段階的縮小）。ピアノの演奏シーンが物語の核心（澪の唯一の武器であり、同時に鷹宮を惹きつける罠）。亡き妻の影を通じた鷹宮の複雑な感情。連弾＝身体的接触の「口実」としての機能
- **感情の流れ**: 
  - 澪: 緊張 → 鷹宮の穏やかさに少し安堵 → 母の話で心が揺れる → ピアノで自分を取り戻す → 連弾の近さにドキドキ → 「奥の部屋」への不安
  - 鷹宮: 紳士的な余裕 → 澪のピアノに心を動かされる → 欲望の静かな昂り → 次の段階へ誘導
- **文量目安**: 約4,500〜5,000字

### シーン6：添い寝——天使の初任務（行為シーン）
- **場所**: 楽園館・2階の寝室
- **時間帯**: 同夜、深夜0時頃〜1時30分頃
- **登場人物**: 澪、鷹宮隆一郎

#### 三層構造

**① 導入：寝室への移行（全体の25%＝約1,500字）**

- 鷹宮に導かれ、2階の寝室へ。廊下にはオレンジ色の間接照明。足元のカーペットが足音を吸い込む
- 寝室の描写。広い部屋。天蓋付きのダブルベッド。シルクのシーツ。ベッドサイドにアロマキャンドル。窓からは月明かり。学院の寮室とは比べものにならない豪奢さ
- 鷹宮が「着替えなくていい。そのままで」と言う。澪は天使の衣装のまま、部屋の中央に立っている。白いキャミソールとレースのガウン。薄い生地の下に、少女の体のラインがうっすらと透けている
- 鷹宮がジャケットを脱ぎ、シャツの袖を折る。「怖がらなくていい。ただ隣で眠るだけだ」。その声は穏やかだが、澪は鷹宮の目の奥に、食事中にはなかった熱のようなものを感じる
- 鷹宮がベッドに腰かけ、澪に手を差し伸べる。「おいで」

**② 行為本体：添い寝と身体接触（全体の55%＝約3,500字）**

- 澪がベッドに上がる。シルクのシーツのひんやりとした感触。鷹宮の隣に横たわる。天蓋から薄い布が垂れている。キャンドルの灯りが二人の影を壁に映す
- 最初は距離がある。澪は仰向け。鷹宮は横向きで澪を見ている。沈黙。時計の針の音だけが聞こえる
- 鷹宮が「こちらを向いて」と囁く。澪が体を横にすると、鷹宮の顔が近い。大人の男の顔。皺。だが目は優しい。澪は目を逸らせない
- 鷹宮の手が澪の髪に触れる。「綺麗な髪だ」。指が黒髪を梳いていく。耳の後ろ、首筋、肩。指先が肌に触れるたびに、澪の体がわずかに震える
- 「冷たい手だな。緊張しているんだね」。鷹宮が澪の手を取り、自分の胸に当てる。シャツ越しに鷹宮の心臓の鼓動が伝わる。安定した、大人のリズム
- 鷹宮が澪を引き寄せる。澪の頭が鷹宮の胸に収まる。大きな腕が背中に回される。抱きしめられている。父親に——いや、父親に抱きしめられた記憶は澪にはない。だからこの腕の温かさが何なのか、わからない
- 澪の体が少しずつ緊張を解いていく。鷹宮の体温、香水の残り香、シャツの布地の感触。安心感に似た何か。だが同時に、「これでいいの？」という疑問が頭の片隅にある
- 鷹宮の手が背中を撫で始める。ガウン越し。肩甲骨から腰へ、ゆっくりと上下する。澪はされるがまま。呼吸が少し深くなる
- 「……お父さんは？」と鷹宮が聞く。「いません。物心つく前に」と澪が答える。鷹宮が「そうか」と呟き、少しだけ力を込めて抱きしめる。その優しさに、澪の目が潤む。泣いてはいけないと思うのに
- 鷹宮の手がガウンの裾から中に入る。薄いキャミソールの上から背中を撫でる。布一枚を隔てた手のひらの温度。澪は息を詰める。だが、拒めない。拒む理由が見つからない——この人は優しい、怖いことはされていない、と自分に言い聞かせる
- 手が背中から脇腹へ移動する。澪の華奢な体を確かめるように。肋骨の浮いた脇腹。鷹宮の大きな手が澪の小さな体を包む。「細いな……ちゃんと食べているか？」。澪が「はい」と小さく答える
- 鷹宮の指先がキャミソールの肩紐に触れる。「少しだけ……いいかな」。澪が頷く前に、鷹宮が肩紐を片方だけ下ろす。白い肩が露出する。鷹宮の唇が肩に触れる。柔らかいキス。澪の体に電流のようなものが走る
- 澪が体を強張らせる。鷹宮が「大丈夫。これ以上はしないよ」と囁く。肩紐を元に戻す。だが、その「これ以上はしない」の境界線を引いたのは鷹宮であり、澪ではない
- 鷹宮が再び澪を胸に抱き寄せる。「今日はこれでいい。よく頑張ったね」。頭を撫でる。澪は「ありがとうございます」と自動的に答えている。何に対して感謝しているのか、自分でもわからない
- 沈黙の中で、二人の呼吸だけが部屋に満ちる。鷹宮の腕の中で、澪はいつの間にかまどろんでいる。安心なのか、疲労なのか、それとも現実から逃げているのか

**③ 余韻：楽園館から寮への帰還（全体の20%＝約1,200字）**

- 目が覚める。腕時計を見ると午前1時過ぎ。鷹宮が「そろそろ戻りなさい」と言う。穏やかな声。ベッドサイドのテーブルに白い封筒が置かれている（今夜の「報酬」）
- 澪が起き上がる。体に鷹宮の香水の匂いが残っている。1階に降りて準備室へ。天使の衣装を脱ぎ、制服に着替え直す。鏡を見る——さっきまで「天使」だった少女が、また「生徒」に戻っている。だが、目の奥の光が少し変わったような気がする
- 楽園館を出る。6月の深夜。夜風が肌を冷やす。鷹宮の体温の残り香が急速に消えていく。森の小道を一人で歩く。月明かりに照らされた白樺の木々。行きとは違い、道を覚えている自分がいる
- 寮に戻る。裏口から入り、音を立てずに廊下を歩く。自分の部屋に入る。彩音は変わらず眠っている。澪は制服を脱いでパジャマに着替え、ベッドに入る
- 天井を見つめる。鷹宮の手の温もり。肩に触れた唇の感触。「怖くなかった」と自分に言い聞かせる。実際、怖くなかった。だが、何かが変わった。昨日までの自分とは違う自分がここにいる
- 封筒の中身を確認する。現金。母に送れる金額。澪はそれを枕の下に入れて、目を閉じる

- **描写のポイント**: 鷹宮の「優しい支配」の構造——暴力も強制もなく、段階的に境界線を越えていく手法。澪が「自分の意思で受け入れた」と思い込む心理構造。添い寝＋軽い身体接触に留め、本番行為はまだ行わない（第三天使の範囲内）。父親不在の少女が「父性」と「性的接触」の区別をつけられない脆さ。行為後の着替え＝天使から生徒へ戻る儀式
- **感情の流れ**: 
  - 澪: 恐怖 → 鷹宮の体温で緩む → 涙（父性への渇望） → 肩へのキスでの動揺 → 安堵と困惑 → 余韻の中の空虚 → 報酬を見て現実に引き戻される
  - 鷹宮: 温かさの演出 → 少女の脆さへの愉悦 → 自制（初回は境界を守る） → 次への布石
- **文量目安**: 約5,000〜7,000字（行為シーン全体）

### シーン7：何食わぬ顔の礼拝（翌朝の二重生活）
- **場所**: 白百合館（寮室）→ 礼拝堂 → 食堂
- **時間帯**: 翌朝6:00〜7:30
- **登場人物**: 澪、彩音、同級生たち
- **出来事**:
  - 朝6時。鐘の音で目が覚める。ほとんど眠れなかった。目の下に薄い隈。だが澪は鏡の前で完璧に身支度を整える。制服を着て、リボンを結んで、髪をブラシで丁寧に整える
  - 彩音が「おはよう〜。澪ちゃん、顔色悪くない？」と心配する。澪は「昨夜ちょっと眠れなくて」と微笑む。嘘ではない。嘘ではないけれど、本当でもない
  - 礼拝堂へ向かう。朝の澄んだ空気。昨夜歩いた森の小道が見える。日の光の下では何の変哲もない散策路。あの道を深夜に歩いた自分が、夢の中の出来事のように感じられる
  - 礼拝堂。ステンドグラスから朝日が差し込み、少女たちの白いブラウスを七色に染める。讃美歌を歌う。隣の彩音と同じ歌を歌いながら、澪は思う——私はもう、この子と同じ場所にいない
  - 礼拝中、ふと前列の上級生に目がいく。昨日の廊下ですれ違った少女。彼女もまた「天使」なのだとわかる。視線が合う。上級生が微かに頷く。「あなたも、こちら側になったのね」という無言の歓迎——あるいは、弔い
  - 食堂で朝食。トーストとスクランブルエッグとサラダ。澪は普通に食べる。彩音と他愛ない話をする。授業の予習。週末の自由時間の計画。何も変わっていないかのように
  - だが、トーストを口に運ぶ手が、一瞬だけ止まる。右肩——昨夜、鷹宮の唇が触れた場所。制服の下で、まだ微かに残っている感覚。澪はトーストを噛み、飲み込み、何事もなかったかのように会話を続ける
  - 食堂を出るとき、枕の下に隠した封筒のことを思い出す。次の外出許可日に、郵便局から母に送金しよう。母はきっと「どこからそんなお金が？」と聞くだろう。「奨学金の追加支給」と答えるつもりだ。嘘が、一つ増えた
- **描写のポイント**: 「何食わぬ顔」の演技。制服＝日常の鎧。昨夜の記憶と今朝の現実が交錯するフラッシュバック的描写。礼拝堂の神聖さと澪の「罪」の対比。上級生＝先輩天使との無言の繋がり。嘘が日常に組み込まれていく最初の一歩
- **感情の流れ**: 疲労と空虚 → 日常に戻る安堵 → 彩音への罪悪感 → 礼拝の讃美歌で心が揺れる → 先輩天使との視線＝孤独の共有 → 食事の中で昨夜の記憶が蘇る → 母への送金を決意＝「これでよかった」と自分を納得させる
- **文量目安**: 約3,000〜3,500字

### シーン8：ピアノの音色（エピローグ）
- **場所**: 礼拝堂
- **時間帯**: 同日の放課後（16時過ぎ）
- **登場人物**: 澪（一人）
- **出来事**:
  - 放課後、澪は一人で礼拝堂に来る。誰もいない。ステンドグラスの天使たちが夕陽に照らされて輝いている
  - 礼拝堂の隅にあるアップライトピアノ。楽園館のスタインウェイとは比べものにならない古い楽器。だが、澪にとってはこちらの方が馴染み深い
  - ピアノの前に座る。昨夜の楽園館で弾いたショパン。同じ曲を弾き始める。だが、音が違う。楽園館のスタインウェイは華やかで完璧な音を出した。この古いピアノは少し音がくぐもっていて、鍵盤が重い。——母のピアノ教室のピアノと、同じ手触り
  - 弾きながら、涙が頬を伝う。泣いている理由がわからない。怖かったのか。悲しかったのか。それとも、鷹宮の腕の中で感じた安心感が、本物ではないと心のどこかでわかっているからか
  - 曲が終わる。澪は鍵盤の上に指を置いたまま、しばらく動かない。ステンドグラスの天使たちが、無言で澪を見下ろしている
  - 深呼吸。涙を拭く。ピアノの蓋を閉じる。「大丈夫。私は大丈夫」。自分に言い聞かせて、礼拝堂を出る
  - 夕暮れの庭園。オレンジ色の空。白樺が風に揺れている。遠くから彩音が「澪ちゃーん！　夕食だよー！」と手を振っている。澪は笑顔を作って駆け寄る
  - 最後の一文：「この日から、白河澪は二つの顔を持って生きることになった。白百合館の優等生と、楽園館の天使。その二つの世界を繋ぐものは、彼女のピアノの音色だけだった」
- **描写のポイント**: 楽園館のスタインウェイと礼拝堂のアップライトピアノ＝二つの世界の象徴。澪の涙＝言語化できない感情の発露。ステンドグラスの天使＝「天使」という言葉の二重の意味。最後は日常に戻る（だが読者は澪の内面を知っている）。ラストの一文で物語の構図を提示
- **感情の流れ**: 一人の時間 → ピアノで感情が溢れる → 涙 → 自分を立て直す → 日常の仮面を被り直す → 二重生活の始まり
- **文量目安**: 約2,500〜3,000字

## EP1 描写方針

### 基本コンセプト
- **視点**: 三人称・澪視点寄り。澪の内面を丁寧に追いながら、鷹宮の行動は「澪から見た」範囲で描く（鷹宮の内面には踏み込まない）
- **文体**: やや抒情的な文語体。五感描写を多用。心理描写は直接的な独白よりも、身体反応や環境描写を通じて暗示する
- **時制**: 過去形基調
- **学園パート**: 明るい色調。白・青・緑を基調とした色彩描写。朝〜夕方の自然光
- **楽園館パート**: 暗い色調に暖色の灯り。オレンジ・琥珀・金を基調。キャンドルの揺らぎ。アロマの香り
- **行為シーンの描写レベル**: 第三天使の範囲（添い寝＋軽い身体接触）。直接的な性器描写はなし。肌の感触、体温、息遣い、香りなどの間接的な官能描写を中心にする

### 重要モチーフ
- **ピアノ**: 澪のアイデンティティ。母との絆。鷹宮との接点。二つの世界を繋ぐもの
- **鏡**: 制服→天使の衣装→制服の変身を映す。アイデンティティの揺れの象徴
- **白樺の並木道**: 入学時に歩いた道。深夜に歩く道。同じ道の二つの顔
- **ステンドグラスの天使**: 礼拝堂の聖なる天使と、楽園館の「天使」の二重の意味
