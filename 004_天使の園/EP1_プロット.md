# EP1「鳥籠の白鍵」詳細プロット

> 白河澪（13歳・中等部2年・第三天使）の楽園館初訪問——消灯後の寮脱出から、パトロン・鷹宮隆一郎との食事会、ピアノの連弾、添い寝（第三天使の初任務）、そして翌日の礼拝堂でのピアノと涙まで。EP0で「特別奨学生プログラム」への参加を決意し、寮室のドアノブに手をかけた澪が、初めて「天使」として夜の世界に足を踏み入れる一夜を描く。

> **前提**: EP0（旧シーン1〜3）にて、聖エステル学院への入学、寮生活と彩音との友情、学院長からの選抜告知が描かれている。EP1はEP0ラスト（消灯後、寮室のドアノブに手をかける澪）から直結する。

## シーン構成（全6シーン）

### シーン1：森の小道（楽園館への道行き）
- **場所**: 白百合館（寮室）→ 寮棟の廊下 → 裏口 → 森の小道 → 楽園館 → 準備室
- **時間帯**: 6月下旬、初任務の夜。22時消灯後〜22時45分頃
- **登場人物**: 澪、寮監、楽園館管理人
- **出来事**:
  - EP0ラストからの直結。ドアノブを握りしめた手が、静かに回される。澪はベッドの中で寝息を立てている彩音を振り返る。月明かりが彩音の頬に青白い影を落としている。「行ってくるね」——声に出せない別れの言葉を胸の中だけで呟く
  - 制服の上にカーディガンを羽織っただけの姿。ポケットには何も入っていない。澪は部屋を出て、音を立てないように扉を閉める。カチリと噛み合う金具の音が、静まり返った廊下に小さく響く
  - 廊下に出ると、寮監が壁際に立って待っている。懐中電灯を手にした影が、澪を見つけて小さく頷く。「こちらへ」。それだけ。会話はない。二人の足音——澪の上履きがリノリウムの床を踏む柔らかい音と、寮監のヒールが規則正しく刻むリズム。月明かりが窓から差し込んで、廊下の床に光の矩形を並べている
  - 寮棟の裏口。鍵を開ける金属音。6月の夜気が澪の頬を撫でる。予想したよりも冷たい。カーディガンの前を合わせる。白樺の木々が月光に照らされて青白い骨のように立ち並んでいる
  - 森の中の小道を歩く。昼間は何度も散策したことのある道。だが夜は同じ道が別の場所のように見える。懐中電灯の光が足元を円形に切り取り、その外側は深い闇。草を踏む足音。虫の声。遠くでフクロウが鳴いている。澪の心臓が耳の奥で打っている
  - 寮監は一定の距離を保って前を歩く。振り返りもしない。澪の緊張を和らげようという気遣いもない。「仕事」として少女を夜の森に導いている。その事務的な態度が、かえって澪の不安を増幅させる
  - 木々の隙間から、暖かいオレンジ色の灯りが漏れてくる。楽園館。昼間、森の奥にぼんやりと見えていた「立入禁止の別棟」。赤レンガの2階建て。校舎と同じ洋式建築だが、ツタが壁面を這い、入口にはアイアンのアーチが薔薇に絡まれている。窓の向こうにシャンデリアの光がゆらめいている
  - 寮監が足を止める。「ここまでです。中で指示を受けてください」。声に感情がない。澪が「ありがとうございます」と答えると、寮監は何も言わずに踵を返す。懐中電灯の灯りが遠ざかっていき、闇に呑まれて消える。澪は一人、楽園館の扉の前に立っている
  - 重い木の扉を押す。温かい空気が顔に触れる。花の香り——百合か、それとも薔薇か——にアロマが混ざった、甘く重い空気。玄関ホールは校舎よりも遥かに豪奢で、大理石の床に澪の上履きが場違いに映る。壁には天使を描いた油彩画が掛かり、天井からはシャンデリアが柔らかい光を落としている
  - 楽園館の管理人（中年女性、落ち着いた物腰）が奥から現れ、微笑む。「白河さんね。待っていたわ。初めてでしょう、緊張してる？」。澪が頷くと、「大丈夫。何も難しいことはないから。まず、お着替えをしましょう」と手を取られ、準備室へ導かれる
  - 準備室。明るい照明、壁一面の鏡、ハンガーラック。管理人が白い衣装を差し出す。「これに着替えてね。制服はここに置いておけばいいから」
  - 着替えの場面。澪が制服を一枚ずつ脱いでいく。赤いリボンを外す。ブラウスのボタンを上から順に外していく。紺のプリーツスカートのホックを外し、下ろす。制服が椅子の上にきちんと畳まれていく——この几帳面さが澪という少女の輪郭を示す。下着姿になった自分を鏡の中に見る。白い肌。華奢な肩。発育途上の薄い胸元。「白河澪」という学院の生徒が、布一枚になって鏡の中に立っている
  - 天使の衣装を身に纏う。白いシルクのキャミソール——肩紐が細く、鎖骨と肩のラインが露出する。膝丈のレースのガウンを羽織る。素足にサテンのスリッパ。管理人が髪を下ろし、耳元に銀の十字架のイヤリングをつけてくれる。指先が耳朶に触れるひんやりとした感触
  - 鏡の前に立つ。数分前まで制服を着ていた中学2年生が、見知らぬ少女に変わっている。白いシルクが澪の黒髪と菫色の瞳を際立たせ、レースのガウン越しに華奢な体のラインがうっすらと透けている。「これが……天使」と澪が呟く。声が自分のものに聞こえない
  - 管理人が鏡越しに微笑む。「綺麗よ」。そして最終確認。「今夜のお相手は鷹宮様。この学院を長く支えてくださっている方よ。最初にお食事と会話をして、あなたのピアノを聴いていただくわ。それから……添い寝をお願いされるかもしれないけれど、それだけ。第三天使のあなたに、それ以上のことをお願いすることはないから。安心して」
  - 「添い寝」という言葉に、澪の指先がガウンの裾を無意識に握る。だが、頷く。「はい、わかりました」。声だけは平静を保つ。母の笑顔が脳裏をよぎる。この一夜で得られるものが、母を少しでも楽にできるなら
  - 管理人に導かれ、準備室を出る。廊下の奥、二つ並んだ扉の片方が開かれる。暖かい光と、微かにウッディな香水の匂いが漂ってくる
- **描写のポイント**: EP0ラスト（ドアノブ）からのシームレスな接続。夜の学園の美しさと不穏さの共存。「生徒→天使」への変身過程を衣装の変化で丁寧に描写（制服を脱ぐ→天使の衣装を纏う＝アイデンティティの転換儀式）。楽園館の豪奢さと校舎・寮との落差。管理人の「安心して」が逆に不安を煽る構造。鏡に映った自分を見る場面がアイデンティティの揺らぎを象徴する
- **感情の流れ**: 緊張と決意 → 彩音の寝顔への罪悪感 → 夜の森の異世界感に心細さ → 楽園館の豪華さへの圧倒 → 着替えの羞恥と非日常感 → 鏡の中の「天使」への戸惑い → 「添い寝」への不安 → 母を想って覚悟を固める
- **文量目安**: 約4,500〜5,000字

### シーン2：食事会（パトロンとの対面）
- **場所**: 楽園館・ダイニングルーム
- **時間帯**: 同夜、23時頃〜23時40分頃
- **登場人物**: 澪、鷹宮隆一郎
- **出来事**:
  - ダイニングルームへ通される。八畳ほどの親密な空間。壁には風景画。円卓にキャンドルが灯り、テーブルには二人分のセッティング。白いクロス。銀のカトラリー。グラスに注がれた赤い液体——澪のグラスはノンアルコールのぶどうジュース、もう片方には深い赤のワイン
  - 奥の椅子から鷹宮隆一郎が立ち上がる。ダークグレーのスーツにシルバーの腕時計。白髪の混じった鉄灰色の髪がオールバックにきちんと整えられている。低い声が穏やかに響く。「初めまして。鷹宮です」。微笑みには威圧感がなく、品のある温かさがある
  - 澪が立ち尽くしたまま挨拶する。「白河澪です。今夜は……よろしくお願いいたします」。声が微かに上ずっている。天使の衣装がこの部屋の空気に溶け込んでいることに、澪自身が戸惑っている。自分だけが場違いなのか、それともこの白い衣装こそがこの場にふさわしいのか
  - 鷹宮が澪の椅子を引く。その動作が自然で、紳士的で、澪はされるがままに座る。「緊張しなくていいよ。今夜はただの食事会だ。美味しいものを食べて、少しお話をするだけ」。低く安定した声。ゆっくりとした話し方。澪の緊張をほぐすことに慣れている——そう感じさせる落ち着き
  - 料理が運ばれてくる。アミューズ、前菜、魚料理、肉料理、デセール。学院の食堂とは次元の違うフレンチのフルコース。澪は食事作法の授業で学んだ通りにナイフとフォークを使う。その手つきに迷いがないことを、鷹宮は静かに観察している
  - 食事中の会話。鷹宮は質問の名手だ。澪が答えやすい話題から始める。「学院の生活はどうですか？」「好きな授業は？」。澪が短く答えると、少し間を置いてから次の質問をする。急かさない。聞き上手。澪は気づかないうちに口数が増えていく
  - 「ピアノを弾かれるそうですね。どんな曲がお好きですか？」。澪の目にわずかな光が灯る。「ショパンが……好きです。ノクターンとか」「なるほど。ショパンか。夜想曲はいいね。哀愁と甘さが同居している」。鷹宮の言葉に教養が滲む。澪は少しだけ安心する——この人は音楽を理解している、と
  - 母の話題へ。「お母様もピアニストだったと聞きました」。鷹宮の声が一段柔らかくなる。澪の手がわずかにナイフを握り直す。「……はい。母は、手を壊してしまって。今はピアノ教室を」。視線を皿に落とす。「でも、母が教えてくれたから、私はピアノが弾けるんです」
  - 鷹宮が静かに頷く。「そうですか。大変でしたね」。声に嘘がない——ように聞こえる。澪の胸の奥で何かが緩む。この人には話してもいい、と無意識に感じ始めている。それこそが鷹宮の技術であることに、澪はまだ気づかない
  - 鷹宮が自分の話をする。「私の妻も、ピアニストでした。10年前に亡くしましてね」。遠い目をして、ワイングラスを傾ける。「あなたの履歴を聞いたとき、少し驚きました。妻の若い頃に、雰囲気が似ている」
  - 澪が「そうだったんですか……」と返す。亡き妻のピアニストという共通項。母を想う少女と、妻を喪った男。二人の間にピアノを通じた共感が芽生える——ように見える。しかし鷹宮が提示する「共感」は、少女の警戒を解く鍵として周到に選ばれたものでもある
  - デセールのクレームブリュレに、鷹宮が自分のスプーンで少しだけすくい、澪の皿の縁に添える。「こちらの方が美味しいから、比べてみて」。その自然な動作に、澪は「ありがとうございます」と反射的に答える。大人の男が少女の皿に手を伸ばすことの意味を、13歳の澪は読み取れない
  - 食事が終わる。鷹宮がナプキンを置いて微笑む。「さて、食後のお楽しみだ。ピアノ、聴かせてもらえますか？」。隣の部屋を示す。澪は「はい」と答える。ピアノがあるなら大丈夫——音楽の前でなら、自分でいられる
- **描写のポイント**: 鷹宮の「紳士的な導入」の精緻さ——質問の選び方、間の取り方、自己開示のタイミングが全て計算されている。しかし澪の視点からはそれが「優しい大人」にしか見えない構造。母と妻という共通の傷を通じた心理的接近。食事作法という「学院で学んだこと」がここで機能する皮肉。デセールの一匙に埋め込まれた距離の侵食
- **感情の流れ**: 
  - 澪: 緊張で身が竦む → 鷹宮の穏やかさに少しずつ安堵 → ピアノの話題で心がほどける → 母の話で感情が揺れる → 鷹宮の亡き妻の話に共感 → 食事を終えてピアノへの期待
  - 鷹宮（澪の目を通して）: 紳士的な余裕 → 澪の教養ある振る舞いへの好感（のように見える） → 妻の話で見せる寂しげな表情 → ピアノへの誘導
- **文量目安**: 約4,000〜4,500字

### シーン3：ピアノ（連弾）
- **場所**: 楽園館・サロン（グランドピアノのある部屋）
- **時間帯**: 同夜、23時40分頃〜翌0時10分頃
- **登場人物**: 澪、鷹宮隆一郎
- **出来事**:
  - ダイニングルームの隣、サロンへ移動する。ダイニングよりも広い部屋。壁の半分が書棚で、残り半分に風景画と鏡が掛かっている。暖炉には火が入っていないが、壁のブラケットランプが琥珀色の光を落としている。部屋の中央にグランドピアノ——スタインウェイ。黒い漆のような塗装が灯りを反射している
  - 澪がピアノを見た瞬間、呼吸が変わる。学院の音楽室にあるヤマハとは違う。楽器というより、ひとつの生き物のような存在感。蓋が開いた状態で、弦が金色に光っている
  - 鷹宮がソファに腰を下ろし、足を組む。「好きな曲を。何でもいい」。ワイングラスを片手に、余裕のある姿勢。澪に選択を委ねる——その「自由」すらも、少女を試す装置であることを、鷹宮は知っている
  - 澪がピアノの前に座る。椅子の高さを調整する。鍵盤に指を置いた瞬間——ダイニングでの緊張が、指先から溶け出していく。これだけは自分のものだ、と体が思い出す
  - ショパンのノクターン第2番変ホ長調。最初の一音が部屋に放たれる。スタインウェイの音は圧倒的に豊かで、学院のピアノとは別次元の響きが空間を満たす。澪の指が鍵盤の上を滑り始める。旋律が夜の空気に溶けていく
  - 演奏する澪の描写。白いシルクのキャミソール、レースのガウン、キャンドルの灯りに照らされた横顔。目を閉じて弾いている。ピアノの前の澪は無防備で、学院での「完璧な優等生」の鎧がすべて外れている。細い指が鍵盤を叩くたびに、小さな肩が微かに揺れる
  - 鷹宮の視線。ソファから澪を見つめるその目に浮かぶ複雑な感情——純粋な音楽への感動と、亡き妻への郷愁と、白いシルクに包まれた少女の姿への欲望が渾然と混ざっている。しかし表情は穏やかなまま。グラスのワインが琥珀色に揺れている
  - 演奏が終わる。最後の和音の余韻が天井に吸い込まれていく。沈黙。数秒後、鷹宮が静かに拍手する。「素晴らしい」。声がわずかに掠れている。「……妻の演奏を思い出しました。あの頃の音に、よく似ている」
  - 澪が「ありがとうございます」と頬を染める。初めてのパトロンに自分のピアノを褒められた——その素朴な喜びが、13歳の少女の顔に浮かんでいる。音楽を通じて認められることは、澪にとって何よりの報酬だ
  - 鷹宮が立ち上がり、ピアノの横に来る。「一つお願いがある。連弾をしませんか」。澪が驚いて見上げる。「鷹宮さんも、弾かれるんですか？」「昔、少し。妻に教わったんだ。下手だけれど」
  - 鷹宮が澪の隣に座る。ピアノの椅子は一つ。長椅子だが、大人の男と少女が並ぶと距離は近い。鷹宮の肩が澪の肩の少し上にある。ウッディな香水の匂いが近い。スーツの布地の感触が、澪の天使の衣装の薄さを意識させる
  - シューベルトの軍隊行進曲。鷹宮がバス側、澪がトレブル側。鷹宮の大きな手が低音部で安定したリズムを刻む。正確で、温かみのある音。「昔少し」にしては達者な腕前。澪はそのことに気づくが、深くは考えない
  - 連弾が進むにつれ、二人の呼吸が同期していく。音楽が二人を一つの流れに組み込む。鷹宮の左手が低音を叩くとき、その腕の動きで肩が澪の肩に触れる。大人の男の体温。硬い肩の骨。澪の心臓が一拍跳ねる。だが指は止まらない。音楽に集中することで、体の反応を意識の外に押しやれる
  - 終盤、鷹宮が意図的にテンポを落とす。最後のフレーズで、鷹宮の右手が鍵盤の中央付近まで来て、澪の左手のすぐ隣に置かれる。指と指の間に数センチ。最後の和音を、二人の手がほぼ重なるように押さえる。鍵盤の上で、52歳の男の手と13歳の少女の手が並んでいる。大きさの違い。骨の太さの違い。その対比
  - 音が消える。鷹宮が「楽しかった」と笑う。それから立ち上がり、澪を見下ろす。「ありがとう。……そろそろ、奥の部屋に行こうか」。穏やかだが、転換を告げる声。ピアノの時間は終わり、夜の本題が始まる
  - 澪がピアノの椅子から立ち上がる。さっきまで鍵盤の上にあった指先が、行き場を失ってガウンの裾を握る
- **描写のポイント**: ピアノ演奏シーンがこのエピソードの核心。澪にとってピアノは唯一の自己表現であり、最も無防備になる瞬間。それを鷹宮が「観客」として享受する構造。連弾は身体的距離の縮小の「口実」として機能する——音楽という名目のもとで肩が触れ、手が近づく。鍵盤の上の二人の手の大きさの対比が、関係の非対称性を視覚的に示す。鷹宮の「そろそろ奥の部屋に」が、音楽の余韻を断ち切る転換点
- **感情の流れ**: 
  - 澪: スタインウェイへの感動 → 演奏に没入し緊張が解ける → 褒められた喜び → 連弾の提案への驚きと小さな嬉しさ → 肩が触れる近さへの動揺 → 音楽で平静を保つ → 「奥の部屋」への不安が蘇る
  - 鷹宮（澪の目を通して）: 音楽を楽しむ穏やかさ → 妻を重ねる郷愁 → 連弾で距離を詰める自然さ → 次の段階への移行

### シーン4：添い寝（天使の初任務）——行為シーン
- **場所**: 楽園館・2階の寝室

### シーン5：帰還（楽園館から寮へ）

- **感情の流れ**: 脱力と余韻 → 鏡の中の自分への違和感 → 制服を着る安堵感 → 冷たい夜風で現実に戻る → 帰り道で記憶が蘇る → 「怖くなかった」ことへの不安 → 彩音の寝息に罪悪感 → 自分のベッドに戻る安心 → 明日への覚悟と嘘の重み

### シーン6：ピアノの音色（エピローグ）
- **場所**: 礼拝堂 → 庭園

  - 放課後、澪は一人で礼拝堂に向かう。彩音には「ちょっとピアノを弾いてくる」とだけ伝える。嘘ではない。ただ、その言葉に込められた意味が、昨日までとは違う
  - 礼拝堂。誰もいない。午後の光がステンドグラスを通って床に色とりどりの模様を描いている。天使を象ったステンドグラスの顔が、逆光で表情が読めない。聖なる天使と、楽園館の「天使」。同じ名前の、まるで違うもの
  - 礼拝堂の隅にあるアップライトピアノ。古い楽器。象牙の一部が黄ばんでいて、鍵盤が少し重い。楽園館のスタインウェイとは比べものにならない。けれど、澪にとってはこちらの方が馴染み深い。母のピアノ教室にあったピアノと同じメーカーで、同じように少し音がくぐもる
  - ピアノの前に座る。昨夜、楽園館のサロンで弾いたショパンのノクターン第2番。同じ曲を弾き始める。——だが、音が違う。スタインウェイは華やかで完璧な響きを返してきた。この古いアップライトは、音が柔らかく丸くて、少しだけ哀しい。母のピアノ教室の音に、よく似ている
  - 弾きながら、指が覚えている。昨夜の鍵盤の感触を。スタインウェイの滑らかな象牙の手触りを。そしてその隣に並んでいた、鷹宮の大きな手を。肩に触れた体温を。背中を撫でた指を
  - 涙が頬を伝い始める。止められない。なぜ泣いているのかわからない。——怖かったのか？　怖くなかった。悲しかったのか？　何が悲しいのかわからない。鷹宮は優しかった。痛いことは何もされていない。ただ、鷹宮の腕の中で感じた安心感が、本物ではないと——心のどこかで、薄い紙の裏に書かれた嘘のように、わかっている
  - あるいは。母のピアノと同じ音がするこの楽器が、自分がどこから来たのかを思い出させるからかもしれない。関節が腫れた母の指。「澪、あなたは才能があるの」。あの声。あの手。その手を救うために、自分は昨夜——
  - 曲が終わる。澪は鍵盤の上に両手を置いたまま、しばらく動かない。涙の雫がドの鍵盤に一粒落ちて、象牙の上で小さく光る。ステンドグラスの天使たちが、色とりどりの光の中で、何も言わずに澪を見下ろしている
  - 深呼吸。目元を手の甲で拭う。鍵盤に落ちた涙の跡を袖で丁寧に拭き取る。ピアノの蓋を静かに閉じる。「大丈夫。私は大丈夫」。声に出さず、唇だけが動く
  - 礼拝堂の重い扉を押して外に出る。夕暮れの庭園。オレンジ色に染まった空。白樺の木々が風に揺れて、葉擦れの音が澪を迎える。世界は昨日と何も変わっていない——変わったのは、澪の側だ
  - 遠くから声が聞こえる。「澪ちゃーん！　夕食だよー！」。彩音が庭園の向こうで手を振っている。風に髪が揺れている。何も知らない笑顔。澪はその笑顔に向かって——一瞬だけ、足が止まる。それから、完璧な微笑みを顔に浮かべて、駆け出す
  - 最後の一文：「この日から、白河澪は二つの顔を持って生きることになった。白百合館の優等生と、楽園館の天使。その二つの世界をつなぐものは、彼女のピアノの音色だけだった。——そして、その音色はもう、昨日までと同じ音ではなかった」
- **描写のポイント**: 楽園館のスタインウェイと礼拝堂のアップライトピアノの対比＝二つの世界の象徴。母のピアノ教室と同じ音がする古いアップライトが、澪の「出自」と「動機」を呼び覚ます。涙の理由を澪自身が言語化できないことが、13歳の少女の限界と複雑な心理を表現する。ステンドグラスの天使＝聖なる「天使」と楽園館の「天使」の二重の意味。ラストの彩音の呼びかけが日常への帰還を象徴し、澪がそこに「完璧な微笑み」で応えることで二重生活の幕開けを示す。最後の一文でEP1全体の構図を提示し、「音色が変わった」ことでEP2以降への橋渡しとする
- **感情の流れ**: 一日をやり過ごした疲労と空虚 → 礼拝堂の静寂に一人になれる安堵 → ピアノを弾いて感情が溢れ出す → 涙（言語化できない感情の発露） → 母のことを思い出して胸が痛む → 涙を拭いて自分を立て直す → 日常の仮面を被り直す → 二重生活の始まり
- **文量目安**: 約3,000〜3,500字

## EP1 描写方針

### 基本コンセプト
- **視点**: 三人称・澪視点寄り。澪の内面を丁寧に追いながら、鷹宮の行動は「澪から見た」範囲で描く（鷹宮の内面には踏み込まない）。鷹宮の真意は行動と台詞の端々から読者に推測させる
- **文体**: やや抒情的な文語体。五感描写を多用。心理描写は直接的な独白よりも、身体反応や環境描写を通じて暗示する
- **時制**: 過去形基調
- **時間軸**: EP1は一夜＋翌日放課後の約20時間を描く。全シーンが連続する一本の時間軸上にある
- **楽園館パート**（シーン1〜5）: 暗い色調に暖色の灯り。オレンジ・琥珀・金を基調。キャンドルの揺らぎ。アロマの香り。シルクの感触
- **学園パート**（シーン6）: 明るい色調。白・青・緑を基調とした色彩描写。午後〜夕方の自然光。ステンドグラスの多色の光
- **行為シーンの描写レベル**: 第三天使の範囲（添い寝＋軽い身体接触）。直接的な性器描写は一切なし。肌の感触、体温、息遣い、香り、心臓の鼓動などの間接的な官能描写を中心にする。処女喪失は描かない（EP2以降に持ち越し）

### 重要モチーフ
- **ピアノ**: 澪のアイデンティティ。母との絆。鷹宮との接点。二つの世界をつなぐもの。楽園館のスタインウェイ（華やか・完璧・他者の世界）と礼拝堂のアップライト（くぐもった音・母の記憶・自分の原点）の対比
- **鏡**: 制服→天使の衣装→制服の変身を映す。着替えのたびに鏡を見る澪は、自分が何者なのかを問い直している
- **白い封筒**: 報酬の象徴。母への想いと「代償」の物質化。枕の下に隠す動作が秘密の重みを描く
- **彩音の寝息**: 「知らない世界」の象徴。澪が二重生活に踏み出したことを、寝息のリズムの変わらなさが際立たせる
- **ステンドグラスの天使**: 礼拝堂の聖なる天使と、楽園館の「天使」の二重の意味。無言で見下ろす存在

### 鷹宮の「優しい支配」の描写指針
- 暴力・脅迫・直接的な強制は一切使わない
- 共感→安心→段階的接触→境界線の提示（「これ以上はしない」）→信頼の獲得、というサイクル
- 澪が「自分の意思で受け入れた」と思い込む余地を、全ての段階に設計する
- 読者には構造が見えるが、澪には見えない——この情報格差がシーンの緊張を生む
- 鷹宮の内面描写は避け、行動と台詞だけで「紳士的な仮面」の裏側を匂わせる
