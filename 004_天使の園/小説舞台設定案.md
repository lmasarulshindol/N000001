# 天使の園（エンジェルガーデン）

> 小説舞台設定案

---

## 1. 世界観概要

名門全寮制女子校「聖エステル学院」。外界から隔絶された広大な敷地の中で、選ばれた少女たちが共同生活を送っている。表向きは上流階級の令嬢を育てるエリート教育機関だが、その裏では、学院が秘密裏に運営する「天使の園（エンジェルガーデン）」と呼ばれるプログラムが存在する。

少女たちは「天使」として、学院を支援するパトロン（富裕層の男性たち）に"奉仕"することで、学費免除と将来の地位を約束される。

### 時代・場所
- **時代**: 現代日本
- **場所**: 長野県の山間部。最寄り駅から車で40分。周囲は深い森に囲まれている
- **外界との接触**: 携帯電話は入学時に没収。外出は月1回の許可制。保護者との面会は学期に1回
- **社会的な認知**: 一般には「躾の厳しいお嬢様学校」として知られている。真の姿を知る者はごく限られている

## 3. 「天使の園」プログラム

### 制度の概要
学院の運営資金の大部分は、富裕層のパトロンからの「寄付」で成り立っている。その見返りとして、パトロンは「天使」に指定された少女との時間を得る。

### 天使の選抜

| 条件 | 詳細 |
|------|------|
| 対象 | 中等部2年生（13歳）以上 |
| 選抜基準 | 容姿・品性・成績を総合的に評価。学院長が最終決定 |
| 人数 | 常時10〜15名が「天使」として活動 |
| 本人への伝え方 | 「特別奨学生プログラム」として説明。最初は食事会やピアノ演奏の「接待」から始まる |
| 段階的な移行 | 接待 → 添い寝 → 身体的奉仕、と段階を踏んで慣れさせる |
| 拒否権 | 建前上は存在するが、拒否すれば特待生資格剥奪=退学となるため事実上の強制 |

### 天使のランク

| ランク | 名称 | 奉仕の範囲 | 報酬 |
|--------|------|-----------|------|
| **第三天使** | セラフ・トワ | 食事・会話の接待、添い寝まで | 学費全額免除 |
| **第二天使** | セラフ・ドゥ | 口での奉仕まで | 学費免除 + 月額小遣い10万円 |
| **第一天使** | セラフ・アン | 全ての奉仕（本番行為含む） | 学費免除 + 月額30万円 + 卒業後の就職斡旋 |
| **大天使** | アルカンジュ | パトロンの「専属」。常時対応 | 学費免除 + 月額50万円 + 卒業後の留学・起業支援 |

### パトロンについて
- 学院に多額の寄付をする富裕層の男性。政財界の重鎮、大企業の経営者、芸術家など
- 入会審査は極めて厳格。身元調査・資産証明・既存会員2名の推薦が必要
- 年会費500万円 + 面会ごとの「寄付」（1回あたり20〜100万円）
- 現在の会員数は約30名

## 4. 学院の日常

### 一日のスケジュール

| 時間 | 内容 |
|------|------|
| 6:00 | 起床 |
| 6:30 | 礼拝堂で朝の礼拝 |
| 7:00 | 朝食（食堂。全員揃って「いただきます」） |
| 8:00〜12:00 | 午前授業（通常教科＋マナー・礼法） |
| 12:30 | 昼食 |
| 13:30〜16:00 | 午後授業（芸術科目・語学・体育） |
| 16:00〜18:00 | 自由時間（部活動・自習・庭園での散策） |
| 18:30 | 夕食 |
| 19:00〜21:00 | 自習時間 |
| 21:30 | 入浴（大浴場。学年ごとの時間割り当て） |
| 22:00 | 消灯 |
| ※天使の活動 | 消灯後、または休日に楽園館で行われる |

### 学院の表の文化
- **規律正しい生活**: 全員が同じ制服を着用。白いブラウスに紺のプリーツスカート。赤いリボン
- **芸術教育の重視**: ピアノ・ヴァイオリン・バレエ・絵画が必修。発表会が定期的に開催される
- **礼法の徹底**: 食事作法、歩き方、お辞儀の角度まで指導される
- **閉鎖的な空気**: 外の情報が極端に少ない。生徒たちは学院の中だけで完結する世界を生きている

### 生徒間の人間関係
- **天使と一般生徒の格差**: 天使に選ばれた生徒は「特別奨学生」として優遇される。一般生徒からの羨望と嫉妬
- **天使同士の連帯**: 秘密を共有する者同士の強い絆。同時に、パトロンの指名を巡る微妙な競争関係
- **先輩後輩**: 上級生の天使が下級生を指導する「姉妹制度」。新人の天使の心理的ケアも担う
- **友情と秘密**: 親友にも天使であることを明かせない（明かしてよいのは天使同士のみ）。二重生活のストレス

## 7. ストーリー構成

### 基本コンセプト
聖エステル学院に入学した少女が、「天使の園」に選ばれ、美しい学院の裏側を知っていく物語。閉鎖的な学園という「箱庭」の中で、少女たちの友情・葛藤・秘密・成長が交錯する。

### 物語の軸
- **箱庭の美と闇**: 外界から隔絶された美しい学園という舞台が、そのまま「逃げ場のない構造」として機能する
- **段階的な侵食**: 接待→添い寝→奉仕と段階を踏む構造が、少女の心理変化を丁寧に描く装置になる
- **少女たちの関係**: 天使同士の連帯と競争、一般生徒との二重生活、先輩から後輩への継承
- **パトロンとの複雑な関係**: 単なる搾取ではなく、中には少女に本気で入れ込む者、父親のように接する者もいる。その曖昧さが物語を豊かにする

### 描写のポイント
- 学園生活の美しい日常描写（授業・寮生活・友人関係）と、夜の楽園館でのシーンのコントラスト
- 制服を着た「生徒」の姿と、楽園館で纏う「天使」の姿の切り替わり
- 閉鎖空間ならではの心理的圧迫と、その中で見つける小さな幸せ

## 8. 性行為シーンの描写方針

### 基本設計思想
- 「天使の園」での奉仕シーンは、段階的なランクアップに沿って描写が深化していく構成
- 学園の清楚な雰囲気と、楽園館での行為の落差が最大の演出ポイント
- 少女の「制服→天使の衣装→裸体」という衣装の変化が、心理的な段階変化を象徴する

### 描写の三層構造

**① 導入（楽園館に向かうまで）**
- 消灯後、暗い廊下を一人で歩く少女。月明かりに照らされる制服姿
- 森の小道を抜けて楽園館へ。夜の森の静けさと、胸の鼓動の対比
- 楽園館で「天使の衣装」に着替える。白いランジェリーとレースのガウン
- パトロンとの対面。食事やワインから始まるゆったりとした時間

**② 行為本体**
- パトロンの年齢・性格・嗜好による多様なシチュエーション
- 少女の反応: ランクに応じた慣れと、それでも消えない緊張
- 環境描写: 楽園館の調度品、アロマの香り、間接照明、シルクの感触
- 学園の「お嬢様」が見せる、普段とは異なる表情・声・仕草

**③ 余韻（行為後）**
- パトロンとの事後の会話（意外と穏やかだったり、冷たかったり）
- 楽園館を出て、夜の森を戻る。制服に着替え直す瞬間の心理
- 寮に戻り、同室の友人が寝息を立てている。何事もなかったかのようにベッドに入る
- 翌朝、何食わぬ顔で礼拝に出席する二重生活

### シーンごとの文量目安
- 導入: 全体の **25〜30%**（楽園館への道行きを丁寧に）
- 行為本体: 全体の **45〜55%**
- 余韻: 全体の **20〜25%**
- 1回の行為シーン全体で **3,000〜5,000字** を目安
