# 官能描写ガイドライン — 普通のつもり

> **用途**: 性行為シーンのセリフ・喘ぎ声・場面説明を、生々しく臨場感ある表現にするための執筆参照。
> **優先順位**: `創作設定.md`（キャラ・話数境界）＞ **本ファイル** ＞ 執筆時の推測。
> **Browser調査**: 2026-06-30（官能小説執筆ノウハウ・喘ぎ声表現論・英語圏 intimacy writing を統合）

---

## 1. 基本方針 — 「文字だけで五感を再現する」

小説は**文字しかない**。アニメや実写と違い、絵・色・声・音楽は読者の想像に委ねる。  
だからこそ、**喘ぎ1行・音1語・地の文1節**に意味を込める（瑞木夕 note／kgmkm 喘ぎ声スキル）。

| 原則 | 内容 |
|------|------|
| **Show, don't tell** | 「興奮した」ではなく「息が、途切れる」「指先が、震える」 |
| **感情が先** | 解剖学の説明より、さえかの「おかしい」と身体のズレ |
| **リズム** | 高まり＝短文。余韻・前戯＝やや長文。**拍**を変える |
| **Beat（間）** | セリフ↔動作↔内心↔環境音を交互に。**同じ型の連打禁止** |
| **Less is more** | 汎用ダーティトークより、キャラに合った**一言** |

## 2. シーンの段階設計（全キャラ共通）

1シーンは **6段階** を意識する。段階が上がるほど、**言葉→ひらがな→母音→無言** へ。

| 段階 | 名称 | セリフ | 喘ぎ | 地の文 | 音 |
|------|------|--------|------|--------|-----|
| 0 | **導入** | 日常会話・冗談 | なし | 五感の予兆（匂い・距離・視線） | 換気扇・足音 |
| 1 | **境界** | 短い確認「……ダメ？」 | 咬み唇・息のみ | 固まる／視線を逸らす | 布・ベルト |
| 2 | **前戯** | 断片（漢字混じり可） | 「ん……」「は、……」 | 触覚・温度・羞恥 | 衣擦れ・水音 |
| 3 | **深入** | 途切れる文 | ひらがな崩れ | 腰・首など弱点 | ばちゃ（浅） |
| 4 | **絶頂前** | 名前・単語のみ | 促音・カタカナ混じり | 同期・弓なり | 軋み・打ち付け |
| 5 | **絶頂** | 1語 or 崩れた1文 | 短い崩れ（1〜2行） | 白くなる・震え | 長い声→静けさ |
| 6 | **事後** | キャラ固有の一言 | 荒い呼吸 | 体液・重さ・余韻 | シャワー・廊下 |

**本作の鉄則**: 段階を**飛ばさない**。特にさえかは、第1話で声なし→第2話小さな声→第3話初絶頂、と**話数で段階上限**がある（`創作設定.md` 感情曲線参照）。

## 3. セリフの書き方

### 3-1. 本作で必須のルール

| ルール | OK | NG |
|--------|----|----|
| **文として成立** | 「だって……はあ、んっ、二人同時なんて、無理……っ、待っ——」 | 「ん、、、」「あ、、、」「深、、、い、、、」 |
| **息は文中に** | 会話の**途中**に呼吸を挟む | 単語ごとに読点で切る |
| **キャラ口調** | 東条＝ですます／樫原＝タメ口短句／悠真＝柔らかい | 全員同じ「もっと」「感じて」 |
| **量** | 1段落にセリフ1〜2行＋地の文 | 喘ぎだけ5行続ける |

### 3-2. 緊張が上がるほど短文（Penwise / SmutWriter）

- **低 tension**: 「今日、カレー、作り直そ。——さえかちゃん、手、伝わってきた」
- **高 tension**: 「……**一度だけ**、……お願い、します」
- **絶頂**: 「……だめ、……**イ**、……ッ！」（1行で完結）

### 3-3. セリフの型（用途別）

| 型 | 例 | 使う場面 |
|----|-----|----------|
| **確認** | 「触っていいですか」「続けていいですか」 | 東条・合意重視 |
| **実況** | 「深い……っ、ぁ、前も……だめ……！」 | 御子柴・3P |
| **指示** | 「脚、開いて」「口、開けて」 | 樫原・直球 |
| **優しめ** | 「りん、前は任せて。——ゆっくり、息、吐いて」 | 相楽・前後 |
| **拒否（曖昧）** | 「痛っ……！ 待って、……」 | さえか・第3話以降 |
| **拒否（明確）** | 「……やめ、て」 | **第2話までのみ**（さえか） |

### 3-4. 攻めのセリフは「スパイス」

- 1ブロック（150〜250字）に**攻めの会話1〜2行**で十分。
- さえか視点では、**相手の低い声**を地の文で受ける：「東条さんが、低く、『いい声』と、言った」

## 4. 喘ぎ声の書き方

### 4-1. 記号と字面（日本語小説向け）

| 記号・技法 | 効果 | 本作での使用 |
|------------|------|--------------|
| **っ**（促音） | 息の詰まり・頂点 | ◎ 積極使用 |
| **……** | 間・余韻 | ◎ 地の文と喘ぎ両方 |
| **――** | 声の途切れ | ◎ 3Pで前後が塞がれる瞬間 |
| **ぁ/ぅ**（小書き） | 声の揺らぎ | ◎ 「あ」だけの単調回避 |
| **カタカナ混じり** | テンション上昇 | △ 絶頂前後のみ（例：「ッ……」） |
| **♡** | 視覚強調 | **× 本作不使用**（起伏が死ぬ） |
| **あ゛**（濁点母音） | 獣性・決壊 | **× 全面禁止** |

### 4-2. 同一音の連打禁止

| 禁止 | 理由 | 代替 |
|------|------|------|
| 「あっ！あっ！あっ！」 | 誰の声か分からない | リズムを変える：「はっ、んっ、はぁっ」 |
| 「んあっ♡ あっ♡ んっ♡」 | 起伏なし | ！と……で強弱 |
| 8文字以上同母音連続 | 単調 | 句読点・促音・単語を挟む |

### 4-3. 言葉→崩れのテンプレ（段階別）

**序盤（漢字・意味あり）**
```
「だって……はあ、んっ、二人同時なんて、無理……っ、待っ——」
```

**中盤（ひらがな増）**
```
「んっ……深い……っ、ぁ、前も……だめ……！」
```

**絶頂（短い崩れ）**
```
「イく……！ 三人、一緒……大翔、悠真、出して……ッ！」
```

**さえか・初絶頂（第3話）**
```
「……だめ、……イ、……ッ！」
```
→ 名前は呼ばない（距離・羞恥）。第5話以降で変化可。

### 4-4. 呼吸音を必ず混ぜる

喘ぎ**だけ**にしない。地の文に呼吸を織る。

```
はあ、ん、はあ。——同じ拍で、重なる。
息が、途切れる。——そのあと、静けさ。
```

**静かなシーン**は喘ぎゼロ・呼吸のみも有効（Olive Spencer／瑞木夕）。

## 5. 場面説明（地の文）— 五感レイヤー

1段落に**最低2種類**の感覚。視覚だけに偏らない（SmutWriter）。

| 感覚 | 本作の語彙例 | 用途 |
|------|--------------|------|
| **視覚** | 白い筋、ネイル、メガネ、汗 | 体位・表情 |
| **触覚** | 掌の温かさ、布越し、1.3倍の首 | さえか弱点 |
| **聴覚** | ばちゃ、軋む、壁越し、低い唸り | **本作の核** |
| **嗅覚** | 石鹸、コーヒー、別の匂い | 日常↔非日常 |
| **内感** | 下腹部の熱、重い、満ち方 | さえか視点 |

### 5-1. 音の三層（見晴台ハウス固有）

```
【人声】  高い息／低い唸り／名前
【物体】  ソファ・ベッド・ばちゃ・ちゅ（舌）
【建物】  壁一枚／フロア／換気扇／廊下足音
```

**壁越し**: 視覚より**聴覚**。第1〜2話はさえかが聞く側。第3話以降「こっちも軋む」で対称。

### 5-2. 動作の書き方

| NG（機械的） | OK（具体的） |
|--------------|--------------|
| 彼は動いた | 腰が、深く押し込まれる |
| 激しく愛し合った | ばちゃ、ばちゃ。短い。規則的。——**こっち**の、心臓と、同じ、拍 |
| 彼女は感じた | 腰が、勝手に、浮く |

**3P**: 前・後ろ・声の**三方向**を1文で束ねる（第1話 Scene2 参照）。

### 5-3. さえか視点の内心（必須）

性行為描写の**35〜40%**は、さえかの「普通／おかしい／向こう・こっち」。

```
身体は、入居初日の向こう側の記憶で、先に反応した。
——おかしい。なのに、止めない。
```

## 6. キャラ別 — 声・セリフ・リズム

### 6-1. 谷口さえか（視点・成長あり）

| 話数 | 声の上限 | 代表 |
|------|----------|------|
| 第1話 | 漏れそう（**出さない**） | 喉まで上がったのを飲み込んだ |
| 第2話 | 小さく1回 | 「……あ、……」／「……やめ、て」 |
| 第3話 | 初絶頂1回 | 「痛っ……！ 待って」「……イ、……ッ！」 |
| 第4話〜 | 段階的上昇 | （執筆時に更新） |

| 要素 | 設計 |
|------|------|
| 基本 | 咬み唇→小さな息→抑えきれない崩れ |
| 弱点 | 首・腰（1.3倍） |
| 絶頂 | 短い。名前は距離がある間は呼ばない |
| 事後 | 「……ごめんなさい」（クセ） |
| 禁止 | あ゛／陽菜型連打／第3話以降の明確「やめて」（第2話まで） |

### 6-2. 御子柴りん

| 要素 | 設計 |
|------|------|
| 基本音 | 「はっ」「んっ」「はぁ」（高め・明るい） |
| 特徴 | **言葉が多い**。実況・名前呼び |
| 3P | 前後で途切れる。「……ん、晴彦……」→ 母音のみ |
| 絶頂 | 「大翔、中……！ 悠真、一緒に……ッ！」 |
| 事後 | 荒い呼吸→笑い。「はぁ、はぁ……もう、今日多い」 |

### 6-3. 東条晴彦

| 要素 | 設計 |
|------|------|
| セリフ | **許可の質問**。「〜ていいですか」を段階ごと |
| 声 | 低い。喘ぎは**最小**。さえかを短く褒める |
| 動き | 遅い。停止を守る |
| 例 | 「谷口さん、——いい声」「痛いなら、止めます」 |

### 6-4. 相楽悠真

| 要素 | 設計 |
|------|------|
| セリフ | 穏やか・関西混じり控えめ。「大丈夫？」「ゆっくり、息、吐いて」 |
| 3P | **前**担当。囁き・優しさ |
| 声 | 低声。さえかを固まらせない |

### 6-5. 樫原大翔

| 要素 | 設計 |
|------|------|
| セリフ | 短いタメ口。「脚、開いて」「可愛い」 |
| 声 | **大きい**。部屋・壁を震わせる |
| 3P | **後**担当。低い唸り・体力 |
| 例 | 「1週間後には一緒だと思うよ」（日常）／「嘘だろ」（第3話） |

## 7. ブロック執筆テンプレ（150〜280字）

```
【環境1行】  場所・時間・二人きり／3P
【導入1行】  日常からの違和感 or さえか内心
【セリフ1行】 キャラ口調・確認 or 短い実況
【動作2行】  具体的部位・音（ばちゃ／軋む）
【喘ぎ1行】  段階に合った崩れ
【内心1行】  おかしい／向こう・こっち
【Beat】     静けさ・シャワー・廊下 —— 次段階へ
```

## 9. 執筆後チェックリスト

### 全体
- [ ] 1段落に感覚が2種類以上あるか
- [ ] 同じ型（セリフ→セリフ→セリフ）の連打がないか
- [ ] 高まりで文が短くなっているか
- [ ] さえかの内心（普通／おかしい）が入っているか

### セリフ
- [ ] 単語ごとの読点切りがないか
- [ ] キャラを目を閉じて読んで判別できるか
- [ ] 話数の「声の上限」を超えていないか

### 喘ぎ
- [ ] 「あ゛」が含まれていないか
- [ ] 同一パターンの8文字以上連続がないか
- [ ] 呼吸音（はあ、ん）が地の文か喘ぎに混ざっているか
- [ ] 絶頂が1シーン1〜2行に収まっているか

### 音・環境
- [ ] ばちゃ／軋む／壁越しのいずれかがあるか
- [ ] 3Pなら前・後・声の三方向が分かるか

### 本作境界
- [ ] さえかの拒否表現が話数ルールに合うか（やめて＝第2話まで）
- [ ] 東条シーンで許可確認があるか
