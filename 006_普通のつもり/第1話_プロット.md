# ブロック0：プロlogue（1,000字）

**時刻**: 入居日の午前〜正午（回想）。本文冒頭。

**場所**: 実家のリビング／新幹線／専門学校近くのカフェ

### 段落構成（5段落×200字）

**P0-1（200字）｜上京の理由**
- 浜松の実家。母が弁当箱を押し付ける。「一人暮らしは大変よ」
- さえか、調理師専門学校合格の通知を見返す。上京は決まっていた
- 弟（15）が「姉ちゃん、都会で彼氏作れよ」と笑う。さえか「……そんな余裕ない」
- **内心**: 恋愛より、まず安定した住まい

**P0-2（200字）｜住居探しの行き詰まり**
- 入学2週間前。学校指定の寮→満室
- 不動産サイト。初期費用30万超のワンルーム→母の顔が曇る
- シェアハウス検索。「個室・家具付き・7万台」
- **「見晴台ハウス」**の写真：白いキッチン、観葉植物、明るい窓。**普通**

**P0-3（200字）｜契約の夜**
- スマホで電子契約。34ページ。「最後まで読みました」チェック
- 別紙Bのポップアップ「親密交流方針への同意」→ **ペットかゴミの細則**と思いタップ
- 署名。指が震えたのは、初めて自分名義の契約だから
- 母に「シェアハウス、ちゃんとしてるところ」と報告。**嘘ではないつもり**

**P0-4（200字）｜荷造り**
- スーツケース1、段ボール2。洋服は地味め。カーディガン2枚
- 下着は白無地のみ。派手なものは持っていない
- 夜、ベッドで募集ページを見返す。**リビングの写真が1枚もない**ことに気づくが、「個室あるからいいか」
- **内心**: 普通のシェア。人とうまくやれば大丈夫

**P0-5（200字）｜入居当日の朝**
- 土曜。新幹線。窓の外が都市に変わる
- 下北沢寄りの住所を地図で確認。閑静な住宅街
- 14時入居予定。荷物を転がしながら、**普通の一歩**だと思う
- **→ Scene 1 へ硬切り**。「普通のつもり、だった」

### プロlogueで埋める伏線

- 別紙B未読
- 家賃の安さの理由（さえかはまだ気づかない）
- 母への報告＝後のルール5番との衝突
- 月曜から学校実習（後で「逃げられない」理由に）

# Scene 1：駅〜玄関前（800字）

**時刻**: 13:40〜14:00  
**場所**: 下北沢寄りの坂道 → 見晴台ハウス外観 → 玄関

### 段落構成（4段落×200字）

**P1-1（200字）｜外観**
- 築10年の一戸建て。外壁は白、小さな庭に観葉植物。**募集写真と同じ**
- さえか、ほっとする。荷物を置き、深呼吸
- インターホンに手を伸ばす——**その前に**、建物の中から声

**P1-2（200字）｜最初の違和感**
- 女性の声。高い。断続的。リビング側の窓はカーテン半開
- さえか、インターホンの指を止める
- 「配達……？ テレビ……？」自分に言い聞かせる
- もう一声。低い男性の声が重なる。**笑い声**
- 耳が熱い。理由は分からない

**P1-3（200字）｜東条の出迎え**
- 勇気を出して押す。チャイム1回
- ドアが開く。細縁メガネの男性（東条）。グレーのTシャツ。**上半身は着ている**
- 「谷口さん？ 待っていました。東条です」
- 低い声。落ち着いている。さえか、お辞儀。「谷口さえかです。今日から……」
- 東条、荷物の視線。「重そうですね。上まで案内します」

**P1-4（200字）｜玄関をくぐる直前**
- 靴を脱ぐ。スリッパが5足。使用感がある
- リビング方向から**肌の色**が見える。白い。動いている
- さえか、カーディガンの裾をつまむ
- 東条「驚かせる形になって申し訳ない。先に、中を——」
- **→ Scene 2**。さえかの足が止まる

# Scene 2：リビング目撃（2,800字）★本話の核

**時刻**: 14:05〜14:15  
**場所**: 玄関ホール → リビング（L字ソファ・キッチン）

### 段落構成（8段落×350字）

**P2-1（350字）｜全景・3P**
- 玄関から2歩。L字ソファ。**御子柴**（茶髪ボブ・タンクトップのみ）がソファの肘に手をついている
- **樫原**（上半身裸）が背後から後背位。**相楽**（エプロン脱落）がソファ前で膝をつき、御子柴の口へ

- さえか、**理解が追いつかない**。スーツケースが滑り落ちる

**P2-2（350字）｜さえかの視線（断片描写）**
- 動くものばかり入る。ばちゃ、ばちゃ。唇の音と打ち付けが重なる
- 御子柴のネイル。振り返ろうとして口に阻まれ、**目だけ合う**
- 樫原「前も後ろも、いっぱいだ」相楽「りん、喉、深いな。ゆっくり呼吸して」
- **内心**: おかしい×3。**でも動けない**

**P2-3（350字）｜東条の位置**
- 東条はさえかの**横**
- 「驚かせてしまいました。説明が後手に回りました」
- 東条「共有スペースでの、居住者間の親密行為です。**人数の制限は、ハウスルール上、ありません**」
- **問題ない、の意味が分からない**

**P2-4（350字）｜3P完結**
- 樫原・相楽、動きを揃えて深くなる
- 「中、……口、……一緒、……！」「二人、同時に、出す」
- 御子柴、二方向から塞がれた声。事後：口元と内側の白い筋
- 御子柴「**三人**、終わった」——恥じている顔ではない

**P2-5〜P2-8** — 逃げる選択・部屋へ（従来通り）

**P3-5（250字）｜慣れ期間**
