## 世界観

### 時代・社会

- **時代**: 現代日本（作中では2026年頃）
- **国**: 日本
- **社会**: 表向きは普通の都市生活。裏では「セックスポジティブ同居」「フリーラブ系シェア」がニッチだが合法として存在する
- **本作との関係**: 国策・学校制度・風俗系列など**既存5作の制度は一切登場しない**

### 「普通」に見せかける仕組み

| 項目 | 内容 |
|------|------|
| 募集元 | 管理会社「ライフリンクス株式会社」 |
| 表の看板 | 「都心アクセス良好・個室付き・月7万円台のシェアハウス」 |
| さえかが見た情報 | 白い壁・観葉植物・キッチン写真のみ。ルール説明PDFは**34ページ中、親密条項は31ページ目の注釈** |
| 入居理由 | 専門学校進学で上京。実家が遠く、寮満室。急ぎで契約した |
| 契約の罠 | 「共有スペースにおける居住者間の親密交流方針（別紙B）」への同意チェックを、さえかは「ペット可とかゴミ出しの話か」と思って承認 |

### シェアハウスの実態

- **昼夜問わず**、リビング・キッチン・風呂上がりの廊下で性行為が起きうる
- 住人全員が「ここはそういう場所」と理解している（さえかだけが知らなかった）
- **個室の鍵はかけられる**が、長期ロックアウトは「距離を置きすぎ」と空気が重くなる
- 叫び声・ベッドの軋み・シャワー室の音は**日常BGM**扱い
- 拒否は形式上可能だが、「なぜ拒むの？」「疲れてるなら休めば？」と**自然に距離が詰められる**圧力がある（暴力ではない）

### 間取り

| 場所 | 用途・描写の要点 |

| **リビング** | L字ソファ・大型テレビ・低いテーブル。**事実上のメイン舞台**。昼も夜もここで始まることが多い |
| **キッチン** | 対面式。料理中に後ろから抱きつかれる、カウンターに座って脚を開く、など |
| **風呂・洗面** | 1浴槽。入浴待ち列。湯上がりタオル一枚のままリビングに戻るのが普通 |
| **さえかの個室** | 6畳・ベッド・デスク・鍵付き。最初の避難所。後半は「待ちきれず開けて出ていく」場所に変化 |
| **他4室** | 男性3・女性1（御子柴）。さえかは**一番奥**の部屋（音が最もよく聞こえる） |

### 壁掲示の「ハウスルール」（入居後に初めて読む）

1. 個室はプライベート。ただし**拒否の意思表示は明示すること**
2. 共有スペースでの親密行為は居住者の自由。**視線を避けるより、参加か退出を選ぶ**
3. 深夜2時以降は大きな声は窓を閉める（それ以前は制限なし）
4. 新規入居者には**1週間の慣れ期間**がある（期間中は「様子見推奨」と書いてあるが、強制力はない）
5. 外部への事情拡散は禁止（SNS・実家への詳細報告など）
