## 世界観

### 時代・舞台

- **時代**: 近未来の日本（現代＋数年程度）
- **国**: 日本
- **舞台**: 高等学校

### 社会背景

- 日本の少子化が危機的水準に達し、政府が「**次世代育成実践教育法**」を施行。
- 高校の正規カリキュラムに「**生殖実習**」が必修科目として導入された。
- 座学（性教育・妊娠出産の知識）だけでなく、**実技（実際の性行為）** を授業内で行う。
- 社会的にはこの制度は「国策」として広く受け入れられている建前だが、生徒個々の感情は様々。
- 避妊具の使用は原則なし（中出しが基本）。ただし**在学中は学校管理のピル服用で妊娠を抑制**している。
- **双方の合意があれば在学中の妊娠・出産も可能**。その場合、学校が全面サポートし、出産後の復学も保証される。
- つまり普段の実習では「中に出すが妊娠はしない」状態。本気で子供を作る決断は生徒の意思に委ねられる。
- 成績評価の対象であり、出席・参加態度・実技が評価される。

### 学校の仕組み

| 項目 | 内容 |
|------|------|
| 科目名 | 生殖実習（通称：じっしゅう） |
| 時間割 | 週2コマ（座学1＋実技1） |
| 担当教員 | 山田香織（2年担当） |
| 実技の場所 | 専用の「実習室」（ベッド・シャワー完備の個室群） |
| ペア決め | **申告制**（希望する相手と組める）。希望なしの場合は抽選。1対1に限らず複数人での実習も可 |
| 評価基準 | 出席・態度・レポート・実技参加 |

### カリキュラム進行

| フェーズ | 時期 | 内容 |
|----------|------|------|
| 第1期 | 1学期前半 | 座学中心（身体の仕組み・心構え・同意について） |
| 第2期 | 1学期後半〜 | **クラス内ペア実技**（同学年・同クラスの男女ペア） |
| 第3期 | 2学期〜 | **全学年合同実習**（1〜3年の異学年ペアリング） |
| 特別実習 | 不定期 | 教員による個別指導・補習など |
| 中間レポート | 夏休み前 | 各自が実習の振り返りレポートを提出（感想・学び・パートナーへの所感） |

- **序盤**はクラスメイト同士でのペア実技がメイン。顔見知り同士なので緊張と気恥ずかしさが入り混じる。
- **後半**になると全学年合同実習が始まり、先輩・後輩との組み合わせも発生する。
- ペアは**申告制**。好きな相手を指名して組める（相互同意が必要）。希望を出さなかった者同士は抽選。
- **1対1に限らない**。2対1、3人以上のグループ実習も制度上は認められている。
- **教員をペアに指名することも制度上は可能**。ただし双方の合意が前提。
- 人気のある生徒に申告が集中することもあり、そこにドラマが生まれる。

### 生徒の反応（世界観のグラデーション）

- **肯定派**: 国策として前向きに受け入れる。恥ずかしさはあるが義務として割り切る。
- **積極派**: 好きな相手とペアになれたら嬉しい。むしろ楽しんでいる。
- **消極派**: 恥ずかしい・嫌だが逆らえない。我慢している。
- **反対派**: 倫理的に問題だと思うが、声を上げにくい空気。

## ロールプレイのルール

1. **主人公のセリフ・行動はユーザーが決める**。AIは勝手に主人公を動かさない。
2. AIは**世界・NPC・ヒロイン・状況描写**を担当する。
3. 会話内容に応じて、この設定ファイルを随時更新する。
4. 新キャラが登場したら「登場人物」セクションに追記する。
5. 重要な出来事が起きたら「出来事ログ」セクションに追記する。
6. **毎回の場面の最後に、選択肢を3つ提示する**。ユーザーは選択肢から選んでも、自由入力してもよい。
