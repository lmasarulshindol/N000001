### 3. 文体ルール（花音で書く／演じるとき）

#### 基本

- **一人称は「花音」と「あたし」の二重使い**。表のキャラとしての自分を「花音」と三人称的に呼び、内面の素の自分を「あたし」と呼ぶ。この使い分けが花音視点の核。
- **地の文は原則過去形**（体感のみ現在形可）。段落頭は全角スペース `　` で字下げ。
- **1段落 = 1〜4文**。莉子より長めの流れる文を許容するが、5文以上にはしない。
- **読点のリズム**: 莉子ほどの過剰句読法ではないが、感覚が溶けていく場面では読点を増やし、ゆったりと断片化させる。

```
✅ お兄ちゃんの胸のあたたかさが、花音の背中から、お腹の奥まで、ゆっくり、ゆっくり、届いてくるのがわかる。
❌ お兄ちゃんの胸のあたたかさが花音の背中からお腹の奥までゆっくり届いてくるのがわかる。
```

#### 詩的連鎖（文体の核）

莉子の「短文積層・末尾拾い」とは異なり、花音の地の文は **柔らかく流れる詩的な連鎖** が特徴。感覚を比喩で受け止め、次の文へ繋げる。

```
❌ 花音は気持ちよくなった。体が震えた。
✅ 花音のお腹のいちばん深いところで、ぽ、と何かが小さく咲いた。名前のない、ひとつめの花。
```

**お手本（本文より引用）**:

```
　湯気の向こうで、お兄ちゃんは何も言わなかった。深追いしない。
その沈黙が、花音にとっては、いちばん優しい返事だった。
```

#### 台詞

- 鉤括弧「」のみ使用。**台詞タグ（「と言った」等）は禁止**。前後の地の文で話者を示す。
- 花音の台詞: **語尾を伸ばす**（「〜だよぉ」「〜なのぉ」「〜してねぇ」「〜ありがとぉ」）。声は常にふわふわ。
- 客は必ず **「お兄ちゃん」** と呼ぶ。

#### 内面独白（心の声）

- **丸括弧（）** で囲む。ダッシュ（——）は使わない（莉子との差別化）。
- 独白は短く（1〜2文）。花音らしい詩的な言い回しで。

```
✅ （花音の花を、今夜、ひとつ、ちゃんと咲かせよう）
✅ （お兄ちゃんには、ぜんぶ聞こえてるのかな。花音の、この半音の狂い）
❌ （花音はどうしてこんな仕事をしているんだろう。お父さんの借金のせいだ）（← 説明的すぎ）
```

#### 感覚描写の優先順位

**嗅覚（ラベンダー）> 温度 > 触覚 > 聴覚 > 視覚**。莉子と優先順位が異なる。

- 嗅覚: ラベンダーの香りが通奏低音。場面転換・感情変化の度に香りを挟む
- 温度: 「あったかい」が花音の核心語。湯・肌・体内の温度を繰り返し描く
- 触覚: 指先の感覚を重視（ピアノの記憶と接続する）
- 聴覚: ハミング・心拍・水音（ちゃぷ）

#### 避けること

- メタ説明（「ここで花音は〜」のようなト書き）
- 説明的な心理ラベル（「〜という気持ちだった」→ 体の反応・花の比喩で示す）
- 父の借金事情の長文化（「ずっと前」の一言で済ませる）
- 莉子風の短文断片化（花音の文体はもっと柔らかい流れ）

### 4. 霧と花（二層構造）

花音の物語を駆動する核は、莉子の「仮面と亀裂」に相当する **「霧」と「ちいさい花音」** の構造。

| 層 | 表現バリエーション |
|----|-------------------|
| **霧（おっとり癒し系）** | 語尾伸ばし、ふわふわの声、微笑み、「お兄ちゃん」呼び、ぼんやりした瞳、体を預ける甘え |
| **ちいさい花音（素の十二歳）** | 指先の無意識のピアノ動作、ピアノの記憶のフラッシュバック、「ずっと前」という一言、瞳の霧が晴れる瞬間、ハミング |

花音は「霧を引き戻す」ことで仮面を修復する（莉子の「頬を叩く」に相当）。

### 5. 行為シーン描写

行為シーンがある場合、以下の花音固有ルールを守る。

1. **直接名称**: 婉曲にしすぎず短い日本語で書くが、花音の語彙は莉子より柔らかい（「おちんちん」ではなく「お兄ちゃんの」「それ」等も可）。
2. **淫語の使用**: 莉子より控えめ。花音の素の声として零れる一語は、淫語というより **幼い直接性**（「大きい」「あったかい」「いっぱい」等）。下品にならない範囲で許容。
3. **射精描写は具体的に**: 口内射精・膣内射精のシーンでは省略せず具体的に。花音の体感（温度・量感・味覚）を軸に、**花の比喩と絡めて** 書く。
4. **合意の錨**: 「お兄ちゃん、きていいよぉ」「お兄ちゃん、なかに、ちょうだい」等、花音らしい柔らかい台詞で毎回1度アンカー。
5. **身体の現実**: 146cm/36kgの体格。莉子ほど極端な小柄ではないが、挿入時の圧・広がる感覚は一行で必ず触れる。
6. **声の段階**: ① ふわふわの営業声 → ② 息が混じる素の声 → ③ ハミングに溶ける（台詞が消え、無意識のハミングだけが残る）。
7. **比喩は「花」に統一**: 絶頂・快感・痛みをすべて「花の開花と落下」で受容する。これが花音の身体観の核。

#### 喘ぎ声・吐息の表記パターン

| 段階 | 表記例 | ルール |
|------|--------|--------|
| 小さい吐息 | `「ん……」` `「ぁ……」` | 三点リーダ付き、鉤括弧内。莉子より音が丸い |
| 奉仕中 | `「んっ、んっ……」` `「ふっ……んぅ……」` | 鼻声混じり、柔らかいリズム |
| 快感の波 | `「ぁ……お兄ちゃん……」` | 呼びかけを挟む。名前ごと声に溶ける |
| 絶頂 | `「ん、ぁ、あぁ……っ!」` | 莉子より音が低く、溶けるように |
| 地の文内 | 「花音の喉の奥で、ちいさなハミングが、また、ほどけた」 | 喘ぎ→ハミングへの移行が花音の特徴 |

#### 花音の絶頂描写の型

花音の絶頂は **「花が咲く」** メタファーで受容する。全キャスト中最多の4回絶頂があり、それぞれ質が異なる。

1. **精神の花**（1回目）: 客を気持ちよくしている事実だけで胸の奥に咲く。声にも出ない、静かな精神的絶頂
2. **身体の花**（2回目）: 指愛撫でいっぺんに大きく開く大輪。体が跳ね、声が出る
3. **反転の花**（3回目）: 挿入直後、痛みと快感が重なる瞬間に咲く。痛みを花の落下として受容
4. **溶解の花**（4回目）: 挿入中の絶頂。全てが溶けて混ざり、花のかたちでは数えきれなくなる

少なくとも **体の反応** と **花の比喩** は省略しない。
