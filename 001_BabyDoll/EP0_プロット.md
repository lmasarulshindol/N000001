# EP0「BabyDollの小さな魔女」詳細プロット

> 佐藤莉子（10歳・BabyDoll最年少キャスト）の入店から「初めてコース」前夜までを描く。母の治療費のために歌舞伎町に身を投じた少女が、小悪魔の仮面を被り、トップキャストへと駆け上がる4ヶ月間。処女のまま「初めてコース」の対象に選ばれ、明日の客を待つ夜で幕を閉じる。

---

## シーン構成（全6シーン）

---

### シーン1：あたしがやらなきゃ（東京下町・母との暮らし）

**場所**: 東京下町の古いアパート（莉子と母の二人暮らし）
**時間帯**: 入店4ヶ月半前。梅雨の夕方
**登場人物**: 莉子、母・佐藤彩花（30代前半）
**文量目安**: 約3,000字

#### 出来事
- 莉子の日常から始まる。学校から帰ると母が台所に立っている。痩せた背中。咳き込む音
- 母・彩花は心臓疾患を抱えている。手術が必要だが費用が足りない。パートの収入では薬代と家賃で精一杯
- 夕食はもやし炒めとご飯。莉子は「おいしい！」と笑う。母も笑うが、目の下の隈が深い
- 食後、母が薬を飲む。莉子はその手を見ている。震えている
- 母が寝た後、莉子はリビングのテーブルに置かれた封筒を見つける。病院からの請求書。金額を見て息を呑む
- 莉子のスマホ。クラスメイトとのLINE。「明日遊ぼう」「ごめん、むり」。友達との距離が開いていく
- ある日、学校帰りの歌舞伎町で「若年就労特別法」のポスターを見かける。莉子はスマホで調べ始める
- BabyDollの求人ページにたどり着く。「10歳から。保護者の同意。月収40〜80万円」。莉子の目が変わる
- 夜、布団の中で天井を見つめる。「お母さんの手術代。あたしがやらなきゃ」

#### 描写のポイント
- 東京下町の生活感。古いアパート、薄い壁、隣の部屋のテレビの音
- 母の病状を莉子がどう見ているか——子供ながらに「時間がない」と理解している
- 10歳の少女がスマホで風俗の求人を調べるという異常さを、淡々と描くことで際立たせる
- 莉子の頭の回転の速さ。感情ではなく計算で動く子

#### 感情の流れ
- 母への愛情と心配 → 経済状況への焦り → 求人を見つけた時の冷静さ → 決意

### シーン2：面談（BabyDollへの応募と入店）

**場所**: 歌舞伎町・BabyDoll 2階受付 → 控え室
**時間帯**: シーン1から2週間後。夏休み直前
**登場人物**: 莉子、店長、受付スタッフ
**文量目安**: 約3,500字

#### 出来事
- 莉子が一人で歌舞伎町に来る。制服ではなく私服（Tシャツにショートパンツ）。ランドセルではなくリュック
- 雑居ビルの前に立つ。心臓がばくばくしている。でも足は止まらない
- エレベーターで2階へ。受付の内装に驚く。「カフェみたい」
- 受付嬢に「面談の予約をした佐藤です」と言う。受付嬢が一瞬驚くが、プロとして対応
- 店長との面談。店長は莉子の年齢と動機を確認する。「お母さんのため？　偉いね。でもお母さんには何て言うの？」
- 莉子「子役の事務所に入ったって言います。オーディションで受かったって」。嘘が自然に出てくる。店長は少し眉を上げるが、頷く
- 保護者同意の問題。莉子が用意した書類（母の署名は莉子が模倣した偽造）。店長は見て見ぬふり——この業界では珍しくないこと
- 健康診断の説明、研修期間の説明。NNサービスの説明を受ける。「生で中に出すってこと？」。莉子が大人びた確認をする。店長は淡々と答える
- 莉子の内面：怖くないわけがない。でも怖いって言ったら終わりだ。あたしは、できる子だから
- 控え室を見せてもらう。ロッカー、ソファ、テレビ。「ここがあたしの職場になるんだ」
- 入店日が決まる。莉子はビルを出て、歌舞伎町のネオンを見上げる。「——やるしかないでしょ」

#### 描写のポイント
- 10歳の少女が風俗店の面談に一人で来る異常さと、莉子の妙な落ち着き
- 店長の反応——慣れている。制度が生んだ日常
- 保護者同意の偽造という「闇」を、莉子が自分でやっていること
- NNサービスの説明を受ける場面で、言葉の意味を理解していく莉子の表情の変化

#### 感情の流れ
- 緊張 → 面談での冷静さ（大人ぶる） → NNの説明で初めて体が強張る → 覚悟を上塗りする

### シーン3：研修と「小悪魔」の誕生

**場所**: BabyDoll 控え室・研修室
**時間帯**: 入店後2週間の研修期間。夏
**登場人物**: 莉子、先輩キャスト・鈴（りん、15歳）、他のキャストたち
**文量目安**: 約3,500字

#### 出来事
- 研修初日。莉子は最年少。他のキャスト（12〜15歳）から物珍しそうに見られる
- 接客マナー研修。お辞儀、言葉遣い、笑顔の作り方。莉子は意外と飲み込みが早い
- 衛生管理研修。避妊（ピル）の説明。莉子が低用量ピルを初めて飲む場面
- 実技研修。マットプレイの基礎をダミー人形で練習。莉子の体が小さすぎて上手くいかない。「くっ……体ちっちゃいのが悪いんだからね」
- 先輩キャスト・鈴との出会い。鈴はBabyDoll最年長の15歳で来月引退。面倒見がいい
- 鈴「莉子ちゃん、10歳でしょ？ すごいね。怖くない？」。莉子「怖くないし。あたし、やると決めたことはやるタイプだから」。鈴が少し笑う。「生意気だなぁ。でも嫌いじゃない」
- 鈴が莉子に接客のコツを教える。「お客さんは、若い子が頑張ってるのを見たいの。でもそれだけじゃつまんない。ちょっとだけ、からかってやるの」
- 莉子がその言葉をヒントに「小悪魔キャラ」を作り上げていく。上目遣い、挑発的なセリフ、ニヤリとした笑い——10歳の無邪気さと大人びた態度のギャップ
- 研修最終日。莉子がキャスト衣装を初めて着る。ピンクの透けシフォンのベビードール。下着なし。胸も下腹部も薄い布越しに丸見え。「……えっ、これ、透けてない？」。鈴「透けてるよ。それがBabyDollの衣装。お客さんはこれが見たくて来るの」。莉子が鏡の前に立つ。ピンクの半透明に包まれた自分の幼い体。平たい胸、くびれのない腰、子供そのものの輪郭が布越しに浮かぶ。「……かわいいじゃん、あたし」。不敵に笑ってみせる。小悪魔の仮面が完成する
- だが寮（希望者用の寮に入っている）の部屋に戻ると、母に電話する。「うん、事務所のレッスン楽しいよ！ お母さん、ちゃんと薬飲んでる？」。声が子供に戻る

#### 描写のポイント
- 研修の具体的な内容で「こういう店で働く」というリアリティを構築
- 莉子の「小悪魔キャラ」が天性ではなく、戦略的に構築されたものであることを示す
- 鈴との姉妹的な関係。年齢は離れているが、同じ境遇の仲間
- 透けベビードールを着た瞬間の変身——「佐藤莉子」から「BabyDollの莉子」へ。透ける裸体が「商品」としての自分を突きつける
- 母との電話で素に戻る瞬間

#### 感情の流れ
- 不安と好奇心 → 研修への適応 → 小悪魔キャラの構築（自己防衛としての仮面） → 母への電話で素に戻る

### シーン4：指名率トップ5（キャストとしての日常）

**場所**: BabyDoll各所（控え室・個室・待合ロビー）
**時間帯**: 入店2ヶ月目〜4ヶ月目。夏〜秋のダイジェスト
**登場人物**: 莉子、鈴、他のキャスト、客（モブ）
**文量目安**: 約3,500字

#### 出来事
- デビューからの日々をダイジェストで描く
- 初めての接客。緊張で手が震えるが、小悪魔の仮面で乗り切る。「ねぇ、おじさん。こんな小さい子でいいの？　変態だね〜」。客が赤面する。莉子は内心ガッツポーズ
- ただし本番は未経験。「お試しコース」「スタンダード（NN非同意なのでコンドーム着用・挿入なし）」を担当
- 入浴→ハンドサービス→口でのサービスまでは経験を重ねる。マットプレイも体の小ささを活かす方法を編み出す
- 莉子の接客スタイルが客に刺さる。「10歳の小悪魔」という唯一無二のポジション
- 指名が増えていく。入店4ヶ月で指名率トップ5入り。莉子の顔がプロフィールパネルで目立つ位置に
- 月収は60万円を超える。母への仕送り（「事務所のギャラ」として）。母の手術の予約が入る
- 控え室での日常。キャスト同士のおしゃべり、嫌な客の情報共有、お菓子の交換
- 鈴の引退が近づく。「鈴ねえ、いなくなったら寂しい」「あたしがいなくても大丈夫でしょ。生意気な妹なんだから」
- ある客が莉子に「本番はしないの？」と聞く。莉子は「あたし、まだ処女だもん。特別なの」と笑ってかわす。だが内心、処女であることの「商品価値」を理解している

#### 描写のポイント
- 「挿入なし」の状態でもプロとして働いている莉子の日常
- 小悪魔接客の具体的な台詞と客の反応で、莉子のキャラクターを確立
- 母の手術費が貯まっていく＝莉子の仕事が「報われている」実感
- 鈴との別れが近い寂しさ——莉子が感情を見せる数少ない場面
- 「処女」が商品として消費される構造を莉子自身が理解している冷徹さ

#### 感情の流れ
- プロとしての成長と自信 → 母の手術への安堵 → 鈴との別れの寂しさ → 処女の「価値」への複雑な感情

### シーン5：「初めてコース」への覚悟

**場所**: BabyDoll控え室 → 莉子の寮室
**時間帯**: 入店4ヶ月目。秋の夜
**登場人物**: 莉子、鈴、店長
**文量目安**: 約3,500字

#### 出来事
- 店長から呼ばれる莉子。「莉子ちゃん、お話があるの」
- 「初めてコース」の説明。「あなたの処女卒業に立ち会えるプラン。90分80,000円。お客さまからのリクエストが溜まってるの」
- 莉子は分かっていた。いずれこの日が来ると。NNサービスへの同意書にサインを求められる
- 「生で、中に出されるってことでしょ？　ピル飲んでるから大丈夫だよね」。大人びた確認。だがペンを持つ指が震えている
- サインする。店長が「ありがとう。日程が決まったら連絡するね」と去る
- 控え室に戻る。鈴がいる（引退直前の最後の出勤日）
- 鈴に打ち明ける。「……あたし、初めてコースやることになった」
- 鈴の表情が変わる。「そっか。……痛いよ。最初は」。初めて鈴が飾らない言葉を使う
- 鈴「でもね、莉子。痛いのは最初だけ。それより——終わった後に、自分が自分じゃないみたいな感じがするの。それが一番きつい」
- 莉子「……大丈夫。あたし、大丈夫だから」。声が小さい
- 鈴が莉子の頭をぽんと撫でる。「大丈夫じゃなくても、泣いていいんだからね。控え室に戻ってきたら、アイス用意しとくから。——って、あたしもう明日でいないんだった」。苦笑い
- 莉子が「ずるい。最後にそういうこと言うの、ずるいよ鈴ねえ」。目が潤む
- 鈴「じゃあ約束。初めてコースが終わったら、LINEして。アイスの代わりにスタンプ送ってあげる」。小指を出す。莉子が小指を絡める
- 寮室に戻った莉子。一人。ベッドに座って天井を見つめる
- スマホで母の写真を見る。母の笑顔。「お母さんの手術、来月だよ。あたしが稼いだお金で。だから——」
- 「だから、あたしの体くらい、安いもんでしょ」。誰にともなく呟く。でも声が震えている
- 布団に潜り込む。枕に顔を埋める。泣いてはいない。泣かない。あたしは泣かない

#### 描写のポイント
- NNサービスへの同意＝生での性行為と中出しを受け入れるという重さ
- 鈴との最後の会話。姉妹のような関係の集大成。小指の約束
- 莉子の小悪魔の仮面が完全に外れる瞬間。「大丈夫」と言いながら声が震える
- 母の写真を見ながらの独白——「あたしの体くらい安いもん」は自己正当化であり、同時に悲鳴
- 「泣かない」という意地が、10歳の少女の精一杯の強さ

#### 感情の流れ
- 覚悟していた現実 → NNサービス同意の重さ → 鈴との別れの感情 → 母を想う決意 → 一人になった時の脆さ → 泣かないという意地

### シーン6：明日の客

**場所**: BabyDoll控え室 → 待合ロビー → 窓の外の歌舞伎町
**時間帯**: シーン5から3日後。金曜日の夕方
**登場人物**: 莉子、受付スタッフ
**文量目安**: 約3,000字

#### 出来事
- いつもの出勤。私服からキャスト衣装に着替える。ピンクの透けベビードール。下着なし。薄いシフォン越しに子供の体が透けて見える。髪をふわりと整える。左耳のピアス穴に小さなハートのピアスを通す
- 鏡の前。小悪魔の笑顔を練習する。「ねぇ、おじさん。あたしが初めてなんだけど、うれしい？」。声に出してみる。完璧。大丈夫
- 受付スタッフが控え室に来る。「莉子ちゃん、明日の初めてコースのお客さまが決まったよ」
- 客の情報。結城健吾。33歳。広告営業。レギュラー会員（初来店）。紹介者はゴールド会員の村瀬
- 「初来店の人なんだ。初めてコースが初来店って、めずらしいね」。受付嬢が少し驚く
- 莉子「初めて同士ってこと？　面白いじゃん」。小悪魔の笑み。だが心臓が跳ねている
- 一人になった控え室。莉子はソファに座って、自分の手を見る。小さい手。まだ子供の手
- 明日、この手で大人の男の人に触れる。そして、この体に——
- 窓の外は歌舞伎町のネオン。ピンク、赤、青。光が街を照らしている。莉子はその光を見つめる
- スマホを取り出す。鈴にLINEを送る。「明日、やるよ。初めてコース」。既読がつく。鈴からスタンプ——ガッツポーズをする猫。それだけで莉子は少し笑える
- 「結城健吾、33歳。——あたしより23歳も上じゃん。おじさんだね」
- 莉子が立ち上がる。窓に映る自分を見る。ピンクの透けベビードールから幼い裸体のシルエットが浮かぶ。ミディアムウェーブの髪。アクアブルーの瞳。10歳。
- 「明日、あたしは女になる」。声に出す。映画みたいな台詞。でも震えていない。もう震えない
- ラスト：莉子は窓に背を向け、控え室を出る。ネオンの光が莉子の小さな背中を照らしている。BabyDollの夜は、まだ始まったばかりだ

#### 描写のポイント
- 「明日の客」の情報で健吾の存在を匂わせる。EP1冒頭への直結
- 小悪魔の仮面を被る瞬間と、一人になった時の素の莉子の交互描写
- 鈴とのLINEのやり取り。離れても繋がっている絆
- 歌舞伎町のネオンが莉子を照らす映像的なラスト
- 「明日、あたしは女になる」——10歳の少女のこの台詞の重さ

#### 感情の流れ
- 日常のルーティン（仮面を被る） → 客の情報で心臓が跳ねる → 一人の時間の不安 → 鈴からのスタンプで少し笑う → 覚悟を決めてネオンに背を向ける

## EP0固有の描写方針

### 莉子の二面性の描き分け

| 場面 | 「小悪魔・莉子」 | 「素の莉子」 |
|------|---------------|-------------|
| 話し方 | 「ねぇ、おじさん♪」。挑発的、生意気 | 「お母さん、薬飲んだ？」。年相応、優しい |
| 表情 | ニヤリ。上目遣い。計算された可愛さ | 口をむっと結ぶ。泣きそうなのを堪える |
| 体の使い方 | わざとらしく寄りかかる、見上げる | 猫背になる、膝を抱える |
| 声のトーン | 少し低め、大人びた | 高め、子供っぽい |

### 通奏低音
- **母の病気**: 全シーンの底流。莉子の行動の全ての動機
- **鈴との絆**: シーン3で出会い、シーン5で別れ、シーン6でLINEで繋がる
- **「泣かない」**: 莉子の意地。EP1で仮面が崩れる伏線

### EP1への接続
- EP0ラスト: 莉子が「明日の初めてコースの客」の情報を受け取る（金曜夕方）
- EP1冒頭: 健吾が歌舞伎町を歩いている（翌日・金曜夜）
- 同じ「金曜の夜」を、EP0は莉子の準備として、EP1は健吾の来店として描く
