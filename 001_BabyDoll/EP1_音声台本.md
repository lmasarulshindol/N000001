# EP1「ベビードールの初恋」音声台本

Irodori-Studio（VoiceDesign モード）で読み上げるための台本。
各セリフに `text`（読み上げ本文）と `caption`（声質・演技指示）を定義する。

## 備考

- **モード**: VoiceDesign（参照音声なし、`--no-ref`）
- **チェックポイント**: `Aratako/Irodori-TTS-500M-v2-VoiceDesign`
- **キャプション**: 各行の `caption` 列をそのまま JSON の `caption` フィールドに使用
- **出力ファイル名案**: `outputs/babydoll_ep1/XXX_キャラ_場面.wav`
- **莉子の声の変化が最重要**: 同一キャラでも場面ごとにキャプションを切り替えることで、小悪魔→素→痛み→安堵の感情変化を表現する
- **合計セリフ数**: 233行（LINEメッセージ含む）
