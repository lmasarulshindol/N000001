# DollGarden（ドールガーデン）

> BabyDoll系列店・高校生専門ソープランド 設定資料

---

## 1. 店舗概要

BabyDoll系列の最終段階として位置づけられる高校生専門店。DollHouseを「卒業」した中学生キャストの転籍先であると同時に、高校生年齢での新規採用も行う。「幼さ」のBabyDoll、「思春期」のDollHouseに対し、DollGardenは**「咲き誇る少女たちの完成形」**をコンセプトに掲げる。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 店名 | DollGarden（ドールガーデン） |
| 業態 | ソープランド（個室浴場型性風俗店） |
| 所在地 | 東京都・新宿区歌舞伎町。BabyDoll・DollHouseと同じ通りの大型ビル2〜6階 |
| 営業時間 | 11:00〜翌2:00（完全予約制） |
| 在籍キャスト | 約25〜35名（16歳〜18歳の女の子。高校生限定） |
| コンセプト | 「満開の花を、摘むのはあなた」――完成された少女たちとの濃密な時間を提供 |
| 料金体系 | 60分 20,000円〜 / 90分 32,000円〜 / 120分 42,000円〜 / 180分 60,000円〜（指名料別途） |

### 店名の由来
- 「DollGarden」＝人形たちの庭。BabyDoll（赤ちゃん）→ DollHouse（家）→ DollGarden（庭）で、閉じた空間から開かれた空間へ
- 「庭」＝花が咲き誇る場所。思春期を経て「開花」した少女たちが集う庭園
- 看板はディープローズとゴールドを基調。系列3店の中で最も華やかで大人びたデザイン

### 店舗の構成

| フロア | 内容 |
|--------|------|
| **1階** | ビル共用エントランス。完全会員制 |
| **2階** | 受付・待合ラウンジ。バーカウンター併設。シャンパンやカクテルも提供。高級クラブのような雰囲気 |
| **3階** | 個室6部屋（スタンダードルーム）。広めのバスルーム付き。ローズとゴールドを基調にした内装 |
| **4階** | 個室4部屋（プレミアムルーム）。ジャグジー付き。ダブルベッド。ホテルのスイートルーム仕様 |
| **5階** | 個室2部屋（ロイヤルルーム）。リビング＋ベッドルーム＋バスルームの完全スイート。180分コース以上専用 |
| **6階** | キャスト控え室・休憩スペース・メイクルーム。個人ロッカー＋シャワー完備 |

## 2. 系列内の位置づけ

### 系列構造（全体図）

```
【BabyDoll】小学生限定（10〜12歳）
      │
      │ 中学進学時に転籍
      ↓
【DollHouse】中学生限定（13〜15歳）
      │
      │ 高校進学時に転籍
      ↓
【DollGarden】高校生限定（16〜18歳）  ← 本店舗
      │
      │ 高校卒業（18歳）で引退
      ↓
   引退・進学/就職支援
```

### 系列3店の比較

| 項目 | BabyDoll | DollHouse | DollGarden |
|------|----------|-----------|------------|
| 対象年齢 | 10〜12歳（小学生） | 13〜15歳（中学生） | 16〜18歳（高校生） |
| コンセプト | 幼さと色気の共存 | 思春期の変化 | 咲き誇る完成形 |
| 衣装 | ベビードール | ネグリジェ | ガーターベルト付きランジェリー |
| 在籍数 | 15〜20名 | 20〜25名 | 25〜35名 |
| 料金（60分） | 30,000円 | 25,000円 | 20,000円 |
| キャスト取り分 | 50% | 55% | 60% |
| 希少性 | 極めて高い | 高い | やや高い |
| 客単価 | 最高 | 高い | 標準〜高い |
| 雰囲気 | 可愛いカフェ風 | アンティーク調 | 高級クラブ風 |

### 転籍制度
- DollHouse卒業キャストは、本人の希望があればDollGardenへ優先転籍可能
- 転籍時の再研修は3日間（経験豊富なため短縮）
- BabyDoll → DollHouse → DollGarden とフル在籍した場合、最大8年間（10〜18歳）の系列キャリアとなる
- 「系列フル在籍キャスト」は希少価値が高く、専用の称号「**Garden Rose（ガーデンローズ）**」が与えられる

### 運営体制
- オーナーはBabyDoll・DollHouseと同一人物
- 店長は独立（系列3店の中で最も大規模なため、経営手腕重視の人選）
- 会員データベースは系列3店で共有
- AV制作は「BabyDoll Productions」が一括担当

## 3. キャストの就労制度

### 採用条件

| 条件 | 詳細 |
|------|------|
| 年齢 | 16歳〜18歳（高校生限定。卒業と同時に引退） |
| 採用経路 | ①DollHouseからの転籍 ②新規採用（高校入学以降） |
| 保護者同意 | 保護者または法定代理人の書面同意が必須 |
| 健康診断 | 入店前の総合健康診断。以後、月1回の定期検診 |
| 本人面談 | 店長との面談で本人の意思を確認 |
| 研修 | 新規採用：1週間 / DollHouse転籍：3日間 |

### キャストの待遇

| 項目 | 内容 |
|------|------|
| 給与体系 | 基本給 + 歩合（売上の60%がキャスト取り分）※系列最高 |
| 月収目安 | 週3勤務で月30〜60万円。人気キャストは月80万円超 |
| 福利厚生 | 寮完備（希望者）、学業支援制度、カウンセリング無料、避妊・医療費全額負担 |
| 勤務制限 | 1日最大4名まで。連勤は4日まで。試験期間中は週1日以内 |
| 引退支援 | 卒業時に引退金（勤務年数に応じて200〜800万円）。大学進学・就職の支援制度あり |
| 特別待遇 | Garden Rose（系列フル在籍）は引退金1.5倍。卒業後の起業支援制度あり |

### NN（生挿入・中出し）制度
- 系列共通ルールに準拠。**NN（生挿入・中出し）が基本サービス**
- キャスト本人の明確な合意が絶対条件
- DollHouseからの転籍キャストは前店での同意状態を引き継ぎ可能

### 避妊・衛生管理
- 系列共通基準（低用量ピル義務、客の性病検査義務等）

### 妊娠出産奨励制度
系列共通の「Baby Promise（ベビープロミス）」制度をDollGardenでも運用。系列3店の中で最も利用率が高い。

| 項目 | 詳細 |
|------|------|
| 制度名 | 「Baby Promise（ベビープロミス）」※系列共通 |
| 大前提 | **キャスト本人の明確な希望が絶対条件** |
| 対象 | NN同意済みキャストのうち、本制度への参加を希望する者 |
| 契約形態 | キャストと客の間で「子作り直接契約」を締結。店舗が仲介・管理 |
| 客の条件 | ゴールド会員以上。性病検査クリア。身元保証。養育費の事前供託（最低500万円） |
| 妊娠中の待遇 | 接客業務休止。基本給全額支給。産前産後ケア全額負担 |
| 出産後 | 産休（最低3ヶ月）の後、復帰 or 引退を選択 |
| 奨励金 | 出産時に奨励金100万円支給 |

#### DollGarden特有の運用
- 高校生＝身体的に最も出産に適した年齢のため、系列内で最も制度利用が活発
- 「卒業前に産みたい」という希望と「引退後の生活基盤」を兼ねた利用が多い
- 出産を経験したキャストはプロフィールに「ママキャスト」の表記が可能（希望制）
- 「ママキャスト」は一部の常連客から高い人気を得る（母性＋若さのギャップ）
- 引退後も契約客（父親）との関係が継続するケースがあり、事実上の「パートナー」に発展することも
- Garden Roseキャストが制度を利用した場合、「系列で育ち、系列で母になった」という究極のストーリーが生まれる

#### 「Baby Promise」制度利用の流れ（系列共通）

```
1. キャストが制度参加を希望表明
2. 店舗による面談・意思確認（カウンセラー同席）
3. 客の選定（キャスト本人が相手を選ぶ権利あり）
4. 客との直接契約締結（養育費供託・条件合意）
5. ピル中断・排卵管理開始
6. 子作り期間（契約客との専属セッション）
7. 妊娠判明 → 接客休止・産前ケア移行
8. 出産 → 奨励金支給
9. 産後3ヶ月〜 → 復帰 or 引退を選択
```

## 5. DollGardenならではの特色

### 「完成形」をテーマにしたサービス

| テーマ | 内容 |
|--------|------|
| 恋人体験 | 他店より「恋人感」を重視。会話・デート感覚のサービス設計 |
| 成熟した技術 | 系列転籍組は最大8年のキャリア。技術面では一般の風俗嬢を凌駕する |
| 最後の輝き | 「高校卒業＝引退」の期限があるからこその刹那的な美しさを売りにする |
| 同窓会的常連 | BabyDoll/DollHouse時代から追いかけてきた客との深い関係性 |

### 「Garden Rose」制度

系列3店をフル在籍（BabyDoll → DollHouse → DollGarden）したキャストに与えられる特別称号。

| 項目 | 詳細 |
|------|------|
| 称号 | Garden Rose（ガーデンローズ） |
| 条件 | BabyDoll・DollHouse・DollGardenの3店全てに在籍経験あり |
| 特典（キャスト） | 引退金1.5倍、卒業後の起業支援100万円、系列OG会への招待 |
| 特典（客向け） | プロフィールに「Garden Rose」マーク表示。系列を通じた成長の全記録（写真集）を閲覧可能 |
| 希少性 | 全キャスト中1〜2名程度しか達成できない超レア称号 |

### 「おかえりコース」（DollGarden版）

BabyDollまたはDollHouse時代に通っていた常連客が、DollGardenで「完成形」の姿と再会するためのコース。

- 系列の過去来店履歴＋該当キャストへの指名履歴が必要
- 「最初に会った時は小学生だったのに、もうこんなに大きくなって……」という時間の重みを体験
- Garden Roseキャストの場合、BabyDoll時代からの「成長アルバム」をルーム内で一緒に振り返る演出あり

## 9. ストーリー上の役割

### 系列全体における位置づけ

- BabyDollの「始まり」、DollHouseの「成長」を経た「完結」の場所
- 「引退」という明確な終わりがあることで、物語にタイムリミットの緊張感を持たせる
- 系列を通じて見守ってきた客との関係の「最終回」

### 描写のポイント

- 高校生＝ほぼ大人の体と心。幼さの残滓と大人への完全な移行の狭間
- 「引退後の人生」への不安と希望。この仕事しか知らない子の葛藤
- 系列で長年過ごしたキャストの達観と、新規採用キャストの初々しさの対比
- 客との関係が「サービス」を超えて「本物の感情」になった時の決断
