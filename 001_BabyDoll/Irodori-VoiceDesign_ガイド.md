# Irodori-TTS VoiceDesign ガイド：莉子×雫 声の出し分け

> Irodori-TTS-600M-v3-VoiceDesign 用  
> BabyDoll 莉子・雫の音声合成設定と、Caption の推奨ルール  
> 作成日：2026-06-23

---

## 1. v3 VoiceDesign の基本構造

v3 では **3つの入力を組み合わせて** 声を制御する。

| 入力 | 役割 | 必須？ |
|------|------|--------|
| **Text** | 読み上げる本文。絵文字を埋め込んで感情・効果音を制御 | 必須 |
| **Reference Voice** | ベースとなる声質（WAV。声のクローン） | 任意（`--no-ref` で省略可） |
| **Caption** | 感情・話し方・距離感・音響環境を日本語で指定 | 任意（VoiceDesign の核心） |

### 声質の固定方法（重要）

VoiceDesign はキャプションだけだと **生成のたびに声質が変わる**。キャラの声を固定するには：

1. **方法A（推奨）**: VoiceDesign で理想の声が出たら、その WAV を **Reference Voice として再利用**。以降は Caption で **演技・感情だけ** を変える
2. **方法B**: `--seed` を固定して同じキャプションで生成（再現性はあるが微妙に揺れる）
3. **方法A+B併用**: Reference Voice + seed 固定 + Caption が最も安定

## 2. 莉子と雫の声質キャプション

### 莉子（佐藤莉子・10歳・小悪魔）

**基本キャプション（声質定義用）**：

```
高めの声の幼い女の子が、自信に満ちた明るい口調で、少し生意気に、テンポよく話している。
```

| 要素 | 設定 | 根拠 |
|------|------|------|
| 声の高さ | 高め | 10歳・133cm |
| 年齢感 | 幼い女の子 | 最年少キャスト |
| 話し方 | テンポよく、生意気 | 小悪魔・からかい系 |
| 印象 | 自信に満ちた、明るい | 大人を翻弄する性格 |

**場面別バリエーション**：

| 場面 | Caption |
|------|---------|
| 営業（小悪魔モード） | 高めの声の幼い女の子が、甘えるような口調で、からかうように、テンポよく話している。 |
| 素の莉子（控え室等） | 高めの声の幼い女の子が、少し疲れた様子で、ぼそっと独り言のように話している。 |
| 感情が揺れる場面 | 高めの声の幼い女の子が、声を震わせながら、小さな声で、必死に平静を保とうとしている。 |
| 怒り・拗ね | 高めの声の幼い女の子が、不満げに、拗ねたように、少し早口で話している。 |
| 喘ぎ・撮影シーン | 高めの声の幼い女の子が、息を荒くしながら、甘い声で、切なげに声を漏らしている。 |

---

### 雫（白井雫・12歳・清楚）

**基本キャプション（声質定義用）**：

```
やや高めの声の少女が、控えめで丁寧な口調で、おずおずと、ゆっくり話している。
```

| 要素 | 設定 | 根拠 |
|------|------|------|
| 声の高さ | やや高め（莉子より低い） | 12歳・148cm。莉子より成長している |
| 年齢感 | 少女 | 「幼い女の子」より大人寄り |
| 話し方 | ゆっくり、丁寧 | 語尾が崩れない。丁寧語キャラ |
| 印象 | 控えめ、おずおず | 恥ずかしがり屋・声が小さい |

**場面別バリエーション**：

| 場面 | Caption |
|------|---------|
| 通常（丁寧語） | やや高めの声の少女が、控えめで丁寧な口調で、少し緊張しながら、ゆっくり話している。 |
| 恥ずかしい場面 | やや高めの声の少女が、恥ずかしそうに、声を小さくして、たどたどしく話している。 |
| 我慢・耐える場面 | やや高めの声の少女が、声を押し殺すように、震えながら、小さな声で耐えている。 |
| 丁寧語が崩れる絶頂 | やや高めの声の少女が、必死に声を抑えようとしながら、甘い声が漏れ出している。 |
| 泣きながら | やや高めの声の少女が、嗚咽を堪えながら、途切れ途切れに、弱々しく話している。 |

---

### 出し分けの差分まとめ

| 差分ポイント | 莉子 | 雫 |
|-------------|------|-----|
| 年齢表現 | 「幼い女の子」 | 「少女」 |
| 声の高さ | 「高めの声」 | 「やや高めの声」 |
| テンポ | 「テンポよく」 | 「ゆっくり」 |
| 態度 | 「自信に満ちた」「生意気」 | 「控えめ」「おずおず」 |
| 感情のベース | 明るい・挑発的 | 丁寧・緊張 |

**この差分を維持すれば、同一 Caption 構文でもキャラが分離する。**

## 3. Caption 推奨ルール（公式準拠）

### 3.1 基本構文

```
【声の高さ】の【年齢感・性別】の声で、【印象・態度】で、【話し方】で、【感情】を込めて話している。
```

**1〜2文に収める**のが安定のコツ。長く書くほど良いわけではない。

### 3.2 指定すべき要素（優先順）

| 優先度 | 要素 | 例 |
|--------|------|-----|
| 1 | 声の高さ・年齢感 | 高めの声の幼い女の子 |
| 2 | 感情・態度 | 苛立ち、申し訳なさ、甘え |
| 3 | 話し方のテンポ | ゆっくり、早口、テンポよく |
| 4 | 距離感 | マイクに近い、囁き、遠くで叫ぶ |
| 5 | 音響環境 | （基本的に指定しない。後述） |

### 3.3 禁止・非推奨事項

#### 矛盾した指示を書かない

```
❌ 低い声だけど高く、怒っているけど穏やかに、早口だけどゆっくり
✅ 低めの声で、怒りを抑えながら、ゆっくり話している
```

モデルがどの要素を優先すべきか判断できず、出力が不安定になる。

#### エコー・リバーブを無意味に指定しない

```
❌ Caption に「エコーがかかった」「残響のある」「広い部屋で」
❌ Text に 📢 を全セリフに挿入
✅ 必要な場面でのみ 📢 を1回だけ挿入
```

📢 は「エコー・リバーブ」の絵文字。**意図しないエコーがかかる原因の大半は、Caption で音響環境を過剰に記述するか、📢 を不用意に使うこと。** 普通のセリフにはエコー系の指示を一切入れない。

#### Reference Voice と Caption で声質を矛盾させない

```
❌ Reference = 低い男性の声 → Caption = 「高い声の少女」
✅ Reference = 高めの少女の声 → Caption = 「怒りを込めて、早口で」（感情・演技だけ変える）
```

Reference で声質を固定し、Caption では **感情・話し方・距離感だけ** を制御するのがベストプラクティス。

#### 複雑すぎる Caption を書かない

```
❌ 高めの声の幼い女の子が、甘えるような口調で、少し拗ねながらも本当は嬉しくて、でも素直になれず照れ隠しをしていて、早口だけど時々ゆっくりになり、囁くような声で、マイクに近い距離感で、エコーが少しかかった明るい部屋で話している。
✅ 高めの声の幼い女の子が、照れ隠しで拗ねたように、少し早口で話している。
```

**要素は3〜4個まで**。それ以上は絵文字制御で補う。

### 3.4 音響環境に関する指針

| 状況 | 対応 |
|------|------|
| 普通のセリフ | **音響環境は一切書かない**（デフォルトでクリアな音声） |
| 電話越し | 📞 を Text に挿入 or Caption に「電話越しの声で」 |
| 囁き | 👂 を Text に挿入 or Caption に「囁くように」 |
| 広い空間でのエコー | 📢 を Text に挿入（**本当に必要な場面だけ**） |
| 屋外の風音 etc. | Caption では制御困難。後処理で対応推奨 |

## 4. 絵文字制御リファレンス（BabyDoll でよく使うもの）

| 絵文字 | 効果 | 莉子の使用場面 | 雫の使用場面 |
|--------|------|---------------|-------------|
| 😏 | からかい・甘え | 小悪魔の営業トーク | ほぼ使わない |
| 🫣 | 恥ずかしそう・照れ | 素の莉子が出る場面 | 客の前で赤くなる場面 |
| 🥺 | 声の震え・自信なさげ | 感情が揺れる場面 | 初接客・怖い場面 |
| 😊 | 嬉しそう・楽しげ | からかい成功時 | 褒められた時 |
| 😠 | 怒り・不満・拗ね | 「言うな！」系 | ほぼ使わない |
| 🥵 | 喘ぎ・うめき | 撮影シーン | 撮影シーン |
| 😮‍💨 | 溜息・吐息 | 呆れ・安堵 | 我慢の後の息 |
| ⏸️ | 間・沈黙 | 素に戻る瞬間の間 | 言葉を探す間 |
| 😭 | 泣き・嗚咽 | 仮面が外れる場面 | 我慢が限界の場面 |
| 🐢 | ゆっくり | — | 丁寧語で慎重に話す |
| ⏩ | 早口 | まくしたてる莉子 | — |
| 🫶 | 優しく | 後輩を安心させる時 | 感謝を伝える時 |
| 👂 | 囁き | 耳元で煽る | 小声で本音を言う |
| 😖 | 苦しげ | — | 行為中の我慢 |
| 😰 | 慌て・緊張 | 素が出てしまった時 | 初めての場面 |
| 💋 | リップノイズ | キスの前後 | キスの前後 |
| 👅 | 舐める音・水音 | フェラ・口移し | フェラ |
| 🤭 | くすくす笑い | からかいの笑い | — |

### 絵文字の使い方ルール

1. **Text 内に直接埋め込む**（Caption ではなく）
2. **効果を強調したい場合は同じ絵文字を重ねる**（例：🥵🥵）
3. **絵文字だらけにしない**。1セリフに1〜2個が適正。3個以上は不安定になりやすい
4. **日本式の顔文字 `(^^)` は使わない**。Unicode 絵文字のみ対応

## 5. 実際の生成例

### 例1：莉子の営業トーク

```
Text:    ねぇ😏 おじさん、こんな小さい子でいいの？⏸️ 変態だね〜🤭
Caption: 高めの声の幼い女の子が、甘えるような口調で、からかうように、テンポよく話している。
```

### 例2：雫の初接客

```
Text:    ……怖いです🥺 でも、来てくださって⏸️ ありがとう、ございます🫣
Caption: やや高めの声の少女が、声を震わせながら、控えめで丁寧な口調で、ゆっくり話している。
```

### 例3：莉子の素（仮面が外れる）

```
Text:    😮‍💨 ……つかれた。⏸️ あたし今日、何人相手したっけ
Caption: 高めの声の幼い女の子が、疲れた様子で、ぼそっと独り言のように、力なく話している。
```

### 例4：雫の絶頂（丁寧語が崩れる）

```
Text:    あっ🥵 ……だめ、……そこ⏸️ 来ます……っ😖
Caption: やや高めの声の少女が、必死に声を抑えようとしながら、甘い声が漏れ出している。
```

### 例5：莉子が雫を安心させる

```
Text:    大丈夫🫶 あたしがいるから。⏸️ ——怖くないでしょ？😊
Caption: 高めの声の幼い女の子が、穏やかに、優しい口調で、ゆっくり話している。
```

## 6. 運用フロー

### 初回セットアップ（キャラの声を確立する）

```
1. VoiceDesign（Caption のみ・--no-ref）で理想に近い声を探す
   → seed を10パターンほど試す
   → 莉子と雫それぞれの「基本キャプション」で生成

2. 理想の声が出た WAV を保存
   → riko_base_voice.wav
   → shizuku_base_voice.wav

3. 以降はこの WAV を Reference Voice として使用
   → Caption では「感情・話し方・距離感」だけ変える
   → 声質は Reference で固定される
```

### セリフ生成（通常運用）

```
1. Text にセリフ本文を入力（絵文字を適宜挿入）
2. Reference Voice にキャラの基本 WAV を指定
3. Caption にそのセリフの「演技指示」だけを書く
4. 生成 → 聞いて調整
   → Caption の感情を微調整
   → 絵文字の位置・個数を調整
   → seed を変えてみる
```

## 7. よくある失敗と対策

| 失敗 | 原因 | 対策 |
|------|------|------|
| 意図しないエコーがかかる | Caption に「広い部屋」「残響」等を書いた / 📢 の誤挿入 | 音響環境を Caption に書かない。📢 は意図的な場面のみ |
| 莉子と雫の声が似てしまう | Caption の差分が不足 | 「幼い女の子」vs「少女」、「テンポよく」vs「ゆっくり」の差を徹底 |
| 声がセリフごとに変わる | Reference Voice なしで生成 | 基本 WAV を Reference に固定する |
| 感情が反映されない | Caption が長すぎる | 要素を3〜4個に絞る。残りは絵文字で |
| 読み間違い | 難読漢字・固有名詞 | ひらがな・カタカナに開く |
| 不自然な息継ぎ | 1文が長すぎる | 句点や改行でセリフを区切る |
