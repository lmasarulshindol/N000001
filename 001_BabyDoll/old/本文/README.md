## 行為本体の共通フロー（全5話）

```text
入浴  ──▶  口での奉仕  ──▶  口内射精（1回目の射精）
                                     │
                                     ▼
                          指愛撫  ──▶  少女の絶頂
                          （感じやすい子は連続絶頂 / 複数回）
                                     │
                                     ▼
                               生挿入（処女膜破り※経験者以外）
                                     │
                                     ▼
                          挿入中 / 射精時同時の絶頂
                                     │
                                     ▼
                              中出し（2回目の射精）
                                     │
                                     ▼
                                   余韻
```

- **口内射精と膣中出しは別段落**で必ず明示的に描写
- 処女喪失（第1・3・4・5話）は痛み・悲鳴・涙を省略せず描写
- 楓（第2話）は経験者として通常挿入

## 共通の描写方針

- **三層構造**: 導入20〜25% → 行為本体50〜60% → 余韻20〜25%
- **NN（生挿入・中出し）**: 全話で描写あり（BabyDollの基本サービス）
- **処女喪失**: 第1・3・4・5話（莉子・ひなた・花音・澪）
- **経験者のプロ初日**: 第2話（楓）
- **視点**: 全話客視点一人称
- **キャスト描写**: 「仮面」と「素」の二重構造
- **セリフ設計**: TTS合成向けに、1行1〜15文字基本、吐息・喘ぎ声を独立行に
