# プロット：ベッド本番（口内射精〜中出し余韻）

> BabyDoll キャスト・**佐藤莉子（10歳）**を主役とする**1セッション内**の行為ライン。  
> 流れ: **オープニングトーク → キス → 口での奉仕 → 口内射精 → 愛撫 → 絶頂（莉子）→ 生挿入おねだり → 正常位 → 後背位 → 対面座位 → 正常位 → 中出しおねだり → 中出し → 余韻**

---

## 0. 既存設定との接続

- **舞台**: ソープランド「BabyDoll」内、個室ベッド。入浴・マットは**別段**（本章では「済んだあと」または**ロング/VIP的に会話多め**で本編開始）として扱うと整合しやすい。
- **制度**: `BabyDoll.md` の**NN（生挿入・中出し）**はキャスト合意が前提。描写では**合意の一言**を短く毎度アンカーにする。
- **莉子の軸**: 小悪魔仮面／プロ意識と、一瞬素が出る**揺れ**。母の事実を**この場で説明**しすぎない（内面1文程度に留めるのが無難）。

---

## 1. 全体の心拍（3幕）

1. **儀式としての前口**（会話とキス）— 緊張と小悪魔の仮面を先に出す。  
2. **口での完了 → 手と言葉での整え**— 客に一度抜かせ、莉子主導の山を作る。  
3. **生への欲と体位の積み上げ → 中出し名乗り → 余韻**— 体位ごとに深さ・見え方・支配/被支配のニュアンスを変え、最後に**営業的おねだり**と**一瞬の本音のズれ**。

---

## 2. ブロック詳細

### 2.1 オープニングトーク

- **目的**: 部屋を「性」一色ではなく**会話の場**に戻し、次の密着（キス）へ橋をかける。  
- **莉子**: いつもの挑発。嫌な箇所・体調の**さりげない確認**（NN面談済みなら軽い一言で足りる）。  
- **会話の芯**: 相手の予定をからかい、**「次、何するか分かりながら口だけ止めてる」**系の釣りで期待感を上げる。  
- **描写**: ベッド端の距離、照明、ベビードール越しの肌。視線の上下で**子ども／娼婦**の揺れを一度だけ入れる。

### 2.2 キス

- **遷移**: 言葉が落ち着いたところで、悪戯口調（例: 「口、借りる？」）で距離ゼロ。  
- **段階**: 触れない口づけ → 唇の奪い合い → 舌（**莉子主導で入る**か**相手が追う**かで性格が出る）。  
- **内面**: 技術としての呼吸の合わせ方 × 一瞬**年相応の照れ**。小悪魔なら、深まった直後の**ニヤリ**で歯止め。  
- **接続**: 口内射精パートのため、唾液とリズムの描写を**後段の口**に繋げる。

### 2.3 口での奉仕

- **ポジション**: 上から見下ろし／**上目**、体格（小柄）を活かした**視点の切り替え**。  
- **段階（執筆用）**: 手首で軸 → 先端（比喩多め）→ 深さ・速度の**二段階** → 髪/耳に触る手の**扱い**（外す・逃がす・仕事として許可）。  
- **制御**: 焦らしと加速を**台詞一つ**で宣言（小悪魔の出番）。

### 2.4 口内射精

- **前**: 悪戯＋短い**最終合意**（「ここに出していいの？」等）。我慢限界のサイン。  
- **中**: 呼吸、喉/舌の**仕事的描写**＋反射的一瞬。  
- **後**: 飲む/出すはキャラと店の運用に合わせる。**飲了後**はうがい・ドリンク・軽口で**次の愛撫**へ。  
- **心拍上の意味**: 一度**射精の「勝ち」**を客に与え、莉子は**挿入用の弾**を切っていない、という力関係のスパイスにも使える。

### 2.5 愛撫

- **位置**: 口直後の過敏帯を避け、**内腿・首・背・胸**から再び亀頭周辺へ戻る猶予。会話再開で**敬語/タメ口**の混ざり。  
- **合意の身体**: 相手の手を胸や腰に当て、**挿入前の触れ方の確認**（台詞少なめ）。

### 2.6 絶頂（莉子）

- **位置づけ**: 挿入前に**指/口**で果たせ、ペース掴み・情動の釣り上げ用。小柄・年齢に合う圧/角度。声は**演技と本物の中間**をトーンで調整。  
- **直後**: 上気したまま**生挿入のおねだり**へ心拍をのせる。

### 2.7 生挿入おねだり

- **台詞の二重**: 営業的フレーバー（「好きにしていいのに」等）＋ 欲の**子どもっぽいせがみ**（作風に合わせて片方に寄せると一貫する）。  
- **同意の見せ方**: NN合意を前提に、**「今日は、ナマで」**のような**短い錨**を一度。  
- **身体**: 潤い、入り方、**最初の入りにくさ**（体格差）を一行。

### 2.8 正常位（1回目）

- **意図**: 顔がよく見える。小悪魔の仮面の**残り方**、目・口元の描写。  
- **展開**: 浅→深、小刻み。体重の乗り方（小柄枠）。**リズムが噛み合う一拍**。  
- **会話・喘ぎ**: 台詞多め。薄い**演技**を匂わせる余地。

### 2.9 後背位

- **意図**: 顔を画面から消し、**声だけ**残す対比。  
- **身体**: 背のライン、腰、角度。小悪魔系なら**罵倒混じり**と**甘え**の混線。  
- **心拍**: 匿名性の快感 → 次の**対面**で顔に引き戻す。

### 2.10 対面座位

- **意図**: 顔と結合の同時。抱擁**面積**最大。  
- **展開**: 額、呼吸。体重移動。ここで**営業台詞が一瞬詰まる**など、本音フック。  
- **中出しへの導線**: 最後の山の**手前**に置くと力学的に揃いやすい。

### 2.11 正常位（2回目・フィニッシュ寄り）

- **意図**: 疲労にくい**締めの体位**。**シーツ/マット**に寝かせ、速度の上乗せ、奥の意思、脚の開き・小さな痙攣。  
- **会話**: 短句・反復に寄せ、**中出しおねだり**へ。

### 2.12 中出しおねだり

- **営業と本音**: 店設定でNNは標準でも、**濃度の違い**（「中に」「このまま」等）を台詞で付ける。小悪魔なら、**最後の一行だけ素**、または**完璧営業で中が空**を一文。  
- **合意の再線**: 暴力に読めない**双方のイエス**（一語可）。

### 2.13 中出し

- **五感**: 圧、熱、量、**溢れ**（年齢・体型的には比喩控えめ）。  
- **直後**: 止まる／抱く／笑い、いずれか。  
- **事後（設定）**: 低用量ピル等の**安心**は、説教にならない**一行**に。

### 2.14 余韻

- `BabyDoll.md` の**③余韻**（シャワー、見送り、控え室の顔）に接続。  
- **会話**: 体のあと＋**どうでもいい雑談**で呼吸を戻す。  
- **莉子**: 廊下まで**営業**→ 扉の向こうは**別章**。

## 3. 時間配分目安（ベッド本番ブロックのみ）

> 全コース60〜90分の**うち、入浴・マットを除いた本番**に相当。章全体の比率は「導入20〜25% / 行為50〜60% / 余韻20〜25%」に合わせて前後章で調整。

| 塊 | 目安 | メモ |
|----|------|------|
| トーク＋キス | 5〜8分 | 入浴抜きならここ厚め |
| 口＋口内 | 8〜12分 | 客1回目 |
| 歯磨き/ドリンク等 | 2〜4分 | 現実ライン |
| 愛撫＋彼女の絶頂 | 6〜10分 | 挿入前の体温 |
| 挿入（4体位） | 15〜25分 | 心拍の山を体位で区切る |
| 中出し・抱擁 | 2〜4分 | |
| 事後＋軽会話 | 3〜6分 | 余韻章へ接続可 |

## 4. 執筆用フック（毎段に1つ意識）

- **仮面**: 小悪魔台詞・上目・意図的な茶化し  
- **亀裂**: 黙る／言葉が出ない／笑いが早口  
- **合意**: NN・中出しが**合意上の行為**だと一瞬で分かる台詞  
- **身体の現実**: 小柄・最年少枠の**入り方・圧・疲労**（一行）
