# BabyDoll（ベビードール）

> 小説舞台設定案

## 1. 世界観概要

少子化と経済格差が極限まで進行した近未来の日本。政府は労働力確保と若年層の経済的自立を目的に、従来の年齢制限を大幅に緩和する「若年就労特別法」を施行した。この法律により、10歳以上の少女が性風俗産業で就労することが合法化された世界。

### 時代設定
- **時代**: 近未来の日本（203X年頃）
- **社会背景**: 少子高齢化により労働人口が激減。社会保障制度が限界を迎え、「働ける者は何歳でも働く」という価値観が広まりつつある
- **法改正**: 「若年就労特別法」により、保護者の同意と本人の意思確認を条件に、10歳以上の性風俗業への就労が合法化
- **建前と本音**: 法律上は「若者の経済的自立と職業選択の自由を尊重する」という建前。実態は、経済的に困窮した家庭の子供たちが流入する構造

### キャストの待遇

| 項目 | 内容 |
|------|------|
| 給与体系 | 基本給 + 歩合（売上の50%がキャスト取り分） |
| 月収目安 | 週3勤務で月40〜80万円。人気キャストは100万円超 |
| 福利厚生 | 寮完備（希望者）、学業支援制度、カウンセリング無料、避妊・医療費全額負担 |

| 引退支援 | 中学進学時に引退金（勤務年数に応じて100〜500万円）。進学支援・姉妹店への転籍斡旋あり |

### NN（生挿入・中出し）基本サービス制度
- BabyDollでは **NN（生挿入・中出し）が基本サービス** として標準提供される
- ただし、**キャスト本人の明確な合意が絶対条件**。キャストが拒否した場合、客は従わなければならない
- キャストは面談時にNNサービスへの同意・不同意を選択でき、いつでも撤回可能
- NN同意済みキャストのプロフィールにはその旨が記載され、客は事前に確認できる
- NN非同意のキャストも在籍可能（コンドーム着用での本番 or 本番なしコースのみ担当）

### 避妊・衛生管理
- キャストは全員、低用量ピルの服用が義務付けられている（費用は店舗負担）
- NNが基本サービスのため、客には入会時および3ヶ月ごとの性病検査を義務付けている
- 検査未提出・期限切れの客にはNN不可（コンドーム着用のみ）
- 個室は毎回消毒・清掃される

### 妊娠出産奨励制度
BabyDollでは、キャスト本人の希望に基づき、客との直接契約による子作り・妊娠・出産が認められている。

| 項目 | 詳細 |
|------|------|
| 制度名 | 「Baby Promise（ベビープロミス）」 |
| 大前提 | **キャスト本人の明確な希望が絶対条件**。店舗や客からの強制は一切不可 |
| 対象 | NN同意済みキャストのうち、本制度への参加を希望する者 |
| 契約形態 | キャストと客の間で「子作り直接契約」を締結。店舗は仲介・管理を行う |
| ピル中断 | 制度利用時は低用量ピルの服用を中断し、排卵日に合わせたスケジュールを組む |
| 客の条件 | ゴールド会員以上。性病検査クリア。身元保証。養育費の事前供託（最低500万円） |
| 妊娠中の待遇 | 妊娠判明後は接客業務を休止。基本給は全額支給。産前産後ケア全額負担 |
| 出産後 | 産休期間（最低3ヶ月）の後、本人の希望で復帰 or 引退を選択可能 |
| 子供の養育 | 契約内容に基づき、客（父親）が養育費を負担。キャストが育てる場合は系列の託児支援あり |
| 奨励金 | 出産時に店舗から奨励金100万円を支給 |

#### 制度の背景
- 少子化対策の国策に沿った形で、「若年出産奨励補助金」が政府から店舗に支給される仕組み
- 客にとっては「自分の子を産んでもらえる」という究極の独占欲の充足
- キャストにとっては経済的保障付きの出産が可能（通常の若年妊娠と異なり、手厚いサポートあり）
- 建前: 「少子化時代の新しい家族のかたち」。実態: 客の欲望とキャストの経済的事情の合致

#### 制度利用の流れ

```
1. キャストが制度参加を希望表明
2. 店舗による面談・意思確認（カウンセラー同席）
3. 客の選定（キャスト本人が相手を選ぶ権利あり）
4. 客との直接契約締結（養育費供託・条件合意）
5. ピル中断・排卵管理開始
6. 子作り期間（通常の接客枠内で、契約客との専属セッション）
7. 妊娠判明 → 接客休止・産前ケア移行
8. 出産 → 奨励金支給
9. 産後3ヶ月〜 → 復帰 or 引退を選択
```

### 「はじめて。」シリーズ

BabyDoll Productionsの看板シリーズ。**入店後の全キャストが必ず撮影する「処女喪失」の記録映像**。

| 項目 | 詳細 |

| 撮影時期 | 入店後、研修を経て初接客の**前**に撮影（処女喪失はAV撮影で行い、初接客時にはすでに非処女） |

| 尺 | 60〜90分（編集後） |

#### 「はじめて。」の流れ

1. **事前インタビュー**（〜10分）: カメラの前で自己紹介。名前、年齢、入店の理由（建前）、今の気持ち。ベビードール姿で撮影
2. **男優との対面**（〜5分）: 相手役との初顔合わせ。緊張の表情を記録
3. **前戯**（〜20分）: キス → 愛撫 → 口での奉仕 → 指での愛撫。カメラは2台体制（引き・寄り）
4. **処女喪失**（〜20分）: 生挿入。処女膜を破る瞬間をクローズアップ。痛み・涙・声の変化を逃さない。中出しまで
5. **事後インタビュー**（〜10分）: 行為直後の感想。汗と涙の残る顔で、カメラに向かって「はじめて」の感想を語る

#### 制作上のルール

- キャストの顔出しは原則OK（入店時の同意書で許諾済み）
- 「はじめて。」は**一度しか撮影できない不可逆コンテンツ**のため、プレミアム価格で販売
- 配信価格: 単品9,800円 / BabyDoll会員は5,000円
- 撮影当日はキャストの体調・精神状態を最優先。撮影中断も可能（ただし実際に中断されたケースはない）
- 「はじめて。」の撮影を終えたキャストのみ、客への「初めてコース」の提供が解禁される（※旧設定の「初めてコース＝処女喪失」は廃止。初めてコースは「はじめて。撮影済みキャストとの疑似初体験プレイ」にリニューアル）

#### 入店〜デビューまでのタイムライン（改訂版）


## 7. 社会的背景の詳細

### 「若年就労特別法」の経緯
1. 少子高齢化による労働力不足が深刻化
2. 児童の経済的貧困率が30%を超える
3. 「子供の貧困を放置するより、安全な環境での就労を認めるべき」という議論が台頭
4. 反対運動も激しかったが、与党が強行採決で成立させた
5. 施行後、風俗業界が「合法的な若年キャスト」を擁する店舗を次々にオープン

### 業界の現状
- BabyDollのような「若年専門店」は全国に約200店舗
- 大手グループが運営するチェーン店から、個人経営の小規模店まで様々
- BabyDollは「少数精鋭・高品質」を掲げる中規模店で、業界内での評判は高い
- 違法店（10歳未満のキャストを使う等）の摘発も相次いでおり、合法店との差別化が重要

### 世論
- **容認派**: 「経済的自立の選択肢が増えた」「厳しい規制の中で安全に働ける」
- **反対派**: 「児童の性的搾取の合法化だ」「貧困対策を就労で解決するな」
- **業界擁護派**: 「違法な闇営業よりも、合法・管理下の方が子供を守れる」
- **社会の空気**: 表立って賛成する人は少ないが、利用者は確実に存在する。「必要悪」として黙認する空気

## 8. ストーリー構成

### 基本コンセプト
BabyDollで働く少女たちの日常と、店を訪れる客との関わりを通じて描かれる群像劇。華やかなサービスの裏にある少女たちの素顔、葛藤、友情、そして時に芽生える本物の感情を描く。

### 物語の軸
- **表と裏のコントラスト**: 接客中のプロフェッショナルな姿と、控え室での素の表情の落差
- **少女たちの事情**: なぜこの仕事を選んだのか。家庭環境、経済的理由、自発的な意思、それぞれのバックストーリー
- **客との関係**: 単なるサービス提供者と客の関係を超えた、人間同士の繋がりが生まれる瞬間
- **成長と選択**: 小学校卒業＝中学進学という「卒業」が近づく中で、引退か姉妹店か、次の人生をどう選ぶかという現実

### 描写のポイント
- ソープランドという舞台設定をリアルに描写しつつ、少女たちの人間性を最優先に
