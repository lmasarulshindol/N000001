# BabyDoll（ベビードール）

> 小説舞台設定案

---

## 1. 世界観概要

少子化と経済格差が極限まで進行した近未来の日本。政府は労働力確保と若年層の経済的自立を目的に、従来の年齢制限を大幅に緩和する「若年就労特別法」を施行した。この法律により、10歳以上の少女が性風俗産業で就労することが合法化された世界。

### 時代設定
- **時代**: 近未来の日本（203X年頃）
- **社会背景**: 少子高齢化により労働人口が激減。社会保障制度が限界を迎え、「働ける者は何歳でも働く」という価値観が広まりつつある
- **法改正**: 「若年就労特別法」により、保護者の同意と本人の意思確認を条件に、10歳以上の性風俗業への就労が合法化
- **建前と本音**: 法律上は「若者の経済的自立と職業選択の自由を尊重する」という建前。実態は、経済的に困窮した家庭の子供たちが流入する構造

## 3. キャストの就労制度

### 採用条件

| 条件 | 詳細 |
|------|------|
| 年齢 | 10歳〜15歳（16歳以降は通常店舗への移籍 or 引退） |
| 保護者同意 | 保護者または法定代理人の書面同意が必須 |
| 健康診断 | 入店前の総合健康診断。以後、月1回の定期検診 |
| 本人面談 | オーナー・店長との面談で本人の意思を確認 |
| 研修 | 入店後2週間の研修期間（接客マナー、衛生管理、心理ケアの基礎） |

### キャストの待遇

| 項目 | 内容 |
|------|------|
| 給与体系 | 基本給 + 歩合（売上の50%がキャスト取り分） |
| 月収目安 | 週3勤務で月40〜80万円。人気キャストは100万円超 |
| 福利厚生 | 寮完備（希望者）、学業支援制度、カウンセリング無料、避妊・医療費全額負担 |
| 勤務制限 | 1日最大3名まで。連勤は3日まで。学業期間中は週2日以内 |
| 引退支援 | 16歳到達時に引退金（勤務年数に応じて100〜500万円）。進学・就職支援あり |

### NN（生挿入・中出し）基本サービス制度
- BabyDollでは **NN（生挿入・中出し）が基本サービス** として標準提供される
- ただし、**キャスト本人の明確な合意が絶対条件**。キャストが拒否した場合、客は従わなければならない
- キャストは面談時にNNサービスへの同意・不同意を選択でき、いつでも撤回可能
- NN同意済みキャストのプロフィールにはその旨が記載され、客は事前に確認できる
- NN非同意のキャストも在籍可能（コンドーム着用での本番 or 本番なしコースのみ担当）

### 避妊・衛生管理
- キャストは全員、低用量ピルの服用が義務付けられている（費用は店舗負担）
- NNが基本サービスのため、客には入会時および3ヶ月ごとの性病検査を義務付けている
- 検査未提出・期限切れの客にはNN不可（コンドーム着用のみ）
- 個室は毎回消毒・清掃される

### キャスト衣装
- **ベビードール型ランジェリー**: 店名を体現する透け素材（シフォン/チュール）のベビードール衣装。大事なところが丸見えで、下着は着けない。キャストごとにイメージカラーが異なる
- 対面・入室時はこの衣装で登場。入浴前に脱ぐ
- キャストが客を迎えに来る際の「第一印象」を決定づける重要な衣装
- デザインはレースとリボンが基調。幼い体を包む透け素材と、そこから透ける裸体のギャップが店のコンセプト

### サービスの流れ（スタンダードコースの場合）

1. **受付・待合**（〜10分）: 受付で会員証提示。待合ロビーでドリンクを飲みながら待機
2. **対面・入室**（〜5分）: キャストがロビーまで迎えに来る。個室へ案内
3. **入浴**（〜15分）: キャストが体を洗ってくれる。スキンシップの始まり
4. **マットプレイ**（〜15分）: 専用マットでのローションプレイ。キャストの全身を使ったサービス
5. **本番**（〜10分）: ベッドに移動しての生挿入中出し（NN標準）
6. **シャワー・見送り**（〜5分）: 二人でシャワーを浴び、キャストが部屋の外まで見送り

## 7. 社会的背景の詳細

### 「若年就労特別法」の経緯
1. 少子高齢化による労働力不足が深刻化
2. 児童の経済的貧困率が30%を超える
3. 「子供の貧困を放置するより、安全な環境での就労を認めるべき」という議論が台頭
4. 反対運動も激しかったが、与党が強行採決で成立させた
5. 施行後、風俗業界が「合法的な若年キャスト」を擁する店舗を次々にオープン

### 業界の現状
- BabyDollのような「若年専門店」は全国に約200店舗
- 大手グループが運営するチェーン店から、個人経営の小規模店まで様々
- BabyDollは「少数精鋭・高品質」を掲げる中規模店で、業界内での評判は高い
- 違法店（10歳未満のキャストを使う等）の摘発も相次いでおり、合法店との差別化が重要

### 世論
- **容認派**: 「経済的自立の選択肢が増えた」「厳しい規制の中で安全に働ける」
- **反対派**: 「児童の性的搾取の合法化だ」「貧困対策を就労で解決するな」
- **業界擁護派**: 「違法な闇営業よりも、合法・管理下の方が子供を守れる」
- **社会の空気**: 表立って賛成する人は少ないが、利用者は確実に存在する。「必要悪」として黙認する空気

## 8. ストーリー構成

### 基本コンセプト
BabyDollに入店した新人キャストの少女を主軸に、**店の「内側」から見た日々**を描く成長譚。客が通り過ぎていく非日常の場所で働く少女たちが、何を感じ、どう変わり、何を選び取るのか——先輩・同僚・店のスタッフとの関わりを通じて、少女自身の人格が形作られていく過程を丁寧に追う。

### 物語の軸
- **「仕事」として性を扱う少女の内面**: 初めての接客、初めての本番、初めての指名。技術・感情・違和感の積み重ね
- **姉妹的な師弟関係**: 教育係の先輩キャストとの関係。指導・反発・信頼・別れ
- **大人たちの眼差し**: 彼女たちを「商品」ではなく一人の人間として見る大人（カウンセラー・店長）の視点
- **卒業までのカウントダウン**: 16歳の引退という必ず訪れる「終わり」が常に物語に影を落とす
- **家の事情と個人の選択**: 入店理由に「家族」が絡む少女たちが、次第に「自分のため」の動機を見つけていく過程

### 視点切り替えの方針
- **主視点**: 新人キャスト少女（物語のメイン）
- **副視点**: 教育係の先輩キャスト（姉貴分）／担当カウンセラー（大人枠）
- **三視点を章ごとにローテーション**することで、同じ出来事を複数の立場から立体的に描く
- 客視点は意図的に抑え、「少女たちの世界」を主役にする

### 描写のポイント
- ソープランドという舞台のリアリティは保ちつつ、**控え室・ロッカー・カウンセリング室**など「接客の外」の時間を重点描写
- 性行為シーンは少女の視点で「仕事の中で何を感じているか」を軸に。客は基本モブ扱い
- 歌舞伎町の夜の空気と、店内の淡いピンクの照明、キャストたちの素顔のコントラスト
- 友情・反発・嫉妬・憧れなど、10代前半の少女たちの感情の機微を丁寧に

## 9. 性行為シーンの描写方針

### 基本設計思想
- 行為シーンは **主人公少女の一人称的内面描写** を軸に構築する
- 「仕事としてどう振る舞うか」「本当は何を感じているか」の二層を常時描く
- 客は基本的に「記号的な存在」として描き、少女の視界・感覚・思考を中心に据える
- 初回・成熟後・特別な客 など、少女の「段階」によって描写の質感を変える

### 描写の三層構造

**① 準備（個室に入るまで）**
- 控え室での身支度。衣装を身につける所作、鏡の中の自分を見る瞬間
- 先輩からの声かけ、同僚との短い会話で気持ちを作る
- 受付からの指名通知を受けたときの感情（緊張・諦め・ふとした期待）
- ロビーへ向かう廊下の足音、照明の色、客を見つけた瞬間の第一印象

**② 行為本体**
- 入浴・マットプレイ・本番の手順を、**少女が頭の中で手順を確認しながら進む** 視点で描写
- 体の各部位の感覚（湯の温度、ローションの滑り、客の手の重さ、粘膜の感覚、体温）
- 仕事モードの「声」と心の中の「本音」のズレを随所に差し込む
- 客の反応を観察する少女の目線（「気持ちよくなってる？」「この人、優しい方かも」等）
- 身体描写: 幼い体の感覚を当事者視点で——胸の未発達、骨盤の狭さ、体の疲れ

**③ 余韻（見送り後）**
- シャワー・後処理。体についた客の名残を洗い流す所作
- 控え室に戻った瞬間の脱力、鏡で自分の顔を確認する習慣
- 同僚との「どうだった？」の会話、食事、スマホを触る時間
- 夜、寮や家に帰った後、ベッドで一日を反芻する沈黙

### シーンごとの文量目安
- 準備: 全体の **20〜25%**
- 行為本体: 全体の **45〜55%**
- 余韻: 全体の **25〜30%**（内面を丁寧に描くため余韻を厚めに）
- 1回の行為シーン全体で **3,500〜6,000字** を目安

### 特別な場面の扱い
- **初体験（処女喪失）**: 単独エピソード級の文量を割く（8,000〜12,000字）。前日譚・当日・翌日の三部構成
- **特殊客対応**: 通常と異なる描写ルールで差別化（例: 粗暴な客は簡潔に／優しい常連は情感豊かに）
- **NN非同意キャストの本番**: コンドーム着用の所作や心理的距離を丁寧に描く
