# 莉子 第0話「面接の日」プロット

> BabyDollの面接・身体検査を受ける11歳の少女・莉子の一日を描く入店前夜譚。

---

## 1. 概要

### あらすじ
11歳の莉子が、ひとりで歌舞伎町のソープランド「BabyDoll」へ面接に向かう。店長面談、カウンセリング、婦人科を含む健康診断——「働く」と決めた日に、彼女は自分の体と意思を大人たちに差し出していく。帰り道、電車の窓に映る自分の顔を見ながら、莉子は「これから始まる日々」をまだ想像もできないまま、小さな覚悟を固める。

### この話のテーマ
**「選ぶ」** ——莉子が自分の意思で、この世界に足を踏み入れることを選ぶ過程。大人たちから差し出される「まだ引き返せる」という言葉と、それでも前に進む11歳の選択。

### 視点
**莉子・一人称寄りの三人称**（章末にカウンセラー視点のワンシーンを挟み、大人側からの"観察"を入れる）

### 文量目安
**全体 10,000〜13,000字**

## 3. シーン構成（全7シーン）

### シーン1: 歌舞伎町へ向かう道（約1,500字）

**場所**: 自宅 → 電車 → 新宿駅 → 歌舞伎町の裏通り
**時間**: 平日の午後（学校を休んだ日／通信制で自由時間 のいずれか）

**描写の骨子**
- 朝、莉子が鏡の前で「普通の服」を選ぶ。白いワンピースと薄いカーディガン。大人っぽく見せたくて、でも子供っぽく見られたくもない、その葛藤
- 母親に告げる言葉（「子役事務所の面接、今日だから」等／※設定確定次第）
- 電車内——平日の昼間、スーツの大人たちに囲まれて小さく座る。揺れるたびに足が浮く
- スマホに残った店からの案内メッセージを何度も確認
- 新宿駅の東口を出た瞬間の、昼でも淀んだ空気
- 歌舞伎町一番街を抜け、細い路地へ。看板の色が派手さを失っていく
- 指定された雑居ビルの前に立つ。1階はシャッターが下りた古着屋。ここで合ってるのか何度も地図を見返す
- エレベーターのボタンに伸ばした指が震える

**心情のピーク**: 「引き返せる。今なら、まだ」→ それでもボタンを押す

### シーン2: 受付・待合ロビー（約1,500字）

**場所**: BabyDoll 2階 受付・待合
**登場**: 莉子、受付スタッフ

**描写の骨子**
- エレベーターの扉が開いた瞬間の第一印象——想像していた"風俗店"と全然違う。淡いピンクと白の内装、観葉植物、ホテルのラウンジのような静けさ
- 受付カウンターの若い女性スタッフが、莉子を見てにこりと笑う。「莉子ちゃん、お待ちしてました」——下の名前で呼ばれたことへの微かな違和感
- 身分証（健康保険証）を預ける。スタッフが書類を確認する間、莉子はロビーの革張りソファに小さく座る
- **足がソファの縁から浮く**——幼い体がこの空間で明らかに異物であることの象徴描写
- ホットココアを出される。マグカップが両手で包み込むほど大きい
- 壁に飾られた抽象画、間接照明、流れる軽いクラシック
- ふと、廊下の奥から少女の笑い声が漏れ聞こえる。すぐに扉が閉まって消える
- 莉子の内心：「ここに、あたしと同じ子たちが、いる」
- スタッフが戻ってきて「店長がお待ちです」と案内

**感情の質感**: 怖さと、妙な"守られている感"の両立。一見の丁寧さが逆に不気味

### シーン3: 店長面談（約2,500字・本話の核①）

**場所**: 2階の個室（店長室／応接室）
**登場**: 莉子、店長

**描写の骨子**

**（前半・挨拶と確認）**
- 店長が立ち上がって莉子を迎える。穏やかな口調、だが目は鋭い
- ソファに向かい合って座る。テーブルを挟んで大人と子供の対比
- 「来てくれてありがとう」から始まる、拍子抜けするほど丁寧な挨拶
- 「今日は3つやることがある。面談、カウンセリング、健康診断。全部終わるのに2〜3時間。大丈夫？」

**（中盤・仕事の実態説明）**
- 店長がまず告げる：「これから話すのは、全部本当のこと。怖くなったら途中で帰っていい」
- 業務の具体的な説明：
  - ソープランドという業態
  - 接客内容（入浴、マットプレイ、本番）
  - **NN（生挿入・中出し）が基本サービスであること**
  - キャスト本人がNNを拒否する権利
  - 避妊ピルの服用が義務であること
  - 1日最大3名まで、連勤制限
- 莉子の内心描写：言葉の一つ一つに体温が下がっていく。**「聞いてはいたけど、大人の口から直接言われると、違う」**
- 「……全部、やるんですか」と小さく聞き返す
- 店長は言葉を選びながら答える。「やる子もいるし、最初はコンドームから、って子もいる。あなたの希望次第」

**（後半・動機確認と意思確認）**
- 「どうしてウチを選んだの？」
- 莉子が答える。吃り気味に、家計の事情を（※具体は設定確定次第）
- 店長は驚かない。「そういう子、多いよ」
- 「今日決めなくていい。保護者同意書を持ってもう一度来てくれたら、研修に入れる」
- **「ただし、ここで働くって決めるなら、あなた自身の意思で決めてほしい。お母さんのため、じゃなくて」**
- 莉子、下を向く。「……考えます」
- 店長の最後の言葉：「研修期間中も、いつでも辞められる。それだけ覚えといて」

**このシーンの要点**:
- 店長を「搾取する大人」ではなく「プロとしての責任者」として描く
- 莉子が**初めて"仕事の中身"を大人の口から聞く瞬間**——知識としては知っていても、現実感が一気に押し寄せる
- 最後の「あなた自身の意思で」が、次話以降ずっと莉子の中に残る種になる

### シーン4: カウンセラー面談（約2,000字・本話の核②）

**場所**: 別室のカウンセリングルーム（柔らかい照明、観葉植物、クッション多め）
**登場**: 莉子、カウンセラー

**描写の骨子**

**（前半・場の雰囲気）**
- カウンセラーが部屋の扉を開けて「入って」と笑う
- 店長の部屋とは明らかに違う空気——柔らかい。ぬいぐるみが隅に置いてある
- 「ここでは仕事の話はしなくていいの。あなた自身のことを話してくれればいい」
- ハーブティーと小さなクッキーが出てくる

**（中盤・莉子の素顔が出る会話）**
- カウンセラーの質問：「学校、楽しい？」「好きな食べ物は？」「最近ハマってることある？」
- 莉子、最初は警戒。でも少しずつ口が開いていく
- 家族の話をする莉子の声が、一瞬だけ子供っぽくなる
- **「お母さん、好き？」**という直球の質問に、莉子が黙り込む
- 「…好きです。だから」と、そこで言葉が途切れる

**（章末・カウンセラー視点への切り替え）**
- 莉子が退室した後の数十秒
- カウンセラーがノートに記録を書き始める。手が一瞬止まる
- 職業的な葛藤の独白：「守りに来たのか、慣らしに来たのか、私は」
- デスクの引き出しから、過去に面談した少女たちのファイルが見える
- 「また一人、来てしまった」
- 窓の外、歌舞伎町の昼の光

**このシーンの要点**:
- 莉子の「家族への罪悪感」が初めて顔を出す場面
- カウンセラー視点を入れることで、**大人側の倫理的な揺れ**を提示し、作品全体のトーンを確立する
- 「守る」と「慣らす」の境界——本作の主題のひとつを最初に提示する

### シーン5: 健康診断（約2,500字・本話の核③）

**場所**: バックヤードの提携健診室
**登場**: 莉子、女性医師、看護師（1名）

**描写の骨子**

**（前半・一般項目）**
- 白衣の女性医師に「よろしくね」と迎えられる
- 身長・体重・視力・聴力・血圧——普通の健康診断の手順
- 体重計の数字を見た医師が一瞬ペンを止める描写（小ささへの瞬きの反応／明言はしない）
- 採血のための腕出し。針が刺さる瞬間、莉子が顔を背ける

**（中盤・婦人科検診）**
- 「次はね、婦人科の診察。初めてだと思うから、ゆっくり説明するね」
- 脱衣。莉子が服を脱いでいく過程を、**本人の感覚で**丁寧に描写
  - ワンピースを脱ぐときの、布が頭を通る暗闇
  - 下着ひとつになったとき、二の腕を自分で抱く仕草
  - 検査着に着替えるまでの数秒の羞恥
- 内診台——初めて見る器具の異様さ
- 医師の声かけ：「足、開いてね。深呼吸して」
- 検査そのものの描写は**即物的に、医療行為として**。痛みや冷たさ、器具の感触
- 莉子は天井の一点を見つめ続ける
- **莉子の内心**：「これが、これから、何度も見られる体なんだ」
- 性感染症検査のための拭取、子宮頸部の観察

**（後半・診察後）**
- 着替え終えた莉子に、医師が「処女膜、まだちゃんとあるよ。初めてコースの候補になると思う」と事務的に告げる
- **莉子、その言葉が"商品説明"として自分に向けられた瞬間、初めて本当の意味で「自分が売られる側にいる」と実感する**
- 医師は悪気なく、業務として告げただけ。そのドライさこそが残る
- 検査結果は後日。ピルの処方も入店手続き後

**このシーンの要点**:
- **本話で最も身体的な描写が濃いシーン**
- 性行為の描写は一切ないが、**「体を差し出すことの実感」**が最も強く出る
- 医師の"プロ意識"と"事務的な言葉選び"が、莉子の体を「医療対象」と「商品」の境界に置く

### シーン6: 控え室のちら見と先輩との邂逅（約1,000字）

**場所**: 3階バックヤード・キャスト控え室（扉越し → 一瞬の対面）
**登場**: 莉子、受付スタッフ、先輩キャスト（14歳・名前は次話以降で確定）

**描写の骨子**
- 健診後、受付スタッフが「ちょっとだけ控え室も見ておく？」と案内
- 扉を開けた瞬間の、ムッとした化粧品・香水・柔軟剤の匂い
- キャストが3人ほど、それぞれ鏡の前で化粧中／スマホを触っている／お菓子を食べている
- 全員、莉子より少しだけ年上の少女たち。普通の女の子に見える
- 莉子、**「あ、普通だ」**と思う。そのギャップに足がすくむ
- 一番奥の鏡の前に座っていた先輩キャストが、ふと振り返る
- 目が合う数秒
- 先輩が小さく笑って、**「頑張んなね」**とだけ言う
- 莉子、思わずお辞儀
- 扉が閉まる。匂いが廊下に薄く残る

**このシーンの要点**:
- 次話以降の**教育係との最初の邂逅**を、記憶に残る短いカットで
- 「普通の女の子たち」の光景が、莉子の中に"自分もこうなる"という予感として刻まれる

### シーン7: 帰路（約1,500字）

**場所**: 雑居ビル前 → 歌舞伎町 → 電車 → 自宅前
**登場**: 莉子（ひとり）、電話越しの母

**描写の骨子**
- 契約書類一式と、保護者同意書のフォーマットを封筒で渡される
- 「次は、お母さんと一緒に。それか同意書だけ持ってきて」
- ビルを出る。歌舞伎町の昼、薄っぺらい陽射し
- 来たときと同じ道を歩いているのに、景色が違って見える
- スマホを握りしめたまま、しばらく歩く
- 電車のホーム。座れない混み具合
- 窓ガラスに映る自分の顔——**さっき内診台の上で見ていた天井と、同じ色**
- 母親に電話する。「合格、したよ」と小さく
- 母の嬉しそうな声。「よかった、頑張ったね莉子」
- 電話を切る
- 窓の外、夕暮れの東京
- 最後の一文：「明日から、あたしは、どうなるんだろう」（※内心のモノローグで締め）

**このシーンの要点**:
- **「決意」ではなく「諦めに近い覚悟」** で終わらせる
- ヒロイックな決意ではなく、11歳の子供が抱えきれない重さを抱えたまま日常に戻っていく、その静けさ
- 母親の無邪気な喜びの声が、莉子の胸に重く残る

## 4. 感情曲線（全体構成）

```
緊張 ┤          ●店長面談
     │        ／          ●健診（内診）
     │      ／               ＼
不安 ┤    ／                    ●＼控え室
     │ ●  ●カウンセラー           ＼
     │道中 （一時的に和らぐ）        ＼ ●帰路（諦めに近い覚悟）
平静 ┼─────────────────────────────────→ 時間
```

- 道中: 漠然とした不安 ⇒ 店長面談: 動揺のピーク
- カウンセラー: 一時的に下がる（大人に受け止められる安堵）
- 健診: 再び上昇、身体的な実感が最大
- 控え室: 予感の形で着地
- 帰路: 静かな下り坂（だが底には覚悟がある）

## 5. 描写上の注意点

### トーン
- **過剰に悲劇化しない**：莉子は泣かない。店側も決して威圧的ではない。**淡々と進んでいく"仕事としての手続き"**の淡白さが逆に重い
- **店の"まともさ"を書く**：合法店として運営されている店の、**表面上は普通の職場**である質感を描く。その普通さこそが本作の怖さ

### 莉子の造形
- 11歳らしい幼さと、**家計を背負う重さによる早熟さ**の両立
- 大人っぽい言葉選びをしようとして時々素が出る
- 泣かない。でも何度も黙り込む
- 自分の体を「商品」として扱われる場面で、**拒絶ではなく受容に近い反応**を示す（そのことの異様さは地の文で示唆）

### 性的描写
- 本話に性行為シーンはなし
- **身体検査シーンが最も身体的**。ただし医療行為の範囲で、即物的に
- 脱衣・内診の描写は**羞恥ではなく淡々とした描写**で統一（次話以降の「接客としての脱衣」との対比になるため）

### 大人たちの描き分け
- **店長**: プロとしての責任感と、淡々とした事務性
- **カウンセラー**: 倫理的な揺れを抱えた"受け止める側"
- **健診医**: 完全な業務人格、悪気のないドライさ
- **受付**: 明るく優しい表面、奥の空気は見せない
- → 4人4色の"大人の女性/男性の仕事の顔"を見せることで、BabyDollの多層性を提示

## 6. 次話（第1話）への繋ぎ

### 解消されていない要素
- 保護者同意書をどうするか（母に本当のことを言うのか／偽造／説得）
- 莉子自身の最終決断は、まだ下されていない（店長が「自分の意思で」と言った言葉が残っている）
- 先輩キャストとの関係はまだ「すれ違い」レベル

### 第1話で描く想定
- 同意書を巡る家庭での出来事
- 研修期間の開始
- 先輩キャストとの正式な師弟関係の始まり
- 初めて"キャスト衣装"を着る日
