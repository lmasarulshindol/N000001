# EP1「ベビードールの初恋」詳細プロット

> 結城健吾（33歳・広告営業）× 佐藤莉子（10歳・BabyDoll最年少キャスト）/ 初来店 × 初めてコース（処女喪失）/ 歌舞伎町の夜
>
> EP0で覚悟を固めた莉子の「初めてコース」当日。初来店の健吾が相手となり、小悪魔の仮面を被った10歳の少女と、罪悪感を抱えた33歳のサラリーマンが出会う。入浴→マットプレイ→処女喪失（生挿入中出し）という一夜を通じて、莉子の仮面が少しずつ崩れ、健吾の心に消えない痕跡を残す。

## シーン構成（全8シーン）

### シーン1：歌舞伎町の夜（日常パート）

**場所**: 歌舞伎町のネオン街 → 雑居ビル前
**時間帯**: 金曜日の夜21時
**登場人物**: 健吾、同僚・村瀬
**文量目安**: 約2,500字

#### 出来事
- 接待の二次会が流れ、同僚の村瀬と歌舞伎町を歩いている
- 村瀬が「お前、最近顔死んでるぞ。いいとこ知ってるから行ってこい」とBabyDollの会員カードを渡す
- 村瀬はゴールド会員。「若い子がいる店だけど、風俗っていうか……なんか、癒されるんだよ」
- 健吾は戸惑う。風俗は学生時代に一度行っただけ。しかも「若い子」という言葉に引っかかる
- 村瀬「いいから。騙されたと思って。今日は特別なコースがあるらしいぜ。初めてコースとかいう」
- 歌舞伎町の裏通り。ネオンの光と客引きの声。雑居ビルの前で村瀬と別れる
- 健吾の独白：何をやってるんだ俺は。金曜の夜に、こんな場所で。でも、家に帰っても暗いワンルームがあるだけだ
- ビルのエレベーターに乗る。指が少し震えている

#### 描写のポイント
- 歌舞伎町の猥雑な空気感。健吾の日常の疲弊と非日常への一歩
- 「初めてコース」という単語を村瀬の口から出すことで、EP0との接続を暗示
- 読者はEP0で莉子側の事情を知っている。健吾は何も知らない——この情報の非対称が緊張を生む

#### 感情の流れ
- 疲労 → 戸惑い → 好奇心と背徳感 → 覚悟

### シーン3：対面・入室（小悪魔の登場）

**場所**: 待合ロビー → 3階個室
**時間帯**: 21:30頃
**登場人物**: 健吾、莉子
**文量目安**: 約3,000字

#### 描写のポイント

#### 感情の流れ
- 健吾: 衝撃（本当に子供だ） → 莉子の小悪魔ぶりに翻弄 → 罪悪感 → 「優しくしてね」で胸が痛む

### シーン4：入浴（関係の変化 → 行為への導入）

**場所**: 個室のバスルーム
**時間帯**: 21:40頃
**登場人物**: 健吾、莉子
**文量目安**: 約3,500字

#### 出来事
- 莉子が「じゃ、おふろ入ろ？」と自然に誘導。バスルームへ

- さっきまで透けて見えていた体が、今は何も隔てるものなく目の前にある。衣装越しの「透け」と「全裸」の落差に健吾は息を呑む
- 健吾は目を逸らそうとするが、逸らせない。莉子が微笑む。「ほら、おじさんも。一人で脱げないの？」
- シャワーで体を流す。莉子が健吾の体を洗い始める。本当に小さな手。泡を立てて胸、背中と洗っていく
- 「おじさん、背中ガチガチだよ。お仕事大変なんだね」。肩甲骨のあたりを丁寧にほぐす。手つきは研修で覚えたものだが、手が小さい分、細かいところまで届く
- 健吾が反応し始める。莉子はそれに気づいて「あ、もう元気になってる♪ おじさん、正直だね〜」。くすくす笑う
- 莉子の体を洗う場面。健吾がボディソープを手に取り、莉子の背中を洗う。小さな背中。肩甲骨が天使の羽のように浮き出ている。肌がすべすべで指が滑る。子供の体だ

#### 描写のポイント
- 莉子の体の描写。10歳の体——胸は平たく、腰のくびれもなく、子供そのもの
- 小悪魔の台詞と子供の体のギャップが最大のインパクト
- 入浴シーンで物理的距離が縮まる。湯気と水音の環境描写
- 健吾の「この子の処女を奪うんだ」という現実の重さ

#### 感情の流れ
- 健吾: 動揺 → 莉子の体への衝撃（子供だ） → 罪悪感 → 触れる喜びと罪の意識の葛藤
- 莉子: 小悪魔の余裕 → 背中を洗われた時にふと素が出る

### シーン5：マットプレイ（行為シーン①：前半）

**場所**: 個室のマットスペース
**時間帯**: 21:55頃
**登場人物**: 健吾、莉子

> 行為シーンの三層構造・前半

**文量目安**: 約5,000字

#### ① 導入（マットへの移行）
- バスルームから出て、マットスペースへ。エアマットにローションが用意されている
- 莉子「マットプレイ、練習したんだよ。おじさんが初めてのお客さん♪」。声は明るいが、指先が少しだけ震えている
- ローションを温める莉子。研修で覚えた手順を一つずつ丁寧にこなす
- 健吾をマットに仰向けに寝かせる。莉子が上から覆いかぶさる

#### ②-A マットプレイ本体
- 莉子の全身を使ったマットプレイ。小さな体がローションでぬるぬると滑る
- 平たい胸で健吾の胸板を滑る。体が小さいからこそ全身が密着する。ローションの温かさと莉子の体温が混じる
- 下半身へのサービス。莉子の小さな手が健吾を握る。指が足りない。「おじさん、おっきい……あたしの手じゃ足りないよ」。困ったような顔が本気に見える
- 莉子がバランスを崩しそうになる。「わっ」と小さく声を上げる。10歳の体にはマットが大きすぎる。健吾が支える。莉子の軽さに驚く
- 莉子「ごめん、まだ下手で……練習通りにいかないなぁ」。悔しそうな顔。プロになりきれない瞬間
- 口でのサービスに移行。小さな口で懸命に咥える。頬を膨らませて、上目遣いで健吾を見る
- 健吾の手が莉子の頭に触れる。柔らかい髪。小さな頭
- 「気持ちいい？」と莉子が顔を上げて聞く。唇が濡れている。アクアブルーの目が真っ直ぐ健吾を見ている
- 健吾が果てる。莉子が口で受け止める。「……ん」。飲み込んで、少し苦そうな顔。すぐに笑顔に戻す。「いっぱい出たね♪ おじさん溜まってたでしょ〜」
- だが目が少し潤んでいる。健吾は気づく。この子は、頑張っている。10歳の子が、必死に「プロ」を演じている

#### 描写のポイント
- マットプレイの技術的未熟さがリアリティ。10歳の体ではマットが大きすぎて上手くいかない
- 「練習通りにいかない」という悔しさ＝莉子のプロ意識と、その裏にある子供の不安
- 口でのサービスで健吾を果てさせることで、莉子の「仕事ができる」面を見せつつ、苦い表情で素を覗かせる

#### 感情の流れ
- 健吾: 快感 → この子にこんなことをさせている罪悪感 → 莉子の頑張りへの胸の痛み
- 莉子: プロモード → 失敗して素が出る → 口でのサービスをやり遂げた達成感 → 少しの苦さ

### シーン6：初めて（行為シーン②：処女喪失〜クライマックス）

**場所**: 個室のベッド
**時間帯**: 22:10頃
**登場人物**: 健吾、莉子

**文量目安**: 約7,000字

- 莉子「ベッド、行こ。……ここからが、"初めて"だよ」。声が少し低くなる。小悪魔の笑みだが、唇が微かに震えている

### シーン8：帰り道（余韻）

**場所**: 歌舞伎町 → 健吾のマンション

**文量目安**: 約3,000字

- スマホを手に取る。BabyDollの予約サイトを開いている自分に気づく。莉子のプロフィールページ。あの小悪魔の笑みの写真。だが健吾はもう、この笑みの裏を知っている
- 「……また、行くのか。俺は」
- 答えは出ない。だが莉子の声が蘇る。「待ってるから」

- 余韻と自問。歌舞伎町の夜景とマンションの対比
- 背中の爪の跡——莉子がしがみついた証拠。10歳の爪は小さい

#### 感情の流れ
- 健吾: 放心 → 反芻 → 罪悪感 → 莉子の声への渇望 → 未解決
- 莉子: 痛みの残響 → 健吾の名前を口にする → 感情移入への自戒 → 再会への微かな期待
