# 第3章：白い寝室の前戯（莉子視点）

　リビングの光が、一度だけ弱まった。

　スタッフがケーブルを巻き、三脚の脚がカーペットの上を滑る。滑る音が、ホテルの静けさに吸われていくみたいだった。あたしを連れていくのは、プロデューサーじゃなく、監督の方を向いた女性スタッフの手のひらだった。

「こちらです。寝室側です」

　扉を開けると、匂いが変わった。リビングより、洗剤と、布と、何か金属みたいな冷たさ。キングサイズのベッドが部屋の真ん中にあって、シーツが白すぎて、最初に目が痛くなる。

　窓は、リビングと同じく夜景を貼ってる。でも白い布のせいで、夜景が遠のいた。遠のいた夜景の前に、あたしがいる。そういう並べ方を、最初から計算してる気がした。

「セット変更、完了です」

　誰かが言って、照明が立ち上がる。パラソルじゃなく、角の硬い光。白いシーツが、さらに白くなる。あたしの影が、布の上に薄く落ちて、すぐに光に削られる。

「莉子ちゃん。ベッド中央。仰向け。……枕は、こちら」

　監督の声は、相変わらず優しい。優しい声で、あたしの背中を押す。

　あたしはベッドに上がる。シーツが、指先に冷たい。仰向けになると、天井が高くて、そこにも小さなスポットの黒い影が見えた。カメラの影。

「足元、こちら。カメラが入ります」

　レンズが、足の先から入ってくる。赤い点が、視界の端でまた止まってる。録ってる。さっきから、ずっと。

「出演者の方、ポジションお願いします」

　男の人たちが、ベッドの周りに並ぶ。さっきまでフレーム外だった二人だけじゃない。数が増えた。三人、四人——数え終わる前に、あたしの両側に手がついた。

　背後から、腕が回る。大きい。熱い。あたしの肩甲骨のあたりに、大人の胸の厚みが押しつけられる。

「っ……」

　声が漏れそうになって、あたしは飲み込む。飲み込んだつもりが、喉で小さく跳ねる。

「いいです。その反応、もう少しだけ大きく」

　耳元で、監督が言う。近い。カメラのない方の耳に、息がかかる。

「……演技、じゃないの？」

　あたしは笑おうとした。笑おうとした顔に、別の手のひらが触れた。頬から、唇へ。親指が、口元をなぞる。

「開けて。……上唇、もう少し」

　開ける。開けたら、指先が、歯の縁を撫でた。撫でられた場所が、急に自分の体じゃなくなる。

「ん……」

　足の方でも、手が動いてる。太もも。膝の内側。そこから、ゆっくり、開かされる。シフォンが、すくりと上がって、冷たい空気が足の間に入る。

「待って、まだ——」

　まだ、って何がまだなのか、自分でも分からない。言葉が途中で細る。

「大丈夫。まだ、前です」

　プロデューサーの声が、ベッドの外からした。大丈夫、の言い方が、契約書みたいに平らだった。

　左からも右からも、手が伸びる。胸の上のシフォンを、指が持ち上げて、また戻す。戻すのに時間がかかる。かかるのは、戻すためじゃない。

「莉子ちゃん。カメラ目線。……おじさんたちに、ひとこと」

　あたしは、レンズを探す。探した先に、黒い丸。丸の奥に、自分の顔。アクアブルーの目が、潤んでる。潤んでるのに、笑ってる。笑ってるつもりなのに、口角が震える。

「おじさんたち……あたし、ちゃんと、可愛がって……」

　言い切れない。

　言い切れないところで、背後の腕が強くなる。強くなるのと同時に、膝の内側の手が、もう一段、上がる。上がった指先が、黒いレースの縁に触れた瞬間、あたしの腹の奥が、きゅっと縮む。

「ぁ……」

　それは、撮影用の声じゃなかった。撮影用の声を、監督が耳元で囁む。

「いいです。その声。もう一回。……“可愛がって”のあと、息を」

　息を。息を吸うと、胸が触れる。誰の手か分からない。分からないほど、手が増える。

　ハンディが、顔の上を回る。回るたびに、赤い点が視界に入る。入るたびに、あたしは自分が画面の中にいることを思い知る。思い知るたびに、小悪魔の台詞が、舌の上で溶ける。

「ねぇ……やっぱ、あたし……」

　やっぱ、の先が出ない。出ないまま、唇に指が戻って、言葉を止める。

「止まらないで。……莉子ちゃんは、強い子」

　スタッフの女の人が、ベッドの端で言った。共感みたいな顔をして。共感のふりをしてるの、分かる。分かるのに、助け船に乗りたくなる。

　あたしは、レンズに向かって笑う。笑うために、目を細める。細めたら、涙の膜が均される。

「……プロ、だもん」

　言った瞬間、太ももの内側を撫でる手が、わざと止まって、また動く。止まり方が、あたしの言葉を試してるみたいだった。

「いいテイクです」

　監督が言う。

「このまま続けます。……衣装、肩紐、落として。自然に」

　誰かの指が、肩を這う。這って、細い紐を外す。外れたシフォンが、胸の上で崩れて、冷たい空気が肌に直に触れる。

　あたしは、白いシーツに沈む。沈むほど、光が強い。強い光の中で、あたしの体は小さく見える。小さく見えるほど、手が大きい。

　夜景は、ガラスの向こうで黒く光ってる。こっちだけが、解剖台みたいに白い。

　赤いRECが、まだ消えない。

　笑顔を、もう一度だけ作ろうとした。作ろうとした唇が、震えて、途中で開く。開いたまま、息だけが出る。

　——仮面、ひび割れた。

　ひび割れたのに、カメラは止まらない。

　監督の声が、優しく言った。

「素晴らしい。……では、本番パートに入ります。莉子ちゃん、準備はいい？」

　あたしは頷く。頷くのが、怖いくらい簡単だった。

「……うん」

　返事が、十歳みたいに細い。

　ベッドの周りの手が、まだ離れない。

　離れないまま、部屋の空気が、もう一段、重くなる。
