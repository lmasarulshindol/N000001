# BabyDoll 初接客オムニバス 本文集【女の子視点版】

> 客視点版（[../README.md](../README.md)）と**同じ出来事・同じ時間軸・同じ台詞**を、  
> BabyDoll所属キャスト5名それぞれの一人称視点から描き直した別バージョン。  
> 原典プロット: [../../プロット_初接客オムニバス.md](../../プロット_初接客オムニバス.md)（第11章新フロー・絶頂設計準拠）

---

## 位置づけ

客視点版が「触っている・入れている」身体感覚中心の描写であるのに対し、  
女の子視点版は「触られている・入れられている」感覚＋仮面の裏側の内心独白を主軸とする。

- 女の子のカギ括弧内セリフ・客のセリフは客視点版と**一字一句一致**
- 地の文・内心独白のみ女の子視点に再構築
- 絶頂回数・流れ・時間軸・行為順序は客視点版と完全同一
- 文量は客視点版の +10〜25%（内面描写分）

## 差別化の設計（客視点版との違い）

| 観点 | 客視点版 | 女の子視点版 |
|------|---------|-------------|
| 主語 | 客（おれ／僕／私） | 女の子本人 |
| 身体感覚 | 触る・入れる側の快感 | 触られる・入れられる側の感覚＋痛み＋内心の動揺 |
| 仮面描写 | 外側から観察 | 仮面を被っている自分を内側から俯瞰する二重視点 |
| 喘ぎ・吐息 | 相手の声として聞く | 自分の声として「なぜ出てしまうのか」の内心込み |
| 背景事情 | 断片的に察する | 行為中もフラッシュバックとして生々しく浮上 |
| 文体リズム | 比較的一定 | 内面崩壊に従って長文→短文→断片化（特に楓・澪） |

## 章立て（各話共通）

客視点版と同じ三幕構成:

1. **前座**（20〜25%）: 出会い／ベビードール／控え室での準備／内面独白
2. **行為本体**（50〜60%）: 口での奉仕→口内射精→指愛撫→絶頂→挿入→絶頂
3. **余韻**（20〜25%）: シャワー／片付け／別れ／内面の整理
