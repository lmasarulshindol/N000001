# DollHouse（ドールハウス）

> BabyDoll系列店・中学生限定ソープランド 設定資料

## 1. 店舗概要

BabyDollの姉妹店として開業。BabyDollを「卒業」した小学生キャストの受け皿であると同時に、中学生年齢での新規採用も行う。「幼さ」が武器だったBabyDollに対し、DollHouseは「少女から女性への変化（思春期）」を最大の魅力として打ち出す。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 店名 | DollHouse（ドールハウス） |
| 業態 | ソープランド（個室浴場型性風俗店） |
| 所在地 | 東京都・新宿区歌舞伎町。BabyDollと同じ通りの別ビル3〜5階 |
| 営業時間 | 12:00〜翌1:00（完全予約制） |
| 在籍キャスト | 約20〜25名（13歳〜15歳の女の子。中学生限定） |
| コンセプト | 「咲きかけの花を、あなたの手で」――思春期の少女たちとの親密な時間を提供 |
| 料金体系 | 60分 25,000円〜 / 90分 38,000円〜 / 120分 50,000円〜（指名料別途） |

### 店名の由来
- 「DollHouse」＝人形の家。BabyDoll（赤ちゃん人形）が成長し、帰る場所としての「家」
- 「イプセンの戯曲『人形の家』」のダブルミーニング。少女たちが「人形」のまま留まるのか、やがて「家」を出るのかという問いかけ
- 看板はラベンダーとクリーム色を基調。BabyDollより少し大人っぽい、アンティーク調のデザイン

## 2. BabyDollとの関係

### 系列構造

```
【BabyDoll】小学生限定（10〜12歳）
      │
      │ 中学進学時に転籍
      ↓
【DollHouse】中学生限定（13〜15歳）
      │
      │ 高校進学時に転籍
      ↓
【DollGarden】高校生限定（16〜18歳）
      │
      │ 高校卒業で引退
      ↓
   引退・進学/就職支援
```

### 転籍制度
- BabyDoll卒業キャストは、本人の希望があればDollHouseへ優先転籍可能
- 転籍時の再研修は1週間（BabyDollでの経験を踏まえた短縮版）
- BabyDoll時代の常連客への「転籍通知」サービスあり（会員限定告知）
- 転籍せず引退を選ぶことも自由。引退金は通常通り支給

### 運営体制
- オーナーはBabyDollと同一人物
- 店長は別人（BabyDoll店長の信頼するベテラン）
- 会員データベースはBabyDollと共有（BabyDoll会員はDollHouseも利用可能）
- AV制作は「BabyDoll Productions」が一括で担当

## 5. DollHouseならではの特色

### 「成長」をテーマにしたサービス

BabyDollが「幼さ」を売りにするのに対し、DollHouseは**「変化する体と心」**をテーマにする。

- **成長記録**: 定期的に身体測定を行い、プロフィールに身長・体重・スリーサイズの変化を記録。常連客は「推しの成長」を追える
- **思春期の変化**: 胸の膨らみ、体毛の変化、体つきの変化——BabyDoll時代にはなかった「女性への変化」が最大の魅力
- **心の変化**: 恋愛や性への意識が芽生える時期。客との関係が「お仕事」から「疑似恋愛」に発展しやすい

### 「おかえりコース」の詳細

BabyDoll時代に指名していた客が、転籍後のキャストと再会するための特別コース。

- BabyDoll会員であったことの確認＋該当キャストへの過去指名履歴が必要
- 「大きくなったね」「おじさんのこと覚えてる？」といった再会の演出
- BabyDoll時代の写真（プロフィール写真）をルーム内に飾り、成長のビフォーアフターを視覚化
- 月2回限定。予約枠が少なく、プラチナ会員優先

## 9. キャラクター設定

### 佐藤 莉子（さとう りこ）― DollHouse時代
**← BabyDollからの転籍組第1号。系列の看板キャスト**

| 項目 | 詳細 |

| 身長 | 152cm（13歳入店時）→ 155cm（15歳卒業時） |

#### 身体的変化の特徴

BabyDoll時代の「まっ平ら」から、思春期の変化が進行中。

- **胸**: 「出てきた」段階。本人は「邪魔」と言いつつ、客の反応を見て武器になると気づく
- **体つき**: 手足が伸びて子供体型から脱却。くびれが生まれ、ヒップに丸みが出始める
- **肌質**: 陶器のような白さは健在だが、汗ばむと艶が出る。幼い「すべすべ」から少女の「しっとり」へ
- **全体印象**: 「幼い」から「少女」へのグラデーション。BabyDoll時代を知る客にとっては「あの子がこんなに」という感動の対象

#### 性格・接客スタイルの変化

- **基本路線**: 小悪魔キャラは健在。ただし「無邪気なからかい」から「計算された挑発」に進化
- **葛藤**: BabyDoll時代は「幼さ」自体が武器だったが、それが使えなくなる。「もう小さい子じゃない」という現実に最初は戸惑う
- **新しい武器の発見**: 「成長していく体を常連客に見せつける」という時間軸の魅力に気づく。「ねぇ、おじさん。あたしの胸、前より大きくなったの分かる？」
- **先輩としての顔**: DollHouseでは最古参格。後輩キャスト（BabyDollからの転籍組含む）への指導役を買って出る。教え方はBabyDoll時代の莉子らしく、「まず客をからかえ。焦らせろ。主導権を渡すな」
- **定番フレーズ**: 「おじさん、あたしのこと小学生の頃から抱いてたんでしょ？ ……ねぇ、今の体と昔の体、どっちが好き？」
- **内面の変化**: 仕事への割り切りがより明確に。BabyDoll時代の「覚悟」が「プロ意識」に昇華。一方で、同年代の普通の中学生への微かな羨望が芽生える

#### DollHouseでの立場

- BabyDollからの転籍第1号。「おかえり。」シリーズ第1作の主演
- BabyDoll時代の常連客がそのまま追いかけてくるため、転籍直後から指名が安定
- 店の「顔」として新規客への対応も任されるエース格
- 後輩たちからは「莉子先輩」と呼ばれる（BabyDollでは「生意気な妹」だったのが逆転）

#### 挿入時の反応パターン（15歳時点）

**① 余裕があるとき（からかいモード）**
- 「んー……おじさんの、前よりちゃんと感じるかも。あたしの中、成長したでしょ？」
- 腰の動かし方を自分でコントロールし、客のペースを奪う。BabyDoll時代より明確に「技術」がある
- 「ねぇ、小学生の頃のあたしと今のあたし……どっちの中が気持ちいい？ ……正直に言ってよ」と、客の心理を突く質問を投げかけながら余裕の笑み
- BabyDoll時代の「無邪気なからかい」が「計算された誘導」に進化している

**② 余裕がなくなってきたとき**
- 「っ……ん、ちょっと……今の、深い……」と、声のトーンが落ちて吐息混じりになる
- BabyDoll時代と違い、声を堪えるのが上手くなっている。唇を舐めて呼吸を整えようとする
- ただし体の反応は隠せない。腰が勝手に動き出し、自分でペースを作れなくなる
- 「……ずるい、そこ知ってるくせに」と、常連客が弱点を突いてきた時だけ、本気で悔しそうな顔をする

**③ 絶頂**
- 「あ……っ、だめ、いく——っ……！」と、BabyDoll時代より低く、甘い声で鳴く。声変わりの途中のような、少女と女の中間の声
- 体を仰け反らせ、シーツを掴む指が白くなる。BabyDoll時代の「しがみつく」から「自分の中で堪える」に変化
- 絶頂の瞬間、目を閉じてぎゅっと眉を寄せる。数秒の沈黙の後、深く息を吐く
- 復帰は早い。「……っふ。おじさん上手くなった？ ……まあ、あたしが育てたようなもんだけど」と、息が荒いまま強がりを言う
- BabyDoll時代ほど「悔しさ」は出さなくなった。代わりに「余裕を崩された後の色気」を自覚し、武器にし始めている
