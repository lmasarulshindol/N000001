# EP1「潮騒のはじまり」詳細プロット

> 蒼汰（14歳・中2）× 紬（13歳・中1）/ 春〜初夏 / 泡児島（鹿児島県の離島）

---

## シーン構成（全8シーン）

### シーン1：船上の少年（日常パート）
- **場所**: 鹿児島港 → フェリー船上 → 泡児島が見えてくる海上
- **時間帯**: 4月上旬、朝〜昼前
- **登場人物**: 蒼汰
- **出来事**:
  - 横浜から夜行バスと新幹線を乗り継いで鹿児島に着いた蒼汰。フェリーに乗り込む
  - 母との別れの回想。「蒼汰、ちゃんとご飯食べるのよ」と泣きそうな顔で笑う母。介護職で疲れた手で蒼汰の頬に触れた。蒼汰は「大丈夫だよ」とだけ返した
  - フェリーのデッキで海を眺める蒼汰。「島興しプログラム」の案内パンフレットを読み返す。生活支援金の額を確認して、小さく息を吐く
  - 同じフェリーに乗っている参加者らしき同年代の少年少女たちの姿。蒼汰は距離を取って一人でいる
  - 3時間の航海の末、水平線に泡児島が現れる。亜熱帯の緑に覆われた小さな島。白い砂浜と青い海。都会しか知らない蒼汰にとっては別世界
  - 「……ここで1年か」。蒼汰の独白。期待はない。ただ、母の負担を減らせるならそれでいい
- **描写のポイント**: 蒼汰の寡黙で達観した性格の提示。母子家庭の経済的事情。フェリーから見える島の美しさと、蒼汰の無感動な反応の対比。読者に「この少年がどう変わるのか」という期待を持たせる
- **感情の流れ**: 淡々 → 母への微かな後ろめたさ → 島を見ても心が動かない空虚さ
- **文量目安**: 約2,500字

### シーン2：入島式と出会い（日常パート）
- **場所**: 泡児島・港町エリア → 施設エリア（宿泊棟の広場）
- **時間帯**: 4月上旬、昼〜午後
- **登場人物**: 蒼汰、紬、プログラム責任者、カウンセラー、他の参加者たち、島民
- **出来事**:
  - フェリーが港に着く。島民が総出で出迎える。手作りの横断幕「ようこそ泡児島へ！」、三線の演奏、花飾り
  - 蒼汰はその歓迎ぶりに戸惑う。過疎の島にとって、自分たちがどれだけ「待たれている」存在かを実感する居心地の悪さ
  - 施設エリアに案内される。宿泊棟は男女別の相部屋形式。カウンセラーから生活ルールの説明
  - 入島式。プログラム責任者の挨拶。「皆さんはこの島の未来です」という言葉に、蒼汰は白けた気持ちになる
  - 参加者の自己紹介タイム。蒼汰は「汐見蒼汰です。横浜から来ました」とだけ言って座る
  - 紬の番。立ち上がった紬は緊張で声が震えている。「は、波照間紬です。東京から……」と言いかけて言葉に詰まる。頬が真っ赤になり、俯いてしまう
  - 蒼汰はその姿をぼんやり見ている。蜂蜜色の髪、右耳のヘアピン、白い肌。都会の匂いがする少女
  - 紬が小さく「……よろしくお願いします」と絞り出して座る。蒼汰は目を逸らす。「俺には関係ない」
  - 入島式後、宿泊棟の部屋に荷物を運ぶ蒼汰。窓から海が見える。波の音が聞こえる。横浜の団地では聞いたことのない音
- **描写のポイント**: 島の歓迎ムードと蒼汰の距離感。紬との「最初の印象」は素っ気ないもの。参加者たちの様子（不安な顔、はしゃぐ顔、緊張した顔）で群像感を出す。島の南国的な空気（花の香り、潮風、眩しい陽光）
- **感情の流れ**: 戸惑い → 白けた気持ち → 紬に対する一瞬の注目 → すぐに無関心に戻る
- **文量目安**: 約3,500字

### シーン3：島の生活と距離（日常パート → 関係の変化）
- **場所**: 学園エリア、浜辺エリア、宿泊棟、港町の商店
- **時間帯**: 4月中旬〜5月（約1ヶ月間のダイジェスト）
- **登場人物**: 蒼汰、紬、島の漁師の老人（源じい）、他の参加者たち
- **出来事**:
  - 島での生活が始まる。学園での授業、共同生活のルーティン
  - 蒼汰は漁師の源じいと仲良くなり、放課後は防波堤で釣りをしている。海を眺めながら本を読む時間が、蒼汰にとっての安らぎ
  - 紬は島の生活に苦戦。虫が出て悲鳴を上げ、暑さにバテ、夜はホームシックで枕を濡らす。他の参加者には平気なふりをしているが、目が赤い朝がある
  - ある日、蒼汰が防波堤で釣りをしていると、紬が一人でふらふら歩いてくる。目が真っ赤。泣いた後だと分かる
  - 紬は蒼汰に気づいて慌てて目を擦る。「……なんでもない」。蒼汰は何も言わず、釣り竿の隣のスペースを顎で示す
  - 紬が恐る恐る座る。しばらく無言。波の音だけが二人の間にある
  - 「……お母さんから電話、来なかった」と紬がぽつりと呟く。蒼汰は返事をしない。ただ、隣にいる
  - 紬が「ここ、コンビニもないし、Wi-Fiも遅いし、虫はいるし」と愚痴を言い始める。蒼汰が「魚はうまい」とぼそっと返す。紬が一瞬きょとんとして、小さく笑う
  - この日から、紬は時々防波堤に来るようになる。蒼汰は特に話しかけないが、追い払いもしない
  - 5月。紬の肌が少し焼けてきた。学園の友達も増えてきた。でも防波堤には相変わらず来る
  - 蒼汰が源じいにもらった魚を宿泊棟のキッチンで捌こうとして失敗しているところに、紬が「私、料理得意だよ」と割って入る。二人で煮魚を作る。他の参加者たちにも振る舞う
  - 蒼汰が紬の煮魚を食べて「……うまい」と言う。紬が嬉しそうに「でしょ？」と胸を張る。蒼汰は、紬が笑うと目が三日月みたいになることに気づく
- **描写のポイント**: 時間経過による二人の距離の変化。「無言の隣にいること」が関係の始まり。島の生活の具体的な描写（授業、食事、釣り、キッチン）。紬が島に馴染んでいく過程と、蒼汰が紬を意識し始める過程の並行
- **感情の流れ**: 無関心 → 紬の孤独に共感 → 隣にいることの心地よさ → 紬の笑顔への微かな高揚
- **文量目安**: 約4,000字

### シーン4：海開き祭の夜（関係の変化）
- **場所**: 浜辺エリア（海開き祭会場）→ 防波堤
- **時間帯**: 7月上旬、夕方〜夜
- **登場人物**: 蒼汰、紬、参加者たち、島民
- **出来事**:
  - 海開き祭。浜辺に提灯が並び、盆踊りの準備。島民と参加者が入り混じって賑やか
  - 紬が浴衣を着ている。島の女性に着付けてもらったらしい。紺地に白い花模様。蜂蜜色の髪を簪でまとめている。いつもと違う紬に蒼汰の心臓が跳ねる
  - 「ど、どう？ 似合ってる？」と紬が蒼汰に聞く。蒼汰は目を逸らして「……別に」と答える。紬が「素直じゃないんだから」とむくれる
  - 祭りを楽しむ二人。焼きそば、かき氷、金魚すくい（紬が全く取れない）。蒼汰が一匹だけ掬ってやると、紬が「すごい！」と歓声を上げる
  - 花火が始まる。浜辺に座って見上げる二人。花火の光が紬の琥珀色の目に映る。蒼汰がその横顔を見つめて、目が離せなくなる
  - 紬が花火に見入りながら「ねえ、蒼汰くん。私、この島に来てよかったかも」と呟く。蒼汰は返事の代わりに花火を見上げる
  - 花火が終わった後、他の参加者たちはまだ祭りの余韻で騒いでいるが、蒼汰と紬は防波堤に二人で座っている
  - 夜の海。月明かりが水面に道を作っている。潮風が紬の髪を揺らす
  - 紬が「お母さんね、最近メール返してくれるようになった。私が島の写真送ったら、『綺麗ね』って」。声が少し震えている
  - 蒼汰が「……よかったじゃん」と返す。紬が「蒼汰くんのお母さんは？」。蒼汰が少し黙ってから「元気にやってる。仕送り分、楽になったって言ってた」
  - 紬が蒼汰の横顔を見て「蒼汰くん、お母さんのこと大好きなんだね」と言う。蒼汰が「……うるさい」と返すが、耳が赤い
  - 紬が笑う。「蒼汰くんって、怒ったふりするとき耳が赤くなるよね」。蒼汰が「帰るぞ」と立ち上がる。紬が「あ、待ってよ！」と慌てて追いかける
  - 宿泊棟への帰り道、紬が蒼汰の袖をそっと掴む。「……暗いから」と言い訳。蒼汰は振り払わない
- **描写のポイント**: 浴衣姿の紬を見た蒼汰の動揺。祭りの華やかさと二人きりの静けさのコントラスト。夜の海と月明かりの美しさ。お互いの家庭の事情を打ち明け合う親密さの深まり。「袖を掴む」という身体的な距離の縮まり
- **感情の流れ**:
  - 蒼汰: 動揺 → 祭りの楽しさ（素直になれない） → 紬の横顔への見惚れ → 紬の言葉に照れ → 振り払えない袖の温もり
  - 紬: 嬉しさ → 島への愛着 → 母への複雑な想い → 蒼汰への親しみと安心感
- **文量目安**: 約3,500字

### シーン5：星祭りの準備（関係の変化 → 行為への接近）
- **場所**: 山間エリアの展望台、浜辺
- **時間帯**: 7月下旬〜8月（数日間のダイジェスト）
- **登場人物**: 蒼汰、紬、カウンセラー
- **出来事**:
  - カウンセラーから「星祭り」の説明。12月の本番に向けて、参加者たちでペアを組んで準備をする。星祭りは「ペアで星を見る」文化があり、島ではカップルが結ばれる行事として知られている
  - カウンセラーがさりげなく蒼汰と紬をペアに組む。二人は気まずそうにしながらも受け入れる
  - 星祭りの飾り（星型の灯籠）を一緒に作る作業。宿泊棟の工作室で、蒼汰が竹を削り、紬が和紙を貼る
  - 作業中の他愛ない会話。紬が「蒼汰くん、器用だね」と褒めると、蒼汰が「団地住まいだと、自分で直さなきゃいけないものが多い」と返す。紬が少し切なそうな顔をする
  - 展望台での星空観測の下見。夜の展望台に二人で上がる。満天の星空。天の川がはっきり見える。紬が息を呑む。「こんなの、見たことない……」
  - 蒼汰も星空に圧倒されている。横浜では見えなかった星の数。「……すげえな」と素直に漏らす。紬が「蒼汰くんが素直に感動するの、初めて見た」と嬉しそう
  - 夜風が冷たい。紬が少し震える。蒼汰が自分のパーカーを紬の肩にかける。「……風邪ひくだろ」。紬が顔を真っ赤にして「あ、ありがとう……」
  - 紬が星を見上げながら「ねえ、蒼汰くん。もし島にずっといられるとしたら、いたい？」と聞く。蒼汰が考え込む。「……分からない。でも、悪くないとは思ってる」
  - 紬がそっと蒼汰の手に触れる。蒼汰の体が一瞬強張る。紬が慌てて手を引っ込めようとするが、蒼汰がその手を掴む。「……冷たい手してんな」。不器用な言い訳。紬の目に涙が光る。「うん。だから……あっためて」
  - 二人が手を繋いだまま、星空を見上げている。潮風が二人の髪を揺らす。蒼汰の心臓が痛いくらいに鳴っている。紬の手は小さくて、震えていて、温かい
  - 別の日。浜辺で二人きりの時、紬が蒼汰に「プログラムのこと……どう思ってる？」と聞く。蒼汰は黙る。紬が「私は……怖いけど、蒼汰くんとなら……」と言いかけて口を噤む。蒼汰の心臓が跳ねる
- **描写のポイント**: 星空の圧倒的な美しさの描写。パーカーを掛ける・手を繋ぐという段階的な身体接触。「プログラム」の本質（子作り）が二人の間に意識されるようになる転換点。紬の「蒼汰くんとなら」が、蒼汰の感情を決定的に揺さぶる
- **感情の流れ**:
  - 蒼汰: 気まずさ → 紬との作業の心地よさ → 星空への感動 → 紬への意識の高まり → 「プログラム」を意識した動揺
  - 紬: 緊張 → 蒼汰との時間の嬉しさ → 手を繋いだ高揚 → 覚悟と恐れの混在
- **文量目安**: 約3,500字

### シーン6：星空の下・導入（行為シーン①：導入）
- **場所**: 浜辺エリア → 浜辺の岩陰
- **時間帯**: 8月中旬、夜
- **登場人物**: 蒼汰、紬
- **出来事**:
  - 夏の夜。参加者たちが浜辺でバーベキューをした後、皆が宿泊棟に帰っていく。蒼汰だけが浜辺に残って海を見ている
  - 紬が戻ってくる。「……やっぱりここにいた」。蒼汰が「なんで戻ってきた」。紬が「蒼汰くんが一人でいるから」
  - 砂浜に並んで座る。波が足元まで来ては引いていく。月明かりが海を銀色に染めている
  - 紬が「ねえ、私のこと……どう思ってる？」と聞く。蒼汰が黙る。長い沈黙
  - 紬が「やっぱ聞かなかったことに——」と言いかけた時、蒼汰が「……知らねえよ。でも、お前がいないと防波堤がやけに静かだなとは思う」
  - 紬の目から涙がこぼれる。「なにそれ……全然ロマンチックじゃない」と笑いながら泣く
  - 蒼汰が紬の涙を指で拭おうとして、手が止まる。紬が蒼汰の手に自分の頬を押し当てる。「……冷たくない。蒼汰くんの手、あったかい」
  - 蒼汰が紬の顔を両手で挟む。紬の琥珀色の目が月明かりの中で揺れている。涙の跡が光っている
  - 蒼汰のキス。不器用で、唇の位置がずれて、紬の唇の端に触れる。紬が「……ずれてる」と囁く。「うるせえ」。蒼汰がもう一度。今度はちゃんと唇に。短い、震えるようなキス
  - 紬が蒼汰の制服の胸元を掴む。「……もっと」。蒼汰の心臓が破裂しそうに鳴っている
  - 二度目のキス。今度は少し長い。潮風が二人の髪を混ぜる。波の音が近い
  - 蒼汰が唇を離す。額を紬の額にくっつけたまま「……場所、変えるか」。紬が小さく頷く
  - 浜辺の奥、大きな岩が風よけになっている場所。月明かりは届くが、宿泊棟からは見えない。砂の上に蒼汰のパーカーを敷く
  - 紬が蒼汰を見上げる。「蒼汰くん……怖い顔してる」。蒼汰が「……怖いのは俺のほうだ」。紬が微笑む。「じゃあ、二人で怖がろう」
  - 蒼汰が紬を抱きしめる。初めて触れる紬の体は細くて、潮の匂いと甘い匂いが混じっている。紬が蒼汰の胸に顔を埋める。心臓の音が聞こえる。「……すごい速い」。「聞くな」
- **描写のポイント**: 不器用な告白と初めてのキス。蒼汰らしい口下手な愛情表現。紬の涙と笑顔の同居。月明かりの海辺の美しさ。二人とも「初めて」であることの緊張感。「怖い」を共有することで生まれる親密さ
- **感情の流れ**:
  - 蒼汰: 内心の葛藤 → 不器用な告白 → キスの緊張 → 覚悟 → 怖さと愛おしさの共存
  - 紬: 勇気を出した問い → 蒼汰の答えに泣き笑い → キスの驚きと幸福 → 覚悟と信頼
- **文量目安**: 約3,500字

### シーン7：星空の下・行為（行為シーン②：行為本体）
- **場所**: 浜辺エリア・岩陰
- **時間帯**: 8月中旬、夜（続き）
- **登場人物**: 蒼汰、紬
- **行為の流れ**:
  - 蒼汰が紬のシャツのボタンに手をかける。手が震えている。紬が「……私がやるよ」と自分でボタンを外し始める。白い肌が月明かりに晒される。鎖骨のライン、ささやかな胸の膨らみを包む白いブラジャー
  - 蒼汰が息を呑む。「見ないで……恥ずかしい」と紬が胸を腕で隠す。蒼汰が「……綺麗だ」と掠れた声で言う。自分でも驚くほど素直な言葉。紬が腕を少しだけ下ろす
  - 蒼汰のシャツも脱ぐ。島生活で少し焼けた肌。紬が蒼汰の胸板に手を当てる。「心臓、まだすごい速い」。「だから聞くなって」
  - 蒼汰が紬を砂の上（パーカーの上）に横たえる。紬のスカートの裾が砂に触れる。月明かりが紬の体を照らしている。蜂蜜色の髪が砂の上に広がる
  - 蒼汰が紬の鎖骨に唇を落とす。紬が「ひっ」と小さく声を上げる。「くすぐったい……でも、やめないで」
  - ブラジャーを外す。紬の胸は小さくて、薄い桜色の乳首が月明かりの中で見える。蒼汰の指が触れると、紬が体を震わせる。「あっ……」。声が波の音に混じる
  - 蒼汰が紬の胸に唇を寄せる。舌先が乳首に触れた瞬間、紬の背中が弓なりに反る。「そ、蒼汰くんっ……なにそれ……変な感じ……」。甘い声が漏れる
  - 蒼汰の手が紬の太腿に触れる。スカート越しに。紬が少し脚を閉じかける。「怖いか」と蒼汰。紬が深呼吸して「……ううん。触って」
  - スカートをたくし上げる。白い下着。紬が真っ赤になって目を瞑る。蒼汰の指が下着の上からそっと触れる。「……濡れてる」。紬が「言わないでっ……！」と蒼汰の肩を叩く

### シーン8：朝凪（余韻）
- **場所**: 浜辺 → 宿泊棟への帰り道 → 宿泊棟
- **時間帯**: 8月中旬、明け方
- **登場人物**: 蒼汰、紬
- **出来事**:
  - 浜辺で目覚める。二人ともパーカーにくるまって砂の上で眠っていた。東の空が薄紫に染まり始めている。朝凪――風が止まり、海面が鏡のように静まっている
  - 紬が先に目を覚ます。隣で眠る蒼汰の寝顔を見て、昨夜のことを思い出し、顔が真っ赤になる。でも、お腹の奥にまだ蒼汰の温もりが残っている気がする
  - 蒼汰が目を開ける。紬と目が合う。一瞬の沈黙。「……おはよう」と蒼汰。「おはよう」と紬。それだけで、昨夜が夢じゃなかったと確認できる
  - 朝日が水平線から昇り始める。二人で並んで朝日を見る。オレンジ色の光が海を染め、紬の蜂蜜色の髪を金色に輝かせる

  - 宿泊棟への帰り道。朝の島は静かで、鳥の声と波の音だけ。ハイビスカスの赤い花が朝露に濡れている
  - 紬が蒼汰の手を繋ぐ。今度は言い訳なしに。蒼汰が少し手に力を込めて握り返す
  - 宿泊棟の前で立ち止まる。「じゃあ、また後で」と紬。男女別の入口で別れなければならない
  - 紬が背伸びして、蒼汰の頬にキスをする。「……昨日の、ずれてたやつのお返し」。蒼汰が耳まで赤くなる

  - 蒼汰が男子棟に入る。窓から朝の海が見える。3ヶ月前、フェリーの上から見た時は何も感じなかった海。今は違う。この島の海は、紬の匂いがする

  - 朝の光が部屋に差し込む。新しい一日が始まる。蒼汰と紬の、島での物語はまだ始まったばかり
- **描写のポイント**: 朝凪の静謐な美しさ。朝日が紬を照らす描写。蒼汰の内面の変化（シーン1の空虚さとの対比）。頬へのキスと「ずれてねえよ」の呼応。「この島に来てよかった」という着地。読者に「この先」を期待させるラスト
- **感情の流れ**: 穏やかな幸福 → 照れ → 紬への愛おしさ → 島への感謝 → 未来への微かな希望

## 描写方針（EP1固有）

### 蒼汰の語り口
- 一人称「俺」。簡潔で飾らない。余計なことは言わない
- 感情が動くと文が短くなる。体言止めや倒置法が増える
- 紬への感情は自覚が遅い。読者には伝わっているが、蒼汰本人は認めたがらない
- 内心のモノローグで「……」が多い。言葉にならない感情を沈黙で表現する
- 母への想いは行動で示す（仕送り、電話の短さ）。言葉にするのが苦手

### 紬の話し方
- 「蒼汰くん」が基本の呼び方（行為中も変わらないが、声のトーンが変わる）
- 一人称は「私」
- 普段は丁寧語混じりだが、感情が高ぶると砕けた口調になる
- 泣き虫だが泣くのを我慢する。目を擦る癖がある
- 嬉しいとき、照れたとき、笑ったときに目が三日月型になる（蒼汰が気づくポイント）

### 島の情景描写
- **五感の活用**: 潮の匂い、波の音、砂の感触、南国の花の香り、日差しの熱さ
- **光の変化で時間の流れを演出**: 昼の白い光 → 夕暮れのオレンジ → 夜の月明かり → 朝日の金色
- **海が二人の関係のメタファー**: 最初は「眺めるだけの海」→「一緒に見る海」→「二人の秘密を知る海」
- **季節の移ろい**: 春の入島（桜はないが、ハイビスカスが咲き始める）→ 夏の盛り（入道雲、蝉、強い日差し）→ 行為後の朝凪（静かな海、朝日）

### 行為シーンの方針
- 行為シーン全体で約10,500字（導入3,500字＋行為本体7,000字）。余韻はシーン8に含む
- 二人とも初体験。不器用さ・手探り感を大切にする
- 蒼汰は口下手なぶん、行動（額へのキス、手を握る）で優しさを見せる
- 紬は恥ずかしがりながらも、蒼汰への信頼から自分を委ねる
- 痛みの描写はリアルに。ただし「蒼汰だから耐えられる」という信頼関係で乗り越える
- 中出しは紬から求める形。「プログラム」の義務ではなく、蒼汰と繋がりたい感情から
- 環境描写（月明かり、波音、星空、砂の感触、潮風）と行為描写を融合させ、「この場所でしかできない体験」として描く
