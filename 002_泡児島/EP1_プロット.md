# EP1「南風の楽園」詳細プロット

> 朝比奈遼一（34歳・ルポライター）× 渡真利海凪（11歳・島の少女）/ 初来島 / 泡児島（人工島リゾート）

---

## 概要

ルポライター・朝比奈遼一が「子作り特区の闇を暴く」体験取材で泡児島に初来島する。ビーチで出会った最年少の少女・渡真利海凪（11歳）の天真爛漫な笑顔に面食らいながらも取材を進める遼一だが、海凪の境遇を知り、記者としての立場が揺らいでいく。その夜、海凪にとって初めてのナイトサービスの相手として一夜を共にすることで、遼一の中のジャーナリストとしての理性が崩壊する。翌朝、海凪の「あたしたちのこと、ちゃんと書いて」という一言に背を押され、島を離れる遼一は、記事を書くことと海凪を守ることの二律背反を抱えたまま東京に帰還する。

※ EP0で島の制度・世界観・雰囲気は読者に提示済み。EP1では遼一の視点を通じた「初来島体験」と海凪との関係に集中する。

## シーン構成（全7シーン）

### シーン1：取材の始まり（日常パート・圧縮版）
- **場所**: 鹿児島の港 → 高速船 → 水平線に浮かぶ泡児島
- **時間帯**: 7月上旬、昼前
- **登場人物**: 遼一、隣の乗客（リピーター）
- **出来事**:
  - 鹿児島の専用ターミナル。泡児島行きの高速船はリゾートクルーズのような豪華さ。乗客は全員男性。スーツ姿のビジネスマン、カジュアルな格好の若者、初老の紳士。誰も目を合わさない
  - 遼一の独白：編集長に「体験しなきゃ書けないだろ。会員登録は済ませてある。経費だ」と送り出された。「子作り特区の闇を暴く」――キャリアを賭けた取材だ。だが、会員証を懐に入れた瞬間から、胸の奥でざわつく何かを無視できない。取材だ。俺はジャーナリストとしてここにいる。体験取材は手段であって目的じゃない。そう自分に言い聞かせる
  - 高速船のデッキ。水平線に泡児島が現れる。白い砂浜、ヤシの木、リゾートホテルの建物群。どこからどう見ても南国の楽園。ここが「児童との性交が合法の島」だとは思えない美しさ
  - 隣の乗客（40代サラリーマン風）がビールを飲みながら「いい島ですよ。3回目です」と話しかけてくる。「最初は罪悪感もありましたけどね。あの子たちの笑顔見てるとね、まあ、制度だし」。遼一は愛想笑いを返しながら、取材ノートにメモを走らせる
  - 島が近づくにつれて、海の色がターコイズブルーに変わっていく。デッキに出ていた乗客たちの表情が緩む。まるで「日常」から「非日常」に切り替わるスイッチが入ったかのように
- **描写のポイント**: 遼一のジャーナリストとしての立場と内面の葛藤の提示。高速船の乗客たちの描写で「こういう男たちが行く島」というリアリティ。島の美しさと目的のギャップ。EP0で世界観は提示済みのため、制度の説明は遼一の内省と乗客の会話で最小限に留める
- **感情の流れ**: 使命感 → 不安と罪悪感 → 島の美しさへの戸惑い
- **文量目安**: 約2,000字

### シーン2：楽園のチェックイン＋海凪との出会い（非日常への移行）
- **場所**: 泡児島・ターミナルエリア → リゾートエリア → ビーチバー
- **時間帯**: 7月上旬、昼〜午後
- **登場人物**: 遼一、コンシェルジュ、ビーチの少女たち、海凪（初登場）
- **出来事**:
  - 島に上陸。ターミナルは空港のようにきれい。コンシェルジュ（30代女性、ホテルマンのような所作）が笑顔で出迎える
  - 会員証の確認、島内ルールの説明。「少女たちの意思を最優先に。無理強いは即退島です」。遼一は記者の目でルールの穴を探しながら聞く
  - リゾートエリアに案内される。南国リゾートそのもの。プール、レストラン、バー。観光客の男たちがくつろいでいる
  - ビーチに出る。白い砂浜。ターコイズブルーの海。そして――水着姿の少女たちが数人、ビーチバーで飲み物を配ったり、観光客と談笑したりしている
  - 遼一は息を呑む。12歳、11歳、14歳。水着を着た少女たちが、大人の男たちの隣で笑っている。その光景は美しくて、同時にどこか壊れている
  - ビーチバーに座る遼一。マンゴージュースを注文。運んできたのは小麦肌の少女。肩下までのゆるいウェーブヘアを高いポニーテールにまとめ、裸足で砂浜を駆けてくる。群青色の大きな瞳、八重歯が覗く笑顔。どう見ても小学生だ
  - 「はい、マンゴージュース！ おにーさん、はじめて？ めっちゃ顔カタいんだけど！」。海凪の第一声。声が大きい
  - 遼一は面食らう。こちらが構える間もなく、海凪は遼一の隣にぴょんと飛び乗って話しかけてくる。「どっから来たの？」「東京!? すごくない!? あたし沖縄の島だよ〜。ちっちゃい島。でもこの島もきれいでしょ？ ねーねー！」
  - まるで迷い込んできた子犬のような海凪。遼一はペンを持つ手をテーブルの下に隠す。取材対象だ。だが、この子の笑顔があまりにも眩しい。そして、あまりにも幼い
  - 海凪が遼一の顔をじっと見て「おにーさん、なんか目がやさしい！ ゆいねぇが言ってたさぁ、目がやさしい人は大丈夫だって」と笑う。遼一は「ゆいねぇ」が年上の少女の名前だと推測する。ここでは少女同士が客の情報を共有しているのだろう
  - コンシェルジュが近づいてきて「朝比奈様、本日のお部屋はコテージ7番です。夜のパートナーはお決まりですか？」。遼一が言葉に詰まる。海凪が「あ、あたし今夜ひまだよ？ おにーさんといっしょがいい！」と屈託なく言う
- **描写のポイント**: 島の「楽園」としての完成度の高さ。少女たちが「普通に」働いている光景の衝撃。海凪の天真爛漫さと遼一の動揺。「取材」の建前が崩れていく予感。海凪の初登場シーンは最重要場面として丁寧に
- **感情の流れ**: 記者としての観察 → 少女たちを見た衝撃 → 海凪の笑顔に面食らう → 「夜のパートナー」という現実
- **文量目安**: 約3,500字

### シーン3：午後のビーチ（関係の変化）
- **場所**: ビーチエリア → レストラン
- **時間帯**: 同日、午後〜夕方
- **登場人物**: 遼一、海凪
- **出来事**:
  - 遼一は海凪を指名する。取材のためだ、と自分に言い聞かせる。海凪は満面の笑みで「やったぁ！ おにーさん、泳ごう泳ごう！」と遼一の手を引っ張る
  - ビーチで過ごす午後。海凪が勢いよく海に飛び込み、遼一を手招きする。「来ないの？ もったいないよ！ ほらほら！」
  - 遼一は最初は渋るが、海凪のあまりの楽しそうな様子に引きずり込まれる。二人で泳ぐ。海凪は魚のように速い。「沖縄の子はみんな泳ぎうまいんだから！」。11歳の子供の体力に、34歳が振り回されている
  - 浜辺で休憩。海凪が濡れたポニーテールの水を絞りながら、遼一の隣にどさっと座る。水着から覗く小麦肌、鎖骨についた水滴、子供らしい細い手足。膝小僧の擦り傷が目に入る。遼一は意識的に目を逸らす
  - 遼一がさりげなく聞く。「海凪は、どうしてこの島に？」
  - 海凪の表情が一瞬だけ曇る。すぐに笑顔に戻して「んー、お金かなぁ。おばぁの借金があってさぁ」。軽い口調で重い話をする。遼一の胸が痛む。まだ11歳だぞ
  - 「でもね、ここ楽しいよ！ ごはんおいしいし、友達いるし、海きれいだし。おにーさんたちもやさしい人多いし」。海凪が空を見上げる。「おばぁが元気なうちに借金ぜーんぶ返して、沖縄に帰るの。それがあたしの目標！」。拳をぐっと握る
  - 遼一は記者として聞くべきことがある。「辛いことは？ ……夜のこととか」。海凪がぱちくりと瞬きする。「あー……あたし、まだ夜のおしごとしたことないんだ」。遼一が驚く。「え？」
  - 海凪が砂を指でいじりながら、少しだけ声のトーンを落とす。「入って3ヶ月でしょ。昼のビーチ接客は最初からやってるんだけど……夜のほうは、ゆいねぇが『焦んなくていいよ』って。あたしも、ちょっとこわくて。ずっと先延ばしにしてたさぁ」
  - 遼一の胸が締まる。つまりこの子は――まだ誰とも。「でもね」と海凪が顔を上げる。群青色の目が真っ直ぐに遼一を見る。「きのう、覚悟きめたの。おばぁの借金、待ってくれないから。赤ちゃん産まないとお金もらえないでしょ？ ……だから、おにーさんが、あたしのはじめてのひとになるんだよ」
  - その言葉が遼一の腹の底に落ちる。初めて。この子の、初めて。11歳の子供が、借金のために、今夜初めて男に体を開く。そしてその相手は――俺だ
  - 海凪が遼一の顔を見て「おにーさん、すっごいかお。だいじょぶだよ、あたし泣かないし！ たぶん！ ……うん、たぶんね！」と笑う。その笑顔が強がりであることは、遼一にも分かる
  - 夕食をレストランで一緒にとる。海凪がミニ浴衣に着替えてきた。白地にハイビスカス柄の膝上丈の浴衣。帯は赤いリボン結び。素足にサンダル。短い裾から小麦色の太腿がまぶしい。「どう？ おにーさん！ かわいい？」。遼一は「似合ってる」と素直に答える。海凪が嬉しそうに八重歯を見せてぴょんと跳ねる。跳ねた拍子に裾がふわりとめくれる。下着を着けていないことに遼一は気づいてしまい、慌てて目を逸らす。海凪は気にしていない様子——島ではそれが当然なのだろう
  - 食事中、海凪がとにかくよく食べる。「島の魚おいしいでしょ！ これ、今朝揚がったやつだよ！ あ、このエビフライもうまいよ！」。遼一は海凪の食べっぷりに笑ってしまう。この子は11歳だ。腹が減って、おいしいものを食べて笑う、普通の子供だ。——なのに今夜、この子の「はじめて」を俺が奪う
  - 夕日が海に沈む。レストランのテラスから二人で見つめる。海凪が「きれい……」と珍しく静かに呟く。その横顔に夕日の赤が映っている。遼一の中で何かが揺れる。この子はまだ、ランドセルを背負っていてもおかしくない年齢だ
  - 海凪がふいに遼一の袖を掴む。小さな指。「……ねぇ、おにーさん。今夜、やさしくしてね？」。冗談めかした声だが、指先は震えている
- **描写のポイント**: 海凪の「子犬」のような元気さとその裏にある事情。「処女」であることの告白が自然な会話の中から出る。遼一が「記者の目」から「一人の男の目」に変わり始める。ビーチ・夕食という時間の中で、今夜が海凪の「初めて」であるという事実が遼一にのしかかっていく
- **感情の流れ**:
  - 遼一: 取材モード → 海凪の身の上に胸が痛む → 「はじめてのひと」の衝撃 → 記者としての自分と男としての自分の乖離 → 覚悟と罪悪感
  - 海凪: 営業の明るさ → 身の上を話す時の一瞬の陰り → 「はじめて」を告白する時の緊張 → 食事の楽しさ → 遼一への好印象と不安の混在
- **文量目安**: 約4,500字

### シーン4：コテージの夜・導入（行為シーン①：導入）
- **場所**: リゾートエリア → コテージ7番
- **時間帯**: 同日、夜20:00頃
- **登場人物**: 遼一、海凪
- **出来事**:
  - 夕食後、海凪が「じゃあ、コテージ行こっか！」と遼一の手をぎゅっと握ってくる。小さくて温かい手。子供の手だ。だが、その手がいつもより少し汗ばんでいる。遼一の心臓が跳ねる
  - コテージへの小道。ヤシの木がライトアップされ、波の音が近い。南国の夜風が肌を撫でる。まるでハネムーンのロケーションだ
  - 海凪がいつもより少し口数が少ない。歩きながらぎゅっと遼一の手を握り直す。「……おにーさん」「ん？」「……なんでもない」。小道の明かりに照らされた横顔は、昼のビーチで見せていた無邪気な表情とは少し違う。緊張している。当然だ。この子にとって初めての夜だ
  - コテージに入る。清潔な白いインテリア。大きなベッド、バスルーム、窓の外に月と海。海凪が照明を落としてムード照明にする。「こっちのほうが落ち着くでしょ？ ……ゆいねぇに教わったの」。手つきはぎこちない。遼一の胸が締まる
  - 海凪が「おふろ、いっしょに入ろ！ ……入るんだよね？ そういうもんなんでしょ？」と笑う。笑顔だが声が少し上擦っている。遼一は一瞬固まる。取材だ。体験取材だ。でも――この子の初めてだぞ
  - バスルーム。海凪がミニ浴衣の帯をするりと解く。浴衣がはだけると、その下は何も着ていない——一瞬で全裸。健康的な小麦肌、まだ胸はほとんど平たく、子供そのものの体つき。遼一は息を呑む
  - 海凪が下着も外す。「おにーさんも脱がないと、おふろ入れないよ？」。けろっとした笑顔を作ろうとしている。だが、腕が無意識に胸の前で交差しかけるのを遼一は見逃さない。この子は裸を見せることにすら慣れていない
  - シャワーで体を流し合う。海凪が遼一の背中を洗う。「おにーさん、背中おっきいねぇ！」。子供の小さな手が肩甲骨をごしごし洗う。一生懸命だ。たぶん、これもゆいねぇに教わったのだろう
  - 遼一が海凪の体を洗う。華奢な背中。肩甲骨が浮き出た、まだ子供の骨格。膝小僧の擦り傷にタオルが触れると「いたっ！」と声を上げる。「くすぐったい！」と海凪が笑う。その笑い声が浴室に響く。一瞬だけ、普通の子供に戻る
  - 湯船に二人で浸かる。狭い浴槽で体が密着する。海凪が遼一の胸に背中を預ける。「あったかい……おにーさん、あったかいね」。子供が親に甘えるような自然さ。だが、海凪の体が微かに強張っているのが伝わってくる
  - 遼一の体が反応している。海凪はそれに気づいている。一瞬びくっと体が跳ねる。だが指摘しない。少し間があって「おにーさん、今夜……ちゃんとしてね？ あたし、おにーさんの赤ちゃんほしいな」
  - その言葉で遼一の頭が真っ白になる。11歳の子供が、「赤ちゃんがほしい」と言っている。初めての夜に。いや――海凪の目は、子供の目じゃない。怯えと覚悟が入り混じった目で、何かを真剣に見つめている
  - 湯船から出て、バスタオルを巻いた海凪がベッドに座る。濡れたポニーテールを解いて、ウェーブヘアが肩に張り付いている。脚を揃えて、膝の上に手を置いている。指先が微かに震えている。「おにーさん、おいで」。手を差し出す。声は明るくしようとしている。でも、少し掠れている
- **描写のポイント**: コテージの密室空間。入浴シーンで距離が縮まるが、海凪の「初めて」の緊張がより強く出る。先輩に教わった手順をなぞろうとする健気さと、隠しきれない恐怖。「子供を産みたい」の一言が持つ重み。遼一の「記者」の仮面が完全に剥がれる瞬間
- **感情の流れ**:
  - 遼一: 緊張 → 海凪の体への動揺 → 「初めて」の子を相手にする罪悪感 → 「赤ちゃんがほしい」の衝撃 → 記者としての理性の崩壊
  - 海凪: 教わった手順をこなそうとする健気さ → 入浴中の安らぎと緊張の交錯 → 「産みたい」に込めた本気と怖さ → 手を差し出す覚悟
- **文量目安**: 約4,000字

### シーン5：コテージの夜・行為（行為シーン②：行為本体 ※処女喪失）
- **場所**: コテージ7番・ベッドルーム
- **時間帯**: 同日、夜21:00頃
- **登場人物**: 遼一、海凪

#### 行為本体

- 遼一が海凪の手を取り、ベッドに引き寄せる。記者ノートはテーブルに置いたまま。今夜はペンを持たない
- 海凪のバスタオルがするりと落ちる。月明かりに照らされた小麦肌。11歳の体は細くて、子供の骨格で、でも柔らかい。海凪が反射的に腕で胸を隠しかける。遼一が「隠さなくていい」と囁く。海凪が恥ずかしそうに腕を下ろす。「……はじめてだから、どうしたらいいかわかんない」
- キス。遼一から。海凪が目を閉じて受け入れる。小さな唇が柔らかい。マンゴーの味がする。海凪が「おにーさん、チューじょうず」と目を開けて笑う。目の端に薄く涙が浮いている。緊張か。遼一が「営業で言ってるだろ」。海凪が「えへへ、チューもはじめて。ほんとだよ」
- 遼一が海凪の首筋に唇を落とす。海凪が「ん……」と小さく声を漏らす。体がびくっと震える。鎖骨の下の小さなほくろに気づく。「ここ、ほくろあるんだ」。海凪が「……見つけちゃった」。声が少し甘くなる
- 胸への愛撫。まだ平たい胸に遼一の手が触れる。海凪が体をびくっと震わせる。「あっ……そこ、へんなかんじ……っ」。薄い桃色の乳首が小さく硬くなる。遼一の指先と舌が交互に刺激する。海凪の背中が反る。「んんっ……なにこれ……からだ、へん……っ」。初めて触れられる感覚に海凪の体が素直に反応する。快感を知らない体が、戸惑いながら感じ始めている

## 描写方針（EP1固有）

### 遼一の語り口
- 一人称「俺」。記者らしい観察眼で周囲を分析する
- 職業意識と欲望の間で揺れる内面描写。「取材だ」という言い訳が徐々に通用しなくなっていく
- 海凪に触れた時の描写は五感を総動員。記者の冷静な観察力が、皮肉にも海凪の体の細部を克明に記録してしまう
- 自嘲と自己嫌悪が混在する内省的な文体
- 海凪の「初めて」を奪うことへの罪悪感が、物語全体を貫く通奏低音

### 海凪の話し方
- 「おにーさん！」が基本の呼び方。元気よく呼ぶ
- 一人称は「あたし」
- 沖縄方言が自然に混じる（「〜さぁ」「〜だばー」「なんくるないさ」等）
- 基本テンション高めでお喋り。語尾が跳ねる。「ねーねー！」「すごくない!?」が口癖
- 核心に触れる話題になると急に口数が減り、海を見つめる
- 泣いた後はぱっと切り替えて笑う。「あたし泣いてないし！」と強がる
- 行為シーンでは「がんばるね」「だいじょぶ」が口癖に。痛くても笑顔を作ろうとする健気さ

### 処女喪失シーンの描写方針
- **段階的な変化**: 指での愛撫（狭さ・処女膜の示唆）→ 挿入時の痛み・涙・出血 → 痛みの中に芽生える快感 → 最終的な受容
- **海凪の覚悟と脆さの両面**: 「おばぁのため」という覚悟と、11歳の子供としての恐怖・痛みの両方を描く
- **遼一の「止まる」描写**: 痛がる海凪に対して遼一が動きを止める場面を複数回入れ、海凪の「つづけて」で再開する。この呼吸が二人の関係性を象徴する
- **出血の扱い**: 行為後に気づく程度の控えめな描写。シーツの赤い染みとして間接的に。海凪自身が「ゆいねぇが言ってた」と受け止めることで、島の先輩後輩関係を暗示
- **痛みから快感へのグラデーション**: 最初は純粋な痛みだが、遼一がゆっくり動く中で「痛いのにきもちいい」という混在状態を経て、最後は快感が勝っていく
