### シーン4：コテージの夜・導入（行為シーン①：導入）

### シーン5：コテージの夜・行為（行為シーン②：行為本体）
- **場所**: コテージ7番・ベッドルーム

- **登場人物**: 遼一、海凪

#### 行為本体

- 遼一が海凪の手を取り、ベッドに引き寄せる。記者ノートはテーブルに置いたまま。今夜はペンを持たない

### シーン7：翌朝の見送り（日常への帰還）
- **場所**: コテージ → ビーチ → ターミナル

- **登場人物**: 遼一、海凪
- **出来事**:

  - 食後、二人でビーチを散歩する。朝のビーチは静かで、観光客はまだいない。海凪と遼一だけ
  - 海凪が遼一の手を繋ぐ。「朝のビーチ、好きなの。仕事じゃない時間だから」。その言葉が刺さる
  - 「ねぇ、おにーさん。あたしの話、だれかに伝えてくれる？ おにーさん、文章書くおしごとなんでしょ？」。海凪が真っ直ぐに遼一を見る。群青色の大きな瞳。「この島のこと。あたしたちのこと。ちゃんと書いて」

- **感情の流れ**:
  - 遼一: 海凪の切り替えの速さへの痛み → 手を繋いだ温もり → 記者としての使命感の再燃 → 島を離れる喪失感
  - 海凪: いつもの明るさ → 「書いて」という本心 → 見送りの寂しさ
- **文量目安**: 約3,000字

### シーン8：記者の帰路（余韻）
- **場所**: 高速船 → 東京のマンション
- **時間帯**: 同日、昼〜夜
- **登場人物**: 遼一
- **出来事**:
  - 高速船のデッキで、遼一は取材ノートを開く。メモはほとんど取れていない。代わりに、海凪の顔が頭から離れない
  - ノートに書き始める。「泡児島。通称"子作りの島"。人工島のリゾート施設に、10〜15歳の少女たちが在住し――」。ペンが止まる。海凪の声が蘇る。「おばぁの借金、あと何回か産めば返せる」
  - 遼一の内面描写。記事に書けば、島は問題視される。閉鎖されるかもしれない。海凪は借金を返せなくなる。書かなければ、島は続く。少女たちはこれからも体を差し出し続ける。どちらを選んでも、誰かが傷つく
  - 東京に戻る。マンションの部屋は暗くて静かだ。波の音も花の香りもない。冷蔵庫のビールを開ける
  - パソコンの前に座る。原稿を書き始める。タイトルは決まらない。「子作り特区の闇」と書いては消す。「楽園の少女たち」と書いては消す
  - スマホに通知。泡児島の会員サイトから。「渡真利海凪からメッセージが届いています」。開くと、海凪が浜辺で撮った自撮り。八重歯の笑顔にピースサイン。「おにーさん！ 今日の海めっちゃきれい！ ぜったいまた来てね😊😊😊」。絵文字が多い。11歳だ
  - 遼一がスマホの画面を見つめる。この笑顔を記事にするのか。この笑顔を守るために書くのか。それとも、この笑顔に会いに行くために島に戻るのか
  - 遼一がパソコンを閉じ、スマホの画面を長く見つめる。部屋の窓の外には東京の夜景。どこにも海は見えない
  - 「……また、行くんだろうな。俺は」
  - 答えは出ない。でも、指は会員サイトの「次回予約」ボタンの上に止まっている

- **感情の流れ**: 混乱 → 書こうとしては止まる → 海凪のメッセージへの動揺 → 未解決のまま夜に沈む
- **文量目安**: 約3,000字

## 描写方針（EP1固有）

### 遼一の語り口
- 一人称「俺」。記者らしい観察眼で周囲を分析する
- 職業意識と欲望の間で揺れる内面描写。「取材だ」という言い訳が徐々に通用しなくなっていく
- 海凪に触れた時の描写は五感を総動員。記者の冷静な観察力が、皮肉にも海凪の体の細部を克明に記録してしまう
- 自嘲と自己嫌悪が混在する内省的な文体

### 海凪の話し方
- 「おにーさん！」が基本の呼び方。元気よく呼ぶ
- 一人称は「あたし」
- 沖縄方言が自然に混じる（「〜さぁ」「〜だばー」「なんくるないさ」等）
- 基本テンション高めでお喋り。語尾が跳ねる。「ねーねー！」「すごくない!?」が口癖
- 核心に触れる話題になると急に口数が減り、海を見つめる
- 泣いた後はぱっと切り替えて笑う。「あたし泣いてないし！」と強がる

### 島の情景描写
- **リゾートの華やかさ**: 白砂ビーチ、ヤシの木、ターコイズブルーの海、カクテル、南国の花
- **夜のコテージ**: 月明かり、波の音、白いシーツ、間接照明。密室の親密さ
- **朝のビーチ**: 静寂、朝日、水平線。「仕事じゃない時間」の穏やかさ
- **光の変化**: 昼の眩しさ → 夕暮れのオレンジ → 夜の月光 → 朝日の金色
- **東京との対比**: 島の色彩豊かさに対して、遼一のマンションはモノクロ的に描写

### 二重構造の演出
- 海凪の「教わった接客」と「子供の素」の境界線を常に揺らす。読者にもどちらか分からない瞬間を作る
- 遼一の「記者」と「客（男）」の二重性。どちらが本当の自分か分からなくなっていく
- 島そのものが「楽園」と「搾取の装置」の二重構造。美しいからこそ残酷
