## 核心コンセプト：「おばぁの宝物は、おばぁを守るために『おんなのこ』になる」

海凪は沖縄の離島でおばぁに育てられた。海と太陽と珊瑚礁が全世界だった。

その世界が壊れるのは、おばぁの借金のせい。——正確には、おばぁが海凪を育てるために重ねた借金のせい。

泡児島は「子作り観光特区」。少女が大人の男と子を作り、出産一時金300万円を受け取れる人工島。海凪はそこに行くことを——おばぁのために——自分で選ぶ。

**「はじめて。」の莉子は仮面を壊された。海凪は仮面すら持っていない。素のまま、裸足のまま、『おんなのこ』になる道を歩き出す。**

EP0は行為を描かない。代わりに描くのは——

- おばぁとの最後の日々。手のひらの温度。線香の匂い。波の音
- 知らない島への船旅。潮風の味が変わる瞬間
- 先輩・結衣の笑顔に隠された優しさと痛み
- 「ナイトサービス」という言葉の意味をまだ知らない海凪の無垢
- そして——知った後の、静かな覚悟

覚悟は一瞬で決まるものじゃない。六日間かけて、波が砂を削るように、少しずつ、少しずつ深くなる。

**EP0のラスト一文で海凪が裸足で歩き出す時、読者はもう知っている。この子の足の下の砂は、沖縄の砂じゃない。**

## シーン1：おばぁの島（~3,500字）

> **本シーンの核心**: 海凪の「全世界」を描く。この世界が壊れる前の、最後の一日。  
> 読者がこの島の匂いと温度を体に刻んでおくことで、泡児島との対比が効く。  
> **失われるものの重さは、それが在った日の描写でしか測れない。**

### 冒頭——朝の浜辺

- 波の音で目が覚める。毎朝。目覚まし時計なんかいらない
- 「おばぁー！ 今日めっちゃ天気いいさぁ！」——裸足で浜辺を駆けてくる。波の音より大きい声
- 空が青い。雲が白い。珊瑚礁の浅瀬がエメラルドに光っている
- 砂が足の裏に吸いつく。あったかい。太陽が昇ったばっかりなのに、もう砂があちこーこー
- 「おばぁ、見て見て、やどかり！ でっかいの！」——手のひらに宿借り。振り向く。にかっと笑う。八重歯が光る

### おばぁとの朝

- 縁側でおばぁが座っている。線香の匂い。仏壇に手を合わせた後
- 「海凪、朝ごはんできてるさ。ゴーヤーチャンプルー」
- 海凪が砂だらけの足のまま上がろうとする。「こら、足洗いなさい」——おばぁの声は低くてあったかい
- ちゃぶ台。二人分。ゴーヤーチャンプルーとじゅーしー（炊き込みご飯）と、もずく酢
- 「いただきます！」——両手を合わせる。ぱくぱく食べる。口いっぱい
- 「おばぁのゴーヤー、にがくないさぁ。あたしの友達はゴーヤーきらいって言うけど、おばぁのは別！」
- おばぁが笑う。目尻の皺が深くなる。「海凪が食べてくれるから、おばぁ嬉しいさ」
- 海凪がおかわりする。おばぁは自分の分を少し海凪の皿に移す。——海凪は気づいていない

### 島の日常

- 学校は島に一つ。全校生徒十二人。海凪は五年生で、同級生は二人
- 通学路は浜沿い。裸足。サンダルは手に持って歩く。「靴なんかいらんさぁ」
- 放課後。海で泳ぐ。素潜り。珊瑚の隙間を覗く。ルリスズメダイが青い
- 「あたし、この島の魚ぜんぶ知ってるよ。名前つけてるし」——得意げ
- 夕方。防波堤に座る。夕日が海に溶ける。空がオレンジからピンクから紫に変わる
- おばぁが迎えに来る。「海凪、帰るよ」。手を繋ぐ。おばぁの手は硬くて、でかくて、あったかい
- 「おばぁ、明日も海行こうね」
- 「行こうね」

### 夜——お守り袋

- 布団の中。おばぁが隣で寝ている
- 海凪はお守り袋を握っている。琉球紅型の布。おばぁが縫ってくれた。中に珊瑚のかけらと五円玉
- 「おばぁ、このお守り、いつ作ってくれたんだっけ」
- 「海凪が三つの時さぁ。熱出して、おばぁ怖くてね。治りますようにって、作ったさ」
- 「……おばぁ、泣いた？」
- 「泣いたさぁ。でーじ泣いた」——笑って言う。もう昔のこと
- 海凪がお守り袋をぎゅっと握る。珊瑚のかけらが布越しにごつごつする
- 「おばぁ。あたし、ぜったいおばぁより先に死なないからね」
- おばぁが海凪の頭を撫でる。ごつごつした指。でもやさしい
- 「海凪、おばぁのたからもんやっさぁ」——毎晩言う。毎晩。呪文みたいに
- 海凪が目を閉じる。波の音が遠い。線香の残り香がする。お守り袋があったかい

**文量ペース**: ここまでで約3,500字。島の匂い・温度・音を全身で焼き付ける

## シーン2：崩れる日常（~4,000字）

> **本シーンの核心**: 海凪の世界が音を立てて壊れる。でも壊れるのは一瞬じゃない。  
> ブローカーの訪問→おばぁの涙→海凪の決意→別れの朝、と段階的に進む。  
> **海凪が「おんなのこになる」決断をするのはこのシーンだが、その意味をまだ理解していない。**  
> **理解していないからこそ、決断が軽い。軽いからこそ、後で重くなる。**

### ブローカーの訪問

- 学校から帰ると、見知らぬ車が家の前に停まっている。白い車。島では見ない車
- 玄関に革靴。島の人はみんなサンダルか裸足。革靴は「外の人」
- 居間に男が座っている。スーツ。本島の匂い。エアコンと香水の匂い
- おばぁが正座している。——海凪は、おばぁが正座しているのを初めて見た
- 「あら、この子が海凪ちゃんですか。可愛いですねぇ」——ブローカーが笑う。歯が白い
- 海凪：「……おばぁ、この人だれ？」
- おばぁ：「海凪、ちょっと外で遊んできなさい」——声が固い
- 海凪：「やだ。おばぁ、顔こわいよ。なにかあったの？」
- ブローカー：「海凪ちゃん、お話していいかな。おばぁのこと、助けたいでしょう？」
- 海凪が座る。おばぁの隣に。おばぁの手を握る。——手が震えていた。おばぁの手が

### 借金の話

- ブローカーが話す。丁寧に。優しい声で。数字を出す
- 「おばぁ様の借金が、これだけあります。返済期限が、来月です」
- 海凪には数字の大きさが分からない。でも、おばぁの手の震えで分かる。でーじやばい
- 「返せなければ、この家を手放していただくことになります」
- おばぁ：「……分かっとるさ」——声が小さい。海凪が初めて聞く、おばぁの弱い声
- 海凪の胸がぎゅっとなる。おばぁが弱い声を出している。おばぁは強い人なのに
- ブローカーが続ける。「ただ——方法がないわけではありません」
- テーブルの上にパンフレットが置かれる。「泡児島」。綺麗な海の写真。リゾートみたい
- 「お嬢さんに、こちらの島で少しの間お仕事をしていただければ——出産一時金として、300万円」
- 海凪：「……しゅっさん？」——意味が分からない。赤ちゃんを産むこと？
- ブローカー：「海凪ちゃんが赤ちゃんを産めば、お金がもらえるんです。おばぁの借金も全部なくなりますよ」
- おばぁ：「やめなさい。この子はまだ——」
- 海凪：「おばぁ、いくら？」——遮る。声が変わっている。子どもの声じゃない
- ブローカー：「300万円です」
- 海凪はその数字が借金より大きいのか小さいのか分からない。でも——おばぁの手の震えが止まる方法があるなら
- 「あたしが行けば、おばぁ、もう泣かなくていいの？」
- 沈黙。波の音が遠くで鳴っている

### おばぁの涙

- ブローカーが帰った後。夕方。二人きり
- おばぁが泣いている。海凪の前で。——海凪が生まれてから一度も見たことがない光景
- 「おばぁ……泣かないで。あたし、大丈夫だよ」
- 「大丈夫じゃないさ。海凪はまだ子どもやっさぁ……おばぁが悪いさ。おばぁがお金なかったから……」
- 海凪がおばぁを抱きしめる。小さい腕で。おばぁの背中は思ったより細い
- 「おばぁ。あたし、行くよ。泡児島」——声が静か。波打ち際の海凪の声
- 「行かんでいい！ おばぁが何とかする！」
- 「おばぁ、嘘つくの下手さぁ」——にへっと笑う。目が濡れている。でも笑っている
- おばぁが海凪の顔を両手で挟む。ごつごつした大きい手で。海凪の頬が歪む
- 「海凪……おばぁのたからもの……」
- 「たからもんだから、おばぁを守りたいのさぁ。……なんくるないさ」
- おばぁが海凪を抱きしめ返す。強く。骨が軋むくらい
- 海凪の手の中で、お守り袋がぎゅっと潰れる。珊瑚のかけらが布越しに刺さる。——痛い。でも離さない

### 最後の夜

- 布団の中。おばぁがもう寝ている。——嘘。寝たふりをしている。肩が震えている
- 海凪は知っている。おばぁは海凪が寝た後で泣く人だ。でも今夜は、寝たふりしながら泣いている。順番が逆になっている
- 海凪がお守り袋を握る。ぎゅっと。珊瑚のかけらの形を指で確かめる
- 目を閉じる。——泡児島。赤ちゃん。300万円。よくわかんない。でも、おばぁが泣かなくなるなら
- 明後日の船で発つ。あさっての朝、この島を出る
- 波の音が聴こえる。いつもと同じ音。でも今夜は、遠い

**文量ペース**: ここまでで約7,500字

## シーン3：泡児島上陸（~3,500字）

> **本シーンの核心**: 沖縄の島と泡児島の「五感の差異」を突きつける。  
> 同じ「島」「海」「砂」なのに、全部が微妙に違う。人工物の不気味な快適さ。  
> 海凪の無邪気さと泡児島の本質のギャップが、読者にだけ見える。  
> **海凪はまだ気づいていない。砂の温度が違うことに。**

### 別れの朝

- 港。小さい漁港。おばぁが見送りに来ている
- 海凪はリュックひとつ。中身は着替えと教科書とお守り袋
- 「おばぁ、泣かないで。あたし、すぐ帰ってくるから」——にかっと笑う。八重歯
- おばぁ：「……手紙、書きなさいよ。おばぁ、待ってるから」
- 海凪：「うん！ いっぱい書く！ 島のこと、全部教えるさぁ」
- 抱きしめる。おばぁの匂い。線香と、潮風と、日焼け止めと、おばぁだけの匂い。全部混ざった、あったかい匂い
- 「おばぁ、あたし、がんばるね」
- おばぁが海凪の手にお守り袋を握らせる。もう一度。確認するように
- 「これ持ってなさい。おばぁの代わりに、海凪を守るから」
- 船が出る。海凪が甲板から手を振る。おばぁが小さくなる。港が小さくなる。島が小さくなる
- 「おばぁーっ！ 行ってくるねーっ！」——声が枯れるまで叫ぶ
- 振り向かない。振り向いたら泣く。だから前を向く。海が広い。知らない方向に、船が進んでいく

### 船の中

- ブローカーが同行している。海凪の隣に座っている。ペットボトルのお茶をくれる
- 「海凪ちゃん、船酔いしない？ 偉いねぇ」
- 海凪：「あたし、島の子だもん。船なんか平気さぁ」——にかっと笑うけど、手の中のお守り袋をぎゅっと握っている
- 窓の外。海の色が変わる。沖縄の海は透明な水色。——段々、深い紺色になる。外洋に出たから
- 「ねぇ、泡児島ってどんなところ？」
- ブローカー：「綺麗な島ですよ。学校もあるし、ビーチもあるし。お友達もたくさんできますよ」
- 海凪：「友達！ やったー！ あたし、人見知りしないから大丈夫！」
- ブローカーが微笑む。パンフレットを見せる。白い砂浜、プール、綺麗な建物
- 海凪：「うわー、リゾートみたい！ おばぁにも見せたかったなぁ」
- ——海凪は気づいていない。パンフレットに「子ども」の写真はあるのに、「大人の女の人」の写真がないことに

### 上陸

- 泡児島が見えてくる。——最初の違和感
- 沖縄の島は緑がぼさぼさに生えている。野生。泡児島は——整っている。芝生が均一。ヤシの木が等間隔。人工
- 「……きれい。でも、なんか、ちがう」——海凪がぽつりと言う
- 船を降りる。桟橋。白い。ピカピカ。おばぁの島の港は木製でガタガタだった
- 砂浜に降りる。裸足で。——砂の感触が違う。サラサラしすぎている。珊瑚のかけらが混じっていない。**人工砂**
- 「……砂、ちがう。おばぁの島の砂じゃない」——海凪が足の裏で感じる、最初の「ここは違う場所だ」
- でも空は青い。海は光っている。波の音がする。——似ているけど違う。コピーみたいな島
- 施設の人が迎えに来る。白い制服。笑顔。「海凪ちゃん、ようこそ泡児島へ！」
- 連れて行かれる。寮。一人部屋。エアコン完備。ベッドがふかふか。窓から海が見える
- 「すご……おばぁの家より全然きれい……」——でもお守り袋を握る手が、ちょっとだけ強くなっている

### 結衣との出会い

- 部屋のドアがノックされる

## 文体ルール（執筆時の指針）

### 基本設定

- 全編海凪視点（一人称「あたし」）
- 短文主体。体言止め多用。感情が動く時は文が短くなる
- 行為シーンがないため、五感描写と内省で密度を出す

### 海凪の語り口ルール

#### 明るい時（デフォルト）

- 文末が跳ねる。「〜さぁ！」「〜じゃん！」「〜だよ！」
- 感嘆符多い。テンション高い
- 比喩は子どもらしく直感的。「でーじおっきい！」「めっちゃきれい！」
- 主語がよく省略される。テンポが速い
- セリフが長い。一息に喋る

```
例：「おばぁー！ 見て見て、やどかり！ でっかいの！ 今までで一番でかいさぁ！」
```

#### 沈む時（波打ち際モード）

- 文が短くなる。体言止め。句点が増える
- 感嘆符が消える。語尾が下がる
- 方言が減る。標準語に近づく
- 比喩が内省的になる。「波みたいに」「遠い」
- セリフが短い。ぽつりぽつり

```
例：「……おばぁ。あたし、おんなのこになるよ。よくわかんないけど。でも、おばぁのために」
```

#### 切り替わりの瞬間

- 明るい→沈むの切り替えは**一文で起こる**
- 「——」ダッシュで接続。前半が明るく、後半が沈む

```
例：「ゆいねぇ、あたし怖くないよ」——にかっと笑う。八重歯。でも——おばぁを安心させるための笑顔と、同じ種類の笑顔
```

### 沖縄方言の使い方ガイド

| 方言 | 意味 | 使用場面 |
|------|------|---------|
| 〜さぁ | 〜だよ、〜ね（語尾） | 日常会話全般。一番頻出 |
| なんくるないさ | なんとかなるさ | 覚悟の場面で使用。回を重ねるごとに意味が重くなる |
| あちこーこー | 熱い、熱々 | 砂の温度描写。沖縄のみ |
| でーじ | とても、すごく | 強調。「でーじこわかった」「でーじ泣いた」 |
| ちゅらかーぎー | 美しい（人・景色） | 島の美しさを語る時 |
| やっさぁ | 〜だよ（断定の語尾） | おばぁの台詞で頻出。「たからもんやっさぁ」 |

**方言の減少ルール**: 海凪が沈む時、方言が減る。標準語に近づく。覚悟の場面で再び「なんくるないさ」が出る時、それは方言ではなく祈りの言葉になっている。

### おばぁの描写ルール

- おばぁのセリフは短い。一文が短い。でも一語一語が重い
- 語尾は「〜さ」「〜さぁ」。海凪より方言が濃い
- **おばぁは海凪の前では泣かない**（原則）。泣くのは海凪が寝た後。シーン2で原則が崩壊する→衝撃の大きさを示す
- おばぁの手の描写を反復する。「硬い」「大きい」「あったかい」「ごつごつ」。この手が海凪の全世界
- 線香の匂いをおばぁの「存在の証」として使う。泡児島で線香の匂いがしない＝おばぁがいない

### 結衣の描写ルール

- 結衣のセリフには「間」がある。言葉を選んでいる。海凪へのダメージを最小限にしようとしている
- 結衣の笑顔には層がある。表面の優しさ、中間の諦め、奥底の痛み。海凪にはまだ表面しか見えない
- **結衣が自分の過去を語る時、時制が過去形になる**。「好きだった」「こわかった」。海凪はその「た」に気づかない
- 結衣は海凪の方言を時々真似する。距離を縮めるため。でも下手。それが少しだけ笑いを生む

### 島の二重描写ルール（沖縄 vs 泡児島）

- 同じ要素（海・砂・波・月・空）を両方の島で描写する。必ず五感の差異を入れる
- 沖縄＝生きた自然。不揃い。匂いが複層的。温度がダイレクト
- 泡児島＝人工自然。均一。無臭に近い。温度が間接的（残り熱、エアコン）
- **海凪は泡児島の違和感に気づくが、言語化できない**。「なんかちがう」止まり
- 読者には分かる。海凪には分からない。そのギャップが不安を生む

### EP1との接続ルール

- EP0のラスト一文で海凪が「歩き出す」。EP1の冒頭は「明日」——ナイトサービス初日
- EP0で登場するモチーフ（お守り袋・裸足・砂の温度・おばぁの手の記憶・なんくるないさ）は、EP1以降で全て再登場し意味が変質する
- 結衣はEP1以降も海凪の支えであり続けるが、結衣自身の物語が徐々に明かされる
- おばぁへの手紙はEP1以降も続く。「嘘は書いていない」の構造が深化する——書けないことが増えていく
