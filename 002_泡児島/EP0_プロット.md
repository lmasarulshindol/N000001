# EP0「南の島の子犬」詳細プロット

> 渡真利海凪（11歳・沖縄離島出身）視点 / 泡児島入島前〜入島3ヶ月後 / 行為シーンなし

**EP0概要**: 本エピソードはヒロイン・渡真利海凪の視点で語られる前日譚である。沖縄の離島でおばぁと暮らしていた海凪が、祖母の借金をきっかけに泡児島へ渡り、島の生活に馴染んでいく過程を描く。天真爛漫な「島の子犬」の愛嬌と、借金返済のために子供の体を差し出す覚悟のコントラストを丁寧に描写する。ラストは主人公・朝比奈遼一と出会う直前で幕を閉じ、EP1へと直接繋がる。

**視点**: 海凪の一人称（「あたし」）。明るく跳ねるような語り口を基調とし、核心に触れる場面では文体が静かに沈む。沖縄方言を自然に混ぜる。

**文量目安**: 15,000〜20,000字（全6シーン合計）

## シーン構成（全6シーン）

### シーン1：おばぁの島（沖縄の離島の日常）

- **場所**: 沖縄本島からさらに離れた小さな離島。おばぁの平屋（赤瓦の古い民家）、港の防波堤、珊瑚の浜辺
- **時間帯**: 入島3ヶ月半前。4月の終わり、朝〜昼
- **登場人物**: 海凪、おばぁ（祖母）、ブローカー（声のみ／気配のみ）
- **出来事**:
  - 冒頭。海凪が朝から裸足で浜辺を走っている。珊瑚の欠片を拾い、ポケットに入れる。潮風が髪をぐちゃぐちゃにする。「おばぁー！ 今日めっちゃ天気いいさぁ！」。何気ない日常の朝
  - おばぁの家。赤瓦の平屋、台所からサーターアンダギーの匂い。おばぁが揚げたてを「ほれ、あちこーこーだから気をつけなさい」と差し出す。海凪がはふはふ食べる。「おばぁのサーターアンダギー、世界一！」
  - 島の描写。人口は百人に満たない。学校は一つだけの小さな分校。海凪のクラスメイトは3人。店は一軒だけの共同売店。でも海と空はどこまでも広い。海凪にとっては「せかいぜんぶ」がここにある
  - おばぁとの二人暮らし。父は海凪が4歳の時に離婚して行方不明。母はその後本島に出て、再婚して別の家庭を持っている。海凪を引き取ったのはおばぁだけ。海凪はおばぁが全世界
  - 午後。防波堤に座って海を眺める海凪。「あたし、おっきくなったら何になるんだろ」。まだ漠然とした将来。海女さんもいいし、お魚屋さんもいい。おばぁと一緒にサーターアンダギー屋さんを開くのが一番の夢
  - 夕方、おばぁの表情が曇る場面。台所で封筒を見つめているおばぁ。海凪が「おばぁ、どうしたの？」と聞くと、おばぁが慌てて封筒を隠して「なんでもないさぁ」と笑う。その笑顔がいつもと違う。海凪はまだ借金のことを知らない
  - 夜。蚊帳の中でおばぁと並んで寝る。おばぁが海凪の髪を梳きながら「みなぎは、おばぁの宝物だねぇ」と呟く。海凪が「おばぁもあたしの宝物さぁ」と返す。ヤモリの声。波の音。世界で一番安全な場所
  - ただし――おばぁの布団の端から覗く封筒。差出人は消費者金融。海凪は気づいていない
- **描写のポイント**: 海凪の「原点」となる風景を五感で刻み込む。おばぁとの温かく満ち足りた暮らし。小さな島の貧しいが豊かな生活。サーターアンダギーの匂い、珊瑚の浜、赤瓦の屋根。この幸せが失われることの予感を、おばぁの曇る表情と封筒で忍ばせる。海凪はまだ何も知らない子供として描く
- **感情の流れ**: 幸福 → 日常の満足感 → おばぁの異変にかすかな不安 → 夜の安心感（ただし読者には不穏さが見える）
- **文量目安**: 約3,000字

### シーン2：崩れる日常（ブローカーの訪問と別れ）

- **場所**: おばぁの家、港
- **時間帯**: シーン1から数週間後。5月中旬、昼〜夕方〜夜
- **登場人物**: 海凪、おばぁ、ブローカー（スーツ姿の中年男性）
- **出来事**:
  - おばぁの異変が明確になっていく数週間の描写。共同売店で買い物を減らすおばぁ。夕飯のおかずが一品減る。おばぁが夜中に起きて電話している声を海凪は聞いている。「……もう少しだけ待ってください」。海凪は布団の中で息を潜める
  - ある日、スーツ姿の男が島に来る。こんな小さな島にスーツの人間は来ない。海凪は分校の帰り道に、おばぁの家の前に停まった見慣れないレンタカーを見つける
  - 玄関の引き戸の向こうから声が聞こえる。「おばあさん、借金のことですが」「お孫さんなら、泡児島で働けます。年齢的にも条件に合う」「出産一時金だけで300万ですよ」。おばぁの声は震えている。「あの子はまだ11で……」「だからこそ需要があるんです」
  - 海凪は全部を理解したわけではない。だけど「おばぁが困っている」「あたしが何かすればおばぁが助かる」ということだけは分かった
  - 男が帰った後、海凪がおばぁに「おばぁ、あたしのこと話してたでしょ？」と切り出す。おばぁが「盗み聞きしたね」と力なく笑う。海凪が「借金って、いくら？」。おばぁが答えられない
  - 海凪が自分で調べたこと（学校の図書室のパソコンで「泡児島」を検索した）。キラキラしたリゾートの写真。「お仕事しながら学校に通える」というパンフレット。11歳の海凪が見つけた情報は、表向きの華やかな部分だけ。本質はまだ理解できていない。「南の島のきれいな合宿学校」というイメージ
  - おばぁに「あたし、行くよ。その島」と告げる海凪。おばぁが顔を歪める。「みなぎ……」。海凪が笑顔で「大丈夫さぁ！ 勉強もできるんでしょ？ お金ももらえるんでしょ？ おばぁの借金、あたしが返す！ なんくるないさ！」
  - おばぁが海凪を抱きしめて泣く。海凪の肩に落ちるおばぁの涙。海凪は泣かない。泣いたらおばぁがもっと辛くなるから
  - 数日後。おばぁが同意書にサインする場面。海凪はその場にいない。おばぁが一人で台所のテーブルに向かい、震える手でペンを持つ。海凪は外で防波堤に座り、水平線を見ている。「あの海の向こうに、あたしが行く島がある」
  - 出発の朝。おばぁが海凪にお守り袋を渡す。古い布で手縫いしたもの。中にはおばぁの髪の毛と、小さな珊瑚の欠片と、折り鶴が入っている。「これ持っていきなさい。おばぁがいつもそばにいるからね」
  - 港で見送り。フェリーが出る。おばぁが港に立って手を振っている。小さくなっていく。海凪は手を振り続ける。おばぁの姿が豆粒になったところで、初めて涙がぼろっとこぼれる。「……泣かないって、決めたのに」。お守り袋をぎゅっと握る
- **描写のポイント**: 日常が壊れていく過程を海凪の視点で描く。借金の詳細は海凪には分からない（＝読者にも全貌は明かさない）。ブローカーの言葉の断片だけが聞こえる恐怖。「あたしが行けばおばぁが助かる」という子供の決意の純粋さと残酷さ。お守り袋は以降の全エピソードを貫くアイテム。別れの場面はこのEPの最大の泣き所
- **感情の流れ**: 不安の蓄積 → ブローカーの声への恐怖 → 「あたしが行く」という決意 → おばぁの涙 → 出発の朝の覚悟 → 港での別れと涙
- **文量目安**: 約4,000字

### シーン3：泡児島上陸（新しい世界）

- **場所**: 沖縄本島の港 → 鹿児島ターミナル → 高速船の中 → 泡児島ターミナル → 居住区の寮
- **時間帯**: 5月下旬、朝〜夕方
- **登場人物**: 海凪、付き添いスタッフ（30代女性）、コンシェルジュ、先輩少女・結衣（ゆい、14歳）
- **出来事**:
  - 沖縄本島からフェリーで鹿児島へ。さらに専用ターミナルから泡児島行きの高速船に乗り継ぐ。長い移動。海凪は船に慣れているので船酔いはしないが、知らない景色が続くことへの緊張がある
  - 高速船の中。乗客はほとんどが大人の男性。スーツの人、カジュアルな人。海凪だけが明らかに子供。付き添いの女性スタッフが隣にいてくれるが、海凪は窓の外の海ばかり見ている。「沖縄の海と色が違う……」。海は同じ海のはずなのに、知らない海に見える
  - 水平線に泡児島が現れる。白い砂浜、ヤシの木、リゾートホテル。「すごい……きれいな島……！」。沖縄の離島とは全然違う。人工的で、ぴかぴかで、まるでテレビで見た外国のリゾートみたい
  - 上陸。ターミナルは空港のようにきれい。冷房が効いている。「沖縄みたいだけど、沖縄じゃない」。花は咲いているのに、匂いがどこか違う。海風はあるのに、おばぁの島の風じゃない
  - チェックイン。身体検査（健康診断）。海凪は検査着に着替えて、身長・体重・血液検査。「137cm、30kg」。看護師が「健康そのものね」と笑う。婦人科の検査もある。海凪にとって初めてで、恥ずかしくて顔が赤くなる
  - ルール説明。コンシェルジュ（30代女性、柔らかい物腰）が丁寧に説明する。「午前はお勉強、午後はお仕事。夜は……もう少し慣れてからね」。海凪は「お仕事って何するの？」と素朴に聞く。コンシェルジュが一瞬言葉を選ぶ。「お客様をおもてなしするの。海凪ちゃんの笑顔で、お客様を幸せにするお仕事よ」
  - 居住区の寮に案内される。4人部屋。すでに3人が使っている部屋。海凪のベッドは窓際。窓から海が見える。「……海、見えるんだ」。少しだけ安心する
  - 同室の先輩少女・結衣（ゆい、14歳）との出会い。結衣は長い黒髪のおっとりした子。「あなたが新しい子？ わぁ、ちっちゃくて可愛い〜！ 何歳？」「11！」「11!? いちばん下の子じゃん！ よろしくね、みーちゃんって呼んでいい？」。海凪が「うん！ ゆいねぇって呼んでいい？」。一瞬で仲良くなる
  - 夜。ベッドに入って、おばぁのお守り袋を枕の下に置く。窓から見える月は、おばぁの島で見た月と同じはずなのに、なぜか遠い。「おばぁ、あたし、ちゃんとやるからね」。目を閉じる。遠くで波の音
- **描写のポイント**: 海凪が「知っている海」から「知らない海」へ渡る旅。沖縄の自然な島と泡児島の人工的な美しさの対比。チェックイン・身体検査で「ここは普通の場所じゃない」という空気を匂わせる。結衣との出会いで安心感を。お守り袋を握りしめて眠る海凪の姿で、ホームシックと覚悟を描く
- **感情の流れ**: 長旅の緊張 → 島の美しさへの感嘆 → 身体検査での戸惑い → 結衣との出会いで笑顔 → 夜のベッドでの孤独と決意
- **文量目安**: 約3,000字

### シーン4：学園の日々と「おしごと」（島での生活）

- **場所**: 泡児島学園（教室）、寮、リゾートエリア・ビーチ
- **時間帯**: 入島から2週間〜1ヶ月半。6月中旬まで。朝〜夜
- **登場人物**: 海凪、結衣、他の少女たち（2〜3人）、教師、観光客（名前なしのモブ数名）
- **出来事**:
  - 泡児島学園の朝。制服はポロシャツとプリーツスカート。海凪は「制服かわいい！」とはしゃぐ。教室は小さくて、生徒は15人ほど。年齢がバラバラ（10歳〜15歳）だが、学力別に分かれている
  - 午前の授業。国語・算数・理科・英語は普通の学校と同じ。海凪は算数が苦手で「うぇ〜」と顔をしかめる。だが国語は得意。「あたし、作文だけは褒められるんだよ！」
  - 性教育の授業。週に2回。人体の仕組み、妊娠のメカニズム、避妊をしないことの意味。海凪は最初、顔が真っ赤になって教科書を顔で隠す。「えっ……これ……えっ……」。周りの先輩たちは慣れた様子。結衣が「最初はびっくりするよね」と笑う
  - マナーの授業。お客様への接し方、会話の仕方、身だしなみ。「笑顔が一番大事よ」と教師が言う。海凪が「笑顔ならまかせて！」と胸を張る。クラスが笑う
  - 昼食。食堂で少女たちが集まって食べる。海凪はあっという間に人気者になる。「みーちゃん、おかわり何杯目〜？」「3杯目！ だっておいしいもん！」。食いしん坊キャラが定着する
  - 午後。初めてのリゾートエリアへの出勤。水着に着替える。海凪の水着は白地にハイビスカス柄のビキニ。結衣が「みーちゃん、似合う〜！」と褒める。海凪はまだ胸がないので、ビキニのトップがぺたんとしている。「おっぱいないのバレバレじゃん……」。結衣が「それが逆にかわいいんだって！」と励ます
  - ビーチに出る。白い砂浜にターコイズブルーの海。「うわぁ……！」。沖縄の海とは違う色だけど、海は海。海凪の目が輝く。走り出す。裸足で砂浜を駆ける。水しぶきを上げて海に飛び込む
  - 観光客との最初の接客。ビーチバーで飲み物を運ぶ仕事から始まる。海凪は緊張しながらも「おにーさん、マンゴージュースどうぞ！」と笑顔で差し出す。客が「ありがとう、元気だね」と笑う。「ちゃんとできた……！」。小さな成功体験
  - 日が経つにつれ、海凪のビーチでの人気が急上昇する。沖縄育ちの海好きが本領を発揮し、「おにーさん、泳ごう！ あたし速いよ！」と客を海に連れ出す。素潜りも得意で、珊瑚の欠片を拾って客にプレゼントする。「あたしの島のおまじない！ 持ってるといいことあるよ！」
  - 常連客がつき始める。「海凪ちゃん、今日もビーチにいる？」と聞かれるようになる。結衣が「みーちゃん、もう人気者じゃん」と笑う
  - 夕方になると水着からミニ浴衣に着替える。膝上丈の白地にハイビスカス柄。帯は赤いリボン結び。下着はなし。結衣に「みーちゃん、パンツはかないの慣れた？」と聞かれ、海凪が「最初ちょっとはずかしかったけど、もう平気！ 風が気持ちいいさぁ」と笑う。ミニ浴衣でレストランやバーの接客をする。座ったりしゃがんだりすると際どいが、少女たちは慣れている
  - 寮での夜。少女たちのおしゃべりタイム。「今日のお客さん、こんなこと言ってた」「あの人やさしかった」。海凪は昼の接客の話で盛り上がるが、ナイトサービスの話になると口をつぐむ。自分はまだ経験がないから
  - おばぁへの手紙を書く海凪。便箋に丁寧な字で。「おばぁへ。南の島の合宿学校、たのしいよ！ 毎日海で泳いでるさぁ。ごはんもおいしいよ。友だちもいっぱいできた。勉強もがんばってるよ。おばぁ、元気にしてる？ サーターアンダギー食べたいなぁ。あたしは元気です。だいすき。みなぎより」。嘘は書いていない。でも、全部は書いていない
  - お守り袋を握って眠る。「おばぁ、あたしちゃんとやってるよ」
- **描写のポイント**: 泡児島での「日常」が形作られていく過程。学園生活の描写で「異常な場所での正常な青春」のギャップを出す。性教育の授業での戸惑いで、海凪がまだ子供であることを強調。ビーチでの接客が海凪に向いていることを自然に示す。おばぁへの手紙で「嘘は書いていないけど全部は書いていない」という11歳の精一杯の嘘を描く
- **感情の流れ**: 新生活への順応 → 授業での戸惑いと楽しさ → ビーチでの解放感 → 接客の成功体験 → 夜の寮での居場所 → おばぁへの思い → 手紙に書けないこと
- **文量目安**: 約3,500字

### シーン5：ナイトサービスへの覚悟（静かな夜）

- **場所**: 寮、ビーチ（夜の波打ち際）
- **時間帯**: 入島から約3ヶ月。8月上旬。夕方〜深夜
- **登場人物**: 海凪、結衣、他の少女（1〜2名）
- **出来事**:
  - 入島3ヶ月。海凪は昼の接客にはすっかり慣れた。ビーチの人気者で、「みーちゃんに会いたい」という指名も増えている。だが、ナイトサービスはまだ一度も経験していない。コンシェルジュが「海凪ちゃんはまだ早いかもしれないから」と配慮してくれている
  - しかし海凪は知っている。自分がナイトサービスをしなければ、本当のお金は稼げないことを。昼の接客の歩合だけでは、おばぁの借金返済には焼け石に水だ
  - ある夜。結衣がコテージから寮に戻ってくる。いつもおっとりしている結衣が、少し疲れた顔をしている。「おかえり、ゆいねぇ」と海凪が声をかける。結衣が「ただいま」と小さく笑う
  - 二人でベッドに並んで座る。海凪が聞く。「ゆいねぇ……ナイトサービスって、どんなかんじ？」。声が小さい。いつもの元気な海凪じゃない
  - 結衣が少し考えてから答える。「最初はね、すごく怖かった。痛いし、なんでこんなことしなきゃいけないのって泣いた」。海凪が息を呑む。「でもね、慣れるの。体も心も。それに、やさしい人だと……全然違うよ。抱きしめてくれる人もいるし、終わった後にお話してくれる人もいる」
  - 「やさしくない人は？」。海凪の声が震えている。結衣が海凪の手を握る。「そういう人もいる。でもね、セキュリティがいるから。本当に嫌な時はボタンひとつで助けが来る。だから、命に関わるようなことはない」
  - 「……痛い？」。海凪の一番の質問。結衣が「最初は痛い。嘘は言わない。でも、ずっとは続かないよ」。海凪がお守り袋をぎゅっと握る
  - 結衣が海凪の頭を撫でる。「みーちゃんは無理しなくていいんだよ」。海凪が首を振る。「ゆいねぇ、おばぁの借金、赤ちゃん産んだら300万もらえるんでしょ？ あと何回か産めば……おばぁの借金、ぜんぶ返せるの」。結衣が一瞬黙る。海凪の言葉が11歳の子供の口から出てきたことの重さ
  - 結衣が何も言えないでいると、海凪が笑顔を作る。「あたし、がんばるさぁ。おばぁのためだもん」。その笑顔が、ビーチで見せる笑顔とは全然違うことに、結衣は気づいている
  - 深夜。海凪は一人で寮を抜け出し、ビーチに行く。月明かりの波打ち際に座る。裸足の足先を波が洗う。いつもの元気な「みーちゃん」はいない。波の音だけが聞こえる
  - 海凪の内面描写。「怖い」という感情を初めてちゃんと認める。あたしはまだ11歳で、男の人とそういうことをしたことがなくて、赤ちゃんができるかもしれなくて、痛いかもしれなくて、怖い。でも。おばぁがあの日泣いていた。おばぁの震える手。おばぁの「ごめんね」という声。あたしがやらなかったら、おばぁはどうなるの
  - お守り袋を取り出す。月明かりに照らされた古い布。おばぁの手縫い。中の珊瑚の欠片を取り出して、掌に乗せる。おばぁの島の珊瑚。「……おばぁ。あたし、がんばるね。あたし、おばぁの宝物だもんね。宝物は、おばぁを守るんだよ」
  - 立ち上がる。水平線を見つめる。月が海に道を作っている。「……よし」。お守り袋を胸に押し当てて、寮に戻る。歩く足取りは、少しだけ大人になっている
- **描写のポイント**: EP0最大の感情的山場。結衣との会話で「ナイトサービス」の現実を間接的に伝える（直接描写はしない）。海凪の「怖い」という感情を丁寧に掘り下げる。夜のビーチで一人座る海凪の横顔――年齢不相応に静かで、美しくて、痛い。お守り袋と珊瑚の欠片は「おばぁとの絆」の象徴。ラストで「少しだけ大人になった足取り」で覚悟を示す
- **感情の流れ**: 日常の中の焦り → 結衣の帰還と問いかけ → 恐怖の直面 → 覚悟の萌芽 → 夜の波打ち際での独白 → お守り袋と約束 → 静かな決意
- **文量目安**: 約3,500字

### シーン6：明日の客（EP1への橋渡し）

- **場所**: 寮、学園の廊下、ビーチ
- **時間帯**: シーン5の翌日〜翌々日の朝。8月上旬
- **登場人物**: 海凪、コンシェルジュ、結衣
- **出来事**:
  - 翌朝。海凪はいつもの時間に起きる。昨夜の覚悟を胸に秘めたまま、いつもの元気な「みーちゃん」モードで朝を迎える。「おはよー！ 今日もいい天気！」。結衣が海凪をじっと見て、「……大丈夫？」。海凪が「なにが？ 元気元気！」と八重歯を見せて笑う。結衣は何も言わない
  - 午前。学園の授業。いつも通り。算数で「うぇ〜」、国語で「やったぁ」。日常は何も変わっていない。でも海凪の中で何かが変わっている
  - 昼休み。学園の廊下でコンシェルジュに呼び止められる。「海凪ちゃん、ちょっといいかしら」。柔らかい笑顔だけど、少し改まった空気
  - 相談室のような小部屋に通される。コンシェルジュが向かい合って座る。「海凪ちゃん、島に来て3ヶ月ね。昼のお仕事、すごく評判いいのよ。ビーチのお客様からの評価がとても高いの」。海凪が「えへへ、そうなの？」と照れる
  - 「それでね。明後日、東京からお客様が来るの。初めて島にいらっしゃる方で……」。コンシェルジュが少し間を置く。「海凪ちゃんの、初めてのナイトサービスのお相手にどうかしら、と思っているの」
  - 海凪の心臓が跳ねる。ドクン、と胸が鳴る。ついにこの日が来た。頭の中で昨夜の覚悟が蘇る。結衣の言葉、波打ち際のお守り袋、おばぁの泣き顔
  - 「どんな人……？」。声が少しかすれる。コンシェルジュが「34歳の男性よ。お仕事は文章を書くお仕事ですって。初来島の方だから、きっと緊張していると思うの。海凪ちゃんの明るさで、リラックスさせてあげてほしいな」
  - 海凪がコンシェルジュの目を見る。群青色の大きな瞳。数秒の沈黙。それから、小さく頷く。「……うん。あたし、やる」
  - コンシェルジュが「無理しなくていいのよ？」と確認する。海凪が首を振って、笑顔を見せる。今度はちゃんとした笑顔。「大丈夫。あたし、覚悟してたから」。11歳が「覚悟」という言葉を使う。コンシェルジュの目が一瞬揺れる
  - 午後。寮に戻って結衣に報告する。「ゆいねぇ、明後日……初めてのナイトサービス、決まった」。結衣が「そう……」と静かに答える。「どんな人？」「東京から来る人。34歳だって。文章書く仕事してるんだって」。結衣が「やさしそうだといいね」。海凪が「うん」
  - 結衣が海凪を抱きしめる。「みーちゃん。嫌だったら途中でやめていいからね。ボタン押せばすぐ人が来るから」。海凪が「うん。……ありがとう、ゆいねぇ」
  - 夕方。海凪は一人でビーチに出る。夕日が水平線に沈みかけている。空がオレンジとピンクに染まっている。波打ち際に立って、水平線を見つめる
  - 海凪の独白。「明後日、東京からおにーさんが来る。あたしの、はじめてのお客さん。……怖い。やっぱり怖い。でも、おばぁ。あたし、やるからね」
  - お守り袋を握る。風が吹いて、ポニーテールがなびく。夕日が海凪の小麦肌をオレンジに染めている。裸足の足元を波が洗う
  - 「おにーさん、どんな人かなぁ」。少しだけ、口元が緩む。怖いけど、ほんの少しだけ好奇心がある。知らない人に会うのは、いつだって少しだけワクワクする。だって、あたし、人見知りしないもん
  - 水平線の向こうに、まだ見ぬ「おにーさん」がいる。明後日、あの海の向こうからやってくる
  - ラストの一文。海凪が深呼吸して、振り返って、寮に向かって歩き出す。

  > あたしは波打ち際に背を向けて、歩き出した。サンダルは持ったまま、裸足で。砂がまだあったかい。明後日になったら、あたしは「おんなのこ」になる。よくわかんないけど、ゆいねぇはそう言ってた。怖いけど、なんくるないさ。おばぁのお守りがあるから。――だから、ぜったい、だいじょうぶ。

- **描写のポイント**: EP1への直接的な橋渡し。コンシェルジュの「34歳、文章を書く仕事」という情報で、読者にはそれが遼一であることが分かる。海凪はまだ知らない相手への不安と、かすかな好奇心を同時に見せる。ラストは「子犬」の明るさと覚悟が共存する一文で閉じる。美しい夕景の中に立つ海凪の姿を印象的に
- **感情の流れ**: いつも通りの朝 → コンシェルジュの言葉に心臓が跳ねる → 覚悟の表明 → 結衣との会話で安心と不安 → 夕暮れのビーチでの独白 → 怖さとかすかな好奇心の共存 → 歩き出す決意
- **文量目安**: 約3,000字

## 描写方針（EP0固有）

### 海凪の語り口
- 一人称「あたし」。語尾が跳ねる明るい文体が基調。「〜さぁ」「〜だよ！」「ねーねー！」が頻出する
- 沖縄方言を自然に混ぜる。「なんくるないさ」「あちこーこー（熱々）」「〜しましょうね（〜するからね、の意味）」
- 核心に触れる場面（借金、ナイトサービス、おばぁの涙）では文体が急に静かになる。短い文。体言止め。「……」の多用
- 子供の語彙で語る。難しい言葉は使わない。「消費者金融」は「お金を貸す会社」、「ブローカー」は「スーツの男の人」
- ただし感情の描写は鋭い。子供だからこそ、嘘のない感覚で世界を捉える

### おばぁの描写
- 海凪の全世界。おばぁの手、おばぁの声、おばぁの匂い（線香とサーターアンダギーの油）を繰り返し挿入する
- お守り袋はおばぁの分身として全シーンに登場させる
- おばぁへの手紙は「嘘は書いていないが全部は書いていない」という子供の精一杯の嘘として描く
- 直接登場はシーン1〜2のみだが、海凪の回想と手紙とお守り袋を通じて全編に存在感を保つ

### 島の二重描写
- **沖縄の離島（シーン1〜2）**: 自然の島。赤瓦、珊瑚、ヤモリ、潮の匂い。貧しいが温かい。色彩は土と海の色
- **泡児島（シーン3〜6）**: 人工の島。白い建物、整備されたビーチ、冷房。美しいが「匂いが違う」。色彩は人工的な白と青
- この対比を通じて「海凪が本当にいるべき場所」はどこかを読者に問いかける

### 覚悟のグラデーション
- シーン2「おばぁの借金、あたしが返す！」→ 子供の勢い。まだ何も分かっていない
- シーン3「あたし、ちゃんとやるからね」→ 不安の中の決意。まだ漠然としている
- シーン4「おばぁ、あたしちゃんとやってるよ」→ 昼の仕事に慣れた自信。でもまだ「本番」はこれから
- シーン5「がんばるね、おばぁ」→ ナイトサービスの現実を知った上での覚悟。怖さを認めた上での決意
- シーン6「あたし、やる」→ 具体的な相手が決まった。もう後戻りできない。最も重い覚悟

### EP1との接続
- シーン6のラストは、EP1シーン2の海凪初登場（ビーチバーで遼一にマンゴージュースを運ぶ場面）の直前で終わる
- EP1のシーン2で海凪が遼一に「おにーさん、はじめて？ めっちゃ顔カタいんだけど！」と声をかける場面を、海凪視点で事前に意味づけする
- EP0を読んだ読者は、EP1の海凪の笑顔の裏にある覚悟と恐怖を知っている。この「知っている」ことが読みの深度を変える
