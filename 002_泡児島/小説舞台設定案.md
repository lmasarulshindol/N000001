# 泡児島（あわじしま）

> 小説舞台設定案

---

## 1. 世界観概要

少子化が極限まで進行した近未来の日本。政府は人口維持のためにあらゆる施策を打ち出しているが、都市部の出生率はなおも低迷を続けている。その中で打ち出された最終手段が、人工島を建設して「子作り観光特区」とする構想であった。

### 時代設定
- **時代**: 近未来の日本（202X年〜203X年頃）
- **社会背景**: 出生率が0.5を下回り、国家存亡の危機。年金制度は事実上崩壊し、地方自治体の7割が消滅寸前
- **法改正**: 「次世代育成緊急特別措置法」が成立。特区として認定された地域では、10代前半からの妊娠・出産が合法化され、政府の支援対象となる
- **社会の受け止め**: 都市部では依然として抵抗感が強いが、「国が滅ぶよりマシ」という危機感から黙認する空気が広がりつつある

## 2. 泡児島の概要

### 地理・基本情報

| 項目 | 詳細 |
|------|------|
| 正式名称 | 泡児島（あわじしま） |
| 所在地 | 九州南方の海上に建設された人工島。鹿児島県に属する |
| 面積 | 約8km²（周囲約12km） |
| 人口 | 常住人口約500人（スタッフ・少女たち）＋観光客が常時200〜300人滞在 |
| アクセス | 鹿児島市内から高速船で約1時間。専用ターミナルあり |
| 気候 | 亜熱帯性気候。年間を通じて温暖で、冬でも10℃を下回ることは稀 |
| 主要産業 | 「子作り観光」が唯一にして最大の産業。宿泊・飲食・リゾート施設が島の経済を支える |

### 島名の由来
- **「泡」**: ソープランド（泡風呂）の「泡」。島の中核サービスが少女たちによる「おもてなし」であることを暗示する
- **「児」**: 児童の「児」。この島の少女たちと、ここで生まれる子供たちを意味する
- **表向きの解釈**: 政府広報では「泡のように新しい命が湧き出る島」という美しい言い換えで説明される。島の正式な観光パンフレットにも「子宝の島」というキャッチコピーが使われている

### 島の地形・エリア構成

| エリア | 特徴 |
|--------|------|
| **港・ターミナルエリア** | 高速船の発着場。島の玄関口。チェックイン施設・会員登録センターを兼ねる |
| **リゾートエリア** | 観光客用の宿泊施設が集まるメインエリア。ホテル・コテージ・レストラン・バー。南国リゾートの華やかさ |
| **ビーチエリア** | 白砂の人工ビーチ。少女たちが水着で接客するビーチバー、マリンアクティビティ。昼間の交流の場 |

| **管理エリア** | 島の運営本部。スタッフ施設、セキュリティ、監視システム |

## 3. 「島興しプログラム」の仕組み

### 制度の概要
政府が推進する「次世代育成緊急特別措置法」に基づき、泡児島が「子作り観光特区」として認定された。島に在住する10代前半の少女たちが、観光客として訪れる成人男性と子作りを行うプログラムが運営されている。

### プログラムの目的
1. **出生数の増加**: 特区内で生まれた子供は政府が全面支援。少女と子供の生活を保障する
2. **観光収益による経済効果**: 観光客の支払う料金が島の運営と少女たちの待遇に充てられる
3. **モデルケースの創出**: 成功すれば全国の過疎地域に展開する計画

### 少女側の制度

| 条件 | 詳細 |
|------|------|
| 年齢 | 10歳〜15歳の少女（保護者の同意が必要） |
| 健康状態 | 事前の総合健康診断をクリアすること |
| 教育 | 島内の「泡児島学園」への通学が義務（午前中は学業、午後以降が接客） |
| 経済支援 | 月額基本給15万円＋接客歩合（40%）。出産時には一時金300万円、育児手当が支給される |
| 福利厚生 | 寮・食事・制服支給。通信制高校の学費補助。カウンセリング。避妊管理は基本的にしない（妊娠が目的のため） |
| 出産・育児 | 島内の医療施設で出産。産後は「島の子」として島全体で育てる共同育児体制。少女が望めば自身で育てることも可能 |

### 観光客（男性）側の制度

| 条件 | 詳細 |
|------|------|
| 資格 | 成人男性（20歳以上）。身分証提示＋性病検査証明が必須 |
| 会員制 | 完全会員制。入会金30,000円。身元調査あり |
| 料金体系 | 宿泊パック: 1泊2日 50,000円〜 / 2泊3日 90,000円〜（少女の指名料・コース料金は別途） |
| コース | デイコース（ビーチ・食事デート）/ ナイトコース（コテージでの一晩）/ ロングステイ（3泊以上の専属） |
| ルール | 暴力・脅迫の厳禁。少女の意思を尊重すること。コンドーム使用は任意（妊娠が目的のため基本は生） |

### サービスの流れ

1. **チェックイン**: ターミナルで会員登録・健康証明提示。島内のルール説明
2. **昼の交流**: ビーチやレストランで少女たちと自然な形で交流。プロフィール帳から気になる子を見つけることもできる
3. **指名・マッチング**: 気に入った少女を指名。少女側の了承を得てペア成立。少女が拒否する権利は保障されている
4. **ナイトサービス**: コテージに二人で入り、一晩を過ごす。入浴→会話→行為という流れが基本
5. **翌朝**: コテージで朝食。少女が見送り。リピーターは同じ少女を指名し続けることも可能

### プログラムの運営体制

| 役職 | 担当 |
|------|------|
| **島長（しまおさ）** | 島の最高責任者。政府から派遣された官僚。表向きは「特区開発推進委員長」 |
| **コンシェルジュ** | 観光客の案内・マッチング調整。「ホテルのフロント」のような役割 |
| **カウンセラー** | 少女たちの心理ケア。定期面談。困っている子の相談窓口 |
| **医療スタッフ** | 島の診療所に常駐。妊娠・出産のケア、健康管理 |
| **セキュリティ** | 観光客のルール違反を監視。少女の安全確保 |
| **教育担当** | 泡児島学園の教師陣。通常の学業＋性教育カリキュラム |

## 4. 島の文化・空気感

### 島の雰囲気
- **リゾート側（観光客向け）**: 南国リゾートの華やかさ。白い砂浜、ヤシの木、カクテルバー。少女たちが水着やワンピースで明るく接客する「楽園」のイメージ
- **居住区側（少女たちの日常）**: 質素だが温かい共同生活。寮での友達との時間、学園での授業、お菓子作り。仕事の「表」と素の「裏」が切り替わる空間

### 少女たちの日常

| 時間 | 内容 |
|------|------|
| 7:00 | 起床。寮で朝食 |
| 8:00〜12:00 | 泡児島学園で授業（通常教科＋性教育・マナー） |
| 12:30 | 昼食 |
| 13:00〜14:00 | 自由時間 / 身支度 |
| 14:00〜 | リゾートエリアで接客開始。ビーチ・レストラン・アクティビティ |

| 20:00〜 | ナイトサービス（指名がある場合、コテージへ） |
| 翌7:00 | コテージからの帰還。寮に戻る |

### 少女たち同士の関係
- **寮の姉妹**: 共同生活で生まれる絆。悩みの共有、仕事の情報交換（「あの人は優しかった」「あの人は少し乱暴」）
- **先輩後輩**: 経験のある先輩が新人に接客のコツや心構えを教える
- **妊娠した子への祝福**: 妊娠は「おめでたいこと」として島全体で祝う文化。妊娠した少女は接客から外れ、出産まで大切にされる
- **朝の帰還**: ナイトサービスから戻った少女を、起きている友達が「おかえり」と迎える。その日の出来事を話したり、黙って寄り添ったり

| 10月 | 収穫祭 | 島の畑で育てたフルーツの収穫祭。少女たちの手作り料理が振る舞われる |

| 通年 | 安産祈願 | 妊娠した少女を島全体で祝福する儀式 |

## 5. 社会的背景の詳細

### 少子化の深刻さ
- 日本全体の出生数が年間30万人を割り込み（現実の2024年時点で約70万人）
- 経済的理由・価値観の多様化・晩婚化が複合的に作用
- 「子供を産むことは個人の自由」という価値観と「国家の存続」の衝突

### 法整備の経緯
1. **第一段階**: 結婚年齢の引き下げ（16歳→14歳）
2. **第二段階**: 「次世代育成緊急特別措置法」の成立。特区における10代前半の出産を合法化
3. **第三段階**: 泡児島の建設。政府主導の人工島プロジェクトとして、「子作り観光特区」の実験場を創出
4. **現在**: 泡児島の「成功」を受け、全国で3つの追加特区が計画中

### 世論
- **賛成派**: 「このままでは国が滅ぶ。非常時には非常手段が必要」「少女たちには手厚い待遇が保障されている」
- **反対派**: 「児童買春の合法化だ」「結局は大人の欲望を正当化しているだけ」「少子化対策は経済支援が先」
- **中間派**: 「理屈は分かるが感情的に受け入れがたい」が最も多い
- **利用者の意識**: 罪悪感を持ちながら来島する者、「国のため」と自分を正当化する者、純粋に少女との出会いを求める者など様々

## 6. ストーリー構成

### 基本コンセプト
泡児島を訪れた主人公（成人男性）が、島の少女と出会い、「子作り観光」という制度の中で人間同士の繋がりが生まれる物語。楽園のようなリゾートの裏にある少女たちの現実、観光客の罪悪感と欲望、そして予想外の感情の交錯を描く。

### 物語の軸
- **楽園と現実のコントラスト**: 南国リゾートの美しさと、「少女と子作りする」という制度の間に生まれるギャップ
- **少女たちの事情**: なぜこの島にいるのか。経済的理由、家庭の事情、自発的な意思。それぞれの背景
- **観光客の葛藤**: 罪悪感と欲望。少女のプロとしての笑顔と素の表情のギャップに翻弄される
- **「仕事」を超えた感情**: サービスとして始まった関係の中に、本物の感情が芽生える瞬間

### 描写のポイント
- 南国人工島の美しい自然描写（海、空、星、花）を物語の基調に
- 少女の「接客モード」と「素の顔」の切り替えを丁寧に
- 行為そのものだけでなく、「なぜこの子がここにいるのか」を丁寧に掘り下げる
- 島の閉鎖性と楽園のイメージの二重構造

## 7. 性行為シーンの描写方針

### 基本設計思想
- 行為シーンは「子作り観光」という制度的枠組みの中で描かれるが、機械的な描写にはしない
- 少女のプロとしての接客スキルと、ふと素が出る瞬間の交錯
- 南国リゾートの開放的な空気感と、行為に至る繊細な心理のコントラストを大切にする
- コテージという密室で、観光客と少女が初めて二人きりになる緊張感

### 描写の三層構造

**① 導入（行為に至るまで）**
- コテージに入る二人。観光客の緊張と期待。少女の営業スマイルの裏にある緊張
- 入浴シーン。少女が体を洗ってくれる。肌と肌の距離が縮まる
- 会話を通じて少女の人間性が見えてくる。「仕事」のはずが、人と人になる瞬間

**② 行為本体**
- 身体描写: 日焼けした肌、汗の質感、吐息、潮の匂いと混じる体温
- 少女の反応: プロとしての振る舞いと、体が正直に反応してしまう瞬間の交錯
- 生での挿入・中出しが基本（妊娠が目的のため）。その行為が持つ重みの描写
- 環境描写: コテージの窓から見える月と海、波の音、南国の夜風

**③ 余韻（行為後）**
- 事後の二人の空気感。少女が淹れてくれるお茶、窓辺での穏やかな会話
- 「これは仕事だ」と割り切ろうとする観光客と、割り切れない何か
- 翌朝、笑顔で見送る少女。営業スマイルなのか本心なのか分からない曖昧さ
- 島を離れるフェリーの上で、観光客が反芻する少女の笑顔

### シーンごとの文量目安
- 導入: 全体の **20〜25%**
- 行為本体: 全体の **50〜60%**
- 余韻: 全体の **20〜25%**
- 1回の行為シーン全体で **3,000〜5,000字** を目安
