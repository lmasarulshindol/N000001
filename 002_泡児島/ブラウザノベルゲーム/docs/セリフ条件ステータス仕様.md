# セリフ条件ステータス仕様

セリフバリエーションを切り替えるための **入力パラメータ一覧** と **派生ステータス** の定義書。
台本データ（`intimate-scenes-data.js` / `intimate-contextual-lines.js` / `character-intimacy-profiles.js`）の条件分岐は、ここで定義したステータスのみを参照する。

---

## 1. パラメータ階層

```
┌───────────────────────────────────────────────────────┐
│ A. キャラ静的属性   ← character-data.js / profile      │
│ B. 関係性ステータス ← gameState.* (進行で変化)         │
│ C. シーン進行       ← intimateSession.* (シーン中変化) │
│ D. 環境ステータス   ← gameState.* (場所・時間)         │
│ E. 派生ステータス   ← A〜D から計算 (純関数)           │
└───────────────────────────────────────────────────────┘
                           ↓
              ╔═══════════════════════════╗
              ║   セリフ選択ロジック       ║
              ║  (条件マッチング → プール) ║
              ╚═══════════════════════════╝
```

## 6. E. 派生ステータス（純関数で計算）

A〜D から計算される **読み取り専用ステータス**。  
`js/dialogue-state-derivers.js` に純関数として実装する。

### 6.1 `relationshipPhase(affection)`

| 段階 | 範囲 | ラベル | 描写 |
| --- | --- | --- | --- |
| `stranger` | 0〜29 | 知人 | 距離感あり・敬語混じり |
| `acquaintance` | 30〜49 | 友達 | 慣れてきた・気軽な会話 |
| `friend` | 50〜69 | 親しい | 信頼・甘えあり |
| `intimate` | 70〜89 | 恋人 | 身体的距離が近い |
| `devoted` | 90〜100 | 心の絆 | 全面降伏・将来の言及 |

### 6.2 `arousalTier(pleasure)`

| 段階 | 範囲 | 描写 |
| --- | --- | --- |
| `calm` | 0〜24 | 平静・前戯入り |
| `warm` | 25〜54 | 高ぶり始め・濡れ始める |
| `hot` | 55〜84 | 余裕がない・フィニッシュ解禁 |
| `burning` | 85〜100 | 限界寸前・自制不能 |

### 6.3 `scenePrivacy(spotId, timeOfDay)`

| 結果 | 説明 | 例 |
| --- | --- | --- |
| `public` | 人目が多い | port × morning/afternoon |
| `semi_private` | 死角・夕方等 | observatory × evening |
| `private` | 完全に二人きり | inn × night/midnight |

### 6.4 `repeatFamiliarity(actionCount)`

| 段階 | 回数 | 描写 |
| --- | --- | --- |
| `first_time` | 0回（初回） | 初々しい |
| `second_time` | 1回（2回目） | 慣れ始め |
| `third_time` | 2回 | 馴染んできた |
| `repeated` | 3回以上 | 境界が溶ける |

### 6.5 `shameLevel(experience, routeProgress, actionCount)`

| 段階 | 条件 | 描写 |
| --- | --- | --- |
| `high` | virgin かつ mainDone なし | 強い羞恥 |
| `medium` | mainDone あり または non-virgin | 戸惑い |
| `low` | creampie 経験あり または 同行動5回+ | 開放的 |

### 6.6 `trustTier(affection, routeProgress)`

| 段階 | 条件 | 用途 |
| --- | --- | --- |
| `wary` | affection<30 かつ main 未経験 | 重い話題禁止 |
| `open` | affection≥30 または main 経験 | 通常会話可 |
| `dedicated` | affection≥70 かつ creampie | 結婚・将来言及可 |

### 6.7 `timeMood(timeOfDay)`

| 値 | 雰囲気 |
| --- | --- |
| `morning` | 清涼・希望 |
| `afternoon` | 活動的・健全 |
| `evening` | センチメンタル |
| `night` | 大胆・解放 |
| `midnight` | 秘め事・濃密 |

### 6.8 `isPreferredSpot(character, spotId)`

`character.stats.preferredSpots.includes(spotId)` → `true` で個性的台詞、`false` で汎用台詞。

### 6.9 `dateContext(gameState)`

`gameState.flags.activeDate` が現在のシーンと一致するか → `'date'` / `'normal'`。

### 6.10 `postFinishMood(finishedAction, profile, affection)`

着弾先がキャラの `preferredFinish` と一致するか / `forbidden` に該当するかで `satisfied` / `mixed` / `disappointed`。

### 6.11 `riskLevel(privacy, timeOfDay)`

`scenePrivacy` × `timeOfDay` から `safe` / `risky` / `forbidden` を導出（背徳感演出用）。

## 7. セリフ条件マトリクス例

「キス」のセリフを選ぶ場合の優先度：

```
1. キャラID (char.id)
2. profile.flavorLines.kiss が存在
   → あれば最優先 (variant 0)
3. relationshipPhase
   → stranger/acquaintance → 戸惑い系
   → friend/intimate/devoted → 甘え系
4. arousalTier
   → calm → ソフト
   → hot/burning → 強め
5. repeatFamiliarity (kissカウンタ)
   → first_time → 初々しい
   → repeated → 求めて止まらない
6. scenePrivacy
   → public → 「誰かに見られちゃう……」
   → private → 「もう、誰もいないから……」
```

## 10. 派生関数 API（`js/dialogue-state-derivers.js`）

```javascript
// 純関数。副作用なし。
DialogueStateDerivers.relationshipPhase(affection);  // → 'stranger' | ... | 'devoted'
DialogueStateDerivers.arousalTier(pleasure);          // → 'calm' | 'warm' | 'hot' | 'burning'
DialogueStateDerivers.scenePrivacy(spotId, time);     // → 'public' | 'semi_private' | 'private'
DialogueStateDerivers.repeatFamiliarity(count);       // → 'first_time' | 'second_time' | 'third_time' | 'repeated'
DialogueStateDerivers.shameLevel(experience, routeProgress, actionCount);
DialogueStateDerivers.trustTier(affection, routeProgress);
DialogueStateDerivers.timeMood(timeOfDay);
DialogueStateDerivers.isPreferredSpot(character, spotId);
DialogueStateDerivers.dateContext(gameState);
DialogueStateDerivers.postFinishMood(finishedAction, profile, affection);
DialogueStateDerivers.riskLevel(spotId, timeOfDay);

// 一括取得（セリフ条件分岐用の入力束を作るヘルパー）
DialogueStateDerivers.snapshot({ character, profile, gameState, session });
// → { phase, arousal, privacy, familiarity, shame, trust, timeMood, isPreferred, date, risk, ... }
```
