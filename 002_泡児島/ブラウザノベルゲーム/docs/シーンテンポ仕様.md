# シーンテンポ仕様 v1.0

**対象**: 泡児島ノベルゲーム  
**目的**: 「選択肢が多すぎて没入感が薄れる」問題を解消するため、選択肢数とテキスト量のバランスを明文化し、実装・台本制作の基準とする。  
**位置付け**: `docs/画面設計書.md`・`docs/UI操作感仕様.md` の補足。本書は **演出テンポと文章量の最低/最大基準** を定義する。

---

## 用語

| 用語 | 定義 |
|------|------|
| ビート | `uiManager.showMessage` 一回分（1クリックで進む単位）。地の文／台詞／システム表記いずれも1ビート扱い |
| アクション | プレイヤーが選択肢から選ぶ1つの行動。1アクション = 複数ビート + 効果 |
| 階層化 | 大カテゴリ→具体プレイの2段選択（例: 「触れる」→ 胸/太もも/耳） |
| 特殊アクション | キャラ別 `specialAction`、または「口でしてもらう」「胸で挟む」など演出重視の専用プレイ |

## R1. 選択肢数の上限

| 場面 | **目標** | **上限** | 構成例 |
|------|----------|----------|--------|
| 前戯（foreplay） | **3〜4択** | 5 | キャラの好み2 + 性感帯特化1 + 「進める／止める」1 |
| 挿入後（piston） | **3択** | 4 | テンポ1 + 体位/キス1 + 「フィニッシュ／止める」1 |
| 余韻（aftercare） | **2〜3択** | 4 | 抱く / 言葉をかける / 終わる |
| 会話フェーズ | **3〜4択** | 5 | 既存通り（好感度上昇話題3 + 戻る1） |
| ハブ | **3〜4択** | 5 | 既存通り |
| キャラ選択 | **3+1** | 5 | 3人ページング + 「他の子」+「マップ」 |

### R1-1. 5択を超えるなら**必ず階層化**

```
（NG）一画面に8個並べる
（OK）「触れる ▼」→ サブメニューで「胸 / 太もも / 耳」
```

階層化したサブメニューも **3〜4択** を守る。

### R1-2. 「メニューに戻る」「やめる」は **末尾固定**

中央配置の中で、最下段に常時固定。プレイヤーが視線で探さずに済むようにする。

### R1-3. 絶頂直前は **2択に縮約**

「もう少し堪える / フィニッシュする」のみ。没入を切らない。
判定: `canClimax(session)` が `true` かつ `pleasure >= 80`。

## R2. アクション1回あたりのビート数

| アクション種別 | **ビート数** | 想定秒数 |
|----------------|-------------|----------|
| 通常アクション | **4〜6** | 15〜30秒 |
| 特殊アクション（性感帯/弱点） | **6〜8** | 30〜45秒 |
| 進行アクション（挿入/絶頂/体位変更） | **3〜5** | 10〜20秒 |
| システム表記のみ | 1 | 即時 |

### R2-1. 推奨ビート構成（通常アクション）

| 順 | 役割 | 例 | 文字目安 |
|----|------|----|----------|
| 1 | **A. 起こす描写**（地の文） | プレイヤー側の動作・空気感 | 30〜60字 |
| 2 | **B. 彼女のリアクション**（台詞） | 性格・性感帯反映の反応 | 30〜80字 |
| 3 | **C. 続く展開**（地の文） | 動きの加速・体の変化 | 40〜80字 |
| 4 | **D. 余韻台詞**（台詞） | 短い1行で締め | 20〜50字 |
| 5 | **E. システム表記** | `（快感 +XX / 好感度 +XX / 性感帯ヒット）` | 1行 |

### R2-2. 特殊アクションは**前置きと余韻**を追加

通常の A〜E に加えて:

- 前: **0. プレイヤーの心の声 / 期待感**（地の文 30〜80字）
- 後: **F. キャラの本音 / 弱さ**（台詞 50〜100字）

## R3. 文章量バランス

| 指標 | 目安 |
|------|------|
| 1ビートあたり最大文字数 | **120字** |
| 1アクション総文字数 | **150〜350字**（通常）／ **300〜500字**（特殊） |
| 台詞:地の文 比率 | **6:4**（彼女の心理を強調） |
| 同じ語尾の連続禁止 | 「……」で終わる行は **2ビート連続まで** |

### R3-1. **回数連動の差分台詞**

同じアクション（例: 「ゆっくり」）でも、`pistonCount % 3` で台詞を切り替える:

| 回数 | 例 |
|------|----|
| 1回目 | 「ん……ゆっくりで、いい……」 |
| 2回目 | 「はぁ……体、覚えてきた……」 |
| 3回目 | 「もう、止まらない、よ……」 |

### R3-2. **快感メーター連動**

`pleasure` 値で台詞のトーンを切替（30未満/30〜60/60〜100）。

## R4. 特殊体位・特殊プレイ

「口でしてもらう」「胸で挟む」「後ろから」など、**前戯とも本番ピストンとも異なるプレイ** は専用ステージ化する。

### R4-1. 出現条件

| プレイ | 条件 |
|--------|------|
| 口でしてもらう | 好感度 30+ かつ 性感帯=唇/lips ≥1.2 のキャラ |
| 胸で挟む | 好感度 50+ かつ 体格条件（age ≥ 13 など） |
| 後ろから | 好感度 40+ かつ position 解放 |
| プレイヤー口で愛撫 | 好感度 60+（信頼関係の証明） |

未解放のプレイは**選択肢に出さない**（UIをすっきり保つ）。

### R4-2. 専用ステージのビート構成

```
[前置き：プレイヤーの心の声]      ← 30〜80字
[A. 起こす描写]                  ← 50〜100字
[B. 彼女のリアクション]           ← 80〜120字
[C. 続く展開]                    ← 60〜100字
[B'. 反復リアクション]            ← 50〜80字
[D. 余韻台詞]                    ← 40〜80字
[F. キャラの本音]                ← 80〜120字
[E. システム表記]                ← 1行
```

→ 合計 **7〜8ビート / 350〜500字**

### R4-3. 「口でしてもらう」標準台本（さくら例）

```
[前置き]
あなたは静かにさくらの頬を撫で、視線で「お願い」と告げた。
さくらは小さく頷き、ゆっくりと身を屈めていく――。

[A]
「……っ、これ、初めて、なの……
でも、あなたが、よろこんで、くれるなら……」

[B]
温かい吐息が先端に触れた瞬間、背筋が痺れる。
ためらう指、震える唇——その全部が、こちらの理性を削っていく。

[C]
「……ん、ぅ……むぅ、ふ……」

[B']
ぎこちない動きの中に、健気さがにじむ。
やがてリズムを覚えはじめ、彼女自身も少しずつ熱を帯びていった。

[D]
「……上手、できてる……？」

[F]
（彼女の頬が紅潮している。プライドを脇に置いて、あなたのために尽くす――
その姿に、こちらの胸がぎゅっと締めつけられた）

[E]
（快感 +28 / 好感度 +8 / 性感帯ヒット：唇）
```

選択肢は **その後** に表示:

```
1. もう少し、続けてもらう
2. 「上手だよ」と褒めて頭を撫でる   ← kink=praise 特殊
3. もう、挿入したい
4. やめる
```

## R5. プレイ階層（カテゴリ）

選択肢を5以下に保つため、以下のカテゴリで階層化する。

### R5-1. 前戯カテゴリ

| 大カテゴリ | サブメニュー（例） |
|-----------|------------------|
| **触れる** | 性感帯1（耳など）/ 性感帯2（首筋）/ 全身を撫でる |
| **キスする** | 唇 / 首筋 / 耳 |
| **特別なこと** | 口でしてもらう / 胸で挟む / プレイヤーが愛撫 |
| **進める** | 挿入する / 別のことをする / メニューに戻る |

→ **第1階層 4択**、各サブメニューも **3〜4択**。

### R5-2. ピストンカテゴリ

| 大カテゴリ | サブメニュー |
|-----------|-------------|
| **動きを変える** | ゆっくり / 激しく / 奥まで |
| **触れながら** | キスしながら / 胸を揉みながら / 耳を舐めながら |
| **体位** | 正常位 / 騎乗位 / 後背位 |
| **フィニッシュ** | 中で / 顔に / 胸に / 堪える |

## R6. 没入を守るUIフラグ

| フラグ | 効果 |
|--------|------|
| `immersiveMode` | 特殊アクション中は HUD（好感度バーなど）を半透明化 |
| `compactChoices` | 絶頂直前など2択時はボタンを大きくして「真ん中ドン」 |
| `autoBeatHint` | 連続ビート中はヒントを「クリック / Enter で進む」のままで点滅維持 |
