# 泡児島ノベルゲーム

近未来の「子作り観光特区」泡児島を舞台にしたブラウザノベルゲーム。
10人以上のキャラクターとの交流を通じて、島での体験を楽しめます。

## 特徴

- **豊富なキャラクター**: 10人以上の個性的な少女キャラクター
- **島内探索システム**: 9つの異なるスポットを自由に移動
- **段階的なフェーズ管理**: 会話→スキンシップ→前戯→本番→アフターケアの5段階
- **完全ローカル動作**: インターネット接続不要
- **レスポンシブ対応**: PC、タブレット、スマートフォンに対応
- **セーブシステム**: 10スロット + オートセーブ + クイックセーブ

## 動作環境

- **対応ブラウザ**: Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **画面解像度**: 320px以上（スマートフォン対応）
- **ストレージ**: 約100MB（画像・音声含む）

## インストール・起動方法

1. このフォルダをローカルに保存
2. `index.html` をブラウザで開く
3. ゲーム開始

## ゲームの流れ

### 1. 島内移動
- マップ画面から好きなスポットを選択
- 各スポットで異なるキャラクターと遭遇

### 2. キャラクター交流
- **会話フェーズ**: キャラクターとの会話で好感度アップ
- **スキンシップフェーズ**: 軽い身体接触（好感度30以上）
- **前戯フェーズ**: より親密な関係（好感度60以上）
- **本番フェーズ**: メインイベント（好感度80以上）
- **アフターケアフェーズ**: 事後の会話

### 3. 好感度システム
- 選択肢によって各キャラクターの好感度が変動
- 好感度が高いほど進行できるフェーズが増加
- 100段階で管理

## 操作方法

### 基本操作
- **マウス/タッチ**: メイン操作
- **クリック/タップ**: テキスト進行、選択肢選択
- **Enter/スペース**: テキスト進行
- **Esc**: メニュー表示
- **Ctrl+S**: クイックセーブ
- **Ctrl+L**: クイックロード

### 設定可能項目
- **テキスト速度**: 3段階（遅い/普通/速い）
- **コンテンツレベル**: 3段階（ソフト/ノーマル/アダルト）
- **BGM音量**: 0-100%
- **効果音音量**: 0-100%
- **オートモード速度**: 5段階

## セーブデータ

### 保存場所
- ローカルストレージ（ブラウザ内）
- 容量: 約1MB以下

### セーブの種類
1. **通常セーブ**: 10スロット、手動保存
2. **オートセーブ**: シーン変更時に自動保存
3. **クイックセーブ**: Ctrl+Sで即座に保存

### データ管理
- **エクスポート**: バックアップファイルとして保存
- **インポート**: バックアップファイルから復元
- **データ削除**: 全データの一括削除

## トラブルシューティング

### よくある問題

1. **ゲームが起動しない**
   - ブラウザのJavaScriptが有効か確認
   - ローカルファイルのアクセス許可を確認
   - 対応ブラウザを使用

2. **画像が表示されない**
   - ファイルパスが正しいか確認
   - 画像ファイルが存在するか確認
   - ブラウザのキャッシュをクリア

3. **セーブできない**
   - ブラウザのローカルストレージが有効か確認
   - 容量制限に達していないか確認
   - プライベートブラウジング無効化

4. **音が出ない**
   - ブラウザの音量設定を確認
   - ゲーム内音量設定を確認
   - 音声ファイルの形式を確認

### サポート情報
- 推奨ブラウザ環境での動作を保証
- ファイアウォールやセキュリティソフトによる制限に注意
- 古いブラウザでは一部機能が制限される場合があります
