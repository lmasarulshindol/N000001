# 第7章「灼熱の独占」プロット

> リーナ回・嫉妬と独占欲。口での奉仕→騎乗位→対面座位

## コンセプト

セレナとミアが膣内射精に移行して急成長する中、「アタシが最初だったのに」という嫉妬と焦燥を抱えたリーナが、ユーマを自分だけのものにしようと自ら主導権を握る。口での奉仕から始まり、騎乗位、対面座位と、リーナが攻める側に回る構成。強がりの鎧が行為の中で溶けていき、嫉妬の奥にある本音——「ユーマに一番大切にされたい」が露わになる。

## 時系列

- 第6章から数日後
- リーナとユーマの膣内射精は既に複数回経験済み（第1章で口、第3章頃に膣内に移行）
- セレナ・ミアの急成長を目の当たりにした直後

### シーン1: 燻る炎（約2,000字）

**場所**: 訓練場
**時間帯**: 午前
**登場人物**: ユーマ、リーナ、セレナ、ミア

**出来事**:
- 訓練場での合同訓練。セレナの回復魔法が劇的に向上しているのを全員が目撃
- ミアの壁が1分以上持続。教官が驚嘆
- リーナの火球も強力だが、セレナとミアの「伸び率」が目立つ
- リーナの訓練に集中できていない。火球の精度が普段より荒い
- ユーマがリーナの様子に気づく。琥珀色の瞳の奥に、いつもと違う光がある
- 訓練後、リーナが一人で的を撃ち続けている。火球が的を通り越して壁に当たる。荒い

**感情の流れ**:
- リーナ：焦燥。自分だって強くなっている。でもセレナとミアの伸びが目立つ。「アタシが最初だったのに」
- ユーマ：リーナの不調に気づく。声をかけるべきか迷う

**描写ポイント**:
- リーナの火球の精度が落ちていることで内面の乱れを可視化
- セレナとミアの成長を見るリーナの横顔。唇を噛んでいる
- 的を外すリーナ。普段なら絶対に外さない

### シーン2: 嫉妬の正体（約2,500字）

**場所**: 食堂→廊下
**時間帯**: 昼
**登場人物**: ユーマ、リーナ、セレナ、ミア、ヴィオラ

**出来事**:
- 食堂でのいつもの食事。4人のヒロインが揃う
- セレナが「ユーマさん、今日の訓練を見てくれましたか」と嬉しそうに話しかける
- ミアが「おにーちゃん、壁1分持ったよ！ 褒めて褒めて！」と飛びつく
- ヴィオラが「次の供給スケジュールだが」と効率的な提案をする
- リーナだけが黙っている。スープを掻き回している。食べていない
- ユーマがリーナに「食べないのか」と声をかける
- リーナ「……別に。食欲ないだけ」— 素っ気ない。いつもの勝ち気さがない
- 食事後、廊下でリーナがユーマの袖を掴む。「……ちょっと来て」
- 声が低い。静かだ。怒っているのか、泣きそうなのか、わからない

**感情の流れ**:
- リーナ：他の3人がユーマに甘える光景が辛い。自分も甘えたいのに、できない
- 嫉妬の正体は「ユーマを取られる恐怖」ではなく「自分が特別じゃなくなる恐怖」

**描写ポイント**:
- 4人の中でリーナだけが沈黙していることの異常さ
- スープを掻き回す手。食べない。リーナが食欲をなくすのは珍しい
- 袖を掴む手の力。小さいが、強い

### シーン3: 「今夜、来て」（約2,000字）

**場所**: 廊下の隅→リーナの部屋の前
**時間帯**: 昼食後
**登場人物**: ユーマ、リーナ

**出来事**:
- リーナがユーマを廊下の隅に連れ出す
- 壁に背を預けて、腕を組む。いつものポーズ。だが目が合わない
- 「……今夜。アタシの部屋に来て」
- ユーマ「供給の予定は明後日だが」
- リーナ「供給じゃない。……いや、供給でもいいけど。とにかく来て」
- 言葉が整理できていない。リーナらしくない
- ユーマが黙って見ている。リーナの琥珀色の瞳が揺れている
- 「……いいから。来て。約束して」
- ユーマ「わかった」
- リーナが踵を返す。赤銅色のポニーテールが揺れる。足音が速い。逃げるように

**感情の流れ**:
- リーナ：自分でも何がしたいのかわからない。ただ、ユーマと二人きりになりたい
- 言葉にできない感情。「供給じゃない」と言いかけて「供給でもいい」と修正する混乱

**描写ポイント**:
- リーナの言葉の不安定さ。普段の切れ味がない
- 「約束して」— リーナがユーマに「約束」を求めるのは珍しい
- 逃げるように去る背中。赤銅色の髪が揺れる

### シーン4: リーナの部屋（約2,500字）

**場所**: リーナの部屋
**時間帯**: 夜
**登場人物**: ユーマ、リーナ

**出来事**:
- ユーマがリーナの部屋を訪ねる。ノックすると「入って」と短い返事
- リーナの部屋。個室とは違う。私物がある。訓練用の手袋。火属性の教本。窓辺に小さな花
- リーナがベッドの端に座っている。髪を下ろしている（珍しい）。いつもの訓練着ではなく、薄手のワンピース
- 「……座んなよ」ベッドの隣を叩く
- ユーマが隣に座る。沈黙。リーナが口を開くのを待つ
- 「……セレナもミアも、強くなったわね」
- 「ああ」
- 「アタシのおかげでしょ。アタシが最初にアンタを見つけたんだから」
- 強がり。でも声が震えている
- 「……ねえ。アタシ、まだ一番？」
- その一言に、全てが詰まっていた。嫉妬。不安。独占欲。そして——愛情
- ユーマの返答。「お前が最初だ。それは変わらない」
- リーナの目が潤む。でも泣かない。泣かないのがリーナだ
- 「……じゃあ、証明して」
- リーナが立ち上がる。ユーマの前に膝をつく。自分から

**感情の流れ**:
- リーナ：不安の吐露→ユーマの言葉で少し安心→でも足りない→「証明して」
- 「証明」＝行為で自分が特別だと確認したい

**描写ポイント**:
- リーナの部屋という私的空間。個室（公的な供給場所）との対比
- 髪を下ろしたリーナ。鎧を脱いだ姿
- 「まだ一番？」の声の震え。リーナの最大の弱さ
- 自分から膝をつく＝口での奉仕を自分から始める

### シーン5: 口での奉仕 — リーナが攻める（約3,500字）

**場所**: リーナの部屋
**時間帯**: 夜
**登場人物**: ユーマ、リーナ

**出来事**:
- リーナが自分からユーマの前に膝をつく
- 「……黙って受けてなさい。アタシがやるから」— 強がりの声。でも手が震えている
- 口での奉仕。リーナは経験がある。初回のぎこちなさはない
- だが今日は違う。いつもより激しい。いつもより深い。感情がこもっている
- 嫉妬を行為にぶつけている。「セレナにもミアにも、こんなことできないでしょ」
- 上目遣い。琥珀色の瞳が挑発的に俺を見上げる。でも瞳の奥は不安で揺れている
- リーナの技術の描写。慣れている。でも今日は「上手にやろう」ではなく「離さない」
- ユーマの反応を見るリーナ。俺が感じているのを確認して、少しだけ安心する
- 射精。リーナが全て飲み下す。口元を拭わない。「……まだ足りない」
- 赤い発光が微かに灯る。だがリーナは満足していない
- 「次はアタシが上」— リーナが立ち上がり、ユーマをベッドに押し倒す

**描写ポイント**:
- 口での奉仕を「攻め」として描く。リーナが主導権を握っている
- 嫉妬が行為の激しさに直結。いつもより深く、いつもより強く
- 上目遣いの琥珀色の瞳。挑発と不安の混在
- 「まだ足りない」— 魔力ではなく、感情が足りない

### シーン6: 騎乗位 — 強がりが溶ける（約3,500字）

**場所**: リーナの部屋
**時間帯**: 夜
**登場人物**: ユーマ、リーナ

**出来事**:
- リーナがユーマを押し倒し、自分が上に跨る
- 「アタシのペースでやるから。……文句ないでしょ」
- 自分から腰を下ろす。挿入。リーナの表情が一瞬だけ歪む。痛みではない。快感
- リーナが動き始める。自分のペースで。最初は強気に
- 「ほら……アタシの方が、上手でしょ……」— 息が荒い。強がりが崩れ始めている
- 腰の動きが次第に乱れていく。自分で制御できなくなっていく
- 声が漏れる。抑えようとして失敗する。「っ……ん……」
- 強がりの仮面が剥がれていく。リーナの素の表情が現れる
- 「……ユーマ……ユーマ……」— 名前を呼ぶ声が甘くなる
- リーナの手がユーマの胸に置かれている。支えるためだったはずの手が、しがみつく手に変わる
- 赤い発光が強くなる。リーナの小麦色の肌が赤い光を帯びる
- リーナの目から涙がこぼれる。「泣いてない……泣いてないから……」
- でも止まらない。腰も涙も止まらない

**描写ポイント**:
- 騎乗位＝リーナが主導権を握る体位。でも快感に負けて主導権が崩れていく
- 「上手でしょ」→声が漏れる→名前を呼ぶ→涙、の段階的崩壊
- 「泣いてない」の反復。リーナの象徴的な台詞
- 赤い発光が感情と連動して強くなる

### シーン7: 対面座位 — 本音（約3,500字）

**場所**: リーナの部屋
**時間帯**: 夜
**登場人物**: ユーマ、リーナ

**出来事**:
- 騎乗位の途中で、リーナの体が崩れる。支えきれなくなる
- ユーマがリーナを抱き起こす。体位が変わる。対面座位
- リーナがユーマの腰の上に座り、正面から抱き合う形
- 距離がゼロになる。顔と顔が近い。リーナの涙がユーマの肩に落ちる
- 「……ばか。こんなの……ずるい……」
- 対面座位でユーマが動く。リーナは抱きついたまま。もう強がれない
- リーナの本音が溢れ出す
- 「アタシが最初だったのに……セレナにもミアにも……ヴィオラにも……」
- 「アタシだけ見てよ……アタシだけのユーマでいてよ……」
- 嫉妬の奥にある本音。「一番大切にされたい」
- ユーマがリーナを抱きしめる。「お前は特別だ」
- 射精の瞬間。赤い光が爆発する。部屋の温度が跳ね上がる
- ランプの炎が天井まで伸びる。リーナの全身が赤い光に包まれる
- 今までで最も強い発光。感情が魔力を増幅している
- リーナがユーマにしがみついたまま、声を上げる。もう抑えない。抑えられない

**描写ポイント**:
- 対面座位＝距離ゼロ。顔が見える。涙が見える。嘘がつけない
- リーナの本音の吐露。嫉妬→独占欲→「一番でいたい」
- 射精と魔力発光のクライマックス。感情が魔力を増幅する設定の最大活用
- ランプの炎が暴走する演出。火属性ならではの

### シーン8: 灰と余熱（約2,500字）

**場所**: リーナの部屋
**時間帯**: 夜（行為後）→翌朝
**登場人物**: ユーマ、リーナ

**出来事**:
- 行為後。リーナがユーマの胸に顔を埋めている。泣き疲れている
- 部屋の温度がまだ高い。ランプの炎が揺れている。シーツが汗で湿っている
- リーナ「……さっきのは忘れて」ユーマ「無理だ」リーナ「……ばか」
- しばらくの沈黙。リーナの呼吸が落ち着いていく
- 「……ねえ。アタシ、重い？」
- ユーマ「重くない」
- リーナ「嘘。絶対重い。嫉妬とか独占欲とか、めんどくさい女でしょ」
- ユーマ「めんどくさい」リーナ「……っ！」ユーマ「だが、嫌いじゃない」
- リーナの拳がユーマの胸を叩く。弱い。全然痛くない
- 「……ばか。そういうこと言うな。期待しちゃうでしょ」
- 翌朝。リーナが先に起きている。髪をポニーテールに結び直している
- いつもの顔。勝ち気な目。でも、昨夜とは何かが違う。柔らかい
- 「……起きたの。遅い」
- ユーマ「お前の部屋だぞ」
- リーナ「うるさい。さっさと出てって。……また来なさいよ」
- 最後の一言だけ、小さかった

**感情の流れ**:
- リーナ：泣き疲れた安堵→恥ずかしさ→「重い女」への自覚→ユーマの言葉で救われる
- 翌朝：いつもの強がりが戻る。でも少しだけ素直になった

**描写ポイント**:
- 「忘れて」「無理だ」のやりとり。リーナとユーマの関係性
- 「重い？」という問い。リーナの自己認識
- 「めんどくさい」「嫌いじゃない」— ユーマなりの愛情表現
- 翌朝のリーナ。鎧を着直す。でも隙間がある
- 「また来なさいよ」— 命令形だが、声が小さい
